# 大模型训练与对齐

## 整体简介

大模型训练与对齐是构建高质量大语言模型的核心流程，涵盖从数据准备、预训练到微调对齐的完整生命周期。本模块聚焦于大模型工程化落地中的关键技术与最佳实践，包括数据处理、模型训练、参数高效微调、偏好对齐等核心环节，旨在帮助开发者系统掌握大模型训练与对齐的全流程设计与优化方法。

## 子主题清单

- [ ] [[预训练流程/预训练流程]] - 涵盖大模型预训练的完整流程，包括数据处理、模型架构设计、训练策略优化等核心内容，是大模型构建的基础环节。
- [ ] [[数据收集与清洗/数据收集与清洗]] - 介绍大模型训练数据的收集、清洗、去重、过滤等关键技术，数据质量直接决定模型最终效果。
- [ ] [[监督微调/监督微调]] - 讲解如何通过监督微调将预训练模型适配到特定任务，包括数据构建、训练策略、评估方法等。
- [ ] [[参数高效微调/参数高效微调]] - 探讨低秩适应（LoRA）、Adapter等参数高效微调技术，实现高效、低成本的模型适配。
- [ ] [[偏好对齐方法/偏好对齐方法]] - 介绍RLHF、DPO、ORPO等偏好对齐技术，将模型输出与人类偏好对齐，提升模型实用性。
- [ ] [[安全对齐与价值对齐/安全对齐与价值对齐]] - 关注大模型的安全性、公平性和伦理问题，包括有害内容过滤、偏见缓解、价值对齐等关键技术。

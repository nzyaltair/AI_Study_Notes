# 凸优化基础

## 1. 概述

凸优化是优化理论中最成熟的分支，研究在凸集上最小化凸函数的问题。其核心优势在于：凸问题的局部最优解即全局最优解，且存在高效的多项式时间算法。凸优化为 [[00_机器学习_综述|机器学习]] 中的 SVM、逻辑回归、LASSO 等模型提供了严格的理论基础，也是理解非凸优化的参照系。

## 3. 核心概念

### 凸集

集合 $\mathcal{C}$ 是凸集，当且仅当对任意 $\mathbf{x}, \mathbf{y} \in \mathcal{C}$ 和 $\theta \in [0,1]$：

$$\theta \mathbf{x} + (1-\theta) \mathbf{y} \in \mathcal{C}$$

**常见凸集**：

- 超平面：$\{\mathbf{x} \mid \mathbf{a}^T\mathbf{x} = b\}$
- 半空间：$\{\mathbf{x} \mid \mathbf{a}^T\mathbf{x} \leq b\}$
- 球：$\{\mathbf{x} \mid \|\mathbf{x}\|_2 \leq r\}$
- 多面体：有限个半空间的交集
- 半正定锥：$\mathbb{S}^n_+ = \{\mathbf{X} \in \mathbb{S}^n \mid \mathbf{X} \succeq 0\}$

**保凸运算**：交集保凸；仿射变换保凸；两个凸集的和（Minkowski 和）保凸。

### 凸函数

函数 $f: \mathcal{C} \to \mathbb{R}$ 是凸函数，当且仅当 $\mathcal{C}$ 为凸集，且对任意 $\mathbf{x}, \mathbf{y} \in \mathcal{C}$ 和 $\theta \in [0,1]$：

$$f(\theta \mathbf{x} + (1-\theta) \mathbf{y}) \leq \theta f(\mathbf{x}) + (1-\theta) f(\mathbf{y})$$

**等价判据**：

- **一阶条件**（可微）：$f(\mathbf{y}) \geq f(\mathbf{x}) + \nabla f(\mathbf{x})^T(\mathbf{y} - \mathbf{x})$，即切平面在函数下方
- **二阶条件**（二阶可微）：$\nabla^2 f(\mathbf{x}) \succeq 0$（Hessian 半正定）

**常见凸函数**：线性函数 $a^Tx$、二次函数 $\frac{1}{2}x^TQx + b^Tx + c$（$Q \succeq 0$）、指数函数 $e^{ax}$、负熵 $x\log x$、范数 $\|x\|$、最大值函数 $\max_i x_i$。

**保凸运算**：非负加权和、逐点最大、仿射复合、逐点上确界。

### 凸优化问题

$$\min_{\mathbf{x}} f(\mathbf{x}) \quad \text{s.t.} \quad g_i(\mathbf{x}) \leq 0, \quad h_j(\mathbf{x}) = 0$$

其中 $f$ 和 $g_i$ 是凸函数，$h_j$ 是仿射函数。关键性质：**局部最优即全局最优**，最优解集是凸集。

### 强凸性

$f$ 是 $\mu$-强凸的，若 $f(\mathbf{x}) - \frac{\mu}{2}\|\mathbf{x}\|^2$ 是凸函数。等价于 $\nabla^2 f(\mathbf{x}) \succeq \mu \mathbf{I}$。强凸性保证唯一最优解和更快的收敛速率。

### Lipschitz 梯度

$\nabla f$ 是 $L$-Lipschitz 的，若 $\|\nabla f(\mathbf{x}) - \nabla f(\mathbf{y})\| \leq L\|\mathbf{x} - \mathbf{y}\|$。等价于 $\nabla^2 f(\mathbf{x}) \preceq L\mathbf{I}$。条件数 $\kappa = L/\mu$ 决定了优化难度。

## 4. 技术原理

### 拉格朗日对偶

**拉格朗日函数**：

$$\mathcal{L}(\mathbf{x}, \boldsymbol{\lambda}, \boldsymbol{\nu}) = f(\mathbf{x}) + \sum_i \lambda_i g_i(\mathbf{x}) + \sum_j \nu_j h_j(\mathbf{x})$$

**对偶函数**：

$$g(\boldsymbol{\lambda}, \boldsymbol{\nu}) = \inf_{\mathbf{x}} \mathcal{L}(\mathbf{x}, \boldsymbol{\lambda}, \boldsymbol{\nu})$$

对偶函数始终是凹函数（无论原问题是否凸），对偶问题始终是凸优化问题。

**弱对偶性**：$g(\boldsymbol{\lambda}, \boldsymbol{\nu}) \leq p^*$（原问题最优值），恒成立。

**强对偶性**：$g(\boldsymbol{\lambda}^*, \boldsymbol{\nu}^*) = p^*$，凸问题在 Slater 条件下成立。

**Slater 条件**（约束规范）：存在严格可行点 $\mathbf{x}$ 使 $g_i(\mathbf{x}) < 0$ 且 $h_j(\mathbf{x}) = 0$。

### KKT 条件

KKT（Karush-Kuhn-Tucker）条件是约束优化最优性的必要条件，在凸问题和强对偶性下也是充分条件：

1. **平稳性**（Stationarity）：$\nabla_{\mathbf{x}} \mathcal{L} = \nabla f(\mathbf{x}^*) + \sum_i \lambda_i \nabla g_i(\mathbf{x}^*) + \sum_j \nu_j \nabla h_j(\mathbf{x}^*) = 0$
2. **原始可行性**（Primal feasibility）：$g_i(\mathbf{x}^*) \leq 0$，$h_j(\mathbf{x}^*) = 0$
3. **对偶可行性**（Dual feasibility）：$\lambda_i \geq 0$
4. **互补松弛性**（Complementary slackness）：$\lambda_i g_i(\mathbf{x}^*) = 0$

### 凸共轭（Fenchel 共轭）

$$f^*(\mathbf{y}) = \sup_{\mathbf{x}} \left( \mathbf{y}^T\mathbf{x} - f(\mathbf{x}) \right)$$

凸共轭将原问题转化为对偶空间中的问题。对偶上升法和对偶分解法的理论基础。Fenchel-Young 不等式：$f(\mathbf{x}) + f^*(\mathbf{y}) \geq \mathbf{y}^T\mathbf{x}$。

### 内点法

通过将不等式约束转化为对数障碍项，将约束问题变为无约束问题：

$$\min_{\mathbf{x}} t f(\mathbf{x}) - \sum_i \log(-g_i(\mathbf{x}))$$

随 $t \to \infty$，障碍问题的解趋近原问题最优解。路径跟踪法（Path-following）在多项式时间内收敛。

## 5. 关键方法与模型

### 支持向量机（SVM）的对偶推导

SVM 原问题：

$$\min_{\mathbf{w}, b} \frac{1}{2}\|\mathbf{w}\|^2 \quad \text{s.t.} \quad y_i(\mathbf{w}^T\mathbf{x}_i + b) \geq 1$$

拉格朗日函数：

$$\mathcal{L} = \frac{1}{2}\|\mathbf{w}\|^2 - \sum_i \lambda_i [y_i(\mathbf{w}^T\mathbf{x}_i + b) - 1]$$

由 KKT 平稳性得 $\mathbf{w} = \sum_i \lambda_i y_i \mathbf{x}_i$，代入后得对偶问题：

$$\max_{\boldsymbol{\lambda}} \sum_i \lambda_i - \frac{1}{2}\sum_{i,j} \lambda_i \lambda_j y_i y_j \mathbf{x}_i^T\mathbf{x}_j \quad \text{s.t.} \quad \lambda_i \geq 0, \ \sum_i \lambda_i y_i = 0$$

对偶形式仅涉及内积，可自然推广到核方法。

### LASSO 回归

$$\min_{\boldsymbol{\beta}} \frac{1}{2}\|\mathbf{y} - \mathbf{X}\boldsymbol{\beta}\|^2 + \lambda \|\boldsymbol{\beta}\|_1$$

L1 正则化产生稀疏解，KKT 条件中互补松弛性导致部分 $\beta_i = 0$。

### 增广拉格朗日法（ALM）

$$\mathcal{L}_\rho = f(\mathbf{x}) + \boldsymbol{\lambda}^T\mathbf{g}(\mathbf{x}) + \frac{\rho}{2}\|\mathbf{g}(\mathbf{x})\|^2$$

交替方向乘子法（ADMM）通过交替更新 $\mathbf{x}$ 和对偶变量，分解复杂约束问题。

## 6. 优势与局限

### 优势

- 全局最优保证：局部最优即全局最优
- 理论完备：收敛性、复杂度有严格证明
- 高效算法：内点法等多项式时间算法
- 对偶理论：提供问题结构分析和分解方法

### 局限

- 深度学习的损失函数高度非凸，凸优化理论不能直接适用
- 实际问题中约束可能不满足 Slater 条件
- 大规模问题中内点法的 Hessian 计算代价高
- 凸性假设在许多实际问题中过强

# 数据集汇总

> 本文件为 AI 开发人员与研究人员的结构化、可操作的数据集指南，系统梳理适用于大模型、多模态、NLP、CV 等方向的高质量公开数据集。

## 概述

本汇总收录了 AI 领域中最具价值的公开数据集，按任务或模态分类组织，强调实用性、可获取性与任务适配性。每个数据集均提供详细信息，包括规模、内容特点、许可协议、获取方式和典型用途，旨在为团队提供数据选型的权威参考。

## 语言预训练数据集

### 经典数据集

- [ ] [[Wikipedia]] - 维基百科数据集
  - **规模**：约 600 万篇文章，180 亿 tokens
  - **内容特点**：多语言，百科知识，结构化信息，已清洗
  - **许可协议**：CC BY-SA 4.0，可商用
  - **获取方式**：[Hugging Face](https://huggingface.co/datasets/wikipedia)
  - **典型用途**：用于 LLM 预训练、知识增强

- [ ] [[BookCorpus]] - 书籍语料库
  - **规模**：约 11,038 本书，8 亿 tokens
  - **内容特点**：英文，小说为主，连续文本
  - **许可协议**：可能存在版权问题，谨慎商用
  - **获取方式**：[GitHub](https://github.com/soskek/bookcorpus)
  - **典型用途**：用于 LLM 预训练、语言建模

### 前沿数据集

- [ ] [[FineWeb]] - 高质量网页文本数据集
  - **规模**：约 136B tokens，多语言
  - **内容特点**：多语言，网页文本，高质量过滤
  - **许可协议**：CC BY-SA 4.0，可商用
  - **获取方式**：[Hugging Face](https://huggingface.co/datasets/HuggingFaceFW/fineweb)
  - **典型用途**：用于 LLM 预训练，替代 C4

- [ ] [[RedPajama]] - 开源预训练数据集
  - **规模**：约 1.2T tokens，多语言
  - **内容特点**：多语言，网页、书籍、论文等混合
  - **许可协议**：Apache 2.0，可商用
  - **获取方式**：[GitHub](https://github.com/togethercomputer/RedPajama-Data)
  - **典型用途**：用于开源 LLM 预训练

### 选型建议

- LLaMA 预训练常用：FineWeb + RedPajama
- 多语言模型：Wikipedia + 多语言新闻语料
- 学术领域：arXiv + ACL Anthology

## 指令微调数据集

- [ ] [[Alpaca]] - 指令微调数据集
  - **规模**：52K 指令-响应对
  - **内容特点**：英文，多样化任务类型
  - **许可协议**：Apache 2.0，可商用
  - **获取方式**：[GitHub](https://github.com/tatsu-lab/stanford_alpaca)
  - **典型用途**：用于 LLM 指令微调，提升模型遵循指令的能力

- [ ] [[ShareGPT]] - 对话式指令数据集
  - **规模**：约 100K 对话回合
  - **内容特点**：多语言，真实用户-助手对话
  - **许可协议**：CC BY-NC-SA 4.0，非商用
  - **获取方式**：[Hugging Face](https://huggingface.co/datasets/anon8231489123/ShareGPT_V3_unfiltered)
  - **典型用途**：用于 LLM 对话能力微调

- [ ] [[OpenAssistant]] - 社区驱动指令数据集
  - **规模**：约 161K 指令-响应对
  - **内容特点**：多语言，多样化任务类型，人工标注质量
  - **许可协议**：Apache 2.0，可商用
  - **获取方式**：[Hugging Face](https://huggingface.co/datasets/OpenAssistant/oasst1)
  - **典型用途**：用于 LLM 指令微调，提升模型交互能力

### 选型建议

- 通用指令微调：Alpaca + OpenAssistant
- 对话能力：ShareGPT + OpenAssistant
- 多语言支持：添加多语言翻译数据集

## 多轮对话数据集

- [ ] [[LMSYS-Chat-1M]] - 真实对话数据集
  - **规模**：约 100 万对话回合
  - **内容特点**：多语言，真实用户-助手对话，多样化话题
  - **许可协议**：CC BY-NC-SA 4.0，非商用
  - **获取方式**：[Hugging Face](https://huggingface.co/datasets/lmsys/lmsys-chat-1m)
  - **典型用途**：用于 LLM 对话能力评估和微调

- [ ] [[DailyDialog]] - 日常对话数据集
  - **规模**：约 13,118 个对话，102,143 句
  - **内容特点**：英文，日常话题，情感标注
  - **许可协议**：CC BY-NC-SA 4.0，非商用
  - **获取方式**：[GitHub](https://github.com/ConvoKit/dailydialog)
  - **典型用途**：用于对话系统训练和评估

- [ ] [[PersonaChat]] - 人格对话数据集
  - **规模**：约 162,000 对话，1.9M 句
  - **内容特点**：英文，基于人格描述的对话
  - **许可协议**：MIT，可商用
  - **获取方式**：[Hugging Face](https://huggingface.co/datasets/bavard/personachat_true_cased)
  - **典型用途**：用于个性化对话系统训练

### 选型建议

- 真实对话场景：LMSYS-Chat-1M + DailyDialog
- 人格化对话：PersonaChat + 自定义人格数据

## 代码生成数据集

- [ ] [[The Stack]] - 代码数据集
  - **规模**：约 6.4TB 代码，350+ 编程语言
  - **内容特点**：多语言代码，开源项目
  - **许可协议**：CC BY-SA 4.0，商用需谨慎
  - **获取方式**：[Hugging Face](https://huggingface.co/datasets/bigcode/the-stack)
  - **典型用途**：用于代码生成模型训练

- [ ] [[CodeSearchNet]] - 代码搜索数据集
  - **规模**：约 6M 代码片段，6 种编程语言
  - **内容特点**：多语言代码，带注释
  - **许可协议**：Apache 2.0，可商用
  - **获取方式**：[GitHub](https://github.com/github/CodeSearchNet)
  - **典型用途**：用于代码搜索、代码理解

- [ ] [[HumanEval]] - 代码生成评估数据集
  - **规模**：164 个编程问题
  - **内容特点**：Python 代码，函数级生成任务
  - **许可协议**：MIT，可商用
  - **获取方式**：[GitHub](https://github.com/openai/human-eval)
  - **典型用途**：用于代码生成模型评估

### 选型建议

- 代码生成训练：The Stack + CodeSearchNet
- 代码生成评估：HumanEval + MBPP

## 图像-文本对齐数据集

- [ ] [[COCO]] - 通用目标检测数据集
  - **规模**：123K 图像，5 句/图，80 类别
  - **内容特点**：英文，日常场景，高质量标注
  - **许可协议**：CC BY 4.0，可商用
  - **获取方式**：[官网](https://cocodataset.org/)
  - **典型用途**：用于图像描述、目标检测

- [ ] [[Flickr30k]] - 图像描述数据集
  - **规模**：31,783 图像，5 句/图
  - **内容特点**：英文，日常场景，多样描述
  - **许可协议**：CC BY-SA 2.0，可商用
  - **获取方式**：[Hugging Face](https://huggingface.co/datasets/flikr30k)
  - **典型用途**：用于图像描述、视觉问答

- [ ] [[Conceptual Captions]] - 网络图像描述数据集
  - **规模**：3.3M 图像-文本对
  - **内容特点**：英文，网络图像，自动收集
  - **许可协议**：可能存在版权问题，谨慎商用
  - **获取方式**：[GitHub](https://github.com/google-research-datasets/conceptual-captions)
  - **典型用途**：用于 CLIP 训练、图像-文本对齐

- [ ] [[WildVision]] - 野生视觉数据集
  - **规模**：10M 图像-文本对
  - **内容特点**：多语言，多样化场景，高质量过滤
  - **许可协议**：CC BY 4.0，可商用
  - **获取方式**：[Hugging Face](https://huggingface.co/datasets/wildvision/wildvision)
  - **典型用途**：用于多模态模型训练、图像-文本对齐

### 选型建议

- 通用图像-文本对齐：COCO + Flickr30k
- 大规模训练：Conceptual Captions + WildVision

## 音频-文本数据集

- [ ] [[LibriSpeech]] - 语音识别数据集
  - **规模**：1000 小时语音，英文
  - **内容特点**：英文，朗读语音，高质量标注
  - **许可协议**：CC BY 4.0，可商用
  - **获取方式**：[官网](http://www.openslr.org/12/)
  - **典型用途**：用于语音识别、语音合成

- [ ] [[Common Voice]] - 多语言语音数据集
  - **规模**：100+ 语言，数万个小时语音
  - **内容特点**：多语言，众包收集，多样化语音
  - **许可协议**：CC0，可商用
  - **获取方式**：[Hugging Face](https://huggingface.co/datasets/common_voice)
  - **典型用途**：用于多语言语音识别、语音合成

- [ ] [[AudioCaps]] - 音频描述数据集
  - **规模**：49,838 音频片段，英文描述
  - **内容特点**：英文，环境声音，描述性文本
  - **许可协议**：CC BY 4.0，可商用
  - **获取方式**：[GitHub](https://github.com/cdjkim/audiocaps)
  - **典型用途**：用于音频-文本对齐、音频描述

### 选型建议

- 英文语音识别：LibriSpeech + Common Voice
- 多语言支持：Common Voice + 语言特定数据集

## 安全对齐数据集

- [ ] [[HarmfulQA]] - 有害问题数据集
  - **规模**：约 30K 问题，多语言
  - **内容特点**：多语言，有害问题分类
  - **许可协议**：Apache 2.0，可商用
  - **获取方式**：[Hugging Face](https://huggingface.co/datasets/allenai/harmful_qa)
  - **典型用途**：用于模型安全评估、对齐训练

- [ ] [[RealToxicityPrompts]] - 毒性提示数据集
  - **规模**：约 100K 提示，英文
  - **内容特点**：英文，毒性提示，评分标注
  - **许可协议**：MIT，可商用
  - **获取方式**：[Hugging Face](https://huggingface.co/datasets/allenai/real_toxicity_prompts)
  - **典型用途**：用于模型毒性评估、安全训练

- [ ] [[HELM]] - 语言模型评估基准
  - **规模**：多任务，多语言
  - **内容特点**：多语言，多维度评估，安全测试
  - **许可协议**：Apache 2.0，可商用
  - **获取方式**：[GitHub](https://github.com/stanford-crfm/helm)
  - **典型用途**：用于模型综合评估，包括安全维度

### 选型建议

- 安全评估：HarmfulQA + RealToxicityPrompts + HELM
- 安全对齐训练：结合人工标注的偏好数据

## 2024–2026 新兴数据集

  - **规模**：约 1B 对话回合，多语言
  - **内容特点**：多语言，高质量对话，蒸馏自大型模型
  - **许可协议**：Apache 2.0，可商用
  - **获取方式**：[Hugging Face](https://huggingface.co/datasets/dclm/dclm)
  - **典型用途**：用于对话模型训练、知识蒸馏

- [ ] [[OpenAssistant Conversations]] - 社区驱动对话数据集
  - **规模**：约 161K 对话，35 语言
  - **内容特点**：多语言，社区贡献，多样化任务
  - **许可协议**：Apache 2.0，可商用
  - **获取方式**：[Hugging Face](https://huggingface.co/datasets/OpenAssistant/oasst1)
  - **典型用途**：用于对话模型训练、指令微调

- [ ] [[LongBench]] - 长上下文基准数据集
  - **规模**：多任务，长上下文
  - **内容特点**：多语言，长上下文任务，多样化格式
  - **许可协议**：Apache 2.0，可商用
  - **获取方式**：[GitHub](https://github.com/OpenLMLab/LongBench)
  - **典型用途**：用于长上下文模型评估

### 数据构建趋势

- **去重策略**：采用先进的文本去重算法，减少数据冗余
- **质量过滤**：使用多维度质量评分，筛选高质量文本
- **多语言扩展**：增加非英语语言覆盖，提升模型多语言能力
- **长上下文构造**：构建长文档数据集，支持模型长上下文理解
- **合成数据**：利用大型模型生成高质量合成数据，补充真实数据

# 生成模型前沿发展

## 1. 概述

生成模型领域正处于快速演进期。扩散模型的统一理论框架正在被重新定义——Flow Matching 统一了扩散模型与归一化流，Consistency Models 打破了多步采样的限制，DiT 架构展示了 Transformer 在生成任务中的可扩展性。本章梳理 2023-2024 年生成模型的核心前沿方向。

## 2. 发展脉络

```
GAN (2014) → VAE (2013) → Diffusion (2020) → Flow Matching (2023) → 统一生成范式
                                           → Consistency Model (2023) → 少步生成
                                           → DiT (2023) → Transformer 架构
                                           → Sora (2024) → 视频生成
```

## 3. 核心前沿方向

### 3.1 Flow Matching — 统一框架

**核心思想**：通过回归向量场来学习将简单分布（高斯）传输到目标分布（数据）的连续流。

**与扩散模型的关系**：
- 扩散模型的逆向 SDE/ODE 是 Flow Matching 的特例
- Flow Matching 更灵活：可以选择任意源分布和目标分布之间的路径
- 可以构造更"直"的路径 → 更少步数采样

**数学框架**：

给定概率路径 $p_t(x)$ 从 $p_0$（噪声）到 $p_1$（数据），定义向量场：

$$u_t(x) = \mathbb{E}\left[\dot{x}_t | x_t = x\right]$$

训练目标（回归向量场）：

$$\mathcal{L}_{FM} = \mathbb{E}_{t, x_t}\left[\|v_\theta(x_t, t) - u_t(x_t)\|^2\right]$$

采样通过 ODE：

$$\frac{dx_t}{dt} = v_\theta(x_t, t)$$

**Conditional Flow Matching**：以数据 $x_1$ 为条件构造路径：

$$x_t = (1-t) \cdot \epsilon + t \cdot x_1, \quad \epsilon \sim \mathcal{N}(0, I)$$

此时 $u_t(x_t | x_1) = x_1 - \epsilon$，训练目标简化为预测 $x_1 - \epsilon$。

### 3.2 Rectified Flow

**核心思想**：通过"拉直"流路径实现高效少步采样。

**方法**：
1. 从两个分布间训练一个（可能是弯曲的）流
2. 通过"重新流动"（reflow）将弯曲路径拉直——用上一次流的输出作为新的配对数据训练新的直线路径
3. 直线路径可以用更少的步数精确求解

**实际效果**：FLUX 和 Stable Diffusion 3 采用 Rectified Flow，实现 4-8 步高质量生成。

### 3.3 Consistency Models

**核心思想**：训练一个模型在 ODE 轨迹的任意点上都能直接映射到轨迹起点（即干净数据），实现一步生成。

**自洽性约束**：对于 ODE 轨迹上的任意两点 $x_t$ 和 $x_s$：

$$f_\theta(x_t, t) = f_\theta(x_s, s)$$

**训练方式**：

1. **一致性蒸馏（CD）**：从预训练扩散模型蒸馏
   - 在轨迹上取相邻两点 $(x_{t}, x_{t-\Delta t})$
   - 用 EMA 模型作为教师：$f_{\theta^-}(x_{t-\Delta t}, t-\Delta t)$
   - 学生模型匹配：$\mathcal{L} = d(f_\theta(x_t, t), f_{\theta^-}(x_{t-\Delta t}, t-\Delta t))$

2. **一致性训练（CT）**：从零训练（无需预训练模型）

**采样**：一步 $f_\theta(x_T, T) \to \hat{x}_0$，或多步（交替去噪和加噪）提升质量。

### 3.4 DiT (Diffusion Transformer)

**核心创新**：用 Transformer 替代 U-Net 作为扩散模型的去噪网络。

**架构设计**：
1. 将潜空间表示分块为 token 序列（类似 ViT）
2. 标准 Transformer blocks 处理 token 序列
3. AdaLN-Zero：通过时间和条件嵌入调制每个 block 的 scale 和 shift
4. 最终 unpatchify 恢复空间结构

**AdaLN-Zero**：

$$y = \text{Attn}(\text{AdaLN}(x, c))$$
$$\text{AdaLN}(x, c) = \gamma(c) \cdot \text{LN}(x) + \beta(c)$$

其中 $\gamma, \beta$ 由条件嵌入 $c$（时间 + 文本）生成，初始化为零（零初始化保证训练开始时为恒等映射）。

**Scaling Law**：DiT 展示了明确的规模定律——更大的模型（DiT-XL）和更多计算产生更好的 FID，验证了 Transformer 在生成任务中的可扩展性。

**影响**：
- Sora 基于 DiT 架构实现视频生成
- Stable Diffusion 3 和 FLUX 采用 DiT
- U-Net 正在被 DiT 逐步取代

### 3.5 Latent Consistency Model (LCM)

将 Consistency Model 应用于潜在扩散模型（如 Stable Diffusion）：
- 在潜空间中执行一致性蒸馏
- 支持 1-4 步快速推理
- 保持 CFG 等条件机制
- 可兼容 LoRA 等 customization

### 3.6 视频生成扩散模型

**核心挑战**：
- 时空一致性：帧间内容连贯
- 长视频生成：内存和计算限制
- 运动控制：合理的物体运动轨迹

**架构方案**：

1. **时空 U-Net**：在 U-Net 中添加时间注意力层
2. **DiT 时空扩展**：将视频分块为时空 token，使用 3D 注意力
3. **级联方案**：先生成关键帧，再插值生成中间帧

**Sora 的方法**（推测）：
- 基于 DiT 架构
- 视频压缩为时空 token
- 大规模训练
- 文本条件引导

### 3.7 3D 生成 — Score Distillation Sampling (SDS)

**核心思想**：利用 2D 扩散模型的先验优化 3D 表示。

**SDS 损失**：

$$\nabla_\theta \mathcal{L}_{SDS} = \mathbb{E}_{t, \epsilon}\left[w(t) (\epsilon_\theta(x_t; t, y) - \epsilon) \frac{\partial x}{\partial \theta}\right]$$

其中 $\theta$ 为 3D 表示参数（如 NeRF、3D Gaussian Splatting），$x = g(\theta)$ 为 3D 渲染的 2D 图像。

**应用**：
- DreamFusion：文本到 3D
- Magic3D：高分辨率 3D 生成
- 3D Gaussian Splatting + SDS：快速 3D 重建和生成

### 3.8 统一多模态生成

**目标**：单一模型处理文本、图像、音频、视频的统一生成。

**方法**：
1. 将所有模态统一为 token 序列
2. 自回归或扩散方式生成
3. 统一的训练目标

**代表**：
- Unified-IO 2：多模态统一 token + 扩散
- Chameleon：Meta 的多模态自回归模型
- Any-to-Any 生成模型

## 4. 技术对比

### 4.1 采样步数 vs 质量

| 方法 | 步数 | 质量 | 训练方式 |
|------|------|------|---------|
| DDPM | ~1000 | 高 | 从零训练 |
| DDIM / DPM++ | 10-50 | 高 | 从零训练 |
| LCM | 1-4 | 中-高 | 蒸馏 |
| Consistency Model | 1-4 | 中-高 | 蒸馏/从零 |
| Rectified Flow | 1-8 | 高 | 从零训练 |
| Flow Matching | 1-10 | 高 | 从零训练 |

### 4.2 架构对比

| 架构 | 主干 | 优势 | 代表模型 |
|------|------|------|---------|
| U-Net | CNN + Attention | 成熟稳定 | SD 1.x/2.x |
| DiT | Transformer | 可扩展性强 | SD3, FLUX, Sora |
| MM-DiT | 多模态 Transformer | 统一文本+图像 | SD3 |

## 6. 前沿趋势总结

### 6.1 理论统一

- **Flow Matching** 统一了扩散模型、连续归一化流和 Rectified Flow
- 噪声预测、score matching、向量场回归被统一在同一框架下
- 不同路径选择（线性、弯曲）是设计选择而非本质区别

### 6.2 架构演进

- U-Net → DiT：Transformer 架构在生成模型中的全面胜利
- Scaling Law 在生成模型中同样适用
- 多模态 Transformer（MM-DiT）统一文本和图像处理

### 6.3 效率优化

- 从 1000 步到 1 步：Consistency Model、Rectified Flow
- 蒸馏技术：从教师模型蒸馏到学生模型
- 量化与剪枝：降低推理成本

### 6.4 应用扩展

- **视频生成**：从图像扩散到时空扩散
- **3D 生成**：SDS 和 3D Gaussian Splatting
- **科学发现**：蛋白质设计（RFdiffusion）、材料生成
- **统一多模态**：Any-to-Any 生成

### 6.5 开放问题

- **评估标准**：缺乏统一的评估框架
- **可控性**：精确控制仍是挑战
- **效率 vs 质量**：少步生成与多步生成的质量差距
- **安全与伦理**：Deepfake、版权、偏见
- **理论基础**：扩散模型的理论性质仍有待深入研究

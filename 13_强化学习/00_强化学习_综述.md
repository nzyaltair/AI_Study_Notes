# 强化学习综述

## 1. 概述

强化学习（Reinforcement Learning, RL）是机器学习的三大范式之一，研究**智能体（Agent）**在与**环境（Environment）**的交互中，通过试错（Trial and Error）学习最优决策策略（Policy），以最大化长期累积奖励（Cumulative Reward）的问题。与监督学习从标注数据中学习输入-输出映射不同，强化学习从环境反馈的奖励信号中学习"做什么才能使长期收益最大化"。

- **解决的问题**：在无法获得精确环境模型或标注数据的序贯决策问题中，如何通过交互反馈学习最优行为策略。覆盖机器人控制、游戏 AI、推荐系统、资源调度、以及大语言模型对齐等场景。
- **核心价值**：强化学习是连接"感知—决策—行动"闭环的核心理论框架。它为最优控制提供了数据驱动的解决方案，也是当前大模型对齐训练（RLHF/GRPO）和推理模型（o1/R1）的理论基石。
- **知识体系范围**：MDP 理论基础 → 值函数与动态规划 → 无模型方法（MC/TD/Q-Learning/SARSA）→ 策略梯度（REINFORCE/Actor-Critic/TRPO/PPO/SAC）→ 深度强化学习（DQN 系列/DDPG/A3C/Rainbow）→ 基于模型的 RL（MuZero/Dreamer）→ 探索与利用 → 离线 RL → 多智能体 RL → 强化学习与大模型 → 工程实践与前沿。

## 3. 核心概念

### 3.1 马尔可夫决策过程（MDP）

强化学习的数学形式化框架。MDP 由五元组 $(\mathcal{S}, \mathcal{A}, P, R, \gamma)$ 定义：状态集 $\mathcal{S}$、动作集 $\mathcal{A}$、状态转移概率 $P(s'|s,a)$、奖励函数 $R(s,a,s')$ 和折扣因子 $\gamma \in [0,1)$。MDP 的马尔可夫性假设——未来只依赖当前状态和动作——是 RL 理论的基石。参见 [[01_强化学习基础与MDP]]。

### 3.2 值函数（Value Function）

值函数衡量某个状态（或状态-动作对）的长期价值：

- **状态值函数** $V^\pi(s)$：在策略 $\pi$ 下从状态 $s$ 出发的期望累积折扣奖励
- **动作值函数** $Q^\pi(s,a)$：在状态 $s$ 执行动作 $a$ 后遵循策略 $\pi$ 的期望累积折扣奖励
- **优势函数** $A^\pi(s,a) = Q^\pi(s,a) - V^\pi(s)$：衡量动作 $a$ 相对于平均水平的优劣

值函数是 RL 评估和学习策略的核心工具。参见 [[02_值函数与动态规划]]。

### 3.3 策略（Policy）

策略 $\pi(a|s)$ 定义智能体在每个状态选择动作的方式：

- **确定性策略** $\pi(s) = a$：直接映射状态到动作
- **随机策略** $\pi(a|s)$：输出动作的概率分布

基于值函数的方法（Q-Learning/DQN）隐式地通过 $\arg\max_a Q(s,a)$ 获得策略；策略梯度方法（REINFORCE/PPO）直接参数化和优化策略 $\pi_\theta$。参见 [[04_策略梯度方法]]。

### 3.4 探索与利用（Exploration vs. Exploitation）

RL 的核心困境：智能体需要在"利用已知最优动作获取高奖励"和"探索未知动作发现更优策略"之间平衡。从简单的 ε-greedy 到基于不确定性的 UCB，再到基于内在奖励的好奇心驱动探索（ICM/RND），探索策略是 RL 性能的关键决定因素。参见 [[07_探索与利用]]。

### 3.5 模型（Model）

环境模型指智能体对状态转移 $P$ 和奖励 $R$ 的内部表征：

- **无模型（Model-Free）**：不学习环境模型，直接从交互数据学习值函数或策略（Q-Learning/DQN/PPO）
- **基于模型（Model-Based）**：学习环境模型并在模型内规划（MuZero/Dreamer）

基于模型的方法样本效率更高但模型偏差可能影响性能；无模型方法更简单但样本效率低。参见 [[06_基于模型的强化学习]]。

### 3.6 on-policy 与 off-policy

- **on-policy**：使用当前策略产生的数据来评估和改进同一策略（SARSA/PPO/TRPO）
- **off-policy**：可以使用与当前策略不同的策略（如旧策略或探索策略）产生的数据来学习目标策略（Q-Learning/DQN/SAC）

off-policy 方法能更高效地利用经验回放池中的数据，但引入分布偏移问题。

## 4. 核心公式速览

### Bellman 期望方程

$$V^\pi(s) = \sum_a \pi(a|s) \sum_{s'} P(s'|s,a) [R(s,a,s') + \gamma V^\pi(s')]$$

$$Q^\pi(s,a) = \sum_{s'} P(s'|s,a) [R(s,a,s') + \gamma \sum_{a'} \pi(a'|s') Q^\pi(s',a')]$$

### Bellman 最优方程

$$V^*(s) = \max_a \sum_{s'} P(s'|s,a) [R(s,a,s') + \gamma V^*(s')]$$

$$Q^*(s,a) = \sum_{s'} P(s'|s,a) [R(s,a,s') + \gamma \max_{a'} Q^*(s',a')]$$

### Q-Learning 更新

$$Q(s_t, a_t) \leftarrow Q(s_t, a_t) + \alpha \left[ r_{t+1} + \gamma \max_{a'} Q(s_{t+1}, a') - Q(s_t, a_t) \right]$$

### DQN 损失函数

$$\mathcal{L}(\theta) = \mathbb{E}_{(s,a,r,s') \sim \mathcal{D}} \left[ \left( r + \gamma \max_{a'} Q_{\theta^-}(s', a') - Q_\theta(s, a) \right)^2 \right]$$

### 策略梯度定理

$$\nabla_\theta J(\theta) = \mathbb{E}_{\tau \sim \pi_\theta} \left[ \sum_t \nabla_\theta \log \pi_\theta(a_t|s_t) \, A^{\pi_\theta}(s_t, a_t) \right]$$

### PPO Clipped 目标

$$\mathcal{L}^{\text{CLIP}}(\theta) = \mathbb{E} \left[ \min \left( r_t(\theta) A_t, \, \text{clip}(r_t(\theta), 1-\epsilon, 1+\epsilon) A_t \right) \right]$$

其中 $r_t(\theta) = \frac{\pi_\theta(a_t|s_t)}{\pi_{\text{old}}(a_t|s_t)}$。

## 5. 方法分类总览

| 类别 | 代表方法 | 特点 | 适用场景 |
|:---|:---|:---|:---|
| 动态规划 | 值迭代、策略迭代 | 需要完整环境模型 | 已知 MDP 的最优控制 |
| 无模型（值函数） | Q-Learning、SARSA、DQN | 不需要模型，off/on-policy 分明 | 离散动作空间 |
| 无模型（策略梯度） | REINFORCE、Actor-Critic、PPO、SAC | 直接优化策略，支持连续动作 | 连续控制、高维动作 |
| 深度强化学习 | DQN、Rainbow、DDPG、A3C | 用神经网络近似值函数/策略 | 视觉输入、高维状态 |
| 基于模型 | MuZero、Dreamer、TD-MPC | 学习世界模型，在模型中规划 | 样本效率要求高 |
| 离线 RL | BCQ、CQL、IQL | 从固定数据集学习，无在线交互 | 医疗、推荐等无法在线探索的场景 |
| 多智能体 RL | MADDPG、QMIX、MAPPO | 处理多智能体协作/竞争 | 博弈、群体协作 |
| RL + LLM | RLHF、GRPO、RLAIF | 用 RL 对齐语言模型 | 大模型对齐与推理 |

## 7. 与其他方向关系

- **与优化理论的关系**：策略梯度本质是随机梯度上升；TRPO/PPO 的信赖域与裁剪是对优化稳定性的约束。参见 [[02_优化理论与方法]]。
- **与大模型训练与对齐的关系**：PPO/GRPO 是 RLHF 的核心优化算法，强化学习是大模型偏好对齐的理论基础。参见 [[11_大模型训练与对齐/04_偏好对齐方法]]。
- **与数学基础的关系**：MDP 基于概率论和马尔可夫链；Bellman 方程是动态规划的核心；策略梯度定理依赖期望和梯度的交换。参见 [[01_数学基础]]。
- **与机器学习的关系**：RL 是机器学习三大范式之一；值函数近似本质是回归问题；模仿学习连接监督学习与 RL。参见 [[03_机器学习]]。
- **与深度学习基础的关系**：深度 RL 用神经网络近似值函数和策略；DQN 的经验回放和目标网络借鉴了深度学习的训练技巧。参见 [[04_深度学习基础]]。
- **与 Agent 系统的关系**：RL 是自主 Agent 决策的核心理论；LLM Agent 的工具调用和规划可视为 RL 的推理时应用。参见 [[19_LLM应用工程/04_Agent系统]]。
- **与 AI 安全与对齐的关系**：RLHF/RLAIF 是价值对齐的主流方法；离线 RL 的分布偏移问题与安全 AI 直接相关。参见 [[17_AI安全与对齐]]。

## 当前发展状态

截至2026年，强化学习领域呈现以下状态：

- **RL 驱动 LLM 推理**：OpenAI o1（2024）和 DeepSeek-R1（2025）证明了 RL 训练可使 LLM 获得长链推理能力。GRPO 成为推理模型训练的主流算法，RL 与 LLM 的融合是当前最活跃的研究方向。
- **世界模型进展**：Dreamer 系列和 TD-MPC2 在连续控制任务上持续改进，世界模型方法在样本效率上逐步接近甚至超越模型无关方法 [待核实]。
- **离线 RL 成熟**：BCQ/CQL/IQL 等方法在推荐系统、医疗决策等无法在线探索的场景中已有生产部署。
- **多智能体 RL**：MAPPO 在大规模多智能体协作中表现优异，但多智能体信用分配和可扩展性仍是开放问题。
- **探索理论深化**：基于计数、不确定性、好奇心的探索方法各有适用场景，统一框架仍缺乏。

## 未来趋势

1. **RL + LLM 深度融合**：推理模型（o1/R1 范式）的成功将推动 RL 在 LLM 训练中的更广泛应用，包括多轮推理、工具使用、Agent 决策的端到端 RL 训练。
2. **世界模型与模型预测控制**：学习精确世界模型用于规划和决策，是从样本效率上突破无模型方法瓶颈的关键方向。
3. **离线 RL 与大模型结合**：利用 LLM 作为世界知识先验，结合离线 RL 进行决策优化，在医疗、金融等领域有广阔应用。
4. **多智能体与博弈论**：随着 [[19_LLM应用工程/00_LLM应用工程_综述|LLM Agent]] 多智能体系统兴起，多智能体 RL 理论将获得新的应用驱动。
5. **安全 RL 与对齐**：约束马尔可夫决策过程（CMDP）和 RL 安全性研究与 [[17_AI安全与对齐/00_AI安全与对齐_综述|AI安全]] 对齐研究深度融合。
6. **样本效率突破**：元学习、课程学习、迁移学习等方法继续推进 RL 的样本效率，使其从游戏/模拟环境走向真实世界应用。

# 统计学习方法

## 背景与发展

统计学习理论（Statistical Learning Theory）由 Vapnik 和 Chervonenkis 在 1970-1990 年代发展，为[[机器学习]]提供了严格的数学基础。它从统计学角度研究学习算法的泛化能力，回答了"为什么从有限样本中学习到的模型能够泛化到未知数据"这一根本问题。支持向量机（SVM）是这一理论最成功的实践产物。

## 核心思想

统计学习理论将学习问题表述为基于经验数据最小化风险泛函的过程。核心是在模型复杂度与经验误差之间取得平衡——模型太简单则欠拟合，太复杂则过拟合。通过结构风险最小化（SRM）原则，在假设空间中按复杂度排序逐层搜索，找到泛化能力的最优保证。

## 技术原理

### 偏差-方差分解

对于回归问题，期望泛化误差可分解为：

$$\mathbb{E}[(y - \hat{f}(x))^2] = \underbrace{(\mathbb{E}[\hat{f}(x)] - f(x))^2}_{\text{Bias}^2} + \underbrace{\mathbb{E}[(\hat{f}(x) - \mathbb{E}[\hat{f}(x)])^2]}_{\text{Variance}} + \sigma^2$$

其中 $\sigma^2$ 为不可约误差。偏差反映模型对真实关系的逼近能力，方差反映模型对训练数据变动的敏感程度。

### VC 维与可学习性

VC 维（Vapnik-Chervonenkis Dimension）度量假设空间 $H$ 能打散（shatter）的最大样本数。它是衡量模型复杂度的关键指标：

- 若 VC 维有限，则假设空间是 PAC 可学习的
- VC 维越大，模型表达能力越强，但泛化保证越弱

**泛化误差界**：以至少 $1-\delta$ 的概率，

$$R(\hat{f}) \leq R_{emp}(\hat{f}) + \sqrt{\frac{h(\ln(2n/h) + 1) - \ln(\delta/4)}{n}}$$

其中 $h$ 为 VC 维，$n$ 为样本数，$R_{emp}$ 为经验风险，$R$ 为真实风险。右边第二项为置信区间，随 $h$ 增大而增大，随 $n$ 增大而减小。

### 结构风险最小化（SRM）

Vapnik 提出的模型选择原则。将假设空间按复杂度划分为嵌套结构 $H_1 \subset H_2 \subset \cdots \subset H_k$，在每个子空间中最小化经验风险，再选择结构风险（经验风险 + 置信区间）最小的模型。SRM 是 SVM 理论的直接基础。

### 正则化理论

正则化通过在损失函数中添加惩罚项控制模型复杂度：

$$\tilde{R}(f) = R_{emp}(f) + \lambda \Omega(f)$$

- **Tikhonov 正则化**（L2）：$\Omega(f) = \|\mathbf{w}\|^2$，对应 Ridge 回归和 SVM 的间隔最大化
- **L1 正则化**：$\Omega(f) = \|\mathbf{w}\|_1$，产生稀疏解，对应 Lasso
- **正则化系数 $\lambda$** 控制偏差-方差权衡，通过交叉验证选择

正则化的贝叶斯解释：L2 正则化等价于参数的高斯先验，L1 正则化等价于拉普拉斯先验。

### PAC 学习

Probably Approximately Correct（PAC）学习框架由 Valiant（1984）提出。一个概念类 $C$ 是 PAC 可学习的，若存在算法 $A$，对任意目标概念 $c \in C$ 和任意分布 $D$，以至少 $1-\delta$ 的概率输出假设 $h$ 使得误差 $\text{error}(h) \leq \epsilon$，且时间和样本量是 $1/\epsilon$ 和 $1/\delta$ 的多项式。

PAC 框架给出了学习问题的计算复杂度视角，是计算学习理论的基石。

### SVM 的统计学习基础

支持向量机是统计学习理论最成功的实践。对于线性可分数据，SVM 寻找最大间隔超平面：

$$\max_{\mathbf{w}, b} \frac{1}{\|\mathbf{w}\|} \quad \text{s.t.} \quad y_i(\mathbf{w}^T\mathbf{x}_i + b) \geq 1, \forall i$$

等价于最小化 $\frac{1}{2}\|\mathbf{w}\|^2$，这正是结构风险最小化的体现——最小化 $\|\mathbf{w}\|$ 等价于控制 VC 维。通过核方法 $K(\mathbf{x}_i, \mathbf{x}_j) = \phi(\mathbf{x}_i)^T\phi(\mathbf{x}_j)$，SVM 在高维特征空间中学习非线性边界而无需显式映射。

## 发展演进

统计学习理论（1970s-1990s）$\to$ SVM（1995）$\to$ 正则化方法发展（Lasso, Elastic Net）$\to$ 核方法扩展 $\to$ 与[[深度学习]]的端到端学习形成互补。深度学习的实践成功推动了对其泛化能力的理论解释研究。

## 关键算法·模型

- **支持向量机（SVM）**：最大间隔分类，核方法非线性化，详见[[监督学习方法]]
- **Lasso / Ridge 回归**：L1/L2 正则化线性模型
- **核方法**：通过核函数隐式映射高维空间，包括 RBF 核、多项式核

## 应用场景

- **小样本高维数据**：SVM 在样本数有限但特征维度高的场景中表现优异（生物信息学、文本分类）
- **可解释性要求高的场景**：正则化线性模型提供系数可解释性（金融风控、医疗统计）
- **理论分析**：为理解学习算法泛化能力提供数学框架

## 与其他技术关系

- 统计学习理论是[[机器学习]]的数学基础
- SVM 等具体算法属于[[监督学习方法]]
- 正则化思想在[[深度学习]]中演化为 Weight Decay、Dropout 等
- 理论基础依赖概率与统计、线性代数和优化方法
- 集成方法中的树模型是表格数据的 SOTA 方法，详见[[集成学习方法]]

# 数学速查表

## 1. 概述

本速查表汇总 AI 学习中最常用的数学公式与概念，作为 [[01_线性代数|线性代数]]、[[02_微积分|微积分]]、[[03_概率与统计|概率与统计]]、[[04_优化方法|优化方法]]、[[05_信息论|信息论]] 的快速参考索引。每个公式标注来源笔记，便于深入查阅。设计目标：**最小冗余、最大检索效率**。

## 2. 发展历史

本表覆盖的公式从经典数学（17-19 世纪微积分、线性代数）到 [[00_大语言模型核心架构_综述|大语言模型]] 时代的特定工具（RoPE、GELU、AdamW），反映了 AI 数学工具栈的持续扩展。详细历史脉络参见各主题笔记。

## 4. 技术原理

### 线性代数（详见 [[01_线性代数|线性代数]]）

| 运算 | 公式 |
|------|------|
| 矩阵乘法 | $C_{ij} = \sum_k A_{ik} B_{kj}$ |
| 点积 | $\mathbf{a} \cdot \mathbf{b} = \sum_i a_i b_i$ |
| 迹 | $\text{tr}(\mathbf{A}) = \sum_i A_{ii}$；$\text{tr}(\mathbf{AB}) = \text{tr}(\mathbf{BA})$ |
| 行列式 | $\det(\mathbf{AB}) = \det(\mathbf{A})\det(\mathbf{B})$ |
| 范数 | L1: $\sum\|x_i\|$；L2: $\sqrt{\sum x_i^2}$；F: $\sqrt{\text{tr}(\mathbf{A}^T\mathbf{A})}$ |

**矩阵分解**：特征分解 $\mathbf{A} = \mathbf{Q}\mathbf{\Lambda}\mathbf{Q}^{-1}$；SVD $\mathbf{A} = \mathbf{U}\mathbf{\Sigma}\mathbf{V}^T$；Cholesky $\mathbf{A} = \mathbf{L}\mathbf{L}^T$

**矩阵微积分**：$\nabla(\mathbf{a}^T\mathbf{x}) = \mathbf{a}$；$\nabla(\mathbf{x}^T\mathbf{A}\mathbf{x}) = 2\mathbf{A}\mathbf{x}$（$\mathbf{A}$ 对称）；$\nabla(\|\mathbf{A}\mathbf{x}-\mathbf{b}\|^2) = 2\mathbf{A}^T(\mathbf{A}\mathbf{x}-\mathbf{b})$

### 微积分（详见 [[02_微积分|微积分]]）

| 概念 | 公式 |
|------|------|
| 链式法则（单变量） | $\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx}$ |
| 链式法则（多变量） | $\frac{\partial y}{\partial x_j} = \sum_i \frac{\partial y}{\partial u_i} \frac{\partial u_i}{\partial x_j}$ |
| 梯度 | $\nabla f = [\frac{\partial f}{\partial x_1}, \dots, \frac{\partial f}{\partial x_n}]^T$ |
| Taylor 展开 | $f(\mathbf{x}) \approx f(\mathbf{a}) + \nabla f^T\Delta\mathbf{x} + \frac{1}{2}\Delta\mathbf{x}^T \mathbf{H} \Delta\mathbf{x}$ |
| Jacobian | $J_{ij} = \frac{\partial f_i}{\partial x_j}$ |
| Hessian | $H_{ij} = \frac{\partial^2 f}{\partial x_i \partial x_j}$ |

**数值稳定**：Log-Sum-Exp $\log\sum \exp(x_i) = \max(x) + \log\sum \exp(x_i - \max(x))$

### 概率与统计（详见 [[03_概率与统计|概率与统计]]）

| 概念 | 公式 |
|------|------|
| 贝叶斯定理 | $P(A\|B) = \frac{P(B\|A)P(A)}{P(B)}$ |
| 期望 | $E[X] = \sum x\,P(x)$（离散）；$\int x\,f(x)\,dx$（连续） |
| 方差 | $\text{Var}(X) = E[X^2] - (E[X])^2$ |
| MLE | $\hat{\theta} = \arg\max_\theta \sum \log P(x_i\|\theta)$ |
| MAP | $\hat{\theta} = \arg\max_\theta [\log P(D\|\theta) + \log P(\theta)]$ |
| 大数法则 | $\bar{X}_n \xrightarrow{P} \mu$ |
| 中心极限定理 | $\sqrt{n}\frac{\bar{X}_n - \mu}{\sigma} \xrightarrow{d} \mathcal{N}(0, 1)$ |

**常用分布**：高斯 $\mathcal{N}(\mu, \sigma^2)$；伯努利 $P(X=1)=p$；均匀 $U(a,b)$

### 信息论（详见 [[05_信息论|信息论]]）

| 概念 | 公式 |
|------|------|
| 熵 | $H(X) = -\sum P(x)\log P(x)$ |
| 交叉熵 | $H(P,Q) = -\sum P(x)\log Q(x) = H(P) + D_{KL}(P\|Q)$ |
| KL 散度 | $D_{KL}(P\|Q) = \sum P(x)\log\frac{P(x)}{Q(x)}$ |
| 互信息 | $I(X;Y) = H(X) - H(X\|Y)$ |
| JS 散度 | $D_{JS} = \frac{1}{2}D_{KL}(P\|M) + \frac{1}{2}D_{KL}(Q\|M)$，$M=\frac{P+Q}{2}$ |
| ELBO | $\log p(x) \geq E_q[\log p(x,z) - \log q(z)]$ |
| 困惑度 | $\text{PPL} = \exp(-\frac{1}{N}\sum \log P(x_i\|ctx))$ |

### 优化方法（详见 [[04_优化方法|优化方法]]）

| 方法 | 更新规则 |
|------|---------|
| SGD | $\theta \leftarrow \theta - \eta g_t$ |
| 动量法 | $v \leftarrow \mu v + \eta g_t$；$\theta \leftarrow \theta - v$ |
| RMSProp | $E[g^2]_t = \rho E[g^2]_{t-1} + (1-\rho)g_t^2$；$\theta \leftarrow \theta - \frac{\eta}{\sqrt{E[g^2]_t+\epsilon}} g_t$ |
| Adam | $\hat{m}_t = \frac{m_t}{1-\beta_1^t}$；$\hat{v}_t = \frac{v_t}{1-\beta_2^t}$；$\theta \leftarrow \theta - \eta\frac{\hat{m}_t}{\sqrt{\hat{v}_t}+\epsilon}$ |
| AdamW | $\theta \leftarrow \theta - \eta(\frac{\hat{m}_t}{\sqrt{\hat{v}_t}+\epsilon} + \lambda\theta_t)$ |

**余弦退火**：$\eta_t = \eta_{min} + \frac{1}{2}(\eta_0 - \eta_{min})(1+\cos\frac{t\pi}{T})$

**KKT 条件**：$\nabla f + \sum \lambda_i \nabla g_i + \sum \mu_j \nabla h_j = 0$，$\lambda_i g_i = 0$，$\lambda_i \geq 0$

### 深度学习公式

**激活函数**：

| 函数 | 公式 | 导数 |
|------|------|------|
| Sigmoid | $\sigma(x) = \frac{1}{1+e^{-x}}$ | $\sigma'(x) = \sigma(x)(1-\sigma(x))$ |
| ReLU | $\max(0, x)$ | $\begin{cases}1 & x>0\\0 & x<0\end{cases}$ |
| GELU | $x\Phi(x) \approx x\sigma(1.702x)$ | — |
| SiLU/Swish | $x\sigma(x)$ | $\sigma(x)(1+x(1-\sigma(x)))$ |
| Softmax | $\frac{\exp(x_i-\max)}{\sum_j \exp(x_j-\max)}$ | $J_{ij} = s_i(\delta_{ij}-s_j)$ |

**损失函数**：MSE $L = \frac{1}{n}\sum(y_i - \hat{y}_i)^2$；交叉熵 $L = -\sum y_i \log \hat{y}_i$

**归一化**：$\hat{x} = \frac{x - \mu}{\sqrt{\sigma^2 + \epsilon}}$，$y = \gamma\hat{x} + \beta$；RMSNorm：$\hat{x} = \frac{x}{\sqrt{\frac{1}{d}\sum x_i^2 + \epsilon}}$

**注意力**：$\text{Attention}(Q,K,V) = \text{softmax}(\frac{QK^T}{\sqrt{d_k}})V$；$\text{MultiHead} = \text{Concat}(\text{head}_1, \dots, \text{head}_h)W^O$

**初始化**：Xavier $\text{Var}(W) = \frac{2}{\text{fan\_in}+\text{fan\_out}}$；He $\text{Var}(W) = \frac{2}{\text{fan\_in}}$

# 正则化与约束优化

## 1. 概述

正则化是在优化过程中引入额外约束或惩罚项，以防止模型过拟合、提升泛化性能的技术。从经典的 L1/L2 正则化到深度学习中的 Dropout，正则化技术贯穿了从 [[00_机器学习_综述|机器学习]] 到 [[00_深度学习基础_综述|深度学习]] 的所有模型训练。约束优化则通过拉格朗日乘子法将约束融入目标函数，是正则化的理论基础。

## 2. 发展历史

| 年代 | 里程碑 | 意义 |
|------|--------|------|
| 1944 | Tikhonov 正则化 | L2 正则化的数学基础（不适定问题） |
| 1951 | KKT 条件 | 约束优化最优性条件 |
| 1996 | LASSO (Tibshirani) | L1 正则化产生稀疏解 |
| 1996 | Weight Decay (Krogh & Hertz) | 神经网络权重衰减 |
| 2012 | Dropout (Hinton et al.) | 随机失活正则化 |
| 2015 | Batch Normalization (Ioffe & Szegedy) | 间接正则化效果 |
| 2016 | Layer Normalization (Ba et al.) | Transformer 标准组件 |
| 2017 | AdamW (Loshchilov & Hutter) | 解耦权重衰减 |
| 2019 | Mixup / CutMix | 数据增强正则化 |
| 2020s | Label Smoothing | 大模型训练标配 |

## 3. 核心概念

- **过拟合**：模型在训练集上表现好但在测试集上泛化差
- **正则化**：在损失函数中加入惩罚项或使用策略限制模型复杂度
- **权重衰减**：直接缩小权重幅值，等价于 L2 正则（在 SGD 中）
- **稀疏性**：L1 正则化使部分参数精确为零，实现特征选择
- **约束优化**：$\min f(x)$ s.t. $g(x) \leq 0$，通过拉格朗日乘子转化

## 4. 技术原理

### 4.1 L2 正则化（Ridge / 权重衰减）

在损失函数中加入权重的 L2 范数平方：

$$\mathcal{L}_{\text{reg}} = \mathcal{L} + \frac{\lambda}{2}\|\mathbf{w}\|^2_2 = \mathcal{L} + \frac{\lambda}{2}\sum_i w_i^2$$

**梯度**：$\nabla \mathcal{L}_{\text{reg}} = \nabla \mathcal{L} + \lambda \mathbf{w}$

**SGD 更新**：$\mathbf{w}_{t+1} = (1 - \eta\lambda)\mathbf{w}_t - \eta\nabla\mathcal{L}$

每步先将权重缩小 $(1-\eta\lambda)$ 倍，再沿梯度方向更新。这就是"权重衰减"名称的由来。

**贝叶斯视角**：L2 正则化等价于参数的零均值高斯先验 $p(\mathbf{w}) = \mathcal{N}(0, \frac{1}{\lambda}\mathbf{I})$，MAP 估计导出 L2 正则。

### 4.2 L1 正则化（LASSO）

$$\mathcal{L}_{\text{reg}} = \mathcal{L} + \lambda \|\mathbf{w}\|_1 = \mathcal{L} + \lambda \sum_i |w_i|$$

**梯度**：$\nabla \mathcal{L}_{\text{reg}} = \nabla \mathcal{L} + \lambda \cdot \text{sign}(\mathbf{w})$

**稀疏性原理**：L1 惩罚在 $w_i = 0$ 处不可微，次梯度包含 0，使得 $w_i = 0$ 成为最优解。几何上，L1 约束的菱形与等损失线的交点更倾向于坐标轴。

**贝叶斯视角**：L1 正则化等价于参数的拉普拉斯先验 $p(w_i) = \frac{\lambda}{2}e^{-\lambda|w_i|}$。

**弹性网络（Elastic Net）**：结合 L1 和 L2：$\lambda_1 \|\mathbf{w}\|_1 + \lambda_2 \|\mathbf{w}\|^2_2$，兼顾稀疏性和稳定性。

### 4.3 权重衰减 vs L2 正则化

在 **SGD** 中两者等价。但在 **Adam** 中不等价：

- **L2 正则**：将 $\lambda\mathbf{w}$ 加入梯度，$\mathbf{w}$ 经过 Adam 的自适应缩放 $\frac{1}{\sqrt{v_t}}$ 后衰减效果被扭曲
- **权重衰减**（AdamW）：直接在参数更新中衰减，独立于自适应学习率

$$\theta_{t+1} = \underbrace{(1 - \eta\lambda)\theta_t}_{\text{权重衰减}} - \underbrace{\frac{\eta}{\sqrt{\hat{v}_t}+\epsilon}\hat{m}_t}_{\text{Adam 更新}}$$

**AdamW 的优势**：权重衰减强度均匀作用于所有参数，不被梯度幅度影响。

### 4.4 Dropout

训练时以概率 $p$ 随机将神经元输出置零：

$$\tilde{\mathbf{h}} = \mathbf{m} \odot \mathbf{h}, \quad m_i \sim \text{Bernoulli}(1-p)$$

**推理时**：不进行 Dropout，但将输出乘以 $(1-p)$（或训练时使用 Inverted Dropout 缩放 $\frac{1}{1-p}$）。

**理论解释**：
- **集成视角**：每次 Dropout 相当于采样一个子网络，最终模型是指数级子网络的隐式集成
- **正则视角**：阻止神经元"共适应"（co-adaptation），迫使每个神经元学习更鲁棒的特征
- **贝叶斯视角**：近似变分推断，Dropout 后的预测可估计模型不确定性（MC Dropout）

### 4.5 Batch Normalization 的正则化效果

Batch Normalization 虽然设计目的是解决内部协变量偏移，但实践中具有显著的正则化效果：

- Mini-batch 统计量引入噪声，类似 Dropout
- 减少对 Dropout 的需求
- 与 Dropout 同时使用时可能反而降低性能

### 4.6 Early Stopping

在验证集损失开始上升时停止训练：

```
Loss
  ↑
  │ Training
  │ ╲
  │  ╲
  │   ╲
  │    ╲___________
  │               ← 理想停止点
  │ Validation          ╱
  │                    ╱
  │                   ╱
  │                  ╱
  │                 ╱
  └──────────────────→ Epoch
```

**理论基础**：提前停止等价于 L2 正则化（Sjöberg & Ljung, 1995），训练迭代次数与正则强度成反比。

### 4.7 Label Smoothing

将 one-hot 标签 $\mathbf{y}$ 替换为平滑标签：

$$\tilde{y}_i = (1-\epsilon) y_i + \frac{\epsilon}{K}$$

其中 $K$ 为类别数，$\epsilon$ 为平滑因子（典型值 0.1）。

**作用**：防止模型过度自信（logit 过大），提升校准性和泛化。是 [[00_大语言模型核心架构_综述|大语言模型]] 训练的标配。

### 4.8 数据增强

通过变换训练数据增加有效样本量：

- **图像**：随机裁剪、翻转、颜色抖动、CutMix、Mixup
- **文本**：回译、同义词替换、EDA
- **语音**：加噪、速度变换、SpecAugment

**Mixup**：$\tilde{\mathbf{x}} = \lambda \mathbf{x}_i + (1-\lambda)\mathbf{x}_j$，$\tilde{y} = \lambda y_i + (1-\lambda) y_j$

**CutMix**：将一张图像的区域替换为另一张图像的对应区域，标签按面积比例混合。

### 4.9 梯度裁剪

限制梯度范数防止梯度爆炸：

**按范数裁剪**（最常用）：

$$\mathbf{g} \leftarrow \frac{\|\mathbf{g}_{\text{threshold}}\|}{\max(\|\mathbf{g}\|, \|\mathbf{g}_{\text{threshold}}\|)} \cdot \mathbf{g}$$

**按值裁剪**：$\mathbf{g} \leftarrow \text{clip}(\mathbf{g}, -c, c)$

[[00_大语言模型核心架构_综述|大语言模型]] 训练中梯度裁剪是标准配置（阈值通常为 1.0）。

### 4.10 约束优化的拉格朗日方法

将约束问题转化为无约束问题：

$$\min f(\mathbf{x}) \quad \text{s.t.} \quad g_i(\mathbf{x}) \leq 0$$

$$\mathcal{L}(\mathbf{x}, \boldsymbol{\lambda}) = f(\mathbf{x}) + \sum_i \lambda_i g_i(\mathbf{x})$$

**拉格朗日对偶上升法**：

$$\mathbf{x}^{k+1} = \arg\min_{\mathbf{x}} \mathcal{L}(\mathbf{x}, \boldsymbol{\lambda}^k)$$

$$\lambda_i^{k+1} = \max(0, \lambda_i^k + \alpha g_i(\mathbf{x}^{k+1}))$$

**增广拉格朗日法（ALM）**：加入二次惩罚项增强收敛性：

$$\mathcal{L}_\rho = f(\mathbf{x}) + \boldsymbol{\lambda}^T\mathbf{g}(\mathbf{x}) + \frac{\rho}{2}\|\mathbf{g}(\mathbf{x})\|^2$$

**ADMM**：交替方向乘子法，将变量分组交替更新，适合分布式约束优化。

## 5. 正则化方法对比

| 方法 | 类型 | 适用场景 | 计算开销 |
|------|------|---------|---------|
| L2 / 权重衰减 | 显式正则 | 通用 | 极低 |
| L1 / LASSO | 显式正则 | 特征选择、稀疏 | 低 |
| Dropout | 随机正则 | 全连接层 | 中 |
| Label Smoothing | 目标正则 | 分类、LLM | 极低 |
| Early Stopping | 隐式正则 | 通用 | 零 |
| 数据增强 | 数据正则 | 图像、语音 | 中 |
| 梯度裁剪 | 稳定性 | RNN、LLM | 低 |

## 6. 优势与局限

### 优势

- 有效防止过拟合，提升泛化性能
- 多种方法可组合使用（如 L2 + Dropout + 数据增强）
- 部分方法有严格的理论解释（贝叶斯先验、约束优化）
- 梯度裁剪等稳定性技术对大模型训练不可或缺

### 局限

- 正则化强度（$\lambda$、$p$ 等）需调参，增加搜索空间
- Dropout 可能减缓训练收敛
- L1 正则在高度相关的特征间随机选择，不稳定
- 过度正则化导致欠拟合

## 9. 前沿发展

- **Stochastic Depth**：随机跳过残差块，Dropout 的层级变体
- **DropConnect**：Dropout 权重而非激活
- **MaxNorm 约束**：限制权重向量的最大范数，与 AdamW 配合
- **Sharpness-Aware Minimization (SAM)**：寻找平坦极小值，提升泛化
- **Weight Noise**：训练时给权重加高斯噪声
- **大模型中的正则化简化**：大模型参数量足够大时，Dropout 作用减弱，主要依赖权重衰减和 Label Smoothing

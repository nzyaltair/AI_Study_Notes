# 大语言模型核心

> 本文件为 大语言模型核心 主题的索引文件，用于组织相关笔记内容。

## 概述

本目录包含大语言模型（LLM）的核心技术组件和设计原理，涵盖从基础架构到前沿技术的完整知识体系。内容从Transformer的核心机制出发，逐步深入到现代LLM的设计模式、优化策略和应用场景，适合AI开发者系统学习LLM技术。

## 主要内容

- [ ] [[LLM整体架构]] - 大语言模型的整体架构设计，包括Transformer基础、模型缩放规律和架构演进
- [ ] [[注意力机制]] - 注意力机制的原理、实现和变体，包括自注意力、多头注意力和高效注意力
- [ ] [[位置编码]] - 位置信息的表示方法，包括绝对位置编码、相对位置编码和旋转位置编码
- [ ] [[向量表示与Embedding]] - 文本的向量表示方法，包括词嵌入、句嵌入和上下文感知嵌入
- [ ] [[分词与Tokenization]] - 文本分词技术，包括Byte Pair Encoding (BPE)、WordPiece和SentencePiece
- [ ] [[长上下文与记忆机制]] - 处理长文本的技术，包括循环机制、检索增强和记忆网络
- [ ] [[推理模型与思维链]] - 大模型的推理能力，包括思维链（CoT）、自洽性和工具使用
- [ ] [[混合专家模型]] - 混合专家（MoE）模型的设计与实现，包括路由机制和扩展策略
- [ ] [[大模型设计模式]] - 现代大模型的设计模式，包括训练策略、优化方法和部署架构

## 学习路径

建议按照以下顺序学习：
1. 注意力机制 → 位置编码 → 向量表示与Embedding → 分词与Tokenization
2. LLM整体架构 → 大模型设计模式
3. 长上下文与记忆机制 → 推理模型与思维链 → 混合专家模型

# 深度学习框架

## 背景与发展

[[深度学习框架]]提供了自动微分、GPU 加速、模型抽象等基础设施，使开发者无需从零实现[[反向传播]]和矩阵运算。框架的演进从 Theano（2007）、Caffe（2013），到 TensorFlow（2015）和 PyTorch（2016），再到 JAX（2020）和 PyTorch 2.0（2023，torch.compile）。框架的选择直接影响开发效率和部署方案。

## 核心思想

通过计算图抽象和自动微分，让开发者只需定义前向传播逻辑，框架自动处理梯度计算和参数更新。这一抽象将数学公式到工程实现的鸿沟填平。

## 技术原理

### 计算图与自动微分

- **静态图**（TensorFlow 1.x）：先编译图再执行，优化空间大但调试困难
- **动态图**（PyTorch/TF 2.x Eager）：边执行边构建图，灵活易调试
- **自动微分**：前向传播记录操作，反向传播沿计算图自动求导

### 主流框架对比

| 特征 | PyTorch | TensorFlow | JAX |
|------|---------|-----------|-----|
| **计算图** | 动态图 | 动态图（Keras） | 函数式 |
| **开发方** | Meta | Google | Google |
| **易用性** | 高 | 高 | 中 |
| **研究生态** | 主导 | 次要 | 增长中 |
| **生产部署** | TorchServe | TF Serving | 较少 |
| **LLM 生态** | HuggingFace 主力支持 | 较少 | Flax/JAX |
| **分布式** | DDP/FSDP | TP Strategy | pmap/pjit |
| **编译优化** | torch.compile | XLA | XLA |

### PyTorch 核心概念

```python
import torch
import torch.nn as nn

class MLP(nn.Module):
    def __init__(self):
        super().__init__()
        self.fc1 = nn.Linear(784, 256)
        self.fc2 = nn.Linear(256, 10)

    def forward(self, x):
        x = torch.relu(self.fc1(x))
        return self.fc2(x)

model = MLP().cuda()
optimizer = torch.optim.AdamW(model.parameters(), lr=1e-3)
loss_fn = nn.CrossEntropyLoss()

for x, y in dataloader:
    x, y = x.cuda(), y.cuda()
    logits = model(x)
    loss = loss_fn(logits, y)
    optimizer.zero_grad()
    loss.backward()      # 自动微分
    optimizer.step()     # 参数更新
```

## 发展演进

Theano（2007）$\to$ Caffe（2013）$\to$ TensorFlow（2015）$\to$ PyTorch（2016）$\to$ TF 2.0（2019，Keras 整合）$\to$ JAX（2020）$\to$ PyTorch 2.0（2023，torch.compile）

PyTorch 在学术研究中占据主导地位，TensorFlow 在企业生产部署中仍有优势，JAX 在函数式和 TPU 场景中增长。

## 关键算法·模型

- **计算图**：静态图（编译优化）与动态图（灵活调试）
- **自动微分**：前向模式与反向模式
- **编译优化**：torch.compile（PyTorch 2.0）、XLA（TensorFlow/JAX）
- **分布式策略**：DDP、FSDP、张量并行、流水线并行

## 应用场景

- **学术研究**：PyTorch 为首，论文复现首选
- **LLM 开发**：PyTorch + HuggingFace Transformers + DeepSpeed
- **计算机视觉**：PyTorch + torchvision / TensorFlow
- **边缘部署**：TensorFlow Lite / PyTorch Mobile
- **科学计算**：JAX 在物理模拟、分子动力学中增长

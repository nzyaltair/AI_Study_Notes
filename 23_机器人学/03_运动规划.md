# 运动规划

## 一句话理解

在考虑运动学/动力学约束和障碍物的条件下，规划机器人从起点到目标的无碰撞路径或轨迹——是机器人从"知道去哪"到"知道怎么去"的核心环节。

## 背景与发展

运动规划是机器人学的核心问题之一。从经典的A*到采样式RRT，再到基于优化的轨迹规划，方法不断演进。

| 年代 | 里程碑 | 作者 | 意义 |
|:---|:---|:---|:---|
| 1968 | A*算法 | Hart等 | 启发式网格搜索 |
| 1986 | 人工势场法 | Khatib | 连续空间规划 |
| 1994 | PRM | Overmars等 | 概率路线图 |
| 1998 | RRT | LaValle | 快速随机树探索 |
| 2002 | CHOMP | Ratliff等 | 协变梯度轨迹优化 |
| 2011 | TrajOpt | Schulman | 序列凸优化 |
| 2016 | 深度学习规划 | 多家 | 端到端运动规划 |

## 核心概念

### 路径规划 vs 轨迹规划

- **路径规划**：找到无碰撞的几何路径 $\mathbf{q}(s), s \in [0, 1]$
- **轨迹规划**：在路径上加入时间信息 $\mathbf{q}(t)$，考虑速度、加速度约束

### 构型空间（Configuration Space, C-Space）

将机器人抽象为构型空间中的一个点，障碍物膨胀为C-空间中的C-障碍物。

$$\mathcal{C}_{free} = \mathcal{C} \setminus \mathcal{C}_{obs}$$

规划问题变为：在 $\mathcal{C}_{free}$ 中找到从 $\mathbf{q}_{start}$ 到 $\mathbf{q}_{goal}$ 的路径。

### 碰撞检测

判断机器人构型是否与障碍物碰撞。方法：

| 方法 | 原理 | 适用 |
|:---|:---|:---|
| 包围盒（AABB/OBB） | 简单几何体近似 | 快速初筛 |
| GJK算法 | 凸体距离 | 凸形状碰撞检测 |
| SAT（分离轴定理） | 投影重叠检测 | 多边形 |
| 球体树 | 层次球体近似 | 复杂形状 |

## 工作原理

### 网格搜索法

将C-空间离散化为网格，使用A*等搜索算法寻找路径。

- 优点：完备、最优（给定合适启发式）
- 缺点：维度灾难——网格数随维度指数增长

### 人工势场法

将目标视为引力源，障碍物视为斥力源，机器人沿合力方向运动：

$$\mathbf{F} = \mathbf{F}_{attr} + \mathbf{F}_{rep}$$

$$\mathbf{F}_{attr} = -\nabla U_{attr}(\mathbf{q}), \quad U_{attr} = \frac{1}{2}k\|\mathbf{q} - \mathbf{q}_{goal}\|^2$$

$$\mathbf{F}_{rep} = -\nabla U_{rep}(\mathbf{q}), \quad U_{rep} = \begin{cases} \frac{\eta}{2}\left(\frac{1}{d(\mathbf{q})} - \frac{1}{d_0}\right)^2 & d(\mathbf{q}) \leq d_0 \\ 0 & d(\mathbf{q}) > d_0 \end{cases}$$

优点：计算快，连续空间。缺点：局部最小值问题。

### 采样式规划

**RRT（Rapidly-exploring Random Tree）**：

```
1. 初始化树 T = {q_start}
2. 随机采样 q_rand ∈ C_free
3. 找到T中最近节点 q_near
4. 从q_near向q_rand扩展 q_new（固定步长）
5. 碰撞检测：若q_new在C_free中，加入T
6. 重复直到q_goal可达
```

RRT*是RRT的渐近最优版本——不断重新连接树中的节点，使路径趋向最优。

**PRM（Probabilistic Roadmap）**：
1. 在C_free中随机采样N个点
2. 连接相邻点（碰撞检测）
3. 使用图搜索（如A*）在路网上查找路径

PRM适合多次查询的场景（预构建路网后反复查询）。

### 轨迹优化

将路径平滑化并加入动力学约束：

$$\min_{\xi} \quad \underbrace{\int_0^T \|\dot{\xi}(t)\|^2 dt}_{\text{平滑性}} + \underbrace{\sum_i \text{cost}_{obs}(\xi(t_i))}_{\text{避障}}$$

方法：CHOMP（协变梯度下降）、TrajOpt（序列凸优化）、STOMP（随机轨迹优化）。

### 最优控制方法

将运动规划建模为最优控制问题：

$$\min_{\mathbf{u}(t)} \int_0^T L(\mathbf{x}, \mathbf{u}, t) dt + V_f(\mathbf{x}(T))$$
$$\text{s.t.} \quad \dot{\mathbf{x}} = f(\mathbf{x}, \mathbf{u}), \quad \mathbf{x}(0) = \mathbf{x}_0$$

求解方法：模型预测控制（MPC）、LQR、直接配点法。

## 应用场景

| 场景 | 规划方法 |
|:---|:---|
| 室内导航 | 网格A*+局部规划器（DWA/TEB） |
| 机械臂抓取 | RRT/RRT*+轨迹优化 |
| 自动驾驶 | Lattice planner + MPC |
| 无人机 | RRT*+MPC |
| 人形机器人 | 全身运动规划 |

## 优势与不足

### 优势
- 采样式规划在高维空间中高效
- 轨迹优化产生平滑、可执行的轨迹
- 多种方法可组合使用

### 不足
- 高维空间（人形机器人）规划仍具挑战
- 动态障碍物场景需要实时重规划
- 完备性和最优性难以同时保证

# 监督学习方法

## 1. 概述

[[06_监督学习方法|监督学习]]是[[00_机器学习_综述|机器学习]]中最常用、最成熟的范式。给定带标注的训练数据 $\{(x_1, y_1), \dots, (x_n, y_n)\}$，目标是学习从输入空间 $\mathcal{X}$ 到输出空间 $\mathcal{Y}$ 的映射 $f: \mathcal{X} \to \mathcal{Y}$，使得对未见过的 $x$，$f(x)$ 能准确预测 $y$。

监督学习分为两大类：
- **分类（Classification）**：$y$ 为离散类别
- **回归（Regression）**：$y$ 为连续值

## 2. 发展历史

| 年代 | 里程碑 | 意义 |
|------|--------|------|
| 1805 | Legendre/Gauss 提出最小二乘法 | 回归分析的数学基础 |
| 1936 | Fisher 提出线性判别分析（LDA） | 经典分类方法 |
| 1958 | Rosenblatt 发明感知机 | 首个可学习的二分类模型 |
| 1958 | Cox 提出逻辑回归 | 概率化二分类模型 |
| 1967 | Cover & Hart 提出 KNN | 基于实例的非参数方法 |
| 1979 | Quinlan 提出 ID3 决策树 | 符号化归纳学习 |
| 1984 | Breiman 等提出 CART | 统一分类与回归树框架 |
| 1992 | Boser, Guyon & Vapnik 引入核方法 SVM | 非线性分类突破 |
| 1995 | Cortes & Vapnik 提出软间隔 SVM | SVM 处理不可分数据 |
| 1996 | Tibshirani 提出 LASSO | L1 正则化自动特征选择 |

## 3. 核心概念

### 3.1 监督学习的形式化

- 训练集：$S = \{(x_1, y_1), \dots, (x_n, y_n)\}$，其中 $x_i \in \mathcal{X},\ y_i \in \mathcal{Y}$
- 假设空间：$\mathcal{H} = \{f_\theta: \mathcal{X} \to \mathcal{Y} \mid \theta \in \Theta\}$
- 损失函数：$L: \mathcal{Y} \times \mathcal{Y} \to \mathbb{R}^+$
- 目标：$\hat{\theta} = \arg\min_{\theta} \frac{1}{n}\sum_{i=1}^{n} L(y_i, f_\theta(x_i))$

### 3.2 生成模型 vs 判别模型

| | 生成模型 | 判别模型 |
|---|---------|---------|
| **建模对象** | 联合分布 $P(X, Y)$ | 条件分布 $P(Y \mid X)$ |
| **决策方式** | 贝叶斯定理：$P(Y|X) \propto P(X|Y)P(Y)$ | 直接建模决策边界 |
| **额外能力** | 可生成新样本、处理缺失值 | 通常分类精度更高 |
| **代表** | 朴素贝叶斯、GDA | 逻辑回归、SVM、决策树 |

### 3.3 判别函数、概率判别与概率生成

Bishop 将监督学习方法分为三类：

1. **判别函数**：直接学习 $f(x)$ 输出类别标签（如 SVM、单层感知机）
2. **概率判别**：学习 $P(Y|X)$ 后做决策（如逻辑回归）
3. **概率生成**：学习 $P(X, Y)$，用贝叶斯定理得到 $P(Y|X)$（如朴素贝叶斯）

概率方法的好处：可给出预测置信度、可结合先验知识、可处理缺失数据。

## 4. 技术原理

### 4.1 线性模型

**线性回归**：$y = \mathbf{w}^T\mathbf{x} + b$

- 损失：MSE $= \frac{1}{n}\sum_i (y_i - \hat{y}_i)^2$
- 闭式解：$\mathbf{w} = (\mathbf{X}^T\mathbf{X})^{-1}\mathbf{X}^T\mathbf{y}$（OLS）
- 梯度下降解：$\mathbf{w}^{(t+1)} = \mathbf{w}^{(t)} - \eta \cdot \frac{2}{n}\mathbf{X}^T(\mathbf{X}\mathbf{w} - \mathbf{y})$

**正则化变体**：

| 方法 | 惩罚项 | 闭式解 | 效果 |
|------|--------|--------|------|
| Ridge（L2） | $\lambda\|\mathbf{w}\|^2$ | $\mathbf{w} = (\mathbf{X}^T\mathbf{X} + \lambda\mathbf{I})^{-1}\mathbf{X}^T\mathbf{y}$ | 权重收缩，处理共线性 |
| Lasso（L1） | $\lambda\|\mathbf{w}\|_1$ | 无闭式解（坐标下降） | 稀疏解，特征选择 |
| Elastic Net | $\lambda_1\|\mathbf{w}\|_1 + \lambda_2\|\mathbf{w}\|^2$ | 无闭式解 | 组稀疏 + 稳定性 |

**逻辑回归**：将线性输出通过 Sigmoid 映射到 $[0,1]$：

$$P(y=1|\mathbf{x}) = \sigma(\mathbf{w}^T\mathbf{x} + b) = \frac{1}{1 + e^{-(\mathbf{w}^T\mathbf{x} + b)}}$$

- 损失：交叉熵 $L = -[y\log\hat{y} + (1-y)\log(1-\hat{y})]$
- 梯度：$\nabla_{\mathbf{w}} L = (\hat{y} - y)\mathbf{x}$ —— 形式简洁
- 多分类扩展：Softmax 回归 $\hat{y}_k = \frac{e^{\mathbf{w}_k^T\mathbf{x}}}{\sum_j e^{\mathbf{w}_j^T\mathbf{x}}}$

**逻辑回归的优势**：概率输出天然校准良好（well-calibrated），是人类可解释的分类基线模型。

### 4.2 决策树

决策树通过递归分治策略将特征空间划分为超矩形区域，形成树形决策结构。

**划分准则**：

| 算法 | 准则 | 公式 | 特点 |
|------|------|------|------|
| ID3 | 信息增益 | $IG = H(D) - \sum \frac{|D_v|}{|D|}H(D_v)$ | 偏向多值特征 |
| C4.5 | 信息增益率 | $IGR = IG / H_A(D)$ | 修正 ID3 偏差 |
| CART | 基尼系数 | $Gini = 1 - \sum_k p_k^2$ | 二叉树，计算高效 |

**信息熵**：$H(D) = -\sum_{k=1}^{K} p_k \log_2 p_k$，度量数据集的不确定性。

**剪枝策略**：

| 策略 | 方法 | 时机 |
|------|------|------|
| **预剪枝** | 限制深度、最少样本数、最小信息增益 | 分裂时判断 |
| **后剪枝** | 代价复杂度剪枝（CCP）：$R_\alpha(T) = R(T) + \alpha|T|$ | 树构建后自底向上 |

决策树的核心优点是可解释性强——每个决策路径对应一条可读的规则。

### 4.3 支持向量机（SVM）

**核心思想**：寻找最大化分类间隔的超平面。在 [[05_统计学习方法|统计学习方法]] 中已经详述其理论基础，这里聚焦算法实现。

**原问题**（软间隔）：

$$\min_{\mathbf{w}, b, \xi} \frac{1}{2}\|\mathbf{w}\|^2 + C\sum_{i=1}^{n}\xi_i$$

$$\text{s.t.} \quad y_i(\mathbf{w}^T\mathbf{x}_i + b) \geq 1 - \xi_i,\ \xi_i \geq 0$$

**参数 $C$ 的影响**：
- $C \to \infty$：不允许误分类 → 硬间隔 → 低偏差高方差
- $C \to 0$：允许大量误分类 → 宽间隔 → 高偏差低方差

**对偶问题**（便于核化和高效求解）：

$$\max_{\alpha} \sum_{i=1}^{n} \alpha_i - \frac{1}{2}\sum_{i,j} \alpha_i \alpha_j y_i y_j K(\mathbf{x}_i, \mathbf{x}_j)$$

$$\text{s.t.} \quad 0 \leq \alpha_i \leq C,\ \sum \alpha_i y_i = 0$$

- SMO（Sequential Minimal Optimization）算法：每次优化两个 $\alpha_i$，高效求解
- 支持向量：$\alpha_i > 0$ 的样本，决定了最终的分类超平面
- 只有支持向量参与预测，其他样本不影响模型

**核函数选择指南**：
- 特征维数 >> 样本数 → 线性核（避免过拟合）
- 两者相当 → RBF 核（默认首选）
- 样本数 >> 特征维数 → 手动构造特征 + 线性核

### 4.4 朴素贝叶斯

基于贝叶斯定理和特征条件独立假设：

$$P(y|\mathbf{x}) \propto P(y) \prod_{i=1}^{d} P(x_i | y)$$

尽管"朴素"假设（特征条件独立）在现实中通常不成立，但朴素贝叶斯在实践中表现惊人地好，原因在于：**即使概率估计不准确，分类决策（比较 $P(y|\mathbf{x})$ 的大小）仍可能正确**。

**三种变体**：

| 变体 | $P(x_i \mid y)$ 分布 | 适用场景 |
|------|---------------------|---------|
| 高斯 NB | $\mathcal{N}(\mu_{y,i}, \sigma^2_{y,i})$ | 连续特征 |
| 多项式 NB | 多项式分布（词频） | 文本分类 |
| 伯努利 NB | 伯努利分布（是否出现） | 二值特征、短文本 |

**拉普拉斯平滑**：$P(x_i|y) = \frac{N_{y,i} + \alpha}{N_y + \alpha \cdot d}$，防止零概率问题。

### 4.5 K 近邻（KNN）

KNN 是一种非参数、懒惰学习方法——没有显式训练阶段，预测时计算测试样本与所有训练样本的距离，取 K 个最近邻投票。

**关键设计选择**：

| 要素 | 选项 | 指南 |
|------|------|------|
| 距离度量 | 欧氏、曼哈顿、余弦 | 欧氏最常用；文本用余弦 |
| K 值 | 奇数，$\sqrt{n}$ 附近 | K 小 → 高方差；K 大 → 高偏差 |
| 权重 | 均匀 / 距离倒数 | 距离加权对近邻更敏感 |

**局限性**：计算复杂度 $O(nd)$ 每次预测需遍历全部训练集；在高维空间中距离趋于均匀（维数灾难）。

## 5. 关键方法/模型

### 5.1 方法对比

| 算法 | 模型类型 | 可解释性 | 数据需求 | 典型场景 |
|------|---------|---------|---------|---------|
| 线性回归 | 参数/判别 | 极高 | 中等 | 回归基线 |
| 逻辑回归 | 参数/概率判别 | 极高 | 中等 | 分类基线 |
| 决策树 | 非参数/判别 | 很高 | 中等 | 可解释分类 |
| SVM | 参数/判别 | 中等 | 小 | 高维小样本 |
| 朴素贝叶斯 | 参数/生成 | 高 | 小 | 文本分类 |
| KNN | 非参数/判别 | 中等 | 大 | 小数据分类 |

### 5.2 经典论文

- Cover & Hart (1967): Nearest Neighbor Pattern Classification — KNN 理论基础
- Quinlan (1986): Induction of Decision Trees — ID3 及决策树方法论
- Breiman et al. (1984): Classification and Regression Trees — CART 专著
- Cortes & Vapnik (1995): Support-Vector Networks — 软间隔 SVM
- Tibshirani (1996): Regression Shrinkage and Selection via the Lasso — LASSO
- Friedman, Hastie & Tibshirani (2010): Regularization Paths for Generalized Linear Models via Coordinate Descent — glmnet 算法

## 6. 优势与局限

### 6.1 算法级优劣

- **线性模型**：可解释性强，训练快速，是天然基线；但只能拟合线性关系
- **决策树**：可解释，无需特征缩放，处理混合类型数据；但容易过拟合，不稳定
- **SVM**：小样本高维数据表现优异，核方法灵活；但大规模数据训练慢，调参敏感
- **朴素贝叶斯**：训练极快，小数据表现好，概率输出；但特征独立假设不现实
- **KNN**：无需训练，理论简单；但预测慢，高维失效

### 6.2 通用局限

- 所有监督方法都依赖高质量标注数据
- 在非结构化数据（图像、语音）上不如[[00_深度学习基础_综述|深度学习]]
- 需要人工特征工程来获得最佳性能

## 7. 应用场景

| 算法 | 典型应用 |
|------|---------|
| 逻辑回归 | 信用评分、广告 CTR 预估、疾病风险预测 |
| 决策树 | 医疗诊断、客户分群规则、风控规则 |
| SVM | 文本分类、图像识别（小数据）、生物信息学 |
| 朴素贝叶斯 | 垃圾邮件过滤、新闻分类、情感分析 |
| KNN | 推荐系统协同过滤、小规模分类任务 |

- 线性模型是[[00_深度学习基础_综述|深度学习]]中神经网络的基础——单层 softmax 等价于多类逻辑回归
- 决策树是[[08_集成学习方法|集成学习]]（随机森林、GBDT）的基学习器
- SVM 的统计学习基础详见 [[05_统计学习方法|统计学习方法]]
- 朴素贝叶斯是理解生成模型和贝叶斯方法的入门
- 模型选择与评估详见 [[03_模型选择与调参|模型选择与调参]]
- 特征工程对这些方法的性能影响至关重要，详见 [[02_特征工程|特征工程]]

## 9. 前沿发展

- **可微分决策树**：用梯度下降训练软决策树，桥接树模型与深度学习
- **SVM 的大规模扩展**：线性 SVM 在大规模文本分类中仍有效（如 LIBLINEAR）
- **结构化预测**：SVM 和逻辑回归扩展到序列标注、图结构预测
- **校准方法**：Platt Scaling、Isotonic Regression 用于校准概率输出
- **Fair ML**：在传统监督方法中加入公平性约束（如公平逻辑回归）

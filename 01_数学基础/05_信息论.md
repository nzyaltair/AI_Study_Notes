# 信息论

## 背景与发展

信息论由 Claude Shannon 于 1948 年在论文《通信的数学理论》中创立，最初研究通信信道中信息的度量与传输极限。20 世纪后半叶，信息论概念被引入统计学和机器学习：交叉熵成为分类任务的标准损失函数，KL 散度用于衡量分布差异，ELBO 成为变分推断的核心。在 [[大语言模型]] 时代，信息论为损失函数设计、模型压缩和概率建模提供核心理论工具。

## 核心思想

信息论量化信息和不确定性。核心概念是熵——衡量随机变量的平均不确定性，以及 KL 散度——衡量两个概率分布的差异。最小化交叉熵等价于最大化似然，这一联系将信息论与统计推断统一起来。

## 技术原理

### 自信息与熵

- **自信息**：$I(x) = -\log P(x)$，概率越小的事件携带信息量越大
- **熵**（离散）：$H(X) = -\sum_{x} P(x) \log P(x) = -E[\log P(X)]$
- **微分熵**（连续）：$H(X) = -\int p(x) \log p(x)\,dx$
- 性质：$H(X) \geq 0$；均匀分布熵最大；确定性分布熵为 0

### 条件熵与联合熵

- **条件熵**：$H(Y|X) = -\sum_{x,y} P(x,y) \log P(y|x)$
- **联合熵**：$H(X,Y) = -\sum_{x,y} P(x,y) \log P(x,y)$
- **链式法则**：$H(X,Y) = H(X) + H(Y|X)$

### 互信息

$$I(X;Y) = H(X) - H(X|Y) = D_{KL}(P(X,Y) \| P(X)P(Y))$$

性质：$I(X;Y) \geq 0$；$I(X;Y) = 0$ 当且仅当 $X$ 与 $Y$ 独立；对称性 $I(X;Y) = I(Y;X)$。

### KL 散度

$$D_{KL}(P\|Q) = \sum_{x} P(x) \log\frac{P(x)}{Q(x)}$$

- **非负**：$D_{KL}(P\|Q) \geq 0$（Gibbs 不等式），当且仅当 $P=Q$ 时为 0
- **不对称**：$D_{KL}(P\|Q) \neq D_{KL}(Q\|P)$
- **前向 KL**（$D_{KL}(P\|Q)$）：使 Q 覆盖 P 的所有支持域
- **后向 KL**（$D_{KL}(Q\|P)$）：使 Q 集中在 P 的高概率区域

### 交叉熵

$$H(P,Q) = -\sum_{x} P(x) \log Q(x) = H(P) + D_{KL}(P\|Q)$$

最小化交叉熵等价于最大化似然：$\min H(P,Q) \iff \max \log P(\text{data}|\theta)$

### 重要不等式

- **吉布斯不等式**：$D_{KL}(P\|Q) \geq 0$
- **数据处理不等式**：对马尔可夫链 $X \to Y \to Z$，$I(X;Z) \leq I(X;Y)$
- **最大熵原理**：在给定约束下，熵最大的分布是最无偏的分布

### 扩展概念

- **JS 散度**：对称化的 KL 散度，$D_{JS} = \frac{1}{2}D_{KL}(P\|m) + \frac{1}{2}D_{KL}(Q\|m)$，$m = \frac{P+Q}{2}$
- **证据下界（ELBO）**：$\log p(x) \geq \mathbb{E}_{q(z)}[\log p(x,z) - \log q(z)]$，变分推断的核心

## 发展演进

Shannon 信息论 (1948) $\to$ 熵编码与数据压缩 $\to$ 统计学习中的信息准则（AIC/BIC）$\to$ 深度学习中的交叉熵损失与知识蒸馏 $\to$ 大模型对齐中的 KL 散度约束（DPO）。

## 关键算法·模型

- **交叉熵损失**：分类任务的标准损失函数
- **KL 散度**：知识蒸馏、DPO 对齐的核心度量
- **ELBO**：变分自编码器（VAE）的理论基础
- **困惑度（Perplexity）**：$= 2^{H(p,q)}$，语言模型评估指标

## 应用场景

- **损失函数设计**：交叉熵用于分类，KL 散度用于生成模型
- **模型压缩**：知识蒸馏用 KL 散度传递大模型知识到小模型
- **[[大语言模型]] 评估**：困惑度衡量语言模型性能
- **变分推断**：ELBO 是 VAE 的理论基础
- **[[强化学习]]**：熵正则化平衡探索与利用
- **特征选择**：互信息最大化选择与目标最相关的特征

## 与其他技术关系

- 建立在 [[概率与统计]] 基础上
- 交叉熵损失是 [[深度学习]] 分类任务的标准损失
- KL 散度是 [[大语言模型]] 对齐方法（如 DPO）的核心度量
- ELBO 是变分推断和 VAE 的理论基础
- 与 [[优化方法]] 密切相关：最小化交叉熵等价于最大似然估计

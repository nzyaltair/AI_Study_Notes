# 视频分析与理解

## 1. 概述

视频分析（Video Analysis）是计算机视觉从静态图像扩展到时序动态场景的领域。视频不仅是图像序列，还包含丰富的**时序信息**——运动、动作、事件、因果关系。视频理解的核心挑战在于如何高效地建模空间特征和时序动态的联合表示。

视频分析的主要任务包括：动作识别（识别视频中的人类动作）、动作定位（定位动作发生的时间段）、目标跟踪（在视频帧间持续跟踪目标）和视频理解（视频问答、视频描述等）。

视频数据的高维度和时序性使其比图像分析更具挑战性——一段 10 秒的视频可能包含数百帧，每帧都是一张高分辨率图像。

## 3. 核心概念

### 3.1 视频表示

- **帧序列**：$V = \{I_1, I_2, \ldots, I_T\}$，$T$ 帧图像
- **时空体**：$V \in \mathbb{R}^{T \times H \times W \times C}$，3D 体数据
- **光流**：$\mathbf{F}_t = (u_t, v_t)$，逐像素的瞬时运动场

### 3.2 任务分类

| 任务 | 输入 | 输出 | 说明 |
|:---|:---|:---|:---|
| 动作识别 | 视频片段 | 动作类别 | "这段视频在做什么" |
| 动作定位 | 未剪辑视频 | (动作, 起止时间) | 定位动作发生时段 |
| 时空动作定位 | 视频 | (动作, 时空管) | 定位动作发生的位置和时段 |
| 多目标跟踪 | 视频 | 轨迹集合 | 逐帧检测+关联 |
| 视频目标分割 | 视频 + 初始标注 | 逐帧掩码 | 在视频中分割目标 |
| 视频描述 | 视频 | 文本描述 | 生成视频内容描述 |
| 视频问答 | 视频 + 问题 | 答案 | 关于视频内容的问答 |

### 3.3 评估指标

| 任务 | 指标 | 说明 |
|:---|:---|:---|
| 动作识别 | Top-1 / Top-5 Accuracy | Clip 级分类准确率 |
| 动作定位 | mAP@IoU | 时段定位精度 |
| 多目标跟踪 | MOTA, IDF1, HOTA | 跟踪准确率、身份一致性、综合指标 |
| 视频分割 | J&F | 区域相似度 J + 轮廓准确度 F |

## 4. 技术原理

### 4.1 双流网络（Two-Stream）

空间流和时间流分别处理外观和运动：

```
视频 → 采样单帧 → 空间流 CNN → 空间特征 ─┐
    → 计算光流  → 时间流 CNN → 运动特征 ─┤→ 融合 → 分类
```

- **空间流**：单帧 RGB 图像分类，捕获外观信息
- **时间流**：输入多帧光流场，捕获运动信息
- **融合**：平均/加权融合两个流的预测

### 4.2 3D 卷积网络（C3D / I3D）

将 2D 卷积扩展到 3D（时间×高度×宽度），直接学习时空特征：

$$O(t,i,j) = \sum_{\Delta t} \sum_{\Delta i} \sum_{\Delta j} V(t+\Delta t, i+\Delta i, j+\Delta j) \cdot K(\Delta t, \Delta i, \Delta j)$$

**C3D**：$3 \times 3 \times 3$ 卷积核，3D 池化。

**I3D（Inflated 3D）**：将 2D 预训练模型（Inception）"膨胀"为 3D：
- 2D $k \times k$ 卷积 → 3D $t \times k \times k$ 卷积
- 权重初始化：沿时间维度平均复制 2D 权重
- 利用 ImageNet 预训练，解决 3D 网络数据不足问题

### 4.3 SlowFast 网络

双速率架构，模拟视网膜的 P 细胞和 M 细胞：

```
视频 → 慢通路（低帧率，大通道）→ 语义特征 ─┐
    → 快通路（高帧率，小通道）→ 运动特征 ─┤→ 融合 → 分类
```

- **Slow 通路**：低帧率（如每 16 帧采 1 帧），高通道数，捕获语义
- **Fast 通路**：高帧率（如每 2 帧采 1 帧），低通道数，捕获运动
- **横向连接**：Fast 通路特征融合到 Slow 通路

### 4.4 视频目标跟踪

#### SORT（Simple Online Realtime Tracking）

```
帧 t 检测 → IoU 匹配上一帧轨迹 → 更新/创建/删除轨迹
```

卡尔曼滤波预测目标位置，匈牙利算法进行检测-轨迹关联。

#### DeepSORT

在 SORT 基础上增加深度外观特征：

$$d(\mathbf{e}_i, \mathbf{e}_j) = \|\mathbf{e}_i - \mathbf{e}_j\|^2$$

关联度量 = IoU 距离 + 外观距离的加权组合：

$$c = \lambda d_{IoU} + (1-\lambda) d_{appearance}$$

#### ByteTrack

关键创新：利用低分检测框。高分检测先匹配，未匹配的轨迹再用低分检测匹配：

```
检测 → 高分检测匹配已有轨迹 → 未匹配轨迹 → 低分检测匹配 → 创建/删除
```

### 4.5 视频目标分割

#### 半监督视频目标分割（VOS）

给定第一帧标注，在后续帧中分割目标：

- **光流传播**：用光流将掩码传播到下一帧
- **记忆网络**：Memory Network 存储历史帧特征，查询当前帧

#### SAM 2 视频分割

SAM 2 将 SAM 扩展到视频，采用流式记忆架构：

```
帧 t → Image Encoder → 当前特征
                        ↓
               Memory Bank（历史帧 + 记忆 token）
                        ↓
               Memory Attention → 当前帧掩码
```

## 6. 优势与局限

### 优势

1. **时序理解**：捕捉图像无法表达的运动和事件信息
2. **运动分析**：光流和轨迹提供物理世界动态信息
3. **应用广泛**：安防监控、自动驾驶、运动分析、视频检索
4. **多模态融合**：视觉+音频+文本的多模态视频理解

### 局限

1. **计算量大**：视频数据维度高，处理成本远超图像
2. **长时序建模困难**：长视频中的远距离时序依赖仍具挑战
3. **标注成本高**：视频标注（逐帧/逐时段）远比图像昂贵
4. **实时性挑战**：实时视频分析需要高效的推理架构
5. **领域迁移困难**：动作识别模型在不同场景间泛化能力有限

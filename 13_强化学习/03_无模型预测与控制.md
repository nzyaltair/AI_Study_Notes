# 无模型预测与控制

## 1. 概述

无模型（Model-Free）强化学习方法不需要已知环境模型（$P$ 和 $R$），直接从与环境交互产生的经验数据中学习值函数或策略。这是实际应用中最常见的 RL 场景——大多数现实问题中我们无法获得精确的转移概率和奖励函数。

无模型方法分为两大类：

- **预测（Prediction）**：给定策略，估计值函数 $V^\pi$ 或 $Q^\pi$
- **控制（Control）**：寻找最优策略 $\pi^*$

本篇介绍蒙特卡洛方法、时序差分学习、Q-Learning、SARSA、n-step TD 和资格迹，这些是所有值函数类 RL 算法的基础。

## 3. 核心概念

### 3.1 蒙特卡洛（MC）与时序差分（TD）

| 维度 | 蒙特卡洛 | 时序差分 |
|:---|:---|:---|
| 更新时机 | 回合结束后 | 每步之后 |
| 目标值 | 实际回报 $G_t$ | 自举估计 $R_{t+1} + \gamma V(S_{t+1})$ |
| 偏差 | 无偏（使用真实回报） | 有偏（使用估计值） |
| 方差 | 高（回报是多项随机变量之和） | 低（只涉及一步随机性） |
| 适用性 | 仅回合制任务 | 回合制和继续型任务均可 |
| 收敛 | 大数定律保证 | 收缩映射保证 |

### 3.2 自举（Bootstrapping）

TD 方法用**当前估计值**更新自身——即用 $V(S_{t+1})$ 的估计来更新 $V(S_t)$。这种"用自己的估计来改善自己的估计"称为自举。MC 不自举（使用完整真实回报），DP 完全自举（使用所有后继状态的期望）。

### 3.3 on-policy 与 off-policy

- **on-policy**：行为策略（产生数据的策略）= 目标策略（被评估/改进的策略）。代表：SARSA
- **off-policy**：行为策略 $\neq$ 目标策略，需要重要性采样纠正。代表：Q-Learning

off-policy 的优势是可以利用旧数据或探索策略产生的数据来学习任意目标策略，数据利用率更高。

## 4. 技术原理

### 4.1 蒙特卡洛方法

MC 方法用**实际回报的样本平均**估计值函数。

**首次访问 MC（First-visit MC）**：只用每个状态在每个回合中首次被访问时的回报：

$$V(s) \leftarrow V(s) + \alpha [G_t - V(s)]$$

**每次访问 MC（Every-visit MC）**：使用每次访问的回报。首次访问 MC 无偏，每次访问 MC 在大样本下也收敛。

**MC 控制**：用 ε-greedy 策略平衡探索，MC 估计 $Q$ 函数：

$$Q(s,a) \leftarrow Q(s,a) + \alpha [G_t - Q(s,a)]$$

MC 控制需要 ε-soft 策略保证所有 $(s,a)$ 对被无限次访问。

### 4.2 时序差分（TD）学习

**TD(0) 预测**：

$$V(S_t) \leftarrow V(S_t) + \alpha \left[ R_{t+1} + \gamma V(S_{t+1}) - V(S_t) \right]$$

其中 $\delta_t = R_{t+1} + \gamma V(S_{t+1}) - V(S_t)$ 称为 **TD 误差**。

**TD(0) 的更新目标**：$R_{t+1} + \gamma V(S_{t+1})$——一个有偏但低方差的估计。

**TD(0) 收敛性**：在 $\alpha$ 满足 Robbins-Monro 条件（$\sum \alpha_t = \infty, \sum \alpha_t^2 < \infty$）下，TD(0) 收敛到 $V^\pi$。

### 4.3 SARSA（on-policy TD 控制）

SARSA 的名字来自五元组 $(S_t, A_t, R_{t+1}, S_{t+1}, A_{t+1})$：

$$Q(S_t, A_t) \leftarrow Q(S_t, A_t) + \alpha \left[ R_{t+1} + \gamma Q(S_{t+1}, A_{t+1}) - Q(S_t, A_t) \right]$$

SARSA 是 on-policy 的——$A_{t+1}$ 由当前策略（如 ε-greedy）采样得到，更新目标包含下一步实际采取的动作。

**SARSA 的行为**：在 Cliff Walking 等环境中，SARSA 学到的策略会"远离悬崖边缘"，因为它考虑了探索时的掉崖风险。

### 4.4 Q-Learning（off-policy TD 控制）

$$Q(S_t, A_t) \leftarrow Q(S_t, A_t) + \alpha \left[ R_{t+1} + \gamma \max_{a'} Q(S_{t+1}, a') - Q(S_t, A_t) \right]$$

Q-Learning 的更新目标使用 $\max_{a'} Q(S_{t+1}, a')$——假设下一步采取最优动作，而非行为策略实际采样的动作。因此 Q-Learning 直接近似 Bellman 最优方程，是 off-policy 的。

**Q-Learning 的行为**：在 Cliff Walking 中，Q-Learning 学到的策略"贴着悬崖边缘走"——因为它假设下一步会走最优路径（不探索），但实际行为策略有 ε 概率掉崖。

### 4.5 期望 SARSA

将 SARSA 中的 $Q(S_{t+1}, A_{t+1})$ 替换为期望值：

$$Q(S_t, A_t) \leftarrow Q(S_t, A_t) + \alpha \left[ R_{t+1} + \gamma \sum_a \pi(a|S_{t+1}) Q(S_{t+1}, a) - Q(S_t, A_t) \right]$$

期望 SARSA 降低了方差（不需要采样 $A_{t+1}$），在许多任务上优于 SARSA 和 Q-Learning。当 $\pi$ 是贪心策略时，退化为 Q-Learning。

### 4.6 n-step TD

MC 和 TD(0) 是两个极端——MC 用直到回合结束的所有奖励，TD(0) 只用一步。n-step TD 在两者之间插值：

**n-step 回报**：

$$G_t^{(n)} = R_{t+1} + \gamma R_{t+2} + \cdots + \gamma^{n-1} R_{t+n} + \gamma^n V(S_{t+n})$$

**n-step TD 更新**：

$$V(S_t) \leftarrow V(S_t) + \alpha \left[ G_t^{(n)} - V(S_t) \right]$$

n-step TD 在偏差和方差之间取得平衡：$n$ 越大越接近 MC（低偏差高方差），$n$ 越小越接近 TD(0)（高偏差低方差）。

### 4.7 TD(λ) 与资格迹

TD(λ) 将所有 n-step 回报以指数衰减权重 $\lambda^{n-1}$ 加权平均，通过**资格迹（Eligibility Traces）**高效实现。

**后向视角（Backward View）**：

资格迹：

$$e_t(s) = \gamma \lambda \, e_{t-1}(s) + \mathbf{1}[S_t = s]$$

更新：

$$V(s) \leftarrow V(s) + \alpha \, \delta_t \, e_t(s), \quad \forall s$$

其中 $\delta_t = R_{t+1} + \gamma V(S_{t+1}) - V(S_t)$。

- $\lambda = 0$：退化为 TD(0)
- $\lambda = 1$：退化为 MC（回合制）

资格迹让所有被访问过的状态按照"最近访问频率"分享 TD 误差的信用分配，是处理信用分配问题的高效机制。

### 4.8 重要性采样（Importance Sampling）

off-policy 方法中，行为策略 $b$ 产生的数据需要用重要性采样比率修正来估计目标策略 $\pi$ 的期望：

$$\rho_{t:T-1} = \prod_{k=t}^{T-1} \frac{\pi(A_k|S_k)}{b(A_k|S_k)}$$

**普通重要性采样**：$\mathbb{E}[G_t] \approx \frac{1}{N} \sum \rho_t G_t$（无偏，高方差）

**加权重要性采样**：$\mathbb{E}[G_t] \approx \frac{\sum \rho_t G_t}{\sum \rho_t}$（有偏，方差低，实践中更稳定）

Q-Learning 不需要显式重要性采样，因为它用 max 操作隐式处理了 off-policy 修正。

## 5. 关键方法/模型

### 5.1 Q-Learning

**论文**：Watkins, *Learning from Delayed Rewards* (1989)

最经典的无模型 RL 算法。off-policy TD 控制，直接近似 Bellman 最优方程。在表格设定下有收敛保证（在适当探索条件下收敛到 $Q^*$）。Q-Learning 是 DQN 的直接前身。

### 5.2 SARSA

**论文**：Rummery & Niranjan, *On-line Q-learning using Connectionist Systems* (1994)

on-policy TD 控制。SARSA 更保守——它学习的是"在当前探索策略下"的值函数，因此在有风险的环境中表现更稳健。

### 5.3 Double Q-Learning

**论文**：van Hasselt, *Double Q-learning* (2010)；van Hasselt et al., *Deep RL with Double Q-learning* (2016)

Q-Learning 的 $\max$ 操作导致**过估计偏差（Overestimation Bias）**：$\mathbb{E}[\max_a Q(s,a)] \geq \max_a \mathbb{E}[Q(s,a)]$。

Double Q-Learning 使用两组独立的 Q 估计：用 $Q_1$ 选动作，用 $Q_2$ 评估值：

$$Y = R + \gamma Q_2(S', \arg\max_{a'} Q_1(S', a'))$$

解耦选择和评估消除过估计偏差。

### 5.4 n-step Bootstrapping 与 Tree Backup

n-step Q-Learning 的 off-policy 版本需要重要性采样或 Tree Backup（无需重要性采样的 off-policy n-step 方法）。

## 6. 优势与局限

### 优势

1. **不需要环境模型**：直接从交互中学习，适用面广
2. **计算高效**：每步更新，无需等待回合结束（TD 方法）
3. **理论基础扎实**：收敛性有理论保证（表格设定）
4. **数据效率**：off-policy 方法可复用旧数据

### 局限

1. **表格表示限制**：只能处理有限小状态空间，无法处理高维连续状态
2. **样本效率低**：相比 DP，需要大量交互样本
3. **探索依赖**：需要适当的探索策略保证所有 $(s,a)$ 被访问
4. **off-policy 不稳定**：与函数近似结合时可能发散（致命三要素：函数近似 + 自举 + off-policy）

## 7. 应用场景

| 场景 | 方法 | 说明 |
|:---|:---|:---|
| GridWorld | Q-Learning / SARSA | 教学/验证基准 |
| 简单控制 | Tabular Q-Learning | 离散状态空间的小问题 |
| 推荐系统 | Q-Learning + 特征工程 | 用户状态离散化后应用 |
| 游戏对战 | SARSA + ε-greedy | on-policy 保证策略稳健性 |

## 8. 与其他技术关系

- **与动态规划的关系**：Q-Learning 是值迭代的采样近似——用样本 $(s,a,r,s')$ 代替 $\sum_{s'} P(s'|s,a)[\ldots]$。SARSA 是策略评估的采样近似。参见 [[02_值函数与动态规划]]。
- **与深度强化学习的关系**：DQN = Q-Learning + 深度神经网络 + 经验回放 + 目标网络。参见 [[05_深度强化学习]]。
- **与策略梯度方法的关系**：Actor-Critic 方法中 Critic 用 TD 学习估计值函数，Actor 用策略梯度更新策略。参见 [[04_策略梯度方法]]。
- **与探索策略的关系**：Q-Learning/SARSA 依赖 ε-greedy 等探索策略保证收敛。参见 [[07_探索与利用]]。

## 9. 前沿发展

- **Q(λ) 与 Retrace(λ)**：更稳定的 off-policy 资格迹方法，降低重要性采样方差
- **Gradient-TD 方法**：在致命三要素下保证收敛的 TD 变体
- **Emphatic TD**：通过强调函数修正 off-policy 学习中的不稳定性
- **与函数近似的结合**：线性函数近似下 TD 的收敛性理论已成熟，非线性（深度网络）下仍需技巧

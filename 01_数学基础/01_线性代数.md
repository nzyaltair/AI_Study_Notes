# 线性代数

## 1. 概述

线性代数研究向量空间及其上的线性变换。在 AI 中，一切数据（图像像素、词向量、特征列）和模型参数（权重矩阵）均以线性代数对象表示，模型的推理与训练本质上是矩阵乘法与线性变换。线性代数是深度学习最核心的数学语言——GPU 的大规模并行矩阵运算能力正是现代 AI 的硬件基础。

## 3. 核心概念

### 向量与向量空间

- **向量**：$n$ 个实数组成的有序数组，列向量 $\mathbf{a} = [a_1, \dots, a_n]^T$
- **内积**：$\langle \mathbf{a}, \mathbf{b} \rangle = \mathbf{a}^T \mathbf{b} = \sum_{i=1}^n a_i b_i$，度量向量间相似度
- **范数**：L2 范数 $\|\mathbf{a}\|_2 = \sqrt{\sum a_i^2}$；L1 范数 $\|\mathbf{a}\|_1 = \sum |a_i|$；Frobenius 范数 $\|\mathbf{A}\|_F = \sqrt{\text{tr}(\mathbf{A}^T\mathbf{A})}$
- **正交**：$\mathbf{a}^T \mathbf{b} = 0$
- **基与维度**：基是向量空间中最大的线性无关向量组，维度为基中向量个数

### 矩阵运算

- **矩阵乘法**：$\mathbf{C} = \mathbf{A}\mathbf{B}$，$c_{ij} = \sum_{k} a_{ik} b_{kj}$
- **Hadamard 积**（逐元素乘）：$(\mathbf{A} \odot \mathbf{B})_{ij} = A_{ij} B_{ij}$
- **特殊矩阵**：单位矩阵 $\mathbf{I}$、对角矩阵、对称矩阵（$\mathbf{A}^T = \mathbf{A}$）、正交矩阵（$\mathbf{A}^T\mathbf{A} = \mathbf{I}$）
- **迹**：$\text{tr}(\mathbf{A}) = \sum_i a_{ii}$，性质 $\text{tr}(\mathbf{A}\mathbf{B}) = \text{tr}(\mathbf{B}\mathbf{A})$
- **行列式**：方阵对空间体积的伸缩因子，$\det(\mathbf{A}\mathbf{B}) = \det(\mathbf{A})\det(\mathbf{B})$，可逆当且仅当 $\det \neq 0$

### 秩与空间

- **秩**：矩阵行/列向量组的最大线性无关组向量个数
- **值域（列空间）**：$\mathcal{R}(\mathbf{A}) = \{\mathbf{y} \mid \mathbf{y} = \mathbf{A}\mathbf{x}\}$
- **零空间**：$\mathcal{N}(\mathbf{A}) = \{\mathbf{x} \mid \mathbf{A}\mathbf{x} = \mathbf{0}\}$
- **秩-零化度定理**：$\dim(\mathcal{R}(\mathbf{A})) + \dim(\mathcal{N}(\mathbf{A})) = n$
- **Gram 矩阵**：$\mathbf{G} = \mathbf{A}^T\mathbf{A}$，对称半正定

### 条件数

- **定义**：$\kappa(\mathbf{A}) = \sigma_{\max} / \sigma_{\min}$（最大与最小奇异值之比）
- **意义**：衡量矩阵求逆的数值稳定性，条件数越大越病态
- **在 AI 中**：条件数大的损失面导致优化困难，BatchNorm/LayerNorm 部分缓解此问题

## 4. 技术原理

### 特征值与特征向量

- **定义**：$\mathbf{A}\mathbf{v} = \lambda\mathbf{v}$，特征向量方向在变换下不变
- **特征分解**：$\mathbf{A} = \mathbf{P}\mathbf{\Lambda}\mathbf{P}^{-1}$（当 $\mathbf{A}$ 有 $n$ 个线性无关特征向量）
- **对称矩阵性质**：特征值全为实数，不同特征值对应特征向量正交，可正交对角化 $\mathbf{A} = \mathbf{Q}\mathbf{\Lambda}\mathbf{Q}^T$
- **迹与行列式**：$\text{tr}(\mathbf{A}) = \sum \lambda_i$，$\det(\mathbf{A}) = \prod \lambda_i$

### 二次型

$f(\mathbf{x}) = \mathbf{x}^T\mathbf{A}\mathbf{x}$，$\mathbf{A}$ 对称。正定（所有特征值 $>0$）或半正定（特征值 $\geq 0$）。

在优化中，Hessian 矩阵的正定性决定驻点类型（极小/极大/鞍点）。

### 奇异值分解（SVD）

任意矩阵 $\mathbf{A} \in \mathbb{R}^{m \times n}$ 可分解为：

$$\mathbf{A} = \mathbf{U}\mathbf{\Sigma}\mathbf{V}^T$$

- $\mathbf{U} \in \mathbb{R}^{m \times m}$：左奇异向量（$\mathbf{A}\mathbf{A}^T$ 的特征向量），正交矩阵
- $\mathbf{\Sigma} \in \mathbb{R}^{m \times n}$：奇异值对角矩阵，$\sigma_1 \geq \sigma_2 \geq \dots \geq 0$
- $\mathbf{V} \in \mathbb{R}^{n \times n}$：右奇异向量（$\mathbf{A}^T\mathbf{A}$ 的特征向量），正交矩阵
- **Eckart-Young 定理**：截断 SVD $\mathbf{A}_k = \sum_{i=1}^k \sigma_i \mathbf{u}_i \mathbf{v}_i^T$ 是秩为 $k$ 的最优低秩近似（Frobenius 范数和谱范数意义下）
- **几何意义**：旋转 → 缩放 → 旋转

### 矩阵微积分

| 函数形式 | 梯度 | 条件 |
|---------|------|------|
| $f(\mathbf{x}) = \mathbf{a}^T \mathbf{x}$ | $\nabla f = \mathbf{a}$ | $\mathbf{a}$ 常数 |
| $f(\mathbf{x}) = \mathbf{x}^T \mathbf{x}$ | $\nabla f = 2\mathbf{x}$ | — |
| $f(\mathbf{x}) = \mathbf{x}^T \mathbf{A} \mathbf{x}$ | $\nabla f = 2\mathbf{A}\mathbf{x}$ | $\mathbf{A}$ 对称 |
| $f(\mathbf{X}) = \text{tr}(\mathbf{A}\mathbf{X})$ | $\nabla f = \mathbf{A}^T$ | — |

- **黑塞矩阵**：$H_{ij} = \frac{\partial^2 f}{\partial x_i \partial x_j}$，对称矩阵，正定性决定局部凸性
- **雅可比矩阵**：$J_{ij} = \frac{\partial f_i}{\partial x_j}$，向量值函数的一阶导数

### 其他重要分解

- **QR 分解**：$\mathbf{A} = \mathbf{Q}\mathbf{R}$（正交 × 上三角），用于最小二乘求解
- **Cholesky 分解**：$\mathbf{A} = \mathbf{L}\mathbf{L}^T$（对称正定矩阵），数值稳定高效
- **Schur 分解**：$\mathbf{A} = \mathbf{Q}\mathbf{U}\mathbf{Q}^T$（上三角），用于特征值计算

## 5. 关键方法/模型

### PCA（主成分分析）

- 通过协方差矩阵的特征分解实现降维
- 选择前 $k$ 个最大特征值对应的特征向量作为主成分
- 等价于在 SVD 框架下对数据矩阵的截断分解

### SVD 低秩近似

- **Eckart & Young (1936)**: *The Approximation of One Matrix by Another of Lower Rank* — 截断 SVD 的最优性证明
- 应用：推荐系统（矩阵分解）、图像压缩、LSA（潜在语义分析）

### 最小二乘法

$$\hat{\mathbf{x}} = (\mathbf{A}^T\mathbf{A})^{-1}\mathbf{A}^T\mathbf{b}$$

- 当 $\mathbf{A}^T\mathbf{A}$ 可逆时唯一解；不可逆时用伪逆 $\mathbf{A}^+ = \mathbf{V}\mathbf{\Sigma}^+\mathbf{U}^T$

### LoRA（Low-Rank Adaptation）

- **Hu et al. (2021)**: *LoRA: Low-Rank Adaptation of Large Language Models*
- $\Delta W = BA$，$B \in \mathbb{R}^{m \times r}$，$A \in \mathbb{R}^{r \times n}$，$r \ll \min(m,n)$
- 仅训练 $2mr$ 个参数而非 $mn$ 个
- 数学基础：预训练权重更新具有低秩结构（Aghajanyan et al., 2020 的观察）

## 6. 优势与局限

**优势**：
- 提供统一的数据表示和计算框架
- 矩阵运算可高度并行化，充分利用 GPU
- 矩阵分解揭示数据的低秩结构，支持降维和压缩

**局限**：
- 大矩阵的精确分解计算量为 $O(n^3)$，需要近似算法
- 高维稀疏矩阵需要特殊数据结构
- 非线性关系不能直接用线性代数表达（需通过核方法或神经网络引入非线性）

## 9. 前沿发展

- **随机化 SVD**：利用随机投影加速大规模矩阵分解（Halko et al., 2011），时间复杂度降至 $O(mnk)$
- **张量分解**：CP 分解、Tucker 分解在多模态学习和压缩中的应用
- **低秩适应的变体**：AdaLoRA（自适应秩分配）、KronA（Kronecker 分解）等
- **GPU 矩阵运算优化**：Tensor Core、混合精度矩阵乘法、FlashAttention 利用分块矩阵计算
- **流式/在线 SVD**：增量更新分解结果，适用于流数据和在线学习

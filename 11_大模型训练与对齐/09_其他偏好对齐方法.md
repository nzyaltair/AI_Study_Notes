# 其他偏好对齐方法

## 1. 概述

DPO的成功催生了一系列直接偏好优化变体。这些方法针对DPO的不同局限进行改进：IPO解决过拟合问题，KTO支持非配对反馈数据，SimPO去除参考模型，ORPO将偏好优化集成到SFT阶段。此外，拒绝采样微调（RFT）提供了一种简单有效的迭代对齐方案。

这些方法共同推动了偏好对齐的简化和普及，使对齐训练的门槛持续降低。

## 2. 发展历史

| 时间 | 方法 | 核心改进 |
|------|------|---------|
| 2023.5 | DPO | RL→分类，去除奖励模型 |
| 2023.8 | IPO | 平方损失，解决过拟合 |
| 2023.10 | KTO | 非配对反馈，前景理论 |
| 2024.2 | RFT | 拒绝采样迭代微调 |
| 2024.5 | SimPO | 去除参考模型 |
| 2024.5 | ORPO | SFT+偏好优化一体化 |

## 3. 核心概念

### 3.1 DPO的局限

DPO虽然简化了对齐流程，但存在以下局限：
- **过拟合风险**：在偏好数据较少时容易过拟合
- **数据格式限制**：需要成对的偏好数据（chosen vs rejected）
- **需要参考模型**：额外维护参考模型增加显存开销
- **离线学习**：受限于训练数据集覆盖

### 3.2 变体改进方向

```
DPO局限 → 变体改进
├── 过拟合 → IPO（平方损失）
├── 数据格式限制 → KTO（非配对反馈）
├── 需要参考模型 → SimPO（去除参考模型）
├── 独立对齐阶段 → ORPO（SFT+对齐一体化）
└── 离线学习 → Iterative DPO / RFT（迭代式）
```

## 4. 技术原理

### 4.1 IPO（Identity Preference Optimization）

IPO用平方损失替代DPO中的sigmoid损失，解决DPO在偏好数据上过拟合的问题：

$$\mathcal{L}_{IPO} = \mathbb{E}\left[\left(\log\frac{\pi_\theta(y_w|x)}{\pi_{ref}(y_w|x)} - \log\frac{\pi_\theta(y_l|x)}{\pi_{ref}(y_l|x)} - \frac{1}{\beta}\right)^2\right]$$

IPO提供了DPO的理论推广框架，可以适配不同的偏好数据假设。其核心贡献是提出了一个通用理论范式，DPO和IPO都是其特例。

### 4.2 KTO（Kahneman-Tversky Optimization）

KTO基于行为经济学中的前景理论（Prospect Theory），**只需二元反馈（好/坏）而非配对比较数据**：

$$\mathcal{L}_{KTO} = \mathbb{E}\left[\sigma\left(\beta \log\frac{\pi_\theta(y_{good}|x)}{\pi_{ref}(y_{good}|x)} - z_{good}\right) + \sigma\left(z_{bad} - \beta \log\frac{\pi_\theta(y_{bad}|x)}{\pi_{ref}(y_{bad}|x)}\right)\right]$$

其中 $z_{good}$ 和 $z_{bad}$ 是基于前景理论的参考点。KTO的核心优势是：
- 不需要配对偏好数据，只需"好/坏"标注
- 数据收集成本更低
- 利用了人类对损失和收益的不对称心理（损失厌恶）

### 4.3 SimPO（Simple Preference Optimization）

SimPO在DPO基础上去除参考模型，使用长度归一化的策略概率作为隐式奖励：

$$\mathcal{L}_{SimPO} = -\mathbb{E}\left[\log\sigma\left(\frac{\beta}{|y_w|}\log\pi_\theta(y_w|x) - \frac{\beta}{|y_l|}\log\pi_\theta(y_l|x) - \gamma\right)\right]$$

其中 $\gamma$ 是奖励margin超参数。SimPO的优势：
- **去除参考模型**：无需加载和前向传播参考模型，节省显存和计算
- **长度归一化**：通过除以输出长度解决DPO中长度偏差问题
- **最简方案**：训练流程最简单，计算成本最低

### 4.4 ORPO（Odds Ratio Preference Optimization）

ORPO将偏好优化集成到SFT阶段，**不需要独立的对齐阶段**：

$$\mathcal{L}_{ORPO} = \mathcal{L}_{SFT} + \lambda \cdot \mathcal{L}_{OR}$$

其中 $\mathcal{L}_{SFT}$ 是标准的SFT损失，$\mathcal{L}_{OR}$ 是基于赔率比（Odds Ratio）的偏好损失：

$$\mathcal{L}_{OR} = -\log\sigma\left(\log\frac{\text{odds}(\pi(y_w|x))}{\text{odds}(\pi(y_l|x))}\right)$$

其中 $\text{odds}(p) = \frac{p}{1-p}$。ORPO的优势：
- **无需参考模型**：赔率比自带归一化
- **无需独立对齐阶段**：SFT和对齐一步完成
- **训练效率高**：单阶段训练

### 4.5 拒绝采样微调（RFT）

RFT是一种简单有效的迭代对齐方法：

1. 用SFT模型对同一指令采样多个回答
2. 用奖励模型或规则筛选高质量回答（拒绝低质量）
3. 将筛选后的数据加入训练集
4. 重新训练模型
5. 重复上述过程

RFT的优势是简单、稳定，不涉及RL。LLaMA 2的后训练流程中使用了RFT。

### 4.6 方法对比总表

| 方法 | 数据格式 | 参考模型 | 训练阶段 | 核心创新 | 计算成本 |
|------|---------|---------|---------|---------|---------|
| DPO | 偏好对 | 需要 | 独立 | RL→分类 | 低 |
| IPO | 偏好对 | 需要 | 独立 | 平方损失防过拟合 | 低 |
| KTO | 好/坏标注 | 需要 | 独立 | 前景理论，非配对 | 低 |
| SimPO | 偏好对 | 不需要 | 独立 | 去参考模型+长度归一化 | 最低 |
| ORPO | 偏好对 | 不需要 | SFT阶段 | SFT+对齐一体化 | 最低 |
| RFT | 生成+评分 | 不需要 | 迭代SFT | 拒绝采样 | 中 |

## 6. 优势与局限

### 优势

- **流程持续简化**：从DPO到SimPO/ORPO，步骤越来越少
- **数据格式灵活**：KTO支持非配对反馈，降低数据收集门槛
- **计算成本降低**：SimPO/ORPO去除参考模型，显存减半
- **理论统一**：IPO提供了涵盖多种方法的通用框架

### 局限

- **离线方法局限**：大多数变体仍是离线学习，探索能力弱
- **超参数敏感**：$\beta$、$\gamma$ 等参数对性能影响大
- **理论保证有限**：去除参考模型后缺少理论正则化保证
- **实验验证不足**：部分新方法在大规模场景下的效果有待验证

## 7. 应用场景

- **快速实验**：SimPO/ORPO流程最简，适合快速验证
- **非配对数据对齐**：KTO适用于只有好/坏标注的场景
- **低资源环境**：去除参考模型的方法适合显存受限场景
- **迭代优化**：RFT/Iterative DPO适合持续优化
- **工业生产**：DPO已成为工业界对齐的主流选择

## 8. 与其他技术关系

- 所有方法都是[[DPO]]的变体或改进
- 与[[RLHF]]/[[GRPO]]不同，大多数为离线学习方法
- RFT可与[[监督微调]]结合，是一种增强SFT
- KTO的前景理论来源于行为经济学
- ORPO将[[监督微调]]和偏好对齐合二为一

## 9. 前沿发展

### 9.1 在线化

- **Iterative DPO**：多轮迭代，每轮生成新偏好数据
- **Online DPO**：实时收集反馈并更新
- **Online KTO/SimPO**：在线版本的各变体

### 9.2 理论统一

- **通用偏好优化框架**：IPO提出的理论范式涵盖多种方法
- **最优传输视角**：从最优传输理论分析对齐方法
- **信息论分析**：从信息论角度理解对齐过程

### 9.3 实践优化

- **自适应超参数**：自动调节 $\beta$、$\gamma$ 等参数
- **多目标对齐**：同时优化多个偏好维度
- **混合方法**：结合不同方法的优势（如DPO + RFT）

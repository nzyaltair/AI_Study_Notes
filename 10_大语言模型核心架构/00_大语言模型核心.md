# 大语言模型核心

大语言模型（LLM）是基于[[Transformer]]的超大规模[[预训练语言模型]]，通过在海量文本上自监督预训练获得通用语言能力，再通过指令微调和对齐技术成为实用的AI助手。

**核心范式**：GPT-3（2020）证明了"Scale is all you need"，ChatGPT（2022）展示了RLHF的威力，DeepSeek-V3/R1（2024-2025）展示了MoE+推理模型的融合创新，共同开启了LLM时代。

## 知识体系结构

本方向按照从基础架构到高级应用的认知逻辑组织，共9个子主题：

### 第一层：基础架构

#### [[LLM整体架构]]

#### [[注意力机制]]
缩放点积注意力、多头注意力、因果掩码、MHA/MQA/GQA/MLA对比、FlashAttention系列、注意力sink、高效注意力变体。

#### [[位置编码]]
正弦编码、可学习编码、相对位置编码、RoPE旋转位置编码、ALiBi、长度外推方法（PI/NTK-aware/YaRN/LongRoPE）、NoPE。

#### [[分词与Tokenization]]
BPE/WordPiece/SentencePiece/Unigram分词、Byte-level BPE、词表设计、Token效率分析、多语言分词优化。

#### [[向量表示与Embedding]]
Word2Vec/GloVe/FastText传统词嵌入、BERT上下文感知嵌入、SimCSE对比学习嵌入、E5/BGE指令微调嵌入、Matryoshka嵌入、Late Chunking。

### 第二层：架构创新

#### [[混合专家模型]]

#### [[长上下文与记忆机制]]

### 第三层：工程与实践

#### [[大模型设计模式]]

## 训练流程全景

```
数据收集与清洗 → 预训练（Next Token Prediction）
  → 监督微调（SFT，指令跟随）
  → 偏好对齐（RLHF/DPO/KTO，人类偏好对齐）
  → 安全对齐（Red Teaming，价值对齐）
  → 部署与推理优化
```

## 现代LLM标准技术栈

| 组件 | 当前最优选择 | 演进路径 |
|------|-------------|---------|
| 架构 | Decoder-only | Enc/Dec → Decoder-only统一 |
| 归一化 | RMSNorm (Pre-Norm) | Post-LN → Pre-LN → RMSNorm |
| 激活函数 | SwiGLU | ReLU → GELU → SwiGLU |
| 注意力 | GQA / MLA | MHA → MQA → GQA → MLA |
| 位置编码 | RoPE | 正弦 → 可学习 → RoPE |
| 前馈网络 | SwiGLU FFN / MoE | Dense FFN → Sparse MoE |
| 长上下文 | YaRN外推 + FlashAttention | 4K → 32K → 128K → 1M+ |

## 核心公式速查

### Scaling Law（Chinchilla）
$$L(N, D) \approx \left(\frac{N_c}{N}\right)^{0.076} + \left(\frac{D_c}{D}\right)^{0.095}, \quad D/N \approx 20$$

### 缩放点积注意力
$$\text{Attention}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V$$

### RoPE
$$q_m^T k_n = |q_m| |k_n| \cos(\theta_m - \theta_n), \quad \theta_i = 10000^{-2i/d}$$

### SwiGLU
$$\text{FFN}_{SwiGLU}(x) = (\text{Swish}(xW_{gate}) \odot xW_{up}) W_{down}$$

### MoE路由
$$y = \sum_{i \in \text{TopK}} \text{softmax}(\text{TopK}(W_g x, k))_i \cdot E_i(x)$$

### MLA（多头潜在注意力）
$$c_K = W_{DKV} x, \quad c_V = W_{DV} x \quad (d_c \ll d \times h)$$

### YaRN外推
$$\theta_i' = \theta_i / s, \quad s = \frac{L_{new}}{L_{train}}, \quad t = 0.1 \log(s) + 1$$

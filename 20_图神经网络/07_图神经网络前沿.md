# 图神经网络前沿

## 1. 概述

GNN 作为深度学习在非欧几里得数据上的核心方法，其前沿研究围绕**理论限制的突破**、**表达能力的提升**和**与新范式（大模型、AI for Science）的融合**展开。本笔记梳理 GNN 领域的关键理论问题（过平滑、过挤压、表达力上限）和前沿发展方向。

核心议题：
- **过平滑**（Over-smoothing）：多层 GNN 使节点表示趋同
- **过挤压**（Over-squashing）：信息瓶颈导致远距离信息丢失
- **表达力**（Expressiveness）：GNN 的图区分能力上限
- **高阶 GNN**：超越 1-WL test 的方法
- **GNN+LLM**：图学习与大语言模型的融合
- **AI for Science**：分子模拟、材料发现等科学应用

## 3. 核心概念

### 3.1 过平滑（Over-smoothing）

**现象**：随着 GNN 层数增加，不同节点的表示越来越相似，最终趋向于与度向量成正比，导致分类性能下降。

**数学描述**：GCN 的传播矩阵 $\hat{A} = \tilde{D}^{-1/2}\tilde{A}\tilde{D}^{-1/2}$ 的最大特征值为 1，对应特征向量为 $\tilde{D}^{1/2}\mathbf{1}$。反复传播 $k$ 次后，信号在所有非主特征向量方向上的分量衰减，只剩主特征向量方向——即所有节点表示趋同。

**度量**：
- **Dirichlet 能量**：$E(h) = \frac{1}{|\mathcal{E}|}\sum_{(u,v) \in \mathcal{E}} \|h_u - h_v\|^2$，能量越低表示越平滑
- **节点相似度**：相邻层节点表示变化的方差

### 3.2 过挤压（Over-squashing）

**现象**：当 GNN 层数增加以覆盖更远距离时，远距离节点的信息在传播过程中被"挤压"到固定大小的向量中，导致信息丢失。

**与过平滑的区别**：
- 过平滑：所有节点表示趋同（表示空间问题）
- 过挤压：远距离信息传播不充分（信息流问题）

**数学分析**：图的结构性质（曲率）决定了信息传播效率。负曲率的图（如瓶颈结构）更容易发生过挤压。

### 3.3 表达力（Expressiveness）

**1-WL 上限定理**（Xu et al., 2019; Morris et al., 2019）：
> 消息传递 GNN 的图区分能力不超过 1-WL test。存在非同构图对，1-WL 无法区分，因此任何 MPNN 也无法区分。

**经典反例**：无法区分的图对包括：
- 两个不同的正则图（同度数、同特征）
- 某些具有对称结构的图

### 3.4 图曲率（Graph Curvature）

**Ricci 曲率**的图上推广——衡量图的"弯曲程度"：
- **正曲率**：图偏向密集（如完全图），信息传播顺畅
- **负曲率**：图偏向树状/瓶颈，信息传播受阻
- **零曲率**：图偏平坦（如网格图）

曲率与过挤压：负曲率区域是信息瓶颈，容易发生过挤压。

## 4. 技术原理

### 4.1 缓解过平滑的方法

| 方法 | 原理 | 公式/机制 |
|:---|:---|:---|
| **残差连接** | 跨层直连保留原始信息 | $h^{(l+1)} = \sigma(\hat{A}h^{(l)}W) + h^{(l)}$ |
| **初始残差** | 连接到初始特征而非上一层 | $h^{(l+1)} = \sigma((1-\alpha)\hat{A}h^{(l)}W + \alpha X)$ |
| **DropEdge** | 随机删除边减缓平滑速度 | $\hat{A}' = \hat{A} \odot M, M_{ij} \sim \text{Bernoulli}(p)$ |
| **PairNorm** | 归一化节点对距离 | $h' = h - \bar{h}, h' = h' / \|h'\|$ |
| **LayerNorm** | 层归一化 | $h' = \gamma \cdot \frac{h - \mu}{\sigma} + \beta$ |
| **APPNP** | 解耦特征变换与传播 | $Z = (1-\alpha)\hat{A}Z + \alpha H^{(0)}$ |
| **GCNII** | 初始残差 + 恒等映射 | $H^{(l+1)} = \sigma((1-\alpha_l)\hat{A}H^{(l)}W + \alpha_l H^{(0)} + \beta_l W)$ |

**GCNII** 是当前最有效的抗过平滑方法之一：$\alpha_l$ 控制初始特征注入比例，$\beta_l$ 控制恒等映射强度。

### 4.2 缓解过挤压的方法

| 方法 | 原理 | 代表 |
|:---|:---|:---|
| **图重连（Graph Rewiring）** | 增加远距离边改善信息流 | DIGL, SDRF |
| **扩散卷积** | 用扩散替代直接传播 | GDC, ADC |
| **多跳聚合** | 直接聚合多跳邻居 | MixHop, N-GCN |
| **全局注意力** | Transformer 直接全局关注 | Graphormer |
| **图曲率优化** | 增加瓶颈区域边改善曲率 | SDRF, BLEND |

**图重连（Stochastic Discrete Ricci Flow, SDRF）**：
1. 计算图的 Ricci 曲率
2. 在负曲率（瓶颈）区域添加边
3. 移除最高正曲率边
4. 迭代直到曲率均匀化

### 4.3 高阶 GNN

超越 1-WL 的方法：

**$k$-GNN**（Morris et al.）：
- 以 $k$ 元组节点为基本单元
- 在 $k$ 元组上定义消息传递
- 表达力 = $k$-WL test
- 代价：$O(n^k)$ 计算复杂度

**子图 GNN**（Bevilacqua et al.; Zhang & Li）：
1. 对每个节点 $v$，提取以其为中心的子图 $G_v$
2. 用 GNN 编码每个子图 $h_{G_v} = \text{GNN}(G_v)$
3. 聚合子图表示得到节点表示 $h_v = \text{Pool}(h_{G_v})$
- 表达力超过 1-WL
- 复杂度取决于子图大小，可调节

**$k$-hop Message Passing**：
- 不同跳数使用不同的聚合函数
- MixHop：$\hat{A}^k$ 的多跳邻居聚合

### 4.4 GNN 可解释性

| 方法 | 原理 | 粒度 |
|:---|:---|:---|
| **GNNExplainer** | 优化子图掩码最大化互信息 | 实例级 |
| **PGExplainer** | 训练全局解释器网络 | 模型级 |
| **SubgraphX** | 蒙特卡洛树搜索找关键子图 | 实例级 |
| **GraphMask** | 边掩码，可学习稀疏掩码 | 模型级 |
| **GradCAM** | 梯度加权的类激活映射 | 实例级 |

**GNNExplainer** 的核心优化：

$$\max_{M} I(Y, G_S) = H(Y) - H(Y | G_S)$$

其中 $G_S$ 是被掩码选择的子图，$M$ 是连续的掩码矩阵。

### 4.5 GNN + LLM 融合

GNN 与大语言模型的融合是当前最前沿的方向：

| 融合范式 | 原理 | 代表 |
|:---|:---|:---|
| **LLM 增强节点特征** | LLM 编码文本属性作为节点特征 | TAPE, GIANT |
| **GNN 增强 LLM** | GNN 提供结构信息辅助 LLM 推理 | GraphGPT, GraphLLM |
| **统一模型** | LLM 直接处理图序列化输入 | InstructGLM, GraphText |
| **对齐训练** | GNN 和 LLM 嵌入空间对齐 | G-Retriever |

**LLM 增强节点特征（TAPE）**：
1. 用 LLM 为每个节点生成文本描述
2. 将描述编码为特征向量
3. 用增强后的特征训练 GNN

**图序列化（InstructGLM）**：
1. 将图结构序列化为文本（如邻接表）
2. 构造自然语言指令（"节点 A 的邻居有 B, C, D..."）
3. 让 LLM 直接进行图推理

## 5. 关键方法与模型

### 5.1 抗过平滑模型对比

| 模型 | 机制 | 可训练层数 | Cora 准确率 |
|:---|:---|:---|:---|
| GCN | 标准 | 2-3 层 | ~81% |
| GCN + 残差 | 残差连接 | 5-10 层 | ~81% |
| APPNP | 解耦传播 | 10+ 层 | ~83% |
| GCNII | 初始残差 + 恒等 | 16-64 层 | ~85% |
| DropEdge | 随机删边 | 5-10 层 | ~82% |

### 5.2 高阶 GNN 对比

| 方法 | 理论表达力 | 复杂度 | 适用场景 |
|:---|:---|:---|:---|
| MPNN (GIN) | = 1-WL | $O(\|\mathcal{E}\|)$ | 通用 |
| $k$-GNN | = $k$-WL | $O(n^k)$ | 小图 |
| 子图 GNN | > 1-WL | $O(n \cdot s)$ | 中等图 |
| Graph Transformer | > 1-WL | $O(n^2)$ | 中等图 |

### 5.3 AI for Science 中的 GNN

| 应用 | 图类型 | 模型 | 特点 |
|:---|:---|:---|:---|
| 分子属性预测 | 分子图 | GIN, SchNet | 原子=节点，键=边 |
| 蛋白质结构 | 残基图 | Equiformer | SE(3) 等变 |
| 材料发现 | 晶体图 | CGCNN, ALIGNN | 周期性图 |
| 药物-药物交互 | 异质图 | R-GCN, DeepDDI | 多关系 |
| 分子动力学 | 3D 分子图 | NequIP, Allegro | 等变神经网络 |

**等变 GNN**（Equivariant GNN）：
- 输入：3D 坐标 + 原子类型
- 要求：旋转/平移/反射等变（E(3) 或 SE(3) 等变）
- 公式：$f(T \cdot x) = T' \cdot f(x)$，其中 $T$ 为变换
- 代表：NequIP, e3nn, Equiformer

## 6. 优势与局限

### 6.1 当前优势

- **理论深化**：过平滑、表达力等理论问题有了深入理解
- **工程成熟**：框架和大规模训练技术支持工业部署
- **跨域应用**：从社交网络到分子模拟的广泛适用性
- **新范式融合**：与 LLM、预训练等新范式的融合正在加速

### 6.2 待解决的挑战

- **深层 GNN**：虽然 GCNII 等方法可训练 64 层，但深层 GNN 的收益仍有限
- **超大图**：十亿级以上图的 GNN 训练仍面临工程挑战
- **动态图实时性**：连续时间动态图的实时推理延迟高
- **跨图泛化**：不同图数据集间的知识迁移仍不成熟
- **GNN+LLM 的有效性**：当前融合方法在复杂图推理上仍不如专用 GNN
- **公平性**：图上的偏见和公平性问题研究不足

## 7. 应用场景

## 9. 前沿发展

### 理论前沿
- **图曲率与信息流**：从微分几何角度理解 GNN 信息传播
- **GNN 泛化理论**：GNN 的 generalization gap 理论分析
- **最优 GNN 深度**：自适应深度选择机制
- **图上的信息瓶颈**：信息论视角的 GNN 分析

### 方法前沿
- **等变 GNN**：SE(3)/E(3) 等变神经网络在分子建模中的应用（Equiformer, NequIP）
- **图扩散模型**：在图域应用扩散模型进行图生成（GraphGD, DiGress）
- **图强化学习**：图上的 RL（如分子优化、网络设计）
- **物理信息 GNN**：将物理方程约束嵌入 GNN（PINN + GNN）

### 应用前沿
- **AI for Science**：GNN 是分子模拟、材料发现、蛋白质工程的核心工具
- **GNN + LLM**：图基础模型、图指令微调、图 RAG
- **时序图实时推理**：低延迟动态图推理系统
- **图联邦学习**：隐私保护的分布式图学习
- **GNN 可信性**：鲁棒性、公平性、隐私保护

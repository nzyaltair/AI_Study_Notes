# 优化算法与学习率调度

## 1. 概述

优化算法是深度学习的"引擎"——它将损失函数的梯度信号转化为参数更新，驱动模型在高维非凸参数空间中寻找优质解。学习率调度则控制优化过程中的步长衰减策略，确保模型稳定收敛到泛化性能良好的局部极小值。

- **优化目标**：$\theta^* = \arg\min_\theta \mathcal{L}(\theta)$，其中 $\theta$ 是百万至千亿维参数向量，$\mathcal{L}$ 是非凸、高维、带噪声的损失函数
- **核心挑战**：非凸地形中的鞍点和局部极小值；不同参数的梯度尺度差异巨大；随机梯度的高方差

## 2. 发展历史

- **1847**：Cauchy 提出梯度下降法（Gradient Descent）
- **1951**：Robbins & Monro 提出随机近似算法（SGD的理论基础）
- **1964**：Polyak 提出动量法（Momentum）
- **1983**：Nesterov 提出加速梯度（NAG），在一阶方法中达到理论最优收敛速率
- **2011**：Duchi et al. 提出 AdaGrad，首次引入参数自适应学习率
- **2012**：Tieleman & Hinton 提出 RMSProp（在Coursera课程中介绍），解决AdaGrad学习率单调衰减问题
- **2014**：Kingma & Ba 提出 Adam，融合Momentum和RMSProp，成为深度学习最广泛使用的优化器
- **2017**：Loshchilov & Hutter 提出 AdamW，修正Adam中Weight Decay的不当实现
- **2023**：Chen et al. 提出 LION（EvoLved Sign Momentum），通过进化搜索发现的更简洁优化器

## 3. 核心概念

### 3.1 随机梯度 vs 全批量梯度

| 方法 | 梯度计算 | 优点 | 缺点 |
|:---|:---|:---|:---|
| **批量梯度下降（BGD）** | $\nabla \mathcal{L} = \frac{1}{N}\sum_{i=1}^N \nabla \ell_i$ | 梯度精确，确定性收敛 | 每步计算 $N$ 个样本，不可行 |
| **随机梯度下降（SGD）** | $\nabla \mathcal{L} \approx \nabla \ell_i$ | 单样本极快 | 梯度高方差，震荡严重 |
| **小批量SGD（Mini-batch SGD）** | $\nabla \mathcal{L} \approx \frac{1}{B}\sum_{i=1}^B \nabla \ell_i$ | **实践中的标准选择** | 需调batch size |

### 3.2 优化算法设计的三个层次

1. **基础梯度下降**：$ \theta_{t+1} = \theta_t - \eta g_t$
2. **一阶矩（动量）**：利用历史梯度的指数加权移动平均平滑更新方向
3. **二阶矩（自适应学习率）**：利用历史梯度平方估计，为每个参数分配独立的学习率

### 3.3 学习率的三重角色

- **收敛速度**：学习率越大收敛越快，但过大导致发散
- **收敛质量**：学习率过小可能陷入尖锐的局部极小值（泛化性能差），适中学习率偏好平坦极小值
- **稳定性**：学习率上限受损失曲面的Lipschitz常数约束：$\eta < 2/L$

## 4. 技术原理

### 4.1 SGD（Stochastic Gradient Descent）

$$\theta_{t+1} = \theta_t - \eta \cdot g_t$$

- **优点**：简单、可解释、对batch size不敏感
- **缺点**：梯度震荡大，收敛慢；对所有参数使用同一学习率，无视参数尺度差异
- **改进方向**：动量（平滑方向）和自适应学习率（个性化学步）

### 4.2 Momentum（动量法）

$$\mathbf{v}_t = \beta \mathbf{v}_{t-1} + (1 - \beta) g_t$$

$$\theta_{t+1} = \theta_t - \eta \mathbf{v}_t$$

- **物理直觉**：如同小球在损失曲面上滚动，动量项保留历史"速度"，平滑梯度震荡
- **超参数**：$\beta = 0.9$ 为经典默认值（即过去约10步梯度的指数加权平均）
- **效果**：在峡谷地形（某一维度梯度极小、另一维度梯度极大）中显著加速收敛

### 4.3 Nesterov Accelerated Gradient（NAG）

$$\theta_{\text{lookahead}} = \theta_t - \eta \beta \mathbf{v}_{t-1}$$

$$\mathbf{v}_t = \beta \mathbf{v}_{t-1} + (1 - \beta) \nabla \mathcal{L}(\theta_{\text{lookahead}})$$

$$\theta_{t+1} = \theta_t - \eta \mathbf{v}_t$$

- **巧思**：先在动量方向"前看一步"，在前看位置计算梯度——"先见之明"使NAG在凸优化中达到一阶方法理论最优收敛速率 $\mathcal{O}(1/t^2)$

### 4.4 AdaGrad

$$\mathbf{r}_t = \mathbf{r}_{t-1} + g_t \odot g_t$$

$$\theta_{t+1} = \theta_t - \frac{\eta}{\sqrt{\mathbf{r}_t} + \epsilon} \odot g_t$$

- **核心思想**：累积所有历史梯度的平方和，梯度大的参数获得更小的有效学习率，梯度小的参数获得更大的有效学习率
- **致命缺陷**：$\mathbf{r}_t$ 单调递增，学习率单调衰减至零，训练后期停滞

### 4.5 RMSProp

$$\mathbf{r}_t = \rho \mathbf{r}_{t-1} + (1 - \rho) g_t \odot g_t$$

$$\theta_{t+1} = \theta_t - \frac{\eta}{\sqrt{\mathbf{r}_t} + \epsilon} \odot g_t$$

- **改进**：用指数移动平均替代累加和，避免学习率单调衰减
- **超参数**：$\rho = 0.999$（对应约1000步的衰减窗口）

### 4.6 Adam（Adaptive Moment Estimation）

**融合Momentum + RMSProp**：

$$\mathbf{m}_t = \beta_1 \mathbf{m}_{t-1} + (1 - \beta_1) g_t \quad \text{(一阶矩估计)}$$

$$\mathbf{v}_t = \beta_2 \mathbf{v}_{t-1} + (1 - \beta_2) g_t \odot g_t \quad \text{(二阶矩估计)}$$

$$\hat{\mathbf{m}}_t = \frac{\mathbf{m}_t}{1 - \beta_1^t}, \quad \hat{\mathbf{v}}_t = \frac{\mathbf{v}_t}{1 - \beta_2^t} \quad \text{(偏差修正)}$$

$$\theta_{t+1} = \theta_t - \frac{\eta}{\sqrt{\hat{\mathbf{v}}_t} + \epsilon} \hat{\mathbf{m}}_t$$

- **偏差修正的必要性**：初始时刻 $\mathbf{m}_0 = \mathbf{v}_0 = 0$，如果不修正，前几步的有效步长会严重低估
- **默认超参数**：$\eta = 0.001$, $\beta_1 = 0.9$, $\beta_2 = 0.999$, $\epsilon = 10^{-8}$
- **地位**：2014-2022年间深度学习领域最广泛使用的优化器

### 4.7 AdamW（Adam with Decoupled Weight Decay）

$$\theta_{t+1} = \theta_t - \eta \lambda \theta_t - \frac{\eta}{\sqrt{\hat{\mathbf{v}}_t} + \epsilon} \hat{\mathbf{m}}_t$$

**问题根源**：传统Adam中，L2正则化 $\frac{\lambda}{2}\|\theta\|^2$ 求导得 $\lambda\theta$，此梯度同样被 $\sqrt{\hat{\mathbf{v}}_t}$ 缩放。这意味着梯度大的参数获得的实际正则化强度反而被削弱——这违反了L2正则化的设计初衷。

**AdamW的解决方案**：将权重衰减（$\eta\lambda\theta_t$）与梯度更新完全**解耦**（Decoupled），直接从参数中减去衰减项，不受二阶矩缩放的影响。

**现代实践**：AdamW是几乎所有现代大语言模型（LLaMA, GPT-4, Gemini, Claude）训练的标准优化器。

### 4.8 优化器选择速查

| 场景 | 推荐优化器 | 说明 |
|:---|:---|:---|
| CNN图像分类 | SGD + Momentum | 泛化性能可能优于Adam |
| Transformer / LLM | AdamW | 大模型训练的事实标准 |
| GAN训练 | Adam | 对超参数不敏感 |
| 强化学习 | Adam / RMSProp | 非平稳目标，需自适应 |
| 新任务快速原型 | Adam/AdamW | 收敛快，少调参 |

---

### 4.9 学习率调度（Learning Rate Scheduling）

学习率调度策略控制训练过程中 $\eta$ 的变化曲线，对最终模型质量的影响不亚于优化器本身的选择。

#### Warmup（预热）

前 $K$ 步学习率从极小值（如 $10^{-7}$）线性增长至目标值：

$$\eta_t = \eta_{\text{target}} \cdot \frac{t}{K}$$

- **必要性**：训练初期模型参数随机初始化，梯度方向和大小不稳定。若一开始就用大学习率，模型容易被"崩飞"到参数空间的不可恢复区域
- **LLM标配**：几乎所有大语言模型训练都使用Warmup（通常为总步数的1%-5%）

#### Cosine Annealing（余弦退火）

$$\eta_t = \eta_{\min} + \frac{1}{2}(\eta_{\max} - \eta_{\min})\left(1 + \cos\left(\frac{t\pi}{T}\right)\right)$$

- **曲线特性**：开始缓慢下降（保留探索能力），中段加速下降（精细收敛），末尾平缓（稳定在极小值邻域）
- **优势**：无超参数（仅需总步数 $T$），在CV和NLP任务上表现优异

#### 其他调度策略

| 策略 | 行为 | 适用场景 |
|:---|:---|:---|
| Step Decay | 每 $N$ 步乘以衰减因子 $\gamma$（如0.1） | 通用，但超参数需手动设定 |
| Linear Decay | $\eta_t = \eta_0(1 - t/T)$ | 简单任务 |
| Cosine with Restarts | 余弦周期结束后重置学习率 | 提升搜索空间覆盖率 |
| Constant | 学习率不变 | 搭配Adam时有时足够 |

**LLM的标准配置**：Warmup → Cosine Decay（或Warmup → Linear Decay）

## 6. 优势与局限

### Adam vs SGD+Momentum

| 维度 | SGD + Momentum | Adam / AdamW |
|:---|:---|:---|
| 收敛速度 | 慢 | **快** |
| 超参数敏感度 | 对学习率极敏感 | 较鲁棒 |
| 泛化性能 | **可能更优**（找到更平坦的极小值） | 可能略差 |
| 大模型训练 | 基本不用 | **事实标准** |
| 调参成本 | 高 | 低 |

### 自适应优化器的局限性

- **泛化差距**：自适应方法倾向于找到尖锐的极小值，泛化性能理论上不如SGD（但大模型时代此差距被数据规模和正则化弥补）
- **二阶矩估计偏差**：$\epsilon$ 的取值（$10^{-8}$ vs $10^{-5}$ vs $10^{-3}$）会微妙影响收敛行为

## 7. 应用场景

- **大语言模型预训练**：AdamW + Warmup + Cosine Decay → 标准三件套
- **视觉模型训练**：SGD+Momentum + Cosine Annealing → 经典组合
- **微调（Fine-tuning）**：AdamW + 较小的固定学习率（$10^{-5} \sim 10^{-4}$）或Cosine Decay
- **GAN训练**：生成器和判别器各用独立Adam，$\beta_1=0.5$（降低动量，适应非平稳博弈）
- **强化学习**：Adam + Constant LR（非平稳目标，无需衰减至零）

## 9. 前沿发展

1. **学习率的理论研究**
   - **$\mu$Transfer**（Yang et al., 2022）：在小模型上搜索最优学习率，可通过理论保证无损迁移至大模型，大幅降低大模型训练的调参成本
   - **Maximal Update Parameterization ($\mu$P)**：确保不同宽度网络的训练动力学一致

2. **新一代优化器**
   - **LION**：只使用梯度的符号（sign），极简设计，在扩散模型和LLM训练中表现优异
   - **Sophia**（2023）：轻量级Hessian对角估计，在语言模型预训练中收敛速度2倍于Adam

3. **学习率调度的自动化**
   - 基于强化学习和贝叶斯优化的自动学习率调度，试图消除"Warmup步数"、"衰减起点"等烦琐的超参数
   - 在线自适应调度：根据梯度的统计特征动态调整学习率，无需预设调度曲线

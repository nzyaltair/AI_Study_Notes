# CV前沿发展：CNN与视觉Transformer的融合时代

## 1. 概述

2020年Vision Transformer（ViT）的问世打破了CNN在计算机视觉领域长达八年的统治地位。此后的研究呈现出三条主线：

1. **ViT对CNN的挑战**：Transformer能否完全取代卷积？
2. **CNN的自我革新**：吸纳Transformer设计哲学，CNN能否回归竞争？
3. **CNN-Transformer融合**：两种范式的最优组合是什么？

当前达成的共识是：**CNN和Transformer各自具有互补的归纳偏置**——CNN天然适合局部特征提取和平移等变性，Transformer擅长全局依赖建模和动态权重调整。最优方案可能是两者的融合，而非取代。

## 2. 发展历史

```
CNN主导时代 (2012-2019)
  │  AlexNet -> VGG -> Inception -> ResNet -> EfficientNet
  │
ViT突破 (2020)
  │  Vision Transformer (Dosovitskiy et al., ICLR 2021)
  │  首次证明纯Transformer可在图像分类上超越CNN
  │  前提：大规模预训练（JFT-300M）
  ▼
ViT优化 (2020-2021)
  │  DeiT: 蒸馏训练，小数据也能训ViT
  │  Swin Transformer: 层次化+窗口注意力，CV最佳论文
  │  PVT, CvT, CoAtNet: 卷积+注意力的混合架构
  ▼
CNN复兴 (2022-2023)
  │  ConvNeXt: CNN架构"现代化"，证明CNN仍可一战
  │  RepLKNet, SLaK: 大卷积核（31×31, 51×51）回归
  │  InternImage: 可变形卷积的大规模应用
  ▼
持续融合 (2023-至今)
     自监督预训练（MAE, SimCLR, DINO）
     大视觉模型（SAM, Dinov2）
     统一架构（CNN+Attention协调共存）
```

## 3. 核心概念

### 3.1 归纳偏置（Inductive Bias）

| 偏置类型 | CNN | Transformer |
|----------|-----|-------------|
| **局部性** | 强（卷积核天然局部） | 弱（全局注意力，需学习局部性） |
| **平移等变性** | 强（权值共享） | 弱（需位置编码辅助） |
| **全局依赖** | 弱（需层层堆叠扩大感受野） | 强（自注意力天然全局） |
| **数据需求** | 中等 | 大（注意力需要大量数据学习） |

### 3.2 自监督预训练

无需人工标注，从数据本身构造监督信号进行预训练，是ViT成功的关键推动力。

| 方法 | 范式 | 核心思路 |
|------|------|----------|
| **SimCLR** (2020) | 对比学习 | 同图不同增广为正样本，拉近表征 |
| **MoCo v3** (2021) | 对比学习+动量队列 | 解决大batch依赖 |
| **MAE** (2022) | 掩码自编码 | 随机遮住75%图像块，重建缺失内容 |
| **DINO/DINOv2** (2021/2023) | 自蒸馏 | Teacher-Student无需负样本的自监督 |
| **BEiT** (2021) | BERT式掩码 | Token级别的掩码预测 |

**为什么MAE对ViT有效？**
- 图像信息密度低（高度冗余），掩码75%仍可重建
- ViT的非重叠Patch划分使掩码重建自然可行
- CNN的卷积核重叠使得掩码信息泄露，MAE不适用于纯CNN

## 4. 技术原理

### 4.1 Vision Transformer (ViT, 2020)

> 论文：Dosovitskiy et al., An Image is Worth 16x16 Words

**核心操作**：将图像切分为 $16\times 16$ 的Patch序列，每个Patch线性投影为一个Token，加上位置编码后直接送入标准Transformer Encoder。

```
图像 -> Patch切分 -> 线性投影 -> [CLS] + Patch Tokens -> Transformer Encoder -> MLP Head -> 类别
```

**关键发现**：
- 在大规模数据（JFT-300M/ImageNet-21K）预训练后，ViT可超越CNN
- 在小数据集（ImageNet-1K）上，ViT因缺乏CNN的归纳偏置而表现不佳
- 使用更多数据时，ViT能自动学习到类似CNN的局部特征（浅层Gabor滤波器）

### 4.2 DeiT (2020)

**核心创新——知识蒸馏训练**：用CNN教师（如RegNet）的软标签指导ViT训练，使ViT在仅ImageNet-1K数据上也能取得有竞争力的结果。引入蒸馏Token（distillation token）与分类Token并行。

**启示**：当数据不足时，CNN的知识可以高效传递到ViT，说明两者的特征表示具有可迁移性。

### 4.3 Swin Transformer (2021)

> 论文：Liu et al., ICCV 2021 Best Paper (Marr Prize)

**核心创新——层次化+窗口注意力**：

1. **层次化结构**：像CNN一样逐层降低分辨率、增加通道数
2. **窗口注意力（W-MSA）**：只在局部窗口内计算自注意力，复杂度从 $O(N^2)$ 降到 $O(N)$
3. **移位窗口（SW-MSA）**：相邻层之间窗口偏移，实现跨窗口信息交互

**意义**：Swin Transformer的输出多尺度特征图使其可以像ResNet一样作为**通用视觉骨干**，直接替代CNN用于检测、分割等下游任务（FPN直接接入），这是ViT的Patch序列输出所不能的。

### 4.4 ConvNeXt (2022)

> [[02_经典CNN架构#4.7 ConvNeXt|详见经典CNN架构笔记]]

核心理念：不是设计新架构，而是用**系统消融实验**回答一个问题——CNN和Transformer的差距究竟来自架构本质，还是来自设计细节差异？

结论：大部分差距来自设计细节（优化器、数据增强、归一化层、激活函数等），将这些改进引入标准ResNet即可得到与Swin Transformer匹敌的纯CNN。

### 4.5 混合架构

| 架构 | 年份 | 设计 |
|------|------|------|
| **CvT** | 2021 | 卷积投影替代ViT的线性投影 |
| **CoAtNet** | 2021 | 浅层卷积(CNN偏置) + 深层注意力(全局建模) |
| **MaxViT** | 2022 | 多轴注意力：窗口+网格注意力交替 |
| **InternImage** | 2023 | 大规模可变形卷积骨干，3096层 |

**CoAtNet的深度-宽度统一法则**：将EfficientNet的复合缩放延伸到混合架构，系统研究了在混合架构中书模的最优方式。

### 4.6 视觉基础模型（Vision Foundation Models）

| 模型 | 年份 | 预训练数据 | 核心能力 |
|------|------|------------|----------|
| **CLIP** | 2021 | 400M图文对 | 图像-文本对齐，零样本分类 |
| **SAM** | 2023 | SA-1B (11M图像) | 提示驱动通用分割 |
| **DINOv2** | 2023 | 142M图像自监督 | 通用视觉特征，无需微调即有强表征 |
| **SAM 2** | 2024 | 视频+图像 | 扩展到视频分割 |

## 5. 关键问题与讨论

### 5.1 CNN是否会被Transformern取代？

**短期**：不会。CNN和Transformer在不同场景各有所长。

| 场景 | 优选 |
|------|------|
| 小数据/小模型 | CNN（归纳偏置优势） |
| 移动端部署 | CNN（算子成熟，量化生态好） |
| 大数据/大模型 | ViT/Swin（扩展性好） |
| 实时检测/分割 | CNN（速度优势） |
| 追求SOTA | 混合架构/Trainformer |

**长期**：边界将模糊。ConvNeXt证明了设计思想比具体操作更重要。未来的"最优架构"可能同时包含卷积和注意力，根据任务自适应选择。

### 5.2 数据规模的作用

CNN和Transformer的核心能力区分可能主要来自数据规模：

- CNN在小数据上好的原因是其**强归纳偏置**（局部性、平移等变性）
- Transformer在大数据上更好的原因是其**弱归纳偏置+更强的学习能力**——"学"到的知识比"注入"的知识更灵活

当数据量继续增长时，Transformer或更弱偏置的架构可能全面超越CNN。

## 6. 优势与局限——范拉对比

### ViT/Transformer的优势
- 全局感受野，天然处理长程依赖
- 动态权重（注意力），根据输入内容调整
- 与NLP统一架构，多模态融合更自然（CLIP/LLaVA）
- 扩展性极好（Scaling行为研究显示Transformer符合幂律）

### ViT/Transformer的局限
- 计算复杂度（全局注意力 $O(N^2)$，虽然线性注意力在改进）
- 小数据集上需额外策略（蒸馏/大量数据增强）
- 推理速度在边缘设备上不及优化后的CNN
- 缺乏平移等变性（位置编码只是近似补充）

## 9. 前沿发展方向

1. **统一视觉架构**：一个模型处理所有视觉任务（分类/检测/分割/生成），类似LLM在NLP中的统一趋势
2. **Scaling Law深化**：视觉模型的缩放律研究——最优计算分配（类似Chinchilla for vision）
3. **选择性状态空间模型**：Mamba/VMamba等选择性SSM在CV中的应用，线性复杂度下与注意力竞争
4. **自监督的规模化**：是否需要标注数据的界限越来越模糊——DINOv2展示了自监督特征的强大迁移能力
5. **3D视觉基础模型**：从2D到3D的扩展（DUSt3R, MASt3R）
6. **开放世界视觉**：不再预设封闭类别集，支持开放词汇/开放世界的识别和理解
7. **视频理解统一**：从2D图像到3D时空视频的基础模型
8. **绿色AI**：在追求性能的同时关注训练和推理的碳足迹——模型效率仍是核心议题

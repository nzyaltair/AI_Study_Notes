# 数学速查表

AI学习中最常用的数学公式和概念快速参考。

## 线性代数

### 基本运算

| 运算 | 符号 | 定义 |
|------|------|------|
| 矩阵乘法 | C = AB | $C_{ij} = \sum_k A_{ik} B_{kj}$ |
| 转置 | $A^T$ | $(A^T)_{ij} = A_{ji}$ |
| 点积 | a·b | $\sum_i a_i b_i$ |
| Hadamard积 | A⊙B | $(A⊙B)_{ij} = A_{ij} B_{ij}$ |
| 迹 | tr(A) | $\sum_i A_{ii}$ |

### 重要分解

- **特征分解**：$A = Q\Lambda Q^{-1}$（方阵）
- **SVD**：$A = U\Sigma V^T$（任意矩阵）
- **Cholesky分解**：$A = LL^T$（对称正定矩阵）

### 范数

- L1: $||x||_1 = \sum |x_i|$
- L2: $||x||_2 = \sqrt{\sum x_i^2}$
- Frobenius: $||A||_F = \sqrt{\text{tr}(A^T A)}$

### 常用恒等式

$(AB)^T = B^T A^T$；$\text{tr}(AB) = \text{tr}(BA)$；$\det(AB) = \det(A)\det(B)$
$\nabla(x^T a) = a$；$\nabla(x^T A x) = 2Ax$（A对称）

## 微积分

### 导数规则

$(fg)' = f'g + fg'$（乘积法则）；$(f \circ g)' = f'(g(x)) \cdot g'(x)$（链式法则）

### 梯度

$\nabla f = [\frac{\partial f}{\partial x_1}, ..., \frac{\partial f}{\partial x_n}]^T$

$\nabla(a^T x) = a$；$\nabla(||x||^2) = 2x$；$\nabla(||Ax-b||^2) = 2A^T(Ax-b)$

### Taylor展开

$f(x) \approx f(a) + \nabla f(a)^T(x-a) + \frac{1}{2}(x-a)^T H(a)(x-a)$

## 概率与统计

### 贝叶斯定理

$P(A|B) = \frac{P(B|A)P(A)}{P(B)}$；$P(\theta|D) = \frac{P(D|\theta)P(\theta)}{P(D)}$

### 期望与方差

$E[X] = \sum x P(x)$（离散）；$Var(X) = E[X^2] - (E[X])^2$

### 常用分布

- **高斯**：$N(\mu, \sigma^2)$，$E[X]=\mu$，$Var(X)=\sigma^2$
- **多元高斯**：$N(\mu, \Sigma)$
- **伯努利**：$P(X=1)=p$，$E[X]=p$，$Var(X)=p(1-p)$

### 估计

- **MLE**：$\theta_{MLE} = \arg\max_\theta \sum \log P(x_i|\theta)$
- **MAP**：$\theta_{MAP} = \arg\max_\theta [\log P(D|\theta) + \log P(\theta)]$

## 信息论

- **熵**：$H(X) = -\sum P(x)\log P(x)$
- **交叉熵**：$H(P,Q) = -\sum P(x)\log Q(x)$
- **KL散度**：$D_{KL}(P\|Q) = \sum P(x)\log\frac{P(x)}{Q(x)}$
- **互信息**：$I(X;Y) = H(X) - H(X|Y)$

## 优化方法

### 梯度下降变体

- **SGD**：$\theta \leftarrow \theta - \eta\nabla L$
- **动量法**：$v \leftarrow \beta v + (1-\beta)\nabla L$，$\theta \leftarrow \theta - \eta v$
- **Adam**：$m_t = \beta_1 m_{t-1} + (1-\beta_1)\nabla L$，$v_t = \beta_2 v_{t-1} + (1-\beta_2)(\nabla L)^2$，$\theta \leftarrow \theta - \eta\hat{m}_t/(\sqrt{\hat{v}_t}+\epsilon)$
- **AdamW**：Adam + 解耦权重衰减

### KKT条件

$\nabla f + \sum \lambda_i \nabla g_i + \sum \mu_j \nabla h_j = 0$；$g_i \leq 0$；$\lambda_i \geq 0$；$\lambda_i g_i = 0$；$h_j = 0$

## 深度学习公式

### 激活函数

- **Sigmoid**：$\sigma(x) = 1/(1+e^{-x})$，$\sigma'(x) = \sigma(x)(1-\sigma(x))$
- **ReLU**：$\text{ReLU}(x) = \max(0,x)$
- **GELU**：$GELU(x) = x\Phi(x) \approx x\sigma(1.702x)$
- **Softmax**：$\text{softmax}(x)_i = \frac{\exp(x_i - \max(x))}{\sum_j \exp(x_j - \max(x))}$

### 损失函数

- **MSE**：$L = \frac{1}{n}\sum(y_i - \hat{y}_i)^2$
- **交叉熵**：$L = -\sum y_i \log \hat{y}_i$

### 归一化

- **BatchNorm**：$\hat{x} = \frac{x - \mu_B}{\sqrt{\sigma_B^2 + \epsilon}}$，$y = \gamma\hat{x} + \beta$
- **LayerNorm**：$\hat{x} = \frac{x - \mu}{\sqrt{\sigma^2 + \epsilon}}$，$y = \gamma\hat{x} + \beta$

### 注意力机制

- **缩放点积注意力**：$\text{Attention}(Q,K,V) = \text{softmax}(\frac{QK^T}{\sqrt{d_k}})V$
- **多头注意力**：$\text{MultiHead} = \text{Concat}(\text{head}_1,...,\text{head}_h)W^O$

### 数值稳定性

- **Log-Sum-Exp**：$\log\sum \exp(x_i) = \max(x) + \log\sum \exp(x_i - \max(x))$
- **梯度裁剪**：$g \leftarrow \frac{c}{\|g\|}g$ 若 $\|g\| > c$

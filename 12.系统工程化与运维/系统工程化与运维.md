# 系统工程化与运维

> 系统工程化与运维主题索引，涵盖模型部署、容器化、监控运维、安全伦理、ONNX导出和研究前沿。

## 概述

将[[大语言模型|LLM]]从实验室推向生产环境，需要系统化的工程能力：模型部署、服务化架构、容器化、监控运维、安全防护和成本控制。系统工程化是AI落地的"最后一公里"。

## 部署架构全景

```
模型训练 → 模型转换(ONNX) → 容器化(Docker) → 服务化(vLLM/TGI) → API网关 → 监控告警
                                                              ↓
                                                         成本控制与安全防护
```

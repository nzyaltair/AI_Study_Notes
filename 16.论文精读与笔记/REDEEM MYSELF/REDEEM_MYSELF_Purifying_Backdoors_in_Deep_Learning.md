
REDEEM MYSELF: Purifying Backdoors in Deep Learning Models using Self Attention Distillation


这篇论文主要探讨了深度学习模型容易受到后门攻击的脆弱性。在后门攻击中，攻击者在模型中植入隐秘的触发器，这些触发器一旦被激活，就会导致模型做出错误分类，而在没有触发器的情况下，模型会正常工作。


该论文提出了一种新的净化方法，称为 **SAGE**（基于自注意力引导的蒸馏）。与以往的方法不同，SAGE 能够利用自注意力机制清除后门，而无需依赖教师模型的监督。传统方法要么会大幅降低模型在干净样本上的性能，要么无法有效应对高级后门攻击。SAGE 通过动态调整学习率，优化模型在干净样本上的预测精度，能够在大幅降低攻击成功率的同时，保持模型性能的高水平。实验表明，SAGE 可将攻击成功率降低 90%，而对干净样本的预测准确率仅下降不到 3%。

SAGE 概述。SAGE 基于自注意力蒸馏机制，模型通过充当老师和学生的身份从自身学习。具体来说，SAGE 由三个关键模块组成，即注意力表示、损失计算和学习率更新。“AT-GEN”表示注意力图生成。C1 和 C2 表示学习率更新的两个指定条件。

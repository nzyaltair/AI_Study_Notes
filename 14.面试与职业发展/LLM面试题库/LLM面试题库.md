# LLM 面试题库

## 目标定位

本文档覆盖大语言模型（LLM）相关岗位（如算法工程师、大模型研发、AI 系统优化等）在技术面试中高频出现的核心问题，兼具深度、广度与实战导向。通过系统性梳理 LLM 领域的关键知识点，帮助求职者系统化备战面试，同时为面试官提供结构化的考察框架。

## 基础理论

### Transformer 架构

#### 问题：请简述 Transformer 架构的核心组件及其作用
- **考察点**：对 Transformer 基本原理的理解
- **分层回答建议**：
  - **基础**：Transformer 由编码器和解码器组成，使用自注意力机制处理序列数据
  - **进阶**：详细说明编码器的多头注意力、前馈神经网络、层归一化等组件
  - **深入**：分析 Transformer 与 RNN/LSTM 的区别，解释并行计算优势

#### 问题：Transformer 为什么需要位置编码？有哪些常见的位置编码方法？
- **考察点**：对位置信息处理的理解
- **分层回答建议**：
  - **基础**：解释 Transformer 缺乏序列位置信息的问题
  - **进阶**：介绍正弦余弦位置编码、学习型位置编码
  - **深入**：分析相对位置编码的优势，如 T5 的相对位置偏置

### 注意力机制

#### 问题：请详细解释自注意力机制的计算过程
- **考察点**：对注意力机制原理的深入理解
- **分层回答建议**：
  - **基础**：计算 Q、K、V，计算注意力分数，归一化，加权求和
  - **进阶**：解释缩放因子的作用，多头注意力的优势
  - **深入**：分析注意力机制的时间复杂度，讨论稀疏注意力的优化

#### 问题：什么是多头注意力？它的优势是什么？
- **考察点**：对多头注意力设计的理解
- **分层回答建议**：
  - **基础**：多头注意力将注意力分为多个子空间并行计算
  - **进阶**：解释如何捕获不同子空间的语义信息
  - **深入**：分析多头注意力的参数共享策略，讨论头数选择的影响因素

### 缩放律（Scaling Laws）

#### 问题：请解释 LLM 的缩放律（Scaling Laws）
- **考察点**：对模型规模与性能关系的理解
- **分层回答建议**：
  - **基础**：说明模型性能随参数、数据、计算量的增长而提升
  - **进阶**：介绍不同的缩放律公式，如 Kaplan 等人的研究
  - **深入**：分析当前缩放律的局限性，讨论数据质量对缩放的影响

## 训练与优化

### 预训练与微调

#### 问题：LLM 的预训练过程是怎样的？与传统 NLP 预训练有何不同？
- **考察点**：对预训练流程的理解
- **分层回答建议**：
  - **基础**：介绍自监督预训练，预测掩码词
  - **进阶**：解释 LLM 预训练的目标函数，如 Causal Language Modeling
  - **深入**：分析数据处理、模型架构选择对预训练效果的影响

#### 问题：指令微调和对话微调有什么区别？如何设计高质量的微调数据？
- **考察点**：对微调方法的理解
- **分层回答建议**：
  - **基础**：解释指令微调的目的是让模型遵循指令
  - **进阶**：对比不同微调数据集的设计策略
  - **深入**：讨论数据质量、多样性对微调效果的影响

### RLHF/DPO/ORPO

#### 问题：请解释 RLHF 的基本原理和训练流程
- **考察点**：对对齐方法的理解
- **分层回答建议**：
  - **基础**：介绍 RLHF 的三个阶段：SFT、奖励模型训练、强化学习
  - **进阶**：解释 PPO 算法在 RLHF 中的应用
  - **深入**：分析 RLHF 的局限性，如奖励模型过拟合

#### 问题：DPO 和 ORPO 相比 RLHF 有什么优势？
- **考察点**：对最新对齐方法的了解
- **分层回答建议**：
  - **基础**：解释 DPO 直接优化偏好排序
  - **进阶**：对比 DPO 和 RLHF 的计算效率
  - **深入**：分析 ORPO 在处理离线数据方面的优势

### 分布式训练

#### 问题：ZeRO 和 FSDP 有什么区别？各适用于什么场景？
- **考察点**：对分布式训练的理解
- **分层回答建议**：
  - **基础**：解释 ZeRO 和 FSDP 的基本原理
  - **进阶**：对比不同级别的参数分片策略
  - **深入**：分析通信开销，讨论混合并行策略

#### 问题：混合精度训练的原理是什么？如何处理梯度下溢问题？
- **考察点**：对训练优化的理解
- **分层回答建议**：
  - **基础**：解释 FP16 和 BF16 的区别
  - **进阶**：介绍损失缩放技术
  - **深入**：分析不同混合精度策略的适用场景

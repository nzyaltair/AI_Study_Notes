# 机器学习

> 本文件为 机器学习 主题的索引文件，用于组织相关笔记内容，建立完整的知识体系框架。

## 概述

机器学习是人工智能的核心分支之一，它致力于研究如何让计算机从数据中学习并做出预测或决策。通过机器学习，计算机可以自动发现数据中的模式和规律，无需显式编程。机器学习技术广泛应用于图像识别、自然语言处理、推荐系统、自动驾驶等多个领域，是现代AI应用的基础。

本主题旨在构建一个完整的机器学习知识体系，帮助学习者掌握从基础概念到高级应用的全方位技能。内容涵盖机器学习基础概念、数据预处理、特征工程、模型算法、模型选择与调参、机器学习流水线以及实战案例等核心模块，为AI开发奠定坚实的机器学习基础。

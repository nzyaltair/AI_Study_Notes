# 线性代数

## 背景

线性代数是人工智能的数学基础语言。从机器学习的特征表示到深度学习的神经网络计算，所有AI模型的核心运算都建立在矩阵和向量之上。理解线性代数是理解任何AI算法工作原理的前提。

## 核心思想

线性代数研究向量空间及其上的线性变换。在AI中，数据以向量/矩阵表示，模型参数以矩阵表示，模型计算本质上是矩阵乘法和线性变换。

## 技术原理

### 向量与向量空间

- **向量**：n个实数组成的有序数组
  - 行向量：$\mathbf{a} = [a_1, a_2, ..., a_n]$
  - 列向量：$\mathbf{a} = \begin{bmatrix} a_1 \\ a_2 \\ \vdots \\ a_n \end{bmatrix}$
- **向量空间**：对加法和标量乘法封闭的向量集合，满足8条公理
- **内积**：$\langle \mathbf{a}, \mathbf{b} \rangle = \mathbf{a}^T \mathbf{b} = \sum_{i=1}^n a_i b_i$
- **范数**：
  - L2范数：$||\mathbf{a}||_2 = \sqrt{\sum_{i=1}^n a_i^2}$
  - L1范数：$||\mathbf{a}||_1 = \sum_{i=1}^n |a_i|$
  - L∞范数：$||\mathbf{a}||_\infty = \max_{i} |a_i|$
- **线性相关/无关**：若存在不全为零的数 $c_1, ..., c_k$ 使得 $c_1\mathbf{a}_1 + ... + c_k\mathbf{a}_k = \mathbf{0}$，则线性相关
- **基与维度**：基是向量空间中最大的线性无关向量组，维度是基中向量个数
- **正交性**：内积为零的向量，$\mathbf{a}^T \mathbf{b} = 0$

### 矩阵与矩阵运算

- **矩阵**：$\mathbf{A} \in \mathbb{R}^{m \times n}$，元素 $a_{ij}$ 位于第i行第j列
- **特殊矩阵**：零矩阵、单位矩阵$\mathbf{I}$、对角矩阵、对称矩阵（$\mathbf{A}^T = \mathbf{A}$）、正交矩阵（$\mathbf{A}^T \mathbf{A} = \mathbf{I}$）
- **矩阵乘法**：$\mathbf{C} = \mathbf{A} \mathbf{B}$，其中 $c_{ij} = \sum_{k=1}^n a_{ik} b_{kj}$
  - 行向量×列向量理解：$c_{ij}$ 是 $\mathbf{A}$ 第i行与 $\mathbf{B}$ 第j列的内积
  - 列向量线性组合理解：$\mathbf{C}$ 第j列是 $\mathbf{A}$ 列向量的线性组合
  - 外积和理解：$\mathbf{C} = \sum_{k=1}^n \mathbf{a}_k \mathbf{b}_k^T$
- **转置**：$(\mathbf{A}^T)_{ij} = a_{ji}$，性质：$(\mathbf{A} \mathbf{B})^T = \mathbf{B}^T \mathbf{A}^T$
- **迹**：$tr(\mathbf{A}) = \sum_{i=1}^n a_{ii}$，性质：$tr(\mathbf{A} \mathbf{B}) = tr(\mathbf{B} \mathbf{A})$

### 行列式

- **定义**：方阵 $\mathbf{A}$ 的行列式 $det(\mathbf{A})$ 是衡量矩阵对空间"体积伸缩因子"的标量
  - 2阶：$\begin{vmatrix} a & b \\ c & d \end{vmatrix} = ad - bc$（平行四边形面积）
  - n阶：$det(\mathbf{A}) = \sum_{j=1}^n (-1)^{i+j} a_{ij} M_{ij}$
- **性质**：
  - $det(\mathbf{A} \mathbf{B}) = det(\mathbf{A}) det(\mathbf{B})$
  - 若 $\mathbf{A}$ 可逆，$det(\mathbf{A}) \neq 0$

### 线性方程组

- **形式**：$\mathbf{A} \mathbf{x} = \mathbf{b}$
- **解的存在性**（Rouché-Capelli定理）：有解当且仅当 $rank(\mathbf{A}) = rank([\mathbf{A} | \mathbf{b}])$
- **求解方法**：
  - 高斯消元法
  - 矩阵求逆：$\mathbf{x} = \mathbf{A}^{-1} \mathbf{b}$
  - LU分解
  - 最小二乘法：$\hat{\mathbf{x}} = (\mathbf{A}^T \mathbf{A})^{-1} \mathbf{A}^T \mathbf{b}$

### 矩阵的秩与零空间

- **秩**：矩阵中行/列向量组的最大线性无关组向量个数
  - $rank(\mathbf{A} \mathbf{B}) \leq \min(rank(\mathbf{A}), rank(\mathbf{B}))$
- **值域（列空间）**：$\mathcal{R}(\mathbf{A}) = \{ \mathbf{y} | \mathbf{y} = \mathbf{A} \mathbf{x} \}$
- **零空间（核）**：$\mathcal{N}(\mathbf{A}) = \{ \mathbf{x} | \mathbf{A} \mathbf{x} = \mathbf{0} \}$
- **秩-零化度定理**：$dim(\mathcal{R}(\mathbf{A})) + dim(\mathcal{N}(\mathbf{A})) = n$
- **Gram矩阵**：$\mathbf{G} = \mathbf{A}^T \mathbf{A}$，对称半正定

### 特征值与特征向量

- **定义**：$\mathbf{A} \mathbf{v} = \lambda \mathbf{v}$，$\lambda$ 为特征值，$\mathbf{v}$ 为特征向量
- **特征方程**：$det(\lambda\mathbf{I} - \mathbf{A}) = 0$
- **特征值分解**：$\mathbf{A} = \mathbf{P} \mathbf{\Lambda} \mathbf{P}^{-1}$
- **对称矩阵性质**：
  - 所有特征值为实数
  - 不同特征值对应特征向量正交
  - 可正交对角化：$\mathbf{A} = \mathbf{Q} \mathbf{\Lambda} \mathbf{Q}^T$
- **迹与行列式**：$tr(\mathbf{A}) = \sum \lambda_i$，$det(\mathbf{A}) = \prod \lambda_i$

### 二次型

- **定义**：$f(\mathbf{x}) = \mathbf{x}^T \mathbf{A} \mathbf{x}$，$\mathbf{A}$ 为对称矩阵
- **正定性**：
  - 正定：$\mathbf{x}^T \mathbf{A} \mathbf{x} > 0$，所有特征值大于0
  - 半正定：$\mathbf{x}^T \mathbf{A} \mathbf{x} \geq 0$，所有特征值非负

### 奇异值分解（SVD）

- **分解**：任意矩阵 $\mathbf{A} \in \mathbb{R}^{m \times n}$ 可分解为 $\mathbf{A} = \mathbf{U} \mathbf{\Sigma} \mathbf{V}^T$
  - $\mathbf{U}$：左奇异向量（$\mathbf{A} \mathbf{A}^T$ 的特征向量）
  - $\mathbf{\Sigma}$：奇异值对角矩阵（$\sigma_i = \sqrt{\lambda_i(\mathbf{A}^T \mathbf{A})}$）
  - $\mathbf{V}$：右奇异向量（$\mathbf{A}^T \mathbf{A}$ 的特征向量）
- **低秩近似**：取前k个最大奇异值得最优近似 $\mathbf{A}_k$（Eckart-Young定理）
- **几何意义**：旋转（$\mathbf{V}^T$）→缩放（$\mathbf{\Sigma}$）→旋转（$\mathbf{U}$）

### 矩阵微积分

- **梯度**：$\nabla_{\mathbf{x}} f(\mathbf{x})$ 是偏导数组成的向量
  - $\nabla_{\mathbf{x}}(\mathbf{a}^T \mathbf{x}) = \mathbf{a}$
  - $\nabla_{\mathbf{x}}(\mathbf{x}^T \mathbf{A} \mathbf{x}) = 2\mathbf{A} \mathbf{x}$（$\mathbf{A}$ 对称）
  - $\nabla_{\mathbf{x}}(||\mathbf{x} - \mathbf{a}||_2^2) = 2(\mathbf{x} - \mathbf{a})$
- **黑塞矩阵**：$H_{ij} = \frac{\partial^2 f}{\partial x_i \partial x_j}$，对称矩阵，正定性决定局部凸性
- **雅可比矩阵**：向量函数对向量的导数，$J_{ij} = \frac{\partial f_i}{\partial x_j}$

## 应用领域

- **[[机器学习]]**：线性回归（$\mathbf{y} = \mathbf{X}\mathbf{w}$）、SVM（内积计算）、PCA降维（特征值分解）
- **[[神经网络基础]]**：前向传播（矩阵乘法）、反向传播（梯度计算）、权重表示
- **[[卷积神经网络]]**：卷积运算本质是矩阵乘法
- **[[Transformer]]**：自注意力机制中的Q、K、V矩阵运算
- **推荐系统**：矩阵分解（SVD）
- **自然语言处理**：词向量表示、主题建模（LSA）

## 与其他技术关系

- 线性代数是[[微积分]]中梯度计算的基础（矩阵微积分）
- 线性代数为[[概率与统计]]中的协方差矩阵、多元高斯分布提供工具
- 线性代数是[[优化方法]]中梯度下降的理论前提
- SVD和低秩近似是[[LoRA]]等参数高效微调技术的数学基础
- 特征值分解在[[信息论]]的主成分分析中有应用

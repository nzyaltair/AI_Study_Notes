## 🔄 完整工具链示例

### 典型PyTorch转换工作流

```bash
# 1. 安装环境（基础）
pip install torch torchvision onnx onnxruntime

# 2. 转换模型
python -c "
import torch
import onnx
model = ...  # 你的PyTorch模型
torch.onnx.export(model, ...)
"

# 3. 检查模型
python -c "import onnx; onnx.check('model.onnx')"

# 4. 性能分析
pip install onnxruntime-tools
python -m onnxruntime.tools.convert_onnx_models_to_ort \
  model.onnx --optimization_level basic
```

### 完整部署流水线

```
PyTorch/TF Model → ONNX Export → ONNX Optimize → Quantize →
TensorRT Engine → ORT Inference → Performance Profiling
```

对应工具：
- Export: `torch.onnx.export` / `tf2onnx`
- Optimize: `onnxoptimizer`, `onnxruntime-tools`
- Quantize: `onnxruntime-tools.quantization`
- TensorRT: `trtexec`
- Inference: `onnxruntime`
- Profiling: `onnxruntime-tools.profile`

## 📥 安装快速参考

```bash
# CPU环境（基础）
pip install onnx onnxruntime

# GPU环境（CUDA 11.x）
pip install onnx onnxruntime-gpu

# 完整工具集
pip install onnx onnxruntime onnxruntime-tools onnx-graphsurgeon \
             onnxoptimizer netron torch torchvision tf2onnx

# 特定硬件后端
pip install onnxruntime-openvino  # Intel
pip install onnxruntime-directml  # Windows GPU
```

**版本兼容性提示**：
- ONNX Runtime版本应与ONNX opset版本匹配
- TensorRT支持ONNX opset 1-13（TRT 8.x）
- 各框架版本可能影响opset支持能力，详见各框架文档

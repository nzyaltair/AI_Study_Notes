# 值函数与动态规划

## 1. 概述

动态规划（Dynamic Programming, DP）是强化学习中最基本的求解方法。当 MDP 的完整模型（转移概率 $P$ 和奖励函数 $R$）已知时，DP 可以通过迭代计算精确求解最优值函数和最优策略。虽然实际应用中很少已知完整模型，但 DP 提供了 RL 所有值函数方法的理论基础——后续的无模型方法（Q-Learning、SARSA、DQN）本质上都是 DP 的采样近似。

本篇介绍值函数的定义、策略评估、策略改进、策略迭代和值迭代，以及它们的收敛性。

## 2. 发展历史

## 3. 核心概念

### 3.1 值函数

值函数是衡量状态或状态-动作对长期价值的函数：

**状态值函数** $V^\pi(s)$：在策略 $\pi$ 下从状态 $s$ 出发的期望回报：

$$V^\pi(s) = \mathbb{E}_\pi \left[ G_t \mid S_t = s \right] = \mathbb{E}_\pi \left[ \sum_{k=0}^{\infty} \gamma^k R_{t+k+1} \mid S_t = s \right]$$

**动作值函数** $Q^\pi(s,a)$：在状态 $s$ 执行动作 $a$ 后，遵循策略 $\pi$ 的期望回报：

$$Q^\pi(s,a) = \mathbb{E}_\pi \left[ G_t \mid S_t = s, A_t = a \right]$$

两者关系：

$$V^\pi(s) = \sum_a \pi(a|s) \, Q^\pi(s,a)$$

### 3.2 最优值函数

**最优状态值函数**：$V^*(s) = \max_\pi V^\pi(s)$

**最优动作值函数**：$Q^*(s,a) = \max_\pi Q^\pi(s,a)$

最优策略 $\pi^*$ 可以从 $Q^*$ 导出：$\pi^*(s) = \arg\max_a Q^*(s,a)$

### 3.3 优势函数

优势函数衡量某动作相对于策略平均水平的优劣：

$$A^\pi(s,a) = Q^\pi(s,a) - V^\pi(s)$$

- $A^\pi(s,a) > 0$：动作 $a$ 优于策略平均
- $A^\pi(s,a) < 0$：动作 $a$ 劣于策略平均
- $A^\pi(s,a) = 0$：动作 $a$ 与策略平均持平

优势函数在策略梯度方法中用于降低梯度估计的方差。参见 [[04_策略梯度方法]]。

## 4. 技术原理

### 4.1 策略评估（Policy Evaluation）

给定策略 $\pi$，计算 $V^\pi$。根据 Bellman 期望方程：

$$V^\pi(s) = \sum_a \pi(a|s) \sum_{s'} P(s'|s,a) \left[ R(s,a,s') + \gamma V^\pi(s') \right]$$

这是一个关于 $V^\pi$ 的线性方程组。迭代求解（同步更新）：

$$V_{k+1}(s) = \sum_a \pi(a|s) \sum_{s'} P(s'|s,a) \left[ R(s,a,s') + \gamma V_k(s') \right]$$

**收敛性**：当 $\gamma < 1$（有限回合任务或折扣设定），$V_k$ 收缩映射收敛到 $V^\pi$。

**收缩映射证明**：定义算子 $T^\pi$：

$$T^\pi V(s) = \sum_a \pi(a|s) \sum_{s'} P(s'|s,a) \left[ R(s,a,s') + \gamma V(s') \right]$$

$$\|T^\pi V_1 - T^\pi V_2\|_\infty \leq \gamma \|V_1 - V_2\|_\infty$$

因为 $\gamma < 1$，$T^\pi$ 是 $\gamma$-收缩映射，由 Banach 不动点定理保证收敛到唯一不动点 $V^\pi$。

### 4.2 策略改进（Policy Improvement）

给定 $Q^\pi$，构造贪心策略 $\pi'(s) = \arg\max_a Q^\pi(s,a)$。

**策略改进定理**：对任意状态 $s$，$V^{\pi'}(s) \geq V^\pi(s)$。

**证明**：

$$V^\pi(s) \leq Q^\pi(s, \pi'(s)) = \sum_{s'} P(s'|s,\pi'(s)) \left[ R + \gamma V^\pi(s') \right]$$

递归展开：

$$V^\pi(s) \leq \sum_{s'} P(s'|s,\pi'(s)) \left[ R + \gamma V^\pi(s') \right] \leq \sum_{s'} P(s'|s,\pi'(s)) \left[ R + \gamma V^{\pi'}(s') \right] = V^{\pi'}(s)$$

### 4.3 策略迭代（Policy Iteration）

交替进行策略评估和策略改进：

```
初始化策略 π₀
重复：
  1. 策略评估：计算 V^π_k
  2. 策略改进：π_{k+1}(s) = argmax_a Q^{π_k}(s,a)
直到 π_{k+1} = π_k
```

**收敛性**：由于策略改进保证 $V^{\pi_{k+1}} \geq V^{\pi_k}$，且有限 MDP 的策略数量有限，策略迭代在有限步内收敛到最优策略。

### 4.4 值迭代（Value Iteration）

将策略评估（只迭代一步）和策略改进合并：

$$V_{k+1}(s) = \max_a \sum_{s'} P(s'|s,a) \left[ R(s,a,s') + \gamma V_k(s') \right]$$

即直接迭代 Bellman 最优方程。

**收敛性**：Bellman 最优算子 $T^*$ 同样是 $\gamma$-收缩映射：

$$\|T^* V_1 - T^* V_2\|_\infty \leq \gamma \|V_1 - V_2\|_\infty$$

因此值迭代收敛到 $V^*$。

### 4.5 策略迭代 vs 值迭代

| 维度 | 策略迭代 | 值迭代 |
|:---|:---|:---|
| 每轮操作 | 完整评估 + 一步改进 | 一步评估 + 一步改进 |
| 收敛速度 | 通常更快（策略空间有限） | 通常更慢（值空间连续） |
| 每轮计算量 | 大（需要收敛策略评估） | 小（单步更新） |
| 实际效率 | 适合小状态空间 | 适合大状态空间 |
| 极限 | 当策略评估退化为单步时，二者等价 | — |

### 4.6 异步动态规划

同步 DP 每轮需要遍历所有状态，在大状态空间中不现实。异步 DP 方法：

- **就地更新（In-place）**：直接用最新值更新，不等所有状态同步
- **优先级扫描（Prioritized Sweeping）**：按值变化大小排序状态，优先更新变化大的
- **实时动态规划（RTDP）**：只在智能体实际访问的状态上更新

异步 DP 是从 DP 到无模型 RL 的过渡——它引入了"按需更新"的思想。

## 5. 关键方法/模型

### 5.1 通用迭代框架

所有 DP 方法都可以看作**贝尔曼算子的不动点迭代**：

- 策略评估：$V \leftarrow T^\pi V$（贝尔曼期望算子）
- 值迭代：$V \leftarrow T^* V$（贝尔曼最优算子）

### 5.2 修改的策略迭代

完整策略评估在实际中过慢。修改的策略迭代只做 $k$ 步评估就进入改进，在精度和效率之间折中。实践中 $k=1$ 退化为值迭代，$k \to \infty$ 为标准策略迭代。

### 5.3 线性规划方法

MDP 的最优值函数可以通过线性规划求解：

$$\min_V \sum_s \mu(s) V(s) \quad \text{s.t.} \quad V(s) \geq \sum_{s'} P(s'|s,a)[R(s,a,s') + \gamma V(s')], \quad \forall s, a$$

其中 $\mu(s)$ 是任意正权重。LP 方法在理论分析中有价值，但对大状态空间不实用。

## 6. 优势与局限

### 优势

1. **理论完备**：收敛性有严格数学保证
2. **精确解**：在已知 MDP 下求得全局最优
3. **基础框架**：所有值函数类 RL 方法的理论基础

### 局限

1. **需要完整模型**：必须已知 $P$ 和 $R$，实际中罕见
2. **维度灾难**：状态空间随变量数指数增长，表格 DP 只适合小问题
3. **计算成本高**：每轮遍历所有状态，大状态空间不可行

## 7. 应用场景

- 小规模 MDP 的精确求解（如 GridWorld、简化的排队系统）
- RL 算法的验证基准（与无模型方法对比验证正确性）
- 理论分析工具（收敛性证明的基础）
- 有精确模拟器的领域（如组合优化问题的 MDP 建模）

## 8. 与其他技术关系

- **与无模型方法的关系**：Q-Learning 是值迭代的采样近似——用 $(s,a,r,s')$ 样本代替 $P(s'|s,a)$ 的求和。SARSA 是策略评估的采样近似。参见 [[03_无模型预测与控制]]。
- **与深度强化学习的关系**：DQN 用神经网络近似 $Q^*$，本质是带函数近似的值迭代。参见 [[05_深度强化学习]]。
- **与搜索的关系**：MCTS 可以看作基于采样的前向动态规划。参见 [[06_基于模型的强化学习]]。
- **与优化理论的关系**：Bellman 算子的收缩映射性质是 DP 收敛性的核心，属于不动点迭代理论。参见 [[02_优化理论与方法]]。

## 9. 前沿发展

- **近似动态规划**：用函数近似替代表格表示，处理大状态空间
- **分布式 DP**：在大规模 MDP（如电网、交通）中并行化 DP 计算
- **线性/凸值函数近似**：用线性函数族或凸松弛近似值函数，获得近似保证

# 卷积神经网络

## 背景

卷积神经网络（CNN）是专门处理网格结构数据（图像、视频）的[[深度学习|深度学习]]架构。它通过模拟人类视觉系统的局部感知和层次化处理原理，实现了对图像特征的自动提取，在2012年AlexNet之后引领了深度学习革命。

## 核心思想

利用**局部连接**和**权值共享**两大核心机制，大幅减少参数量同时保留空间结构信息。通过多层卷积堆叠，从低级边缘特征逐步构建到高级语义特征。

## 技术原理

### 核心组件

| 组件 | 作用 | 关键参数 |
|------|------|---------|
| **卷积层** | 特征提取 | 卷积核大小、步长、填充、通道数 |
| **池化层** | 下采样、不变性 | 池化窗口大小、步长（最大池化/平均池化） |
| **全连接层** | 分类/回归 | 输出维度 |
| **BatchNorm** | 加速训练、稳定收敛 | — |
| **激活函数** | 引入非线性 | ReLU为主 |

### 卷积运算

输出尺寸公式：$O = \frac{I - K + 2P}{S} + 1$

其中 $I$=输入尺寸，$K$=卷积核大小，$P$=填充，$S$=步长

### 感受野

感受野（Receptive Field）指特征图上某个点对应输入图像的区域大小。深层网络通过层层堆叠扩大感受野，使网络能"看到"更大的图像区域。

### 经典架构演进

| 架构 | 年份 | 核心创新 | 深度 |
|------|------|---------|------|
| **LeNet-5** | 1998 | 首个CNN，手写数字识别 | 7层 |
| **AlexNet** | 2012 | ReLU、Dropout、GPU训练 | 8层 |
| **VGG** | 2014 | 小卷积核(3×3)堆叠 | 16-19层 |
| **GoogLeNet** | 2014 | Inception模块，并行卷积 | 22层 |
| **ResNet** | 2015 | 残差连接，解决梯度消失 | 152层+ |
| **DenseNet** | 2017 | 密集连接，特征复用 | — |
| **EfficientNet** | 2019 | 复合缩放策略 | — |
| **ConvNeXt** | 2022 | 用CNN对标Transformer性能 | — |

### 残差连接（ResNet）

$$\mathbf{y} = F(\mathbf{x}, \{W_i\}) + \mathbf{x}$$

残差连接使梯度能直接反向传播到浅层，解决了深层网络梯度消失问题，使训练上百层的网络成为可能。

## 发展演进

LeNet（1998）→ AlexNet（2012，深度学习革命）→ VGG/GoogLeNet（2014）→ ResNet（2015，残差连接）→ EfficientNet（2019）→ Vision Transformer挑战（2020+）→ ConvNeXt（2022，CNN回归）

## 应用领域

- **图像分类**：ImageNet、细粒度分类
- **目标检测**：YOLO、Faster R-CNN、DETR
- **图像分割**：U-Net（医学）、DeepLab（语义分割）
- **特征提取**：作为[[多模态模型|多模态模型]]的视觉编码器
- **视频分析**：3D CNN（动作识别）

## 与其他技术关系

- CNN的卷积思想与[[线性代数|线性代数]]的矩阵运算密切相关
- ResNet的残差连接思想被[[Transformer与预训练|Transformer]]继承
- CNN作为视觉编码器，与[[多模态模型|多模态模型]]结合
- 训练技巧详见[[训练技巧与稳定性]]
- 在图像领域面临Vision Transformer的竞争，但ConvNeXt证明CNN仍有竞争力

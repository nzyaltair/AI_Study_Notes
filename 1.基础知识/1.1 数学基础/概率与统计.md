# 概率与统计

## 背景

概率与统计是人工智能和机器学习的核心数学基础，为AI算法提供了处理不确定性和建模数据分布的理论工具。从贝叶斯推断到最大似然估计，概率统计方法贯穿于机器学习的所有领域。

## 核心思想

概率论研究随机现象的规律，统计学从数据中推断总体分布。在AI中，概率统计用于建模数据分布、处理不确定性、设计损失函数和进行统计推断。

## 技术原理

### 概率基本要素

- **样本空间**（$\Omega$）：随机实验所有可能结果的集合
- **事件**（$A$）：样本空间的子集
- **概率公理**：
  1. 非负性：$P(A) \geq 0$
  2. 规范性：$P(\Omega) = 1$
  3. 可列可加性：互不相容事件之并的概率等于各事件概率之和
- **条件概率**：$P(A|B) = \frac{P(A \cap B)}{P(B)}$
- **独立性**：$P(A \cap B) = P(A)P(B)$

### 随机变量

- **累积分布函数（CDF）**：$F_X(x) = P(X \leq x)$
- **概率质量函数（PMF，离散）**：$p_X(x) = P(X = x)$
- **概率密度函数（PDF，连续）**：$f_X(x) = \frac{dF_X(x)}{dx}$

### 期望与方差

- **期望**：
  - 离散：$E[X] = \sum_x x \, p_X(x)$
  - 连续：$E[X] = \int x \, f_X(x) dx$
  - 性质：$E[aX + bY] = aE[X] + bE[Y]$；若独立，$E[XY] = E[X]E[Y]$
- **方差**：$Var[X] = E[(X - E[X])^2] = E[X^2] - (E[X])^2$
  - 性质：$Var[aX] = a^2 Var[X]$；若独立，$Var[X+Y] = Var[X] + Var[Y]$

### 常见概率分布

| 分布 | 类型 | PMF/PDF | 期望 | 方差 |
|------|------|---------|------|------|
| 伯努利 | 离散 | $p^x(1-p)^{1-x}$ | $p$ | $p(1-p)$ |
| 二项式 | 离散 | $\binom{n}{x}p^x(1-p)^{n-x}$ | $np$ | $np(1-p)$ |
| 泊松 | 离散 | $e^{-\lambda}\frac{\lambda^x}{x!}$ | $\lambda$ | $\lambda$ |
| 均匀 | 连续 | $\frac{1}{b-a}$ | $\frac{a+b}{2}$ | $\frac{(b-a)^2}{12}$ |
| 指数 | 连续 | $\lambda e^{-\lambda x}$ | $\frac{1}{\lambda}$ | $\frac{1}{\lambda^2}$ |
| 正态 | 连续 | $\frac{1}{\sqrt{2\pi}\sigma}e^{-\frac{(x-\mu)^2}{2\sigma^2}}$ | $\mu$ | $\sigma^2$ |

### 贝叶斯定理

$$P(Y|X) = \frac{P(X|Y)P(Y)}{P(X)}$$

- **先验概率** $P(Y)$：观察数据前对Y的信念
- **似然函数** $P(X|Y)$：给定Y时观察到X的概率
- **后验概率** $P(Y|X)$：观察到X后对Y的更新信念
- **证据** $P(X)$：归一化常数

### 多元概率分布

- **协方差矩阵**：$\Sigma_{ij} = Cov[X_i, X_j]$，对称半正定
- **多元高斯分布**：$\mathbf{X} \sim \mathcal{N}(\mu, \Sigma)$
  $$f(\mathbf{x}) = \frac{1}{(2\pi)^{n/2}|\Sigma|^{1/2}} \exp(-\frac{1}{2}(\mathbf{x}-\mu)^T\Sigma^{-1}(\mathbf{x}-\mu))$$
  - 边缘分布和条件分布仍为高斯
  - 线性变换：$\mathbf{A}\mathbf{X} + \mathbf{b} \sim \mathcal{N}(\mathbf{A}\mu + \mathbf{b}, \mathbf{A}\Sigma\mathbf{A}^T)$

### 统计推断

#### 大数法则与中心极限定理

- **大数法则**：样本均值依概率收敛于总体均值，$\bar{X}_n \xrightarrow{P} \mu$
- **中心极限定理**：$\sqrt{n}\frac{\bar{X}_n - \mu}{\sigma} \xrightarrow{d} \mathcal{N}(0, 1)$

#### 最大似然估计（MLE）

$$\hat{\theta}_{MLE} = \arg\max_\theta \sum_{i=1}^n \log P(x_i|\theta)$$

- 无偏性：$E[\hat{\theta}] = \theta$
- 一致性：$\hat{\theta}_n \xrightarrow{P} \theta$
- 渐近正态性：$\sqrt{n}(\hat{\theta}_{MLE} - \theta) \xrightarrow{d} \mathcal{N}(0, I(\theta)^{-1})$

#### 贝叶斯推断

$$P(\theta|\mathbf{X}) = \frac{P(\mathbf{X}|\theta)P(\theta)}{P(\mathbf{X})}$$

- **最大后验估计（MAP）**：$\hat{\theta}_{MAP} = \arg\max_\theta [\log P(\mathbf{X}|\theta) + \log P(\theta)]$
- **共轭先验**：后验与先验属同一分布族（如Beta-Bernoulli、Gaussian-Gaussian）
- **全贝叶斯**：计算完整后验分布，考虑参数不确定性

#### 近似推断

- **蒙特卡洛方法**：通过随机采样估计期望，$E_p[f(X)] \approx \frac{1}{n}\sum f(x_i)$
- **重要性采样**：使用提议分布 $q(x)$ 采样，权重 $w(x) = p(x)/q(x)$
- **MCMC**：Metropolis-Hastings算法、Gibbs采样
- **变分推断**：用简单分布 $q(\theta)$ 近似后验 $p(\theta|D)$，最大化ELBO

### 信息论概念

（详见[[信息论]]）

- 熵：$H(X) = -\sum P(x)\log P(x)$
- 交叉熵：$H(P,Q) = -\sum P(x)\log Q(x)$
- KL散度：$D_{KL}(P\|Q) = \sum P(x)\log\frac{P(x)}{Q(x)}$

## 应用领域

- **[[机器学习]]**：概率模型、贝叶斯分类器、最大似然训练
- **[[深度学习]]**：交叉熵损失函数、Dropout（贝叶斯近似）
- **生成模型**：GAN（最小化JS散度）、VAE（最大化ELBO）
- **[[大语言模型]]**：自回归语言建模（下一个token的概率分布）
- **不确定性估计**：贝叶斯神经网络、蒙特卡洛Dropout

## 与其他技术关系

- 概率统计依赖[[线性代数]]（协方差矩阵、多元高斯分布）
- 概率统计为[[信息论]]提供概率基础
- 贝叶斯定理是[[机器学习|朴素贝叶斯]]和[[大语言模型|RLHF]]的核心
- 最大似然估计等价于最小化交叉熵损失（[[信息论]]）
- 变分推断是VAE等生成模型的理论基础

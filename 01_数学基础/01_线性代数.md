# 线性代数

## 背景与发展

线性代数源于 18-19 世纪求解线性方程组与研究线性变换的需要。矩阵概念由 Sylvester 命名、Cayley 发展为系统理论，SVD 由 Beltrami 与 Jordan 独立提出。20 世纪中期以来，线性代数成为 [[机器学习]] 与 [[深度学习]] 的核心数学语言：数据以向量/矩阵表示，模型参数以矩阵表示，模型计算本质上是矩阵乘法与线性变换。

## 核心思想

线性代数研究向量空间及其上的线性变换。在 AI 中，一切数据（图像像素、词向量、特征列）和模型参数（权重矩阵）均以线性代数对象表示，模型的推理与训练均归结为矩阵运算与矩阵微积分。

## 技术原理

### 向量与向量空间

- **向量**：$n$ 个实数组成的有序数组，行向量 $\mathbf{a} = [a_1, \dots, a_n]$，列向量为其转置
- **内积**：$\langle \mathbf{a}, \mathbf{b} \rangle = \mathbf{a}^T \mathbf{b} = \sum_{i=1}^n a_i b_i$
- **范数**：L2 范数 $\|\mathbf{a}\|_2 = \sqrt{\sum a_i^2}$；L1 范数 $\|\mathbf{a}\|_1 = \sum |a_i|$
- **正交**：$\mathbf{a}^T \mathbf{b} = 0$
- **基与维度**：基是向量空间中最大的线性无关向量组，维度为基中向量个数

### 矩阵运算

- **矩阵乘法**：$\mathbf{C} = \mathbf{A}\mathbf{B}$，$c_{ij} = \sum_{k} a_{ik} b_{kj}$，可理解为行向量与列向量的内积，或列向量的线性组合
- **特殊矩阵**：单位矩阵 $\mathbf{I}$、对角矩阵、对称矩阵（$\mathbf{A}^T = \mathbf{A}$）、正交矩阵（$\mathbf{A}^T\mathbf{A} = \mathbf{I}$）
- **迹**：$\text{tr}(\mathbf{A}) = \sum_i a_{ii}$，性质 $\text{tr}(\mathbf{A}\mathbf{B}) = \text{tr}(\mathbf{B}\mathbf{A})$
- **行列式**：方阵对空间体积的伸缩因子，$\det(\mathbf{A}\mathbf{B}) = \det(\mathbf{A})\det(\mathbf{B})$，可逆当且仅当 $\det \neq 0$

### 秩与空间

- **秩**：矩阵行/列向量组的最大线性无关组向量个数
- **值域（列空间）**：$\mathcal{R}(\mathbf{A}) = \{\mathbf{y} \mid \mathbf{y} = \mathbf{A}\mathbf{x}\}$
- **零空间**：$\mathcal{N}(\mathbf{A}) = \{\mathbf{x} \mid \mathbf{A}\mathbf{x} = \mathbf{0}\}$
- **秩-零化度定理**：$\dim(\mathcal{R}(\mathbf{A})) + \dim(\mathcal{N}(\mathbf{A})) = n$
- **Gram 矩阵**：$\mathbf{G} = \mathbf{A}^T\mathbf{A}$，对称半正定

### 特征值与特征向量

- **定义**：$\mathbf{A}\mathbf{v} = \lambda\mathbf{v}$
- **特征分解**：$\mathbf{A} = \mathbf{P}\mathbf{\Lambda}\mathbf{P}^{-1}$
- **对称矩阵**：特征值全为实数，不同特征值对应特征向量正交，可正交对角化 $\mathbf{A} = \mathbf{Q}\mathbf{\Lambda}\mathbf{Q}^T$
- **迹与行列式**：$\text{tr}(\mathbf{A}) = \sum \lambda_i$，$\det(\mathbf{A}) = \prod \lambda_i$

### 二次型

$f(\mathbf{x}) = \mathbf{x}^T\mathbf{A}\mathbf{x}$，$\mathbf{A}$ 对称。正定（所有特征值 $>0$）或半正定（特征值 $\geq 0$）。

### 奇异值分解（SVD）

任意矩阵 $\mathbf{A} \in \mathbb{R}^{m \times n}$ 可分解为：

$$\mathbf{A} = \mathbf{U}\mathbf{\Sigma}\mathbf{V}^T$$

- $\mathbf{U}$：左奇异向量（$\mathbf{A}\mathbf{A}^T$ 的特征向量）
- $\mathbf{\Sigma}$：奇异值对角矩阵，$\sigma_i = \sqrt{\lambda_i(\mathbf{A}^T\mathbf{A})}$
- $\mathbf{V}$：右奇异向量（$\mathbf{A}^T\mathbf{A}$ 的特征向量）
- **低秩近似**：取前 $k$ 个最大奇异值得最优近似（Eckart-Young 定理）
- **几何意义**：旋转 $\to$ 缩放 $\to$ 旋转

### 矩阵微积分

- **梯度**：$\nabla_{\mathbf{x}}(\mathbf{a}^T\mathbf{x}) = \mathbf{a}$，$\nabla_{\mathbf{x}}(\mathbf{x}^T\mathbf{A}\mathbf{x}) = 2\mathbf{A}\mathbf{x}$（$\mathbf{A}$ 对称）
- **黑塞矩阵**：$H_{ij} = \frac{\partial^2 f}{\partial x_i \partial x_j}$，正定性决定局部凸性
- **雅可比矩阵**：$J_{ij} = \frac{\partial f_i}{\partial x_j}$

## 发展演进

线性代数从解方程组发展出矩阵论，20 世纪 SVD 和特征分解成为数值分析核心工具。在 [[深度学习]] 时代，GPU 大规模矩阵乘法成为训练的基础设施，低秩分解技术又催生了 LoRA 等参数高效微调方法。

## 关键算法·模型

- **PCA**：特征值分解实现降维
- **SVD 低秩近似**：推荐系统、压缩
- **最小二乘法**：$\hat{\mathbf{x}} = (\mathbf{A}^T\mathbf{A})^{-1}\mathbf{A}^T\mathbf{b}$
- **LoRA**：$\Delta W = BA$，$r \ll \min(m,n)$，基于低秩分解的参数高效微调

## 应用场景

- **[[机器学习]]**：线性回归（$\mathbf{y} = \mathbf{X}\mathbf{w}$）、SVM 内积、PCA 降维
- **[[深度学习]]**：前向传播（矩阵乘法）、反向传播（梯度计算）、权重表示
- **[[Transformer]]**：自注意力中 Q、K、V 矩阵运算
- **推荐系统**：矩阵分解（SVD）
- **NLP**：词向量表示、主题建模（LSA）

## 与其他技术关系

- 为 [[微积分]] 中矩阵微积分提供基础
- 为 [[概率与统计]] 中协方差矩阵、多元高斯分布提供工具
- 为 [[优化方法]] 中梯度下降提供理论前提
- SVD 低秩近似是 [[LoRA]] 等参数高效微调的数学基础
- 特征值分解在 [[信息论]] 相关的 PCA 中有应用

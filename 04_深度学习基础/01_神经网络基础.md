# 神经网络基础

## 1. 概述

人工神经网络（Artificial Neural Network, ANN）是深度学习的核心计算模型，由大量互联的人工神经元组成，通过层级化非线性变换实现对复杂函数的逼近。神经网络的设计灵感来源于生物神经系统的信息处理机制，但在数学本质上是复合函数的优化求解器。

- **解决的问题**：神经网络解决的核心问题是**从数据中自动学习映射函数**。给定输入 $\mathbf{x}$ 和目标输出 $\mathbf{y}$，通过参数化的复合非线性函数 $\hat{\mathbf{y}} = f_\theta(\mathbf{x})$ 不断逼近真实的 $\mathbf{x} \to \mathbf{y}$ 映射关系。
- **与生物神经网络的区别**：现代深度学习的神经网络是高度简化、高度工程化的数学模型，不应被理解为生物神经系统的忠实模拟。

## 2. 发展历史

神经网络的演进经历了"兴起→寒冬→复兴→爆发"的周期：

- **联结主义萌芽（1943-1969）**：1943年McCulloch-Pitts提出M-P神经元模型；1958年Rosenblatt感知机实现可学习的线性分类；1969年Minsky证明感知机的非线性局限，进入第一次寒冬
- **反向传播复兴（1986-1995）**：BP算法解决多层网络训练问题；通用逼近定理给出理论保证；但因算力不足和SVM等浅层方法的竞争，1990s中期再次遇冷
- **深度学习破茧（2006-2012）**：Hinton 深度信念网络(DBN) 以逐层预训练突破深层训练瓶颈；GPU算力成熟；2012年AlexNet引爆新纪元
- **工业化与规模化（2012-至今）**：ResNet(2015)解决超深网络退化；Transformer(2017)重构序列建模范式；Scaling Law(2020)确立"大力出奇迹"的大模型路径

## 3. 核心概念

### 3.1 人工神经元（Artificial Neuron）

神经元是神经网络的最小计算单元，完成"线性加权→偏置→非线性激活"三步操作：

$$z = \sum_{i=1}^{n} w_i x_i + b = \mathbf{w}^T \mathbf{x} + b$$

$$a = f(z)$$

其中 $\mathbf{w} \in \mathbb{R}^n$ 为权重向量（Weight），$b \in \mathbb{R}$ 为偏置（Bias），$f(\cdot)$ 为非线性激活函数（Activation Function）。

- **权重的意义**：$w_i$ 的大小和符号决定了第 $i$ 个输入特征对神经元输出的影响程度和方向
- **偏置的意义**：$b$ 控制神经元的激活阈值，使决策边界不必经过原点
- **激活函数的意义**：引入非线性，使网络具备逼近任意复杂函数的能力

### 3.2 网络层（Layer）

层是神经元的集合，同一层内的神经元接收相同的输入但拥有独立的参数：

- **输入层（Input Layer）**：直接将原始数据传入网络，不进行任何计算
- **隐藏层（Hidden Layer）**：位于输入层和输出层之间，负责特征提取和变换。网络深度即隐藏层的数量
- **输出层（Output Layer）**：输出最终的预测结果，维度由任务决定（回归为1维，$K$分类为$K$维）

### 3.3 多层感知机（Multi-Layer Perceptron, MLP）

MLP是最基础的前馈神经网络架构，由输入层、若干个全连接隐藏层和输出层顺序堆叠而成。

**数学表示**（以两层隐藏层为例）：

$$\mathbf{h}_1 = f(\mathbf{W}_1 \mathbf{x} + \mathbf{b}_1)$$

$$\mathbf{h}_2 = f(\mathbf{W}_2 \mathbf{h}_1 + \mathbf{b}_2)$$

$$\hat{\mathbf{y}} = g(\mathbf{W}_3 \mathbf{h}_2 + \mathbf{b}_3)$$

其中 $g(\cdot)$ 为输出层激活函数（分类任务通常为Softmax，回归任务为恒等映射）。

**关键设计要点**：
- 层间**全连接**：每个神经元与上一层所有神经元相连，参数量为 $n_{in} \times n_{out}$
- 隐藏层维度（宽度）和隐藏层数量（深度）是核心超参数
- 所有可学习参数 $\{\mathbf{W}_1, \mathbf{b}_1, \dots, \mathbf{W}_L, \mathbf{b}_L\}$ 通过反向传播和梯度下降联合优化

### 3.4 通用逼近定理（Universal Approximation Theorem, UAT）

> **定理**（Hornik et al., 1989）：对于任意闭区间上的连续函数 $F(\mathbf{x})$ 和任意 $\epsilon > 0$，存在一个仅含**单隐藏层**的前馈神经网络，只要隐藏层神经元数量足够多且激活函数非多项式，就能以 $\epsilon$ 精度逼近 $F(\mathbf{x})$。

**三层含义**：

1. **存在性保证**：神经网络表达能力的理论上界是无限的——没有神经网络"学不会"的连续函数
2. **深度的必要性**：UAT本身仅保证了单隐层的存在性。但后续研究表明，深层网络能以**指数级更少的神经元**表达同样的函数，通过特征复用（Feature Reuse）实现参数效率的巨大提升
3. **未回答的问题**：UAT不说明如何从有限样本中学习到泛化良好的参数，也不保证梯度下降能找到全局最优解

**深度与宽度的本质区别**：

| 维度 | 单隐层宽网络 | 多隐层深网络 |
|:---|:---|:---|
| 逼近效率 | 可能需要指数级神经元 | 可通过层级组合高效表达 |
| 特征学习 | 扁平的单次映射 | 层次化特征抽象 |
| 泛化性能 | 易过拟合 | 层级归纳偏置（Hierarchical Inductive Bias）有利于泛化 |
| 训练难度 | 浅层梯度容易传播 | 深层面临梯度消失/爆炸 |

## 4. 技术原理

### 4.1 前向传播的信息流

神经网络的前向传播是从输入到输出的逐层计算过程：

```
输入 x
  ↓
Layer 1: z^(1) = W^(1)x + b^(1)  →  a^(1) = f(z^(1))
  ↓
Layer 2: z^(2) = W^(2)a^(1) + b^(2)  →  a^(2) = f(z^(2))
  ↓
  ...
  ↓
Layer L: z^(L) = W^(L)a^(L-1) + b^(L)  →  ŷ = g(z^(L))
```

每一层完成两个操作：
1. **仿射变换**：$\mathbf{z}^{(l)} = \mathbf{W}^{(l)} \mathbf{a}^{(l-1)} + \mathbf{b}^{(l)}$，实现特征空间的线性投影与平移
2. **非线性激活**：$\mathbf{a}^{(l)} = f(\mathbf{z}^{(l)})$，引入对特征的非线性扭曲

连续堆叠这两步操作使网络能够学习到高度复杂的决策边界。

### 4.2 深层网络为何需要非线性

如果所有激活函数都是线性的（$f(x)=x$），多层线性变换的复合仍是线性的：

$$\mathbf{W}_3(\mathbf{W}_2(\mathbf{W}_1\mathbf{x} + \mathbf{b}_1) + \mathbf{b}_2) + \mathbf{b}_3 = \mathbf{W}'\mathbf{x} + \mathbf{b}'$$

无论堆叠多少层，表达能力等价于单层线性模型。非线性激活函数是深层网络表达能力的**充要条件**。

### 4.3 网络容量与泛化

- **参数数量** $P$ 衡量网络的理论拟合容量（Capacity）
- **有效容量**受优化算法、正则化和数据量的共同制约
- 深度学习经验法则：$P \gg N$（参数量远大于样本数）时网络仍能良好泛化，此现象挑战传统统计学习理论的 VC 维上界，是当前理论研究的重要课题

## 5. 关键方法/模型

### 5.1 里程碑模型演进

| 模型 | 年份 | 论文 | 核心贡献 |
|:---|:---|:---|:---|
| **AlexNet** | 2012 | *ImageNet Classification with Deep CNNs* | 5层卷积+3层全连接；首次在大规模视觉任务上验证GPU训练可行性；引入ReLU、Dropout、数据增强 |
| **VGG** | 2014 | *Very Deep Convolutional Networks* | 统一使用3×3小卷积核堆叠替代大卷积核；证明深度（16-19层）对视觉任务的关键价值 |

| **ResNet** | 2016 | *Deep Residual Learning for Image Recognition* | 残差连接 $\mathbf{a}^{(l)} = f(\mathbf{z}^{(l)}) + \mathbf{a}^{(l-1)}$ 使152层网络训练成为可能；奠定所有后续深层架构的设计基础 |
| **DenseNet** | 2017 | *Densely Connected Convolutional Networks* | 极致特征复用：每层与所有前层直接连接 |
| **Transformer** | 2017 | *Attention Is All You Need* | 以自注意力机制完全取代RNN的时序迭代；释放序列并行计算能力；成为LLM的标准架构 |

## 6. 优势与局限

### 核心优势
- **层次化表征能力**：浅层学习边缘/纹理等低级物理特征，深层组合为语义部件和物体轮廓，形成从具体到抽象的特征金字塔
- **端到端联合优化**：特征提取与任务目标在统一损失函数下同步更新，避免多阶段流水线的累积误差
- **极佳的扩展性**：Scaling Law表明，只要持续增加模型容量和数据规模，性能几乎可以无限提升
- **通用性**：同一架构（如MLP/Transformer）可处理视觉、语言、语音等多种模态数据

### 理论与工程局限
- **黑盒特性**：数十亿参数的复合非线性函数难以进行可追溯的逻辑推导
- **灾难性遗忘（Catastrophic Forgetting）**：连续学习新任务时，新梯度更新会覆盖旧任务形成的参数表征
- **对抗脆弱性**：人眼不可察觉的微小输入扰动可导致顶级模型输出完全荒谬的结果
- **训练不稳定性**：深层网络的梯度消失/爆炸问题，对初始化和超参数高度敏感

## 7. 应用场景

- **结构化数据建模**：MLP在精准营销、信用评分、推荐系统中的特征交互与分类
- **信号级特征感知**：一维残差网络直接处理振动信号、心电（ECG）、音频频谱的异常检测
- **LLM核心组件**：大语言模型的FFN层本质上是隐藏维扩增4倍的高维MLP，占据模型约2/3的参数量
- **多模态感知骨干**：作为特征投影器（Projector）连接不同模态的嵌入空间

## 8. 与其他技术关系

- **与数学基础的关系**：矩阵乘法是前向传播的算力瓶颈；链式法则是反向传播的数学内核
- **与优化理论的关系**：一阶梯度算法（SGD/Adam/AdamW）是神经网络训练的引擎（详见 [[06_优化算法与学习率调度]]）
- **与激活函数/损失函数的关系**：激活函数决定了网络的非线性表达能力，损失函数定义了优化目标（详见 [[02_激活函数与损失函数]]）
- **与后续架构的关系**：MLP的前馈机制和层级堆叠思想被CNN、RNN、Transformer全盘继承，是卷积层、注意力层等高级组件的原子操作

## 9. 前沿发展

1. **新型前馈架构探索**
   - **MLP-Mixer**（2021）：证明完全不用卷积或注意力，仅在通道维度和空间维度交替做全连接就能在CV任务上取得一流性能，重塑了人们对MLP表达上限的认知
   - **KAN (Kolmogorov-Arnold Networks)**（2024）：将可学习激活函数置于边上而非节点上，用样条函数替代固定激活函数，在科学计算任务上展示出更好的可解释性和参数效率

2. **物理学启发的新架构**
   - **Physics-Informed Neural Networks (PINN)**：将Navier-Stokes方程、薛定谔方程等物理守恒律作为正则化项注入损失函数，使数据稀缺时仍能严格在物理约束下拟合

3. **神经正切核理论（Neural Tangent Kernel, NTK）**
   - 在无限宽极限下，神经网络的训练动力学等价于核方法，为理解深层网络在非凸面上为何能规避局部极小值、找到泛化良好的全局最优解提供了解析数学视角

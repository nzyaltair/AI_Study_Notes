# 无监督学习方法

## 1. 概述

[[07_无监督学习方法|无监督学习]]从无标注数据 $\{x_1, \dots, x_n\}$ 中发现内在的结构、模式或分布特征，无需人工标注。其核心挑战在于：**没有"正确答案"作为监督信号，算法必须从数据自身中发现有意义的结构**。

无监督学习涵盖四大类任务：
- **聚类（Clustering）**：将样本分组成"簇"
- **降维（Dimensionality Reduction）**：将高维数据映射到低维空间
- **密度估计（Density Estimation）**：学习数据的概率分布
- **关联规则（Association Rules）**：发现数据中的频繁模式

在[[00_深度学习基础_综述|深度学习]]时代，无监督学习演变为**自监督学习**，成为大模型预训练的核心范式。

## 2. 发展历史

| 年代 | 里程碑 | 意义 |
|------|--------|------|
| 1901 | Pearson 提出 PCA | 最早的降维方法 |
| 1957/1982 | Lloyd 提出 K-means（1982 年正式发表） | 最广泛使用的聚类算法 |
| 1967 | MacQueen 命名 "K-means" | K-means 术语确立 |
| 1977 | Dempster 等提出 EM 算法 | 含隐变量模型的通用求解框架 |
| 1993 | Agrawal 等提出 Apriori 算法 | 关联规则挖掘经典方法 |
| 1996 | Ester 等提出 DBSCAN | 基于密度的任意形状聚类 |
| 2000 | Han 等提出 FP-Growth | 避免多次数据库扫描的关联规则 |
| 2008 | van der Maaten 提出 t-SNE | 高维数据可视化标杆 |
| 2018 | McInnes 等提出 UMAP | 保留更多全局结构的降维可视化 |
| 2018+ | 自监督学习兴起（BERT、SimCLR） | 无监督学习进入预训练时代 |

## 3. 核心概念

### 3.1 聚类有效性度量

- **内部指标**（无真实标签）：轮廓系数（Silhouette Score）、Davies-Bouldin 指数、Calinski-Harabasz 指数
- **外部指标**（有真实标签）：调整兰德指数（ARI）、归一化互信息（NMI）、同质性/完整性
- **稳定性**：对数据扰动的鲁棒性

### 3.2 距离/相似度度量

| 度量 | 适用场景 | 特点 |
|------|---------|------|
| 欧氏距离 | 连续特征 | 最常用，球面假设 |
| 曼哈顿距离 | 高维稀疏 | 对异常值更鲁棒 |
| 余弦相似度 | 文本、高维 | 关注方向而非大小 |
| 相关系数 | 时序数据 | 度量线性关系强度 |
| Jaccard 相似度 | 集合/二值 | 两集合交集/并集 |
| 编辑距离 | 字符串 | 序列相似度 |

## 4. 技术原理

### 4.1 聚类算法

#### K-means

目标：最小化簇内平方误差和（WCSS）：

$$\text{WCSS} = \sum_{k=1}^{K} \sum_{x \in C_k} \|x - \mu_k\|^2$$

**Lloyd 算法**：
1. 随机初始化 $K$ 个中心 $\mu_1, \dots, \mu_K$
2. **分配步**：$C_k = \{x_i : \|x_i - \mu_k\| \leq \|x_i - \mu_j\|, \forall j\}$
3. **更新步**：$\mu_k = \frac{1}{|C_k|}\sum_{x \in C_k} x$
4. 重复 2–3 至收敛（中心不再变化或 WCSS 变化极小）

**关键改进**：

- **K-means++**：智能初始化，首个中心随机选，后续中心按 $D(x)^2$（与最近已有中心的距离平方）概率采样 → 近似最优的 $O(\log K)$ 竞争比
- **Mini-Batch K-means**：每次用随机小批样本更新中心 → 大规模数据加速

**如何选择 K**：
- **肘部法则**：绘制 WCSS vs K 曲线，找"肘点"
- **轮廓系数**：$s(i) = \frac{b(i) - a(i)}{\max(a(i), b(i))}$，越接近 1 越好
- **Gap 统计量**：比较聚类紧密度与零分布下的期望值

#### DBSCAN（Density-Based Spatial Clustering of Applications with Noise）

基于密度连通性的聚类，无需预设簇数量。

**核心概念**：
- **$\epsilon$-邻域**：以样本为中心、半径 $\epsilon$ 的范围
- **核心点（Core Point）**：$\epsilon$-邻域内包含 $\geq$ MinPts 个样本
- **边界点（Border Point）**：在核心点的 $\epsilon$-邻域内但自身不是核心点
- **噪声点（Noise Point）**：既非核心点也非边界点

**算法流程**：从每个未访问的核心点出发，通过密度可达关系扩展聚类，直到所有核心点被访问。

**参数选择**：
- **MinPts**：经验值 $\geq$ 维度 $+1$，通常取 4–10
- **$\epsilon$**：绘制 k-distance 图（每个点到其第 MinPts 近邻的距离排序），取"拐点"

**优势**：发现任意形状簇、自动识别噪声、无需预设 K
**局限**：对 $\epsilon$ 敏感、密度差异大的数据效果差、高维数据中距离失效

#### 高斯混合模型（GMM）

将数据建模为 $K$ 个高斯分布的加权混合：

$$p(x) = \sum_{k=1}^{K} \pi_k \cdot \mathcal{N}(x \mid \mu_k, \Sigma_k)$$

其中 $\sum \pi_k = 1$，每个高斯分量有自己的均值 $\mu_k$ 和协方差 $\Sigma_k$。

**EM 算法求解**：

**E 步**（期望步）：计算每个样本属于每个分量的责任度（后验概率）

$$\gamma(z_{ik}) = \frac{\pi_k \cdot \mathcal{N}(x_i \mid \mu_k, \Sigma_k)}{\sum_{j=1}^{K} \pi_j \cdot \mathcal{N}(x_i \mid \mu_j, \Sigma_j)}$$

**M 步**（最大化步）：基于责任度更新参数

$$\mu_k = \frac{\sum_i \gamma(z_{ik}) x_i}{\sum_i \gamma(z_{ik})}, \quad \Sigma_k = \frac{\sum_i \gamma(z_{ik})(x_i - \mu_k)(x_i - \mu_k)^T}{\sum_i \gamma(z_{ik})}, \quad \pi_k = \frac{\sum_i \gamma(z_{ik})}{n}$$

EM 保证对数似然单调不减，收敛到局部最优。

**GMM vs K-means**：
| 特性 | K-means | GMM |
|------|---------|-----|
| 聚类类型 | 硬聚类 | 软聚类（概率归属） |
| 簇形状 | 各向同性球形 | 任意椭圆形 |
| 参数估计 | 启发式迭代 | EM（有理论保证） |
| 计算复杂度 | $O(nKd)$ | $O(nKd^2)$（全协方差） |

#### 层次聚类

构建样本间的层次结构树（Dendrogram），分为凝聚法（自底向上）和分裂法（自顶向下）。

**簇间距离度量**：

| 方法 | 定义 | 特点 |
|------|------|------|
| **单连接**（Single） | $\min_{a \in A, b \in B} d(a,b)$ | 链式效应，细长簇 |
| **全连接**（Complete） | $\max_{a \in A, b \in B} d(a,b)$ | 紧凑簇，对离群点敏感 |
| **平均连接**（Average） | $\frac{1}{|A||B|}\sum_{a,b} d(a,b)$ | 折中方案 |
| **Ward 方法** | 最小化合并后方差增量 | 倾向产生大小均匀的簇 |

**凝聚法步骤**：每个样本自成一簇 → 合并最近的两个簇 → 更新距离矩阵 → 重复至仅剩一个簇。

### 4.2 降维算法

#### 主成分分析（PCA）

PCA 寻找数据方差最大的正交方向（主成分），将数据投影到低维空间。

**推导**：
1. 数据中心化：$\tilde{x}_i = x_i - \bar{x}$
2. 计算协方差矩阵：$S = \frac{1}{n-1}\sum_i \tilde{x}_i \tilde{x}_i^T$
3. 特征分解：$S = V\Lambda V^T$，取前 $k$ 个最大特征值对应的特征向量
4. 投影：$z_i = V_k^T \tilde{x}_i$

**方差解释率**：第 $j$ 个主成分解释的方差比例为 $\lambda_j / \sum_i \lambda_i$，累积解释率用于决定保留的维度数。

**PCA 的局限**：仅捕获线性关系，对异常值敏感。

#### t-SNE（t-Distributed Stochastic Neighbor Embedding）

t-SNE 是主要用于**可视化**的非线性降维方法。核心思想是在高维和低维空间中分别定义样本间的相似度概率分布，最小化两者的 KL 散度。

**关键设计**：
- 高维空间：用高斯分布建模局部相似度
- 低维空间：用 **t 分布**（重尾分布）建模，避免"拥挤问题"——同类点不会过度拥挤
- 梯度下降优化低维坐标

**重要注意事项**：
- t-SNE 不保留全局距离，只保局部结构——簇之间的距离无意义
- 对困惑度（perplexity）参数敏感（典型值 5–50）
- 结果不可逆——无法将新数据映射到已有嵌入
- 首先用 PCA 降至 50 维再 t-SNE 可降低噪声和加速

#### UMAP（Uniform Manifold Approximation and Projection）

UMAP 基于拓扑数据分析（Riemannian 几何 + 代数拓扑），是 t-SNE 的现代替代。

**相比 t-SNE 的优势**：
- 更快（可处理百万级样本）
- 更好地保留全局结构
- 结果具有可重复性（随机种子固定时）
- 可学习映射函数，将新样本投影到已有嵌入

**原理简述**：在数据上构建模糊单纯集（fuzzy simplicial set）表示高维拓扑结构，在低维空间中优化等效表示。

### 4.3 关联规则挖掘

#### Apriori 算法

**核心概念**：
- **项集（Itemset）**：一个或多个项目的集合
- **支持度（Support）**：$P(A \cap B)$，项集在数据中出现的频率
- **置信度（Confidence）**：$P(B \mid A) = \frac{\text{Support}(A \cup B)}{\text{Support}(A)}$
- **提升度（Lift）**：$\frac{P(B \mid A)}{P(B)}$，$>1$ 表示正相关，$=1$ 表示独立

**先验原理**（Apriori Property）：频繁项集的子集必频繁。利用此单调性剪枝搜索空间。

**算法流程**：
1. 找出所有频繁 1-项集
2. 由频繁 $k$-项集生成候选 $(k+1)$-项集（连接步）
3. 扫描数据，过滤不满足最小支持度的候选（剪枝步）
4. 重复 2–3 直到无法生成新候选项集
5. 从频繁项集中生成满足最小置信度的关联规则

#### FP-Growth

通过 FP-tree（频繁模式树）压缩数据库，只需两次扫描数据。比 Apriori 更高效，避免了候选生成-测试的多轮迭代。

### 4.4 异常检测

| 方法 | 原理 | 优势 | 局限 |
|------|------|------|------|
| **Isolation Forest** | 异常点更易被随机分割隔离 | 无分布假设，高维可用 | 对局部异常不敏感 |
| **LOF**（局部离群因子） | 比较样本密度与邻居密度 | 处理局部密度差异 | 参数敏感 |
| **One-Class SVM** | 学习包围正常数据的边界 | 核方法灵活 | 调参复杂 |
| **椭圆包络** | 假设正常数据服从高斯分布 | 简单快速 | 仅适合凸分布 |
| **DBSCAN 噪声检测** | 不属于任何簇的点为异常 | 无需额外算法 | 依赖聚类质量 |

## 5. 关键方法/模型

### 5.1 方法选择指南

| 任务 | 首选方法 | 何时换用 |
|------|---------|---------|
| 球形簇聚类 | K-means++ | 非球形 → DBSCAN/GMM |
| 任意形状聚类 | DBSCAN | 密度差异大 → 层次聚类 |
| 软聚类/概率 | GMM + EM | 需硬分配 → K-means |
| 线性降维 | PCA | 非线性 → t-SNE/UMAP |
| 2D/3D 可视化 | UMAP | 极重局部结构 → t-SNE |
| 异常检测 | Isolation Forest | 局部异常 → LOF |
| 关联规则 | FP-Growth | 教学理解 → Apriori |

## 6. 优势与局限

### 6.1 优势

- **无需标注**：最大优势——标注成本为零
- **探索性**：发现未知模式和结构，提供数据洞察
- **预处理价值**：聚类和降维常用于监督学习的特征预处理
- **可扩展性**：部分算法（如 Mini-Batch K-means）可处理大规模数据

### 6.2 局限

- **评估困难**：无标注意味着缺乏客观的评价标准
- **主观性强**：K 的选择等依赖领域知识
- **初始化敏感**：K-means、EM 等对初始值敏感，容易陷入局部最优
- **高维失效**：距离度量在高维空间中趋于均匀（维数灾难），聚类效果差

| 应用领域 | 任务 | 方法 |
|---------|------|------|
| 用户分析 | 用户分群、画像 | K-means、GMM |
| 异常检测 | 欺诈检测、故障监测 | Isolation Forest、LOF |
| 推荐系统 | 协同过滤、物品聚类 | K-means、关联规则 |

| 图像处理 | 图像压缩、去噪 | PCA、自编码器 |

## 8. 与其他技术关系

- 降维思想与 [[00_深度学习基础_综述|深度学习]] 中的自编码器（Autoencoder）一脉相承
- GMM 的 EM 算法是理解隐变量模型（包括 HMM、LDA）的基石
- K-means 的向量量化思想用于词袋模型的视觉词袋（BoVW）
- 聚类结果可作为 [[02_特征工程|特征工程]] 中的新特征（聚类标签、距离）
- 自监督学习是无监督学习在 [[00_深度学习基础_综述|深度学习]] 时代的进化——BERT 的 MLM 和 SimCLR 的对比学习本质上都是无监督/自监督范式
- 评估指标理论依赖 [[03_概率与统计|概率与统计]]

## 9. 前沿发展

- **深度聚类**：结合自编码器和聚类的联合优化（如 DEC、IDEC）
- **自监督表示学习**：对比学习（SimCLR、MoCo）、掩码自编码器（MAE）——无监督预训练已成为视觉领域主流
- **图聚类**：GNN 用于图结构数据的聚类（谱聚类 + 深度学习）
- **可微分聚类**：将聚类目标嵌入端到端可训练框架
- **大规模降维**：PCA 的随机化近似（Randomized SVD）、在线 t-SNE
- **子空间聚类**：在高维数据中发现存在于不同低维子空间中的簇
- **多视图聚类**：整合来自多个数据源或特征的聚类信息

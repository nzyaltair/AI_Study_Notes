# 无监督学习方法

## 背景

无监督学习从无标记数据中发现潜在结构和模式，无需人工标注。在数据探索、异常检测、推荐系统等场景中发挥重要作用，也是自监督学习和[[深度学习]]预训练的理论基础。

## 核心思想

给定无标记数据 $\{\mathbf{x}_1, \dots, \mathbf{x}_n\}$，发现数据的内在结构：聚类（分组）、降维（压缩）、密度估计（分布建模）或关联规则（模式发现）。

## 技术原理

### 聚类算法

#### K-means

将数据划分为K个簇，最小化簇内平方误差和：

$$SSE = \sum_{i=1}^{k} \sum_{\mathbf{x} \in C_i} \|\mathbf{x} - \boldsymbol{\mu}_i\|^2$$

- 算法：随机初始化K个中心 → 分配样本到最近中心 → 更新中心 → 重复至收敛
- 优点：简单高效；缺点：需预设K、对初始值敏感、只适合凸形簇
- 改进：K-means++（智能初始化）、Mini-Batch K-means（大规模数据）

#### 层次聚类

构建聚类树（Dendrogram），分为：
- **凝聚式**（自底向上）：每步合并最近的两个簇
- **分裂式**（自顶向下）：递归分裂

簇间距离度量：单连接、全连接、平均连接、Ward方法

#### DBSCAN

基于密度的聚类，无需预设簇数：
- 核心点：$\epsilon$-邻域内至少有 MinPts 个点
- 可发现任意形状的簇，自动识别噪声点
- 优点：无需预设K、处理噪声；缺点：对参数敏感、高维效果差

#### 高斯混合模型（GMM）

概率聚类，假设数据由K个高斯分布混合生成：

$$P(\mathbf{x}) = \sum_{i=1}^{k} \pi_i \mathcal{N}(\mathbf{x} | \boldsymbol{\mu}_i, \boldsymbol{\Sigma}_i)$$

- 使用EM算法求解：E步计算责任度，M步更新参数
- 软聚类：给出样本属于各簇的概率
- 可用于密度估计和异常检测

### 降维算法

#### 主成分分析（PCA）

通过线性变换找到数据方差最大的方向：

- 计算协方差矩阵的特征值和特征向量
- 取前k个最大特征值对应的特征向量作为主成分
- 投影：$\mathbf{Z} = \mathbf{W}^T \mathbf{X}$
- 优点：无参数、计算高效；缺点：只捕获线性关系

#### t-SNE

非线性降维，特别适合高维数据可视化：
- 高维空间用高斯分布建模相似度
- 低维空间用t分布建模（避免拥挤问题）
- 保持局部结构，适合2D/3D可视化
- 缺点：计算慢、不可逆（新数据无法直接映射）

#### UMAP

基于拓扑数据分析的降维方法：
- 比t-SNE更快、保留更多全局结构
- 可用于新数据映射（transform）
- 适合大规模数据

### 关联规则

#### Apriori算法

发现频繁项集和关联规则（"啤酒与尿布"）：
- 支持度：$P(A \cap B)$
- 置信度：$P(B | A) = P(A \cap B) / P(A)$
- 提升度：$P(B|A) / P(B)$
- 先验原理：频繁项集的子集必频繁

#### FP-Growth

通过FP-tree压缩数据，避免Apriori的多次扫描，更高效。

### 异常检测

| 方法 | 原理 | 适用场景 |
|------|------|---------|
| **统计方法** | 假设分布，检测偏离点 | 一维数据 |
| **距离方法** | 远离密集区域的点 | 多维数据 |
| **密度方法**（LOF） | 局部密度异常 | 密度不均数据 |
| **Isolation Forest** | 随机隔离，路径短=异常 | 高维数据 |

## 发展演进

K-means（1967）→ 层次聚类 → DBSCAN（1996）→ PCA（1901，早期） → t-SNE（2008）→ UMAP（2018）→ 自监督学习（[[深度学习]]时代）

无监督学习在深度学习时代演变为自监督学习（如BERT的掩码语言模型），成为[[大语言模型|大模型]]预训练的核心范式。

## 应用领域

- **客户分群**：市场营销中的用户画像
- **异常检测**：金融欺诈检测、工业缺陷检测
- **数据可视化**：高维数据的2D/3D可视化
- **推荐系统**：协同过滤中的用户/物品聚类
- **生物信息学**：基因表达分析、蛋白质结构

## 与其他技术关系

- 降维思想与[[深度学习|自编码器]]一脉相承
- GMM的EM算法是理解[[大语言模型|隐变量模型]]的基础
- 聚类是[[特征工程|特征构造]]的手段之一
- 自监督学习是无监督学习在深度学习中的进化
- 理论基础依赖[[概率与统计]]和[[线性代数]]

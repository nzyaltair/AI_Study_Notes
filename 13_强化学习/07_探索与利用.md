# 探索与利用

## 1. 概述

探索与利用（Exploration vs. Exploitation）是强化学习中最核心的困境之一：智能体需要在**利用已知最优动作获取即时高奖励**（exploitation）和**探索未知或较少尝试的动作以发现更优策略**（exploration）之间取得平衡。

过度利用导致智能体陷入局部最优——错过尚未发现的更优策略；过度探索则浪费样本在已知次优动作上。在稀疏奖励环境（如 Montezuma's Revenge）中，探索策略的质量直接决定了 RL 能否学到有效策略。

本篇从多臂老虎机问题出发，系统介绍从简单到高级的探索策略。

## 3. 核心概念

### 3.1 遗憾（Regret）

在多臂老虎机中，衡量探索策略质量的指标是**累积遗憾**：

$$R_T = T \cdot \mu^* - \mathbb{E}\left[\sum_{t=1}^{T} \mu_{a_t}\right]$$

其中 $\mu^*$ 是最优臂的期望奖励，$\mu_{a_t}$ 是时刻 $t$ 选择的臂的期望奖励。好的探索策略使 $R_T$ 随 $T$ 的增长尽可能慢。

- ε-greedy：$R_T = O(T)$（线性增长，次优）
- UCB / Thompson Sampling：$R_T = O(\sqrt{T \ln T})$（次线性，渐近最优）

### 3.2 内在奖励（Intrinsic Reward）

在复杂环境中，外部奖励稀疏时，智能体需要**内在奖励**驱动探索。内在奖励衡量"新颖性"或"不确定性"——访问新颖或不确定的状态获得更高内在奖励：

$$R_{\text{total}}(s, a) = R_{\text{ext}}(s, a) + \beta \cdot R_{\text{int}}(s, a)$$

内在奖励的设计是探索策略研究的核心。

### 3.3 探索的分类

| 类型 | 机制 | 代表方法 |
|:---|:---|:---|
| 随机探索 | 随机扰动动作 | ε-greedy、高斯噪声、熵正则化 |
| 基于不确定性 | 利用值估计的不确定性 | UCB、贝叶斯 UCB |
| 基于后验采样 | 从后验分布采样 | Thompson Sampling |
| 基于好奇心 | 预测误差/信息增益 | ICM、RND |
| 基于记忆 | 新颖性检测 | Episodic Memory、NGU |
| 参数空间探索 | 网络权重噪声 | NoisyNet、Variational Policy |

## 4. 技术原理

### 4.1 ε-greedy

最简单的探索策略：

$$a_t = \begin{cases} \arg\max_a Q(s,a) & \text{概率 } 1-\epsilon \\ \text{随机动作} & \text{概率 } \epsilon \end{cases}$$

- $\epsilon$ 固定：简单但次优，遗憾线性增长
- $\epsilon$ 退火：$\epsilon$ 从高到低递减，前期探索后期利用
- $\epsilon$ 自适应：根据不确定性调整（如 DECSTR）

### 4.2 UCB（Upper Confidence Bound）

**论文**：Auer et al., *Finite-time Analysis of the Multiarmed Bandit Problem* (2002)

选择上置信界最大的臂：

$$a_t = \arg\max_a \left[ \hat{Q}(a) + c \sqrt{\frac{\ln t}{N(a)}} \right]$$

- $\hat{Q}(a)$：臂 $a$ 的平均奖励（利用项）
- $c\sqrt{\ln t / N(a)}$：不确定性奖金（探索项），$N(a)$ 是臂 $a$ 被选择的次数
- 少选的臂不确定性高，获得更多探索机会

**UCB 的直觉**："乐观面对不确定性"（Optimism in the Face of Uncertainty）——假设不确定的臂可能更好，主动尝试。

### 4.3 Thompson Sampling

**论文**：Thompson, *On the likelihood that one unknown probability exceeds another* (1933)

从每个臂的后验分布中采样，选择采样值最大的臂：

$$a_t = \arg\max_a \tilde{\mu}_a, \quad \tilde{\mu}_a \sim P(\mu_a | \text{data})$$

- 乐观的臂（后验分布偏高）更可能被选中
- 不确定的臂（后验分布宽）有更大概率被采样到高值

Thompson Sampling 在理论和实践中都表现优异，是贝叶斯最优探索的经典方法。

### 4.4 好奇心驱动探索（ICM）

**论文**：Pathak et al., *Curiosity-driven Exploration by Self-Supervised Prediction* (2017)

ICM（Intrinsic Curiosity Module）的核心是用**预测误差**作为内在奖励：

1. **逆向模型（Inverse Model）**：给定 $(s_t, s_{t+1})$，预测动作 $a_t$。这迫使网络学习与动作相关的特征。
2. **前向模型（Forward Model）**：给定 $(\phi(s_t), a_t)$，预测 $\phi(s_{t+1})$。
3. **内在奖励**：$R_{\text{int}} = \frac{1}{2} \| \hat{\phi}(s_{t+1}) - \phi(s_{t+1}) \|^2$

**关键设计**：逆向模型确保特征空间 $\phi$ 只编码"受动作影响"的信息，忽略不可控的随机背景（如电视噪声），避免好奇心被噪声误导。

### 4.5 随机网络蒸馏（RND）

**论文**：Burda et al., *Exploration by Random Network Distillation* (2019)

RND 用一个**固定随机网络**作为目标，训练一个**预测网络**去模仿它：

1. 目标网络 $f^*$：随机初始化，参数固定
2. 预测网络 $\hat{f}$：学习 $\hat{f}(s) \approx f^*(s)$
3. 内在奖励：$R_{\text{int}}(s) = \| \hat{f}(s) - f^*(s) \|^2$

**直觉**：预测网络在见过的状态上拟合好（低误差），在未见过的状态上拟合差（高误差）。预测误差自然地衡量了状态的新颖性。

**RND 的优势**：
- 实现简单（两个网络 + MSE 损失）
- 不需要逆向模型
- 对随机噪声更鲁棒（随机网络提取的特征比像素级预测更抽象）
- 在 Montezuma's Revenge 等困难探索游戏上取得突破

### 4.6 NGU 与 Agent57

**论文**：Badia et al., *Never Give Up: Learning Directed Exploration Strategies* (2020)

NGU 结合了两种探索信号：

1. **Episodic 好奇心**：当前回合内的短期新颖性（基于 episodic memory 的 k-NN 密度估计）
2. **Lifelong 好奇心**：跨回合的长期新颖性（RND 式预测误差）

内在奖励：$R_{\text{int}} = \frac{1}{\sqrt{N(s)}} \cdot R_{\text{episodic}}(s)$

**Agent57**（Badia et al., 2020）：NGU + 自适应探索/利用平衡，是首个在所有 57 款 Atari 游戏上超越人类水平的算法。

### 4.7 熵正则化

在策略梯度方法中，通过在目标函数中加策略熵奖励鼓励随机性（探索）：

$$J(\theta) = \mathbb{E}[R] + \beta \, \mathcal{H}(\pi_\theta(\cdot|s))$$

SAC 将此推向极致——最大熵 RL 将策略熵作为优化目标的一部分，实现持续探索。参见 [[04_策略梯度方法]]。

### 4.8 NoisyNet

在网络权重中注入可学习噪声，实现参数空间的探索。与动作空间探索（ε-greedy）相比，参数空间探索更高效——同一状态的扰动具有一致性。参见 [[05_深度强化学习]]。

## 5. 探索策略对比

| 方法 | 类型 | 适用场景 | 优势 | 局限 |
|:---|:---|:---|:---|:---|
| ε-greedy | 随机 | 通用 | 简单 | 探索效率低 |
| UCB | 不确定性 | 老虎机/小状态 | 理论保证 | 高维状态难扩展 |
| Thompson Sampling | 贝叶斯 | 老虎机 | 渐近最优 | 后验计算复杂 |
| ICM | 好奇心 | 视觉RL | 自监督 | 对噪声敏感 |
| RND | 蒸馏 | 视觉RL | 简单鲁棒 | 特征可能退化 |
| 熵正则化 | 随机 | 策略梯度 | 与优化统一 | 探索力度有限 |
| NoisyNet | 参数空间 | 通用 | 高效 | 噪声规模难调 |

## 6. 优势与局限

### 优势

1. **突破稀疏奖励**：好奇心等内在奖励让智能体在无外部奖励时也能学习
2. **逃离局部最优**：系统性探索发现更优策略
3. **理论保证**：UCB/Thompson Sampling 有遗憾界保证

### 局限

1. **探索效率**：在高维连续空间中，即使是高级探索策略也可能效率不足
2. **内在奖励尺度**：内在/外在奖励的平衡系数 $\beta$ 难以调参
3. **好奇心退化**：预测模型一旦训练充分，所有状态的预测误差趋近于零，好奇心消失
4. **噪声敏感性**：随机环境中的不可预测部分会持续产生高预测误差，误导探索

## 7. 应用场景

| 场景 | 推荐方法 | 说明 |
|:---|:---|:---|
| 多臂老虎机 | UCB / Thompson Sampling | 理论最优 |
| Atari 稀疏奖励游戏 | RND / NGU | Montezuma's Revenge 等 |
| 连续控制 | 熵正则化 (SAC) | 内置探索 |
| 大模型 RLHF | 熵正则化 | 防止策略过早确定化 |
| 机器人探索 | ICM + 内在奖励 | 真实环境中样本昂贵 |

## 8. 与其他技术关系

- **与值函数方法的关系**：ε-greedy 是 Q-Learning/DQN 的默认探索策略。参见 [[03_无模型预测与控制]]、[[05_深度强化学习]]。
- **与策略梯度的关系**：熵正则化和最大熵 RL（SAC）将探索融入优化目标。参见 [[04_策略梯度方法]]。
- **与离线 RL 的关系**：离线 RL 无法在线探索，探索问题转化为分布外（OOD）动作的处理。参见 [[08_离线强化学习]]。
- **与信息论的关系**：好奇心和信息增益本质上是信息论概念——减少对环境的不确定性。参见 [[01_数学基础]]。

## 9. 前沿发展

- **基于计数的探索**：密度估计（k-NN、哈希）在连续状态空间中的推广
- **变分探索**：用变分推断估计模型不确定性，理论更严谨
- **Go-Explore**：显式存档感兴趣状态，从存档点重新探索，在 Montezuma's Revenge 上取得突破
- **大模型辅助探索**：用 LLM 的语义理解引导探索方向（如 LLM 提供语言化的探索目标）
- **自监督预训练 + RL**：预训练表示提供更好的状态编码，间接改善探索
- **元学习探索策略**：学习能快速适应新任务的探索策略

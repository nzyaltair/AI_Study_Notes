# 学习率调度与训练策略

## 1. 概述

学习率是深度学习训练中最重要的超参数，直接影响收敛速度和最终性能。学习率调度（Learning Rate Scheduling）策略通过在训练过程中动态调整学习率，在训练早期保持快速收敛、在后期精细调优，是现代深度学习训练不可或缺的组件。[[00_大语言模型核心架构_综述|大语言模型]] 训练中，学习率调度策略（如余弦退火 + 预热）已成为标准配置。

## 2. 发展历史

| 年代 | 里程碑 | 意义 |
|------|--------|------|
| 2010s | Step Decay | 经典的阶梯式衰减策略 |
| 2015 | Cyclical Learning Rates (Smith) | 周期性学习率，打破单调递减 |
| 2016 | SGDR / Warm Restarts (Loshchilov) | 余弦退火 + 热重启 |
| 2017 | One-Cycle Policy (Smith & Topin) | 单周期策略，超级收敛 |
| 2017 | Warmup | Transformer 训练引入预热 |
| 2019 | RAdam | 自适应修正预热问题 |
| 2020s | WSD / WSDS | 大模型训练中的新型调度策略 |

## 3. 核心概念

- **学习率** $\eta$：每次参数更新的步长
- **预热（Warmup）**：训练初期从一个极小学习率线性增大到目标值
- **退火（Annealing）**：训练后期逐渐减小学习率
- **热重启（Warm Restart）**：周期性地将学习率重置为较大值
- **超级收敛（Super-convergence）**：通过特定调度策略在远少于常规训练的 epoch 内达到更高精度

## 4. 技术原理

### 4.1 常数学习率

最简单的策略，全程使用固定学习率 $\eta_t = \eta_0$。适用于短训练或实验调试，但训练后期容易在最优点附近震荡，无法收敛到更优解。

### 4.2 阶梯衰减（Step Decay）

每隔固定 epoch 将学习率乘以衰减因子：

$$\eta_t = \eta_0 \cdot \gamma^{\lfloor t / S \rfloor}$$

典型设置：$\gamma = 0.1$，$S = 30$（每 30 epoch 衰减 10 倍）。经典 CNN 训练（如 ResNet）常用此策略。

**直觉**：训练初期用大学习率快速接近最优点，在验证损失停滞时减小学习率做精细调整。

### 4.3 指数衰减

$$\eta_t = \eta_0 \cdot e^{-\lambda t}$$

或连续版本 $\eta_t = \eta_0 \cdot \gamma^t$。衰减更平滑，但需要调 $\lambda$。

### 4.4 余弦退火（Cosine Annealing）

$$\eta_t = \eta_{\min} + \frac{1}{2}(\eta_{\max} - \eta_{\min})\left(1 + \cos\left(\frac{t\pi}{T}\right)\right)$$

学习率从 $\eta_{\max}$ 按余弦曲线平滑下降到 $\eta_{\min}$。

**优点**：
- 平滑过渡，无突变
- 训练后期缓慢减小，有利于精细调优
- 是当前 [[00_大语言模型核心架构_综述|大语言模型]] 训练的标准调度策略

**变体——半周期余弦**：仅使用余弦的前半周期（从 $\cos(0)=1$ 到 $\cos(\pi)=-1$），实现单调递减。

### 4.5 预热（Warmup）

训练前 $T_{\text{warmup}}$ 步内将学习率从 0（或极小值）线性增大到目标值：

$$\eta_t = \eta_{\max} \cdot \frac{t}{T_{\text{warmup}}}, \quad t \leq T_{\text{warmup}}$$

预热结束后切换到其他调度策略（如余弦退火）。

**为什么需要预热**：
- 训练初期参数随机初始化，梯度方向噪声大
- Adam 等优化器的二阶矩估计在初期不准确
- 大 batch 训练中初期梯度方差更大
- Transformer 中 LayerNorm + 残差连接在初始化阶段可能不稳定

**典型设置**：LLM 预训练中 warmup 通常占总训练步数的 0.1%-2%。

### 4.6 余弦退火 + 预热（Warmup + Cosine Annealing）

[[00_大语言模型核心架构_综述|大语言模型]] 训练的标准配置：

```
学习率
  ↑
  │     ╭─── η_max
  │    ╱  ╲
  │   ╱    ╲
  │  ╱      ╲
  │ ╱        ╲
  │╱          ╲─── η_min
  └──────────────────→ 训练步数
   warmup   cosine
```

- 预热阶段：线性增大 $\eta$ 从 $0$ 到 $\eta_{\max}$
- 退火阶段：余弦衰减 $\eta$ 从 $\eta_{\max}$ 到 $\eta_{\min}$

**典型参数**（GPT-3 级别）：$\eta_{\max} = 6 \times 10^{-4}$，warmup = 2000 步，$\eta_{\min} = 0.1 \eta_{\max}$。

### 4.7 热重启（SGDR / Warm Restarts）

$$\eta_t = \eta_{\min} + \frac{1}{2}(\eta_{\max} - \eta_{\min})\left(1 + \cos\left(\frac{(t \bmod T)\pi}{T}\right)\right)$$

每 $T$ 步将学习率重置为 $\eta_{\max}$，然后重新余弦退火。

**直觉**：周期性"跳出"当前局部最优，探索新的损失谷地。类似模拟退火中的温度重启。

**优点**：提升泛化性能、可结合集成（不同周期的模型快照形成集成）。

### 4.8 One-Cycle 策略

Smith & Topin (2017) 提出的单周期策略：

1. **升温阶段**：学习率从 $\eta_{\min}$ 线性增大到 $\eta_{\max}$
2. **降温阶段**：学习率从 $\eta_{\max}$ 余弦衰减到 $\eta_{\min}$
3. **最后微调**：学习率进一步降至 $\eta_{\text{final}} \ll \eta_{\min}$

同时配合动量反向变化（学习率增大时动量减小，反之亦然）。

**超级收敛**：One-Cycle 在远少于常规训练的 epoch 内达到更高精度，使用比常规更大的学习率。

### 4.9 WSD 调度（Warmup-Stable-Decay）

大模型训练中出现的新型调度策略：

1. **Warmup**：线性预热
2. **Stable**：保持常数学习率（大部分训练时间）
3. **Decay**：最后阶段衰减（余弦或线性）

**优点**：稳定阶段允许灵活调整训练长度（可中途延长），比余弦退火更适合"训练到中途决定是否继续"的场景。

### 4.10 多阶段调度

复杂训练流程中的分段调度：

- **预训练**：Warmup + Cosine Annealing
- **微调**：较小的常数学习率 + 线性衰减
- **指令微调/RLHF**：更小的学习率，更短的预热

## 5. 学习率选择方法

### 学习率范围测试（LR Range Test）

Smith (2015) 提出的经验方法：

1. 从极小学习率（如 $10^{-7}$）开始
2. 每个 batch 指数增大学习率
3. 绘制 Loss-LearningRate 曲线
4. 选择 Loss 开始快速下降的学习率作为初始值

**经验法则**：选择 Loss 下降最快点（最陡斜率）的学习率，或 Loss 最低点前 1-2 个数量级。

### 超参数搜索

- **网格搜索**：在预设网格上穷举，计算量大
- **随机搜索**（Bergstra & Bengio, 2012）：比网格搜索高效
- **贝叶斯优化**：Optuna、Hyperopt，用高斯过程/TPE 建建模
- **Population Based Training（PBT）**：Ray Tune 中实现的进化搜索

## 6. 优势与局限

### 优势

- 合理的调度策略可显著提升模型性能和训练效率
- 预热有效解决大 batch 训练初期的不稳定
- 余弦退火平滑过渡，无需选择衰减时机
- 热重启提升泛化并支持快照集成

### 局限

- 调度策略本身也有超参数（warmup 步数、$\eta_{\min}$ 比例等）
- 余弦退火需要预先知道总训练步数，灵活性受限
- 不同任务/架构的最优策略可能不同
- 过度依赖经验，缺乏理论指导

## 7. 应用场景

- **[[00_大语言模型核心架构_综述|大语言模型]] 预训练**：Warmup + Cosine Annealing 为标准配置
- **[[00_深度学习基础_综述|深度学习]] 分类任务**：Step Decay 或 Cosine Annealing
- **快速实验**：One-Cycle 实现超级收敛
- **微调**：较小的常数学习率 + 线性衰减
- **超参数优化**：LR Range Test 快速确定学习率范围

## 8. 与其他技术关系

- 学习率调度直接控制 [[02_一阶优化方法]] 和 [[03_自适应学习率方法]] 的步长
- 预热是 [[07_大模型训练优化]] 中大 batch 训练的必要组件
- One-Cycle 配合动量调整，与 [[02_一阶优化方法]] 中的动量法紧密相关
- 梯度裁剪（[[06_正则化与约束优化]]）与学习率调度互补，共同控制训练稳定性
- 分布式训练中，不同节点的学习率调度需同步（见 [[08_分布式优化与并行训练]]）

## 9. 前沿发展

- **Schedule-Free 优化器**（Defazio et al., 2024）：无需预设调度策略的自适应方法
- **WSD/WSDS**：大模型训练中可变长度训练的新型调度
- **自动调度搜索**：用元学习/RL 自动选择最优调度
- **学习率与 Scaling Laws 的关系**：Chinchilla 等工作揭示了学习率随模型规模的变化规律
- **Min LR 比例的经验研究**：不同模型规模下 $\eta_{\min}/\eta_{\max}$ 最优比例的实证分析

# 序列标注模型

## 1. 概述

序列标注（Sequence Labeling）是 NLP 中的基础任务类型——为输入序列中的每个元素（通常是词或字符）分配一个标签。分词、词性标注、命名实体识别（NER）、分块（Chunking）等核心 NLP 任务都可以归结为序列标注问题。

序列标注模型的发展代表了 NLP 方法论的核心演进：从基于规则到统计模型（HMM），再到判别式模型（CRF），最后到神经网络（BiLSTM-CRF、BERT）。这一演进路径也映射了整个 NLP 领域从规则到统计到深度学习的范式变迁。理解序列标注模型的原理，是理解 NLP 方法论的基础。

## 3. 核心概念

### 3.1 序列标注的形式化

给定输入序列 $\mathbf{x} = (x_1, x_2, \ldots, x_T)$，寻找最优标签序列 $\mathbf{y} = (y_1, y_2, \ldots, y_T)$：

$$\hat{\mathbf{y}} = \arg\max_{\mathbf{y}} P(\mathbf{y} | \mathbf{x})$$

关键区别在于如何建模 $P(\mathbf{y} | \mathbf{x})$。

### 3.2 生成式 vs 判别式

| 维度 | 生成式（HMM） | 判别式（CRF） |
|:---|:---|:---|
| 建模目标 | $P(\mathbf{x}, \mathbf{y})$ | $P(\mathbf{y}|\mathbf{x})$ |
| 利用特征 | 仅词本身 | 任意上下文特征 |
| 独立性假设 | 强（观测条件独立） | 弱（无观测独立性假设） |
| 推理 | Viterbi | Viterbi（类似） |
| 训练 | 最大似然 | 最大条件似然 |

### 3.3 标签方案

序列标注需要将实体级标注转化为词级标签：

**BIO 方案**：
```
张 三 在 北 京 工 作
B-PER I-PER O B-LOC I-LOC O O
```

**BIOES 方案**（更精细）：
```
张 三 在 北 京 工 作
S-PER  O  S-LOC     O O
```

## 4. 技术原理

### 4.1 隐马尔可夫模型（HMM）

HMM 将标签序列视为隐状态，词序列视为观测：

$$P(\mathbf{x}, \mathbf{y}) = \prod_{t=1}^{T} \underbrace{P(y_t | y_{t-1})}_{\text{转移概率}} \cdot \underbrace{P(x_t | y_t)}_{\text{发射概率}}$$

**两个假设**：

1. **一阶马尔可夫假设**：当前状态只依赖前一状态 $P(y_t | y_{1:t-1}) = P(y_t | y_{t-1})$
2. **观测独立假设**：当前观测只依赖当前状态 $P(x_t | y_{1:t}, x_{1:t-1}) = P(x_t | y_t)$

**Viterbi 解码**（$O(T \cdot K^2)$，$K$ 为标签数）：

$$\delta_t(j) = \max_{i} [\delta_{t-1}(i) \cdot P(y_t=j | y_{t-1}=i)] \cdot P(x_t | y_t=j)$$

### 4.2 最大熵马尔可夫模型（MEMM）

MEMM 是判别式模型，建模 $P(y_t | y_{t-1}, x_1, \ldots, x_T)$：

$$P(\mathbf{y} | \mathbf{x}) = \prod_{t=1}^{T} P(y_t | y_{t-1}, \mathbf{x})$$

**标注偏置问题（Label Bias）**：MEMM 在每个位置做局部归一化，导致低熵状态（转移少的状态）的路径被偏好。

### 4.3 条件随机场（CRF）

CRF 通过全局归一化解决了 MEMM 的标注偏置问题。

**线性链 CRF**：

$$P(\mathbf{y} | \mathbf{x}) = \frac{1}{Z(\mathbf{x})} \exp\left(\sum_{t=1}^{T} \left[\sum_k \lambda_k f_k(y_t, \mathbf{x}, t) + \sum_l \mu_l g_l(y_{t-1}, y_t, \mathbf{x}, t)\right]\right)$$

- **状态特征** $f_k(y_t, \mathbf{x}, t)$：位置 $t$ 的标签与输入的特征函数
- **转移特征** $g_l(y_{t-1}, y_t, \mathbf{x}, t)$：相邻标签之间的转移特征
- **配分函数** $Z(\mathbf{x}) = \sum_{\mathbf{y}} \exp(\ldots)$：全局归一化

**训练**：最大化对数似然，使用 L-BFGS 优化。梯度计算需要前向-后向算法。

**解码**：Viterbi 算法（与 HMM 类似，但用特征分数替代概率）。

### 4.4 BiLSTM-CRF

BiLSTM-CRF（Huang et al., 2015）将深度学习与 CRF 结合：

```
词嵌入 → 字符级CNN → BiLSTM（前向+后向）→ 拼接 → 线性层 → CRF → 标签序列
```

- **BiLSTM**：自动学习丰富的上下文特征表示，替代人工特征工程
- **CRF**：建模标签间的转移约束（如 B-PER 后不能接 I-LOC）

**为什么仍需要 CRF？** BiLSTM 独立预测每个位置的标签，缺乏序列级的约束。CRF 层通过学习转移矩阵，确保标签序列的全局合法性。

**损失函数**：

$$L = -\log P(\mathbf{y}^* | \mathbf{x}) = -\left[\text{score}(\mathbf{x}, \mathbf{y}^*) - \log Z(\mathbf{x})\right]$$

### 4.5 BERT-CRF

用预训练的 BERT 替换 BiLSTM 作为编码器：

```
子词序列 → BERT → 隐藏层 → 线性投影 → CRF → 标签序列
```

BERT 的双向自注意力提供了比 BiLSTM 更强的上下文建模能力，预训练知识也大幅提升标注精度。

## 5. 关键方法对比

| 方法 | 建模方式 | 特征来源 | NER F1 | 优势 | 局限 |
|:---|:---|:---|:---|:---|:---|
| HMM | 生成式 | 统计概率 | ~85% | 简单、速度快 | 独立假设过强 |
| MEMM | 判别式（局部） | 人工特征 | ~87% | 利用上下文特征 | 标注偏置 |
| CRF | 判别式（全局） | 人工特征 | ~89% | 全局最优、特征灵活 | 特征工程依赖人工 |
| BiLSTM-CRF | 神经网络 | 自动学习 | ~90% | 端到端、无特征工程 | 需要标注数据 |
| BERT-CRF | 预训练+微调 | 预训练迁移 | ~92% | SOTA、少样本好 | 计算开销大 |

### CRF vs BiLSTM-CRF：何时仍需 CRF？

即使有强大的 BiLSTM/BERT，CRF 层仍有价值：

1. **标签约束**：CRF 学习标签转移规则（如 I-PER 不能出现在 O 之后）
2. **序列级优化**：CRF 优化整个序列的联合概率，而非逐位置独立
3. **长距离标签依赖**：CRF 转移矩阵隐式建模标签间的长程约束

## 6. 优势与局限

### 优势

1. **通用性**：一个框架解决分词、POS、NER 等多种任务
2. **理论成熟**：HMM/CRF 有完善的概率论基础和高效的训练/解码算法
3. **可解释**：CRF 的特征权重和转移矩阵可分析

### 局限

1. **标注成本**：需要大量人工标注数据
2. **标签方案限制**：BIO 等方案难以处理嵌套实体
3. **LLM 冲击**：大模型在零样本/少样本场景下减少了对专门标注模型的需求

## 7. 应用场景

- **中文分词**：B/M/E/S 标注
- **词性标注**：每词标注词性
- **命名实体识别**：BIO/BIOES 标注实体边界和类型
- **分块（Chunking）**：识别名词短语、动词短语等
- **分句**：标注句子边界

## 9. 前沿发展

- **Span-based 方法**：直接枚举 span 并分类，避免标签方案的限制
- **Prompt-based 标注**：将序列标注转化为生成任务，利用 LLM 的零样本能力
- **统一信息抽取**：一个模型处理 NER、RE、Event 等多种抽取任务
- **低资源标注**：结合主动学习和远程监督减少标注需求
- **多任务学习**：联合训练分词、POS、NER 等相关任务

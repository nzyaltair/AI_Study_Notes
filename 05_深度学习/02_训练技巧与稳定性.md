# 训练技巧与稳定性

## 背景与发展

[[深度学习]]模型训练是一个复杂的优化过程，面临梯度消失/爆炸、过拟合、训练不稳定等挑战。训练技巧随模型规模增长不断演进：从单卡训练技巧（初始化、BatchNorm、Dropout）发展到千卡分布式训练体系（ZeRO、张量并行）。掌握训练技巧是从能跑通模型到训练出高质量模型的关键工程能力。

## 核心思想

通过归一化、正则化、学习率调度、梯度管理等技术，使[[神经网络]]训练更稳定、更高效、泛化性能更好。这些技术本质上是在控制训练过程中的梯度流和参数更新幅度。

## 技术原理

### 归一化技术

| 方法 | 原理 | 适用场景 |
|------|------|---------|
| **BatchNorm** | 对每个 mini-batch 的每个特征归一化 | CNN、MLP |
| **LayerNorm** | 对每个样本的所有特征归一化 | [[序列模型]]、[[Transformer]]、LLM |
| **RMSNorm** | LayerNorm 简化版，只做缩放不含中心化 | 现代 LLM（Llama 系列） |
| **GroupNorm** | 将通道分组归一化 | 小 batch 场景 |

**LayerNorm 公式**：$\mathbf{y} = \gamma \cdot \frac{\mathbf{x} - \mu}{\sqrt{\sigma^2 + \epsilon}} + \beta$

### 正则化方法

| 方法 | 原理 | 应用 |
|------|------|------|
| **Dropout** | 训练时随机丢弃神经元 | MLP、CNN |
| **DropPath** | 随机丢弃整个残差分支 | [[Transformer]]、ViT |
| **Weight Decay** | L2 正则化，限制权重大小 | 通用 |
| **Label Smoothing** | 软化标签，防止过自信 | 分类任务 |
| **数据增强** | 增加数据多样性 | CV（翻转/裁剪）、NLP（回译） |

### 梯度管理

- **梯度裁剪**（Gradient Clipping）：限制梯度范数 $\|\nabla L\| \leq \text{max\_norm}$，防止梯度爆炸，LLM 训练标配
- **梯度累积**（Gradient Accumulation）：小 batch 模拟大 batch，$\text{有效batch} = \text{micro\_batch} \times \text{accumulation\_steps}$
- **梯度检查点**（Gradient Checkpointing）：以时间换空间，减少显存占用

### 学习率调度

| 策略 | 公式/描述 | 适用场景 |
|------|---------|---------|
| **Step Decay** | 每 N 步乘以 $\gamma$ | 通用 |
| **Cosine Annealing** | $\eta_t = \eta_{min} + \frac{1}{2}(\eta_{max} - \eta_{min})(1 + \cos(\frac{t\pi}{T}))$ | CV、NLP |
| **Warmup** | 前 N 步线性增大 | LLM 训练标配 |
| **Warmup + Cosine** | Warmup 后余弦退火 | LLM 最常用 |

### 混合精度训练

- 使用 FP16/BF16 减少显存、加速训练
- FP16 需要 Loss Scaling 防止梯度下溢
- BF16 指数位与 FP32 相同，无需 Loss Scaling
- 现代 GPU（A100/H100）原生支持 BF16

### 分布式训练

| 策略 | 原理 | 适用场景 |
|------|------|---------|
| **数据并行** | 每张卡完整模型 + 部分数据 | 模型放得下单卡 |
| **模型并行** | 模型拆分到多卡 | 大模型 |
| **流水线并行** | 按层切分，流水执行 | 超大模型 |
| **ZeRO** | 优化器状态/梯度/参数分片 | LLM 训练 |
| **张量并行** | 矩阵运算切分到多卡 | LLM（Megatron-LM） |

## 发展演进

随机初始化 $\to$ Xavier/He 初始化 $\to$ BatchNorm（2015）$\to$ LayerNorm $\to$ Dropout $\to$ 混合精度 $\to$ 分布式训练（ZeRO/张量并行）。训练技巧从单卡优化发展到千卡分布式训练体系。

## 关键算法·模型

- **归一化**：BatchNorm、LayerNorm、RMSNorm
- **正则化**：Dropout、DropPath、Weight Decay、Label Smoothing
- **学习率调度**：Warmup + Cosine Annealing
- **分布式**：ZeRO、张量并行、流水线并行

## 应用场景

- **[[卷积神经网络]]训练**：BatchNorm、数据增强、迁移学习
- **[[序列模型]]训练**：梯度裁剪、LayerNorm
- **LLM 训练**：BF16 混合精度、ZeRO 分布式、Warmup+Cosine、梯度裁剪

## 与其他技术关系

- 归一化技术是[[深度学习基础|神经网络]]训练稳定性的核心
- 分布式训练是大模型训练的工程基础
- 实验管理与复现是训练工程化的延伸，详见[[实验管理与复现]]
- 模型压缩（剪枝/量化/蒸馏）与训练优化相关

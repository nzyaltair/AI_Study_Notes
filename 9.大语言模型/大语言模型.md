# 大语言模型

> 本文件为大语言模型主题的索引文件，整合核心架构、训练对齐、推理优化和评测基准。

## 概述

大语言模型（LLM）是基于[[Transformer架构详解|Transformer]]的超大规模[[预训练语言模型|预训练语言模型]]，通过在海量文本上自监督预训练获得通用语言能力，再通过指令微调和对齐技术成为实用的AI助手。GPT-3（2020）证明了"Scale is all you need"，ChatGPT（2022）展示了RLHF的威力，开启了LLM时代。

## 训练流程全景

```
数据收集与清洗 → 预训练（Next Token Prediction）
  → 监督微调（SFT，指令跟随）
  → 偏好对齐（RLHF/DPO，人类偏好对齐）
  → 安全对齐（Red Teaming，价值对齐）
  → 部署与推理优化
```

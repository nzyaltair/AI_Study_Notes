# 训练技巧与稳定性

## 背景

深度学习模型训练是一个复杂的优化过程，面临梯度消失/爆炸、过拟合、训练不稳定等挑战。掌握训练技巧是从能跑通模型到训练出高质量模型的关键工程能力，也是[[大语言模型|LLM]]训练的基础。

## 核心思想

通过归一化、正则化、学习率调度、梯度管理等技术，使深度网络训练更稳定、更高效、泛化性能更好。

## 技术原理

### 归一化技术

| 方法 | 原理 | 适用场景 |
|------|------|---------|
| **BatchNorm** | 对每个mini-batch的每个特征归一化 | CNN、MLP |
| **LayerNorm** | 对每个样本的所有特征归一化 | [[序列模型|RNN]]、[[Transformer与预训练|Transformer]]、[[大语言模型|LLM]] |
| **RMSNorm** | LayerNorm简化版，只做缩放 | 现代LLM（Llama系列） |
| **GroupNorm** | 将通道分组归一化 | 小batch场景 |

**LayerNorm公式**：$\mathbf{y} = \gamma \cdot \frac{\mathbf{x} - \mu}{\sqrt{\sigma^2 + \epsilon}} + \beta$

### 正则化方法

| 方法 | 原理 | 应用 |
|------|------|------|
| **Dropout** | 训练时随机丢弃神经元 | MLP、CNN |
| **DropPath** | 随机丢弃整个残差分支 | Transformer、ViT |
| **Weight Decay** | L2正则化，限制权重大小 | 通用 |
| **Label Smoothing** | 软化标签，防止过自信 | 分类任务 |
| **数据增强** | 增加数据多样性 | CV（翻转/裁剪）、NLP（回译） |

### 梯度管理

- **梯度裁剪**（Gradient Clipping）：限制梯度范数 $\|\nabla L\| \leq \text{max\_norm}$
  - 防止梯度爆炸，[[大语言模型|LLM]]训练标配
- **梯度累积**（Gradient Accumulation）：小batch模拟大batch
  - $\text{有效batch} = \text{micro\_batch} \times \text{accumulation\_steps}$
- **梯度检查点**（Gradient Checkpointing）：以时间换空间，减少显存

### 学习率调度

| 策略 | 公式/描述 | 适用场景 |
|------|---------|---------|
| **Step Decay** | 每N步乘以γ | 通用 |
| **Cosine Annealing** | $\eta_t = \eta_{min} + \frac{1}{2}(\eta_{max} - \eta_{min})(1 + \cos(\frac{t\pi}{T}))$ | CV、NLP |
| **Warmup** | 前N步线性增长 | [[大语言模型|LLM]]训练标配 |
| **Warmup + Cosine** | Warmup后余弦退火 | LLM最常用 |
| **OneCycle** | 先增后减 | 快速训练 |

### 优化器选择

| 优化器 | 特点 | 适用场景 |
|--------|------|---------|
| **SGD + Momentum** | 稳定但需调参 | CV（ResNet系列） |
| **Adam** | 自适应学习率，快速收敛 | 通用 |
| **AdamW** | Adam + 解耦权重衰减 | [[大语言模型|LLM]]标配 |
| **Lion** | 符号动量，内存高效 | 前沿研究 |

### 混合精度训练

- 使用FP16/BF16减少显存、加速训练
- FP16需要Loss Scaling防止梯度下溢
- BF16（Brain Float16）指数位与FP32相同，无需Loss Scaling
- 现代GPU（A100/H100）原生支持BF16

### 分布式训练

| 策略 | 原理 | 适用场景 |
|------|------|---------|
| **数据并行** | 每张卡完整模型+部分数据 | 模型放得下单卡 |
| **模型并行** | 模型拆分到多卡 | 大模型 |
| **流水线并行** | 按层切分，流水执行 | 超大模型 |
| **ZeRO** | 优化器状态/梯度/参数分片 | [[大语言模型|LLM]]训练 |
| **张量并行** | 矩阵运算切分到多卡 | LLM（Megatron-LM） |

## 发展演进

随机初始化 → Xavier/He初始化 → BatchNorm（2015）→ LayerNorm → Dropout → 混合精度 → 分布式训练（ZeRO/张量并行）

训练技巧随模型规模增长不断演进，从单卡训练技巧发展到千卡分布式训练体系。

## 应用领域

- **[[卷积神经网络|CNN]]训练**：BatchNorm、数据增强、迁移学习
- **[[序列模型|RNN]]训练**：梯度裁剪、LayerNorm
- **[[大语言模型|LLM]]训练**：BF16混合精度、ZeRO分布式、Warmup+Cosine、梯度裁剪

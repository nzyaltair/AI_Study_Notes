# 卷积神经网络（CNN）综述

## 1. 概述

卷积神经网络（Convolutional Neural Network, CNN）是一类专门为处理**网格结构数据**（如图像、视频、音频频谱）而设计的深度学习架构。其核心思想源于生物视觉系统的层级处理机制——通过**局部感受野**（Local Receptive Field）与**权值共享**（Weight Sharing）两大设计原则，CNN能以远少于全连接网络的参数量，高效提取空间层次化特征。

CNN的核心贡献在于：将传统图像处理的**手工特征设计**（SIFT、HOG等）替换为**端到端的可学习特征提取**，从原始像素直接学习到任务所需的语义表征，推动了计算机视觉乃至整个人工智能领域的范式转变。

## 2. 发展历史

| 时期 | 里程碑 | 核心意义 |
|------|--------|----------|
| 1959-1962 | Hubel & Wiesel 猫视觉皮层实验 | 发现简单细胞/复杂细胞的层级感受野机制，为CNN提供生物学灵感 |
| 1980 | 福岛邦彦 Neocognitron | 首个模拟视觉皮层层级处理的神经网络 |
| 1989 | LeCun 提出反向传播训练CNN | 首次将反向传播应用于卷积网络，用于手写邮政编码识别 |
| 1998 | LeNet-5 | 首个实用CNN，成功应用于银行支票手写数字识别 |
| 2012 | AlexNet | ImageNet竞赛中错误率骤降~10%，点燃深度学习革命 |
| 2014 | VGGNet / GoogLeNet | 深度堆叠 vs. 多尺度并行，奠定深层CNN设计范式 |
| 2015 | ResNet | 残差连接使训练152层网络成为可能，准确率首次超越人类 |
| 2017 | DenseNet / MobileNet | 密集特征复用与移动端高效推理两条路径并行发展 |
| 2019 | EfficientNet | 系统化的网络宽度/深度/分辨率复合缩放策略 |
| 2020 | Vision Transformer (ViT) | Transformer首次在图像分类上超越CNN |
| 2022 | ConvNeXt | CNN架构设计"现代化"，证明纯CNN仍可与ViT竞争 |

**关键转折点**：2012年AlexNet的成功标志着深度学习时代的开端，其关键推动因素包括大规模标注数据集（ImageNet）、GPU并行计算能力、以及ReLU/Dropout等训练技巧的成熟。

## 3. 核心概念

### 3.1 基本术语

| 术语 | 定义 |
|------|------|
| **卷积核（Kernel/Filter）** | 可学习的权重矩阵，在输入上滑动执行局部特征提取 |
| **特征图（Feature Map）** | 卷积运算的输出，每个通道对应一个卷积核的响应 |
| **步长（Stride）** | 卷积核每次滑动的像素距离 |
| **填充（Padding）** | 在输入边界补零以控制输出尺寸 |
| **感受野（Receptive Field）** | 特征图上某点对应输入图像的区域大小 |
| **通道（Channel）** | 特征的深度维度（RGB图像为3通道，中间层可为数百通道） |

### 3.2 核心设计原则

- **局部连接**：每个神经元只连接到输入的局部区域，而非全连接，极大减少参数量
- **权值共享**：同一卷积核在整个输入上共享权重，保证平移等变性
- **层次化表征**：浅层学习边缘/纹理，中层学习形状/部件，深层学习语义/物体

### 3.3 解决的问题

CNN主要解决计算机视觉中需要**特征学习**与**模式识别**的任务，包括：图像分类、目标检测、语义分割、实例分割、姿态估计、图像生成等。

## 4. 技术原理

### 4.1 卷积运算本质

CNN中使用的操作严格来说是**互相关**（Cross-correlation）而非数学上的卷积，但遵循惯例统称"卷积"。对于输入 $I$ 和卷积核 $K$：

$$O(i,j) = \sum_m \sum_n I(i+m, j+n) \cdot K(m,n)$$

输出尺寸：$O = \left\lfloor \frac{I - K + 2P}{S} \right\rfloor + 1$

### 4.2 网络整体结构

典型CNN由三种基本层交替堆叠而成：

```
输入 → [[卷积层 → 激活函数 → 池化层] × N] → 全连接层 → 输出
```

- **卷积层**：特征提取
- **激活函数**：引入非线性（ReLU为主流）
- **池化层**：降采样，增强平移不变性
- **全连接层**：汇总全局信息，输出分类/回归结果
- **BatchNorm**：批归一化，加速收敛、稳定训练

## 6. 优势与局限

### 优势
- **参数效率高**：权值共享大幅减少参数量，降低过拟合风险
- **平移等变性**：同一特征在图像不同位置产生相同响应
- **层次化表征**：自动学习从低级到高级的特征抽象
- **可迁移性强**：预训练特征在多种视觉任务中泛化良好

### 局限
- **缺乏全局建模能力**：卷积核感受野有限，长距离依赖需要深层堆叠
- **对旋转/尺度敏感**：不天然具备旋转等变性（需数据增强补偿）
- **静态权重推理**：卷积核参数固定，无法像注意力机制根据输入动态调整
- **计算效率挑战**：大卷积核或深层网络的计算量和内存消耗仍然可观

## 9. 前沿发展

1. **CNN与Transformer融合**：Swin Transformer、CoAtNet等混合架构取两者之长
2. **ConvNeXt与CNN现代化**：借鉴Transformer设计哲学（LayerNorm、GELU、大卷积核），改造纯CNN使其与ViT匹敌
3. **自监督预训练**：MAE（Masked Autoencoder）、SimCLR等方法使CNN/ViT在海量无标注数据上预训练
4. **高效推理**：模型剪枝、量化、知识蒸馏让CNN在边缘设备持续发挥作用
5. **3D视觉**：3D卷积用于点云、体素、多视图的表征学习
6. **开放世界视觉**：从闭集分类到开放词汇检测/分割（如Grounding DINO、SAM）

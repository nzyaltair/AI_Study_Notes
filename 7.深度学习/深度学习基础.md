# 深度学习基础

## 背景

深度学习是[[机器学习]]的子集，通过多层神经网络自动学习数据的层次化特征表示。它克服了[[特征工程|人工特征工程]]的瓶颈，在图像、语音、自然语言等领域取得了革命性突破。2012年AlexNet在ImageNet上的成功标志着深度学习时代的开启。

## 核心思想

深度学习通过多层非线性变换，将原始数据逐步转化为更高层次的抽象表示。每一层学习到的特征都是对前一层特征的组合，形成从低级到高级的特征层次结构。

## 技术原理

### 神经元模型

单个神经元是深度学习的基本单元：

$$y = f\left(\sum_{i=1}^n w_i x_i + b\right) = f(\mathbf{w}^T\mathbf{x} + b)$$

其中 $f$ 为激活函数，$\mathbf{w}$ 为权重，$b$ 为偏置。

### 前向传播与反向传播

- **前向传播**：输入数据逐层通过网络的计算过程
  - $\mathbf{a}^{(l)} = f(\mathbf{W}^{(l)}\mathbf{a}^{(l-1)} + \mathbf{b}^{(l)})$
- **反向传播**：利用链式法则从输出层向输入层计算梯度
  - $\frac{\partial L}{\partial \mathbf{W}^{(l)}} = \frac{\partial L}{\partial \mathbf{a}^{(l)}} \cdot \frac{\partial \mathbf{a}^{(l)}}{\partial \mathbf{W}^{(l)}}$
- **计算图与自动微分**：现代框架（PyTorch/TensorFlow）自动构建计算图并求导

### 激活函数

| 激活函数 | 公式 | 特点 |
|---------|------|------|
| **Sigmoid** | $\frac{1}{1+e^{-x}}$ | 输出(0,1)，梯度消失 |
| **Tanh** | $\frac{e^x-e^{-x}}{e^x+e^{-x}}$ | 输出(-1,1)，零中心 |
| **ReLU** | $\max(0, x)$ | 计算快，缓解梯度消失，但有死亡ReLU问题 |
| **Leaky ReLU** | $\max(0.01x, x)$ | 解决死亡ReLU |
| **GELU** | $x \cdot \Phi(x)$ | [[大语言模型|LLM]]中常用 |
| **SwiGLU** | $\text{Swish}(xW_1) \otimes (xW_2)$ | 现代LLM标配 |

### 损失函数

| 任务类型 | 损失函数 | 公式 |
|---------|---------|------|
| 回归 | MSE | $\frac{1}{n}\sum(y_i - \hat{y}_i)^2$ |
| 二分类 | 交叉熵 | $-\sum[y_i\log\hat{y}_i + (1-y_i)\log(1-\hat{y}_i)]$ |
| 多分类 | Softmax交叉熵 | $-\sum y_i\log\frac{e^{z_i}}{\sum_j e^{z_j}}$ |
| 序列 | CTC | 变长序列对齐 |
| 对比学习 | InfoNCE | 拉近正样本、推远负样本 |

### 优化算法

- **SGD**：$\theta \leftarrow \theta - \eta \nabla L$
- **Momentum**：$\mathbf{v} \leftarrow \beta\mathbf{v} + \nabla L$，$\theta \leftarrow \theta - \eta\mathbf{v}$
- **Adam**：结合动量与自适应学习率
  - $\mathbf{m} = \beta_1\mathbf{m} + (1-\beta_1)\nabla L$（一阶矩）
  - $\mathbf{v} = \beta_2\mathbf{v} + (1-\beta_2)(\nabla L)^2$（二阶矩）
  - $\theta \leftarrow \theta - \eta \frac{\hat{\mathbf{m}}}{\sqrt{\hat{\mathbf{v}}}+\epsilon}$
- **AdamW**：Adam + 解耦权重衰减（[[大语言模型|LLM]]训练标配）

### 参数初始化

- **Xavier初始化**：$\text{Var}(W) = \frac{2}{n_{in}+n_{out}}$（适合tanh）
- **He初始化**：$\text{Var}(W) = \frac{2}{n_{in}}$（适合ReLU）
- 良好的初始化防止梯度消失/爆炸，加速收敛

## 发展演进

感知机（1958）→ 多层感知机（1980s）→ 反向传播（1986）→ 深度信念网络（2006）→ AlexNet突破（2012）→ [[卷积神经网络|CNN]]/[[序列模型|RNN]] → [[Transformer与预训练|Transformer]]（2017）→ [[大语言模型|LLM]]

## 关键架构

- **前馈神经网络（MLP）**：全连接层堆叠，适合表格数据
- **[[卷积神经网络|卷积神经网络]]**：局部连接+权值共享，适合图像
- **[[序列模型|循环神经网络]]**：时序依赖建模，适合文本/语音
- **Transformer**：自注意力机制，并行处理序列（详见[[Transformer与预训练]]）

## 应用领域

- **计算机视觉**：图像分类、目标检测、图像分割
- **自然语言处理**：文本分类、机器翻译、问答系统
- **语音处理**：语音识别、语音合成
- **推荐系统**：深度推荐模型
- **科学计算**：蛋白质结构预测（AlphaFold）

- 深度学习是[[机器学习]]的子集，但通过端到端学习减少了对[[特征工程]]的依赖

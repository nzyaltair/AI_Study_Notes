# 谱域图卷积

## 1. 概述

谱域图卷积（Spectral Graph Convolution）从**图信号处理**出发，在图拉普拉斯矩阵的特征空间中定义卷积操作。它是图神经网络的理论起源——GCN、ChebNet 等经典模型都源自谱域推导，然后逐步简化为高效的空域实现。

谱域方法的核心思路：将图信号变换到频率域（通过图傅里叶变换），在频域上施加可学习的滤波器，再变换回节点域。这提供了严格的数学基础，但直接计算需要特征分解（$O(n^3)$），因此后续工作通过多项式近似大幅降低计算复杂度，最终演化为今天广泛使用的空域 GCN。

## 3. 核心概念

### 3.1 从经典卷积到图卷积

**经典卷积定理**：时域卷积 = 频域逐点乘积

$$(f * g)(t) = \mathcal{F}^{-1}(\hat{f}(\omega) \cdot \hat{g}(\omega))$$

将此定理推广到图上：用图傅里叶变换替代经典傅里叶变换。

### 3.2 图傅里叶变换

基于归一化图拉普拉斯 $L_{sym} = U\Lambda U^T$ 的特征分解：

- **正变换**：$\hat{x} = U^T x$（节点域 → 谱域）
- **逆变换**：$x = U\hat{x}$（谱域 → 节点域）
- 特征向量 $u_1, \dots, u_n$ 构成图上的"频率基"
- 特征值 $\lambda_1 = 0 \leq \lambda_2 \leq \dots \leq \lambda_n$ 对应频率

### 3.3 图频率的物理意义

- **低频分量**（小 $\lambda$）：信号在相邻节点间变化平缓（如社区结构）
- **高频分量**（大 $\lambda$）：信号在相邻节点间变化剧烈（如社区边界）
- GCN 的归一化传播 $\tilde{D}^{-1/2}\tilde{A}\tilde{D}^{-1/2}$ 本质上是一个**低通滤波器**

## 4. 技术原理

### 4.1 Spectral CNN（Bruna et al., 2013）

最原始的谱图卷积定义：

$$x *_{\mathcal{G}} g_\theta = U g_\theta(\Lambda) U^T x$$

其中 $g_\theta(\Lambda) = \text{diag}(\theta_1, \dots, \theta_n)$ 是可学习的对角滤波器。

**多通道扩展**（输入 $C_{in}$ 通道，输出 $C_{out}$ 通道）：

$$H_{:,j}^{(l+1)} = \sigma\left(\sum_{i=1}^{C_{in}} U g_{\theta_{ij}}(\Lambda) U^T H_{:,i}^{(l)}\right), \quad j = 1, \dots, C_{out}$$

**问题**：
- 需要计算 $L$ 的特征分解，复杂度 $O(n^3)$
- 滤波器参数量 $O(n)$，与图大小相关
- 滤波器依赖特定图结构，不可迁移到不同图

### 4.2 ChebNet（Defferrard et al., 2016）

用 **Chebyshev 多项式**近似谱滤波器，避免特征分解：

$$g_\theta(\Lambda) \approx \sum_{k=0}^{K} \theta_k T_k(\tilde{\Lambda})$$

其中 $\tilde{\Lambda} = \frac{2}{\lambda_{max}}\Lambda - I$ 将特征值归一化到 $[-1, 1]$，$T_k$ 是第 $k$ 阶 Chebyshev 多项式（递推：$T_0(x)=1, T_1(x)=x, T_k(x)=2xT_{k-1}(x)-T_{k-2}(x)$）。

**关键性质**：利用 Chebyshev 多项式的递推性，卷积变为：

$$g_\theta * x \approx \sum_{k=0}^{K} \theta_k T_k(\tilde{L}) x$$

其中 $\tilde{L} = \frac{2}{\lambda_{max}}L - I$。

- 不需要特征分解，计算复杂度 $O(K|\mathcal{E}|)$（$K$ 为多项式阶数，$|\mathcal{E}|$ 为边数）
- $K$ 阶多项式对应 $K$ 跳局部性——滤波器只聚合 $K$ 跳邻居
- 参数量 $O(K)$，与图大小无关

### 4.3 GCN（Kipf & Welling, 2017）

GCN 对 ChebNet 做了进一步简化——**一阶近似**：

**简化假设**：
1. $K = 1$（仅一阶邻居）
2. $\lambda_{max} \approx 2$（线性近似）
3. 令 $\theta = \theta_0 = -\theta_1$（共享参数）

推导：

$$g_\theta * x \approx \theta_0 x + \theta_1 (L - I) x = \theta(I + D^{-1/2}AD^{-1/2}) x$$

令 $\tilde{A} = A + I$（加自环），$\tilde{D}_{ii} = \sum_j \tilde{A}_{ij}$，重归一化：

$$g_\theta * x = \theta \tilde{D}^{-1/2}\tilde{A}\tilde{D}^{-1/2} x$$

**逐层传播规则**（加入可学习权重 $W$ 和非线性激活 $\sigma$）：

$$\boxed{H^{(l+1)} = \sigma\left(\tilde{D}^{-\frac{1}{2}} \tilde{A} \tilde{D}^{-\frac{1}{2}} H^{(l)} W^{(l)}\right)}$$

**设计直觉**：
- 加自环 $\tilde{A} = A + I$：保留节点自身信息
- 对称归一化 $\tilde{D}^{-1/2}\tilde{A}\tilde{D}^{-1/2}$：平衡高低度节点的贡献（高度节点不会主导聚合）
- 本质是谱域一阶低通滤波器

### 4.4 APPNP（Klicpera et al., 2019）

APPNP 利用 PageRank 的思想，**解耦特征变换与图传播**：

**PageRank 传播**：

$$Z = \alpha(I - (1-\alpha)\tilde{P})^{-1} X$$

其中 $\tilde{P} = \tilde{D}^{-1/2}\tilde{A}\tilde{D}^{-1/2}$，$\alpha$ 为传送概率。

近似为 $K$ 步迭代：

$$Z^{(k)} = (1-\alpha)\tilde{P} Z^{(k-1)} + \alpha X, \quad Z^{(0)} = X$$

**优势**：可以传播很远（大 $K$）而不过平滑，因为每步都通过 $\alpha X$ 注入原始特征。

### 4.5 谱滤波器的频率响应

各方法的频率响应特性：

| 方法 | 滤波器类型 | 频率响应 |
|:---|:---|:---|
| Spectral CNN | 任意 | 全频段可学习 |
| ChebNet | $K$ 阶多项式 | $K$ 阶多项式近似 |
| GCN | 一阶线性 | 低通滤波器 |
| APPNP | PageRank | 带残差的低通 |

## 5. 关键方法与模型

### 5.1 GCN 完整前向传播

```
输入: 邻接矩阵 A, 节点特征 X, 标签掩码
1. 预处理: Ã = A + I, D̃ = degree(Ã), P = D̃^{-1/2} Ã D̃^{-1/2}
2. H^{(0)} = X
3. For l = 0, 1, ..., L-1:
     H^{(l+1)} = σ(P · H^{(l)} · W^{(l)})
4. 输出: Z = softmax(H^{(L)})
```

### 5.2 GCN 的两种视角

| 视角 | 解释 |
|:---|:---|
| 谱域 | 一阶 Chebyshev 近似的低通滤波器 |
| 空域 | 对称归一化的邻居均值聚合 |

GCN 的优雅之处在于：从谱域推导出发，得到了简洁的空域实现，使两种视角统一。

### 5.3 归一化方式对比

| 归一化 | 公式 | 特点 |
|:---|:---|:---|
| 对称归一化 | $\tilde{D}^{-1/2}\tilde{A}\tilde{D}^{-1/2}$ | GCN 使用，平衡度数 |
| 行归一化 | $\tilde{D}^{-1}\tilde{A}$ | GraphSAGE mean 聚合 |
| 无归一化 | $\tilde{A}$ | GIN sum 聚合 |

## 6. 优势与局限

### 6.1 优势

- **理论严格**：从图信号处理出发，有明确的频域解释
- **局部性**：$K$ 阶多项式保证 $K$ 跳局部性，计算高效
- **统一性**：GCN 同时具有谱域和空域解释
- **参数高效**：GCN 每层仅一个权重矩阵 $W$

### 6.2 局限

- **计算瓶颈**：Spectral CNN 需要特征分解，不可扩展到大图
- **过平滑**：GCN 在 2-3 层后性能开始下降
- **固定权重**：GCN 的邻居权重由度数决定，不可学习
- **谱域依赖**：滤波器理论上依赖图的拉普拉斯特征分解，跨图泛化受限

## 7. 应用场景

- **半监督节点分类**：GCN 的经典应用（Cora, CiteSeer, PubMed）
- **图信号去噪**：利用低通滤波特性去除图上的噪声
- **社交网络分析**：社区检测、用户分类
- **引文网络**：论文主题分类

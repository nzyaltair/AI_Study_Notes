# 开源项目收藏

> 本文件为 AI 开发人员的高价值开源项目索引，聚焦活跃、高质量、具备工程落地价值的项目。

## 概述

本索引收录了 AI 工程实践中最具价值的开源项目，涵盖大模型、多模态、训练框架、推理优化、评估工具等核心方向。所有项目均经过筛选，优先考虑 GitHub Stars ≥ 3k、持续维护（近 6 个月有 commit）、文档完善的项目。

## 大语言模型

### 基础模型实现

- [ ] [[Qwen]] - 通义千问：阿里达摩院开发的大语言模型，支持多语言，具有强代码能力和数学推理能力
  - **官方链接**：[Qwen](https://github.com/QwenLM/Qwen)
  - **核心功能**：支持多种尺寸模型（0.5B-14B），提供完整训练和推理工具链
  - **适用场景**：通用 NLP 任务、代码生成、数学推理、多语言翻译
  - **主要优势**：中文优化、代码能力强、支持长上下文
  - **生态影响力**：工业界广泛采用，社区活跃度高

### 微调框架

- [ ] [[LLaMA-Factory]] - 一站式大模型微调工具，支持多种微调方法和模型架构
  - **官方链接**：[LLaMA-Factory](https://github.com/hiyouga/LLaMA-Factory)
  - **核心功能**：支持 LoRA、QLoRA、全参数微调等多种方法，内置数据处理和评估工具
  - **适用场景**：快速模型适配、领域知识注入、指令微调
  - **主要优势**：低显存需求、多模型支持、配置简单、文档完善
  - **生态影响力**：被广泛用于学术研究和工业界落地

## 高效推理

- [ ] [[vLLM]] - 高性能大语言模型推理引擎，专注于高吞吐和低延迟
  - **官方链接**：[vLLM](https://github.com/vllm-project/vllm)
  - **核心功能**：连续批处理、KV Cache 优化、PagedAttention 技术
  - **适用场景**：大模型服务部署、高并发 API 调用、低延迟推理
  - **主要优势**：单卡高吞吐、易于集成、支持多种模型
  - **生态影响力**：被多家云服务提供商采用，社区活跃度高

- [ ] [[Text Generation Inference (TGI)]] - Hugging Face 官方推理引擎，优化文本生成任务
  - **官方链接**：[TGI](https://github.com/huggingface/text-generation-inference)
  - **核心功能**：模型量化、批处理优化、流式输出
  - **适用场景**：生产级文本生成服务、API 部署
  - **主要优势**：企业级稳定性、与 Hugging Face 生态深度集成
  - **生态影响力**：被广泛应用于商业产品中

## 训练框架

- [ ] [[DeepSpeed]] - Microsoft 开发的深度学习训练优化库，专注于大规模模型训练
  - **官方链接**：[DeepSpeed](https://github.com/microsoft/DeepSpeed)
  - **核心功能**：ZeRO 优化、混合精度训练、流水线并行
  - **适用场景**：超大模型训练、分布式训练加速
  - **主要优势**：显著降低显存使用、提高训练速度、易于集成
  - **生态影响力**：被众多研究机构和企业采用

- [ ] [[Hugging Face Transformers]] - 最流行的 NLP 模型库，提供丰富的预训练模型和工具
  - **官方链接**：[Transformers](https://github.com/huggingface/transformers)
  - **核心功能**：模型加载、微调、推理、评估
  - **适用场景**：快速模型应用、研究实验、生产部署
  - **主要优势**：模型种类丰富、API 友好、社区活跃
  - **生态影响力**：AI 领域最广泛使用的库之一

## 多模态模型

- [ ] [[Diffusers]] - Hugging Face 开发的扩散模型库，专注于生成模型
  - **官方链接**：[Diffusers](https://github.com/huggingface/diffusers)
  - **核心功能**：稳定扩散、ControlNet、Text-to-Image、Image-to-Image
  - **适用场景**：图像生成、编辑、风格转换
  - **主要优势**：易于使用、模型丰富、性能优化
  - **生态影响力**：多模态生成领域标准工具

## Agent 框架

- [ ] [[LangChain]] - 大语言模型应用开发框架，专注于连接模型与外部工具
  - **官方链接**：[LangChain](https://github.com/langchain-ai/langchain)
  - **核心功能**：链式调用、工具集成、记忆管理、文档处理
  - **适用场景**：构建复杂 AI 应用、RAG 系统、Agent 系统
  - **主要优势**：模块化设计、丰富的集成、社区活跃
  - **生态影响力**：Agent 开发的事实标准框架

- [ ] [[LlamaIndex]] - 数据感知型 LLM 应用框架，专注于结构化数据处理
  - **官方链接**：[LlamaIndex](https://github.com/run-llama/llama_index)
  - **核心功能**：数据索引、查询转换、路由选择、评估
  - **适用场景**：知识库问答、数据分析、个性化推荐
  - **主要优势**：数据处理能力强、查询优化、易于扩展
  - **生态影响力**：企业级 RAG 应用的首选框架

## 评估工具

- [ ] [[OpenCompass]] - 开源大模型评测工具链，支持多维度评估
  - **官方链接**：[OpenCompass](https://github.com/open-compass/opencompass)
  - **核心功能**：多维度评测、自动评估、结果分析
  - **适用场景**：模型选型、性能对比、能力评估
  - **主要优势**：评测维度全面、支持中文模型、自动化程度高
  - **生态影响力**：被广泛用于学术研究和工业界评测

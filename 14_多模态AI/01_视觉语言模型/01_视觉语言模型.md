# 视觉语言模型

## 背景与发展

视觉语言模型（VLM）连接视觉与语言两个模态，使模型能"看图说话"和"按文绘图"。VLM 的发展经历了从早期双塔架构到桥接式架构，再到原生多模态统一模型的演进。

CLIP（2021）开创了对比学习对齐范式，通过大规模图文对训练实现零样本分类。BLIP-2（2023）引入 Q-Former 桥接视觉编码器与 LLM。LLaVA（2023）以简单的投影层将视觉编码器接入开源 LLM，实现了多模态对话。GPT-4V/Gemini（2023）代表了原生多模态大模型的前沿，从头训练统一处理图文输入。

## 核心思想

通过跨模态对齐，将视觉信息映射到语言模型的语义空间，使 LLM 能够理解和推理视觉内容。不同架构范式以不同方式实现这一对齐：对比学习直接对齐语义空间，桥接模块提取视觉特征并投影到 LLM 输入空间，原生融合在训练阶段即统一处理多模态。

## 技术原理

### 架构范式

| 范式 | 架构 | 代表模型 |
|------|------|---------|
| 对比学习 | 双塔编码器 + 对比损失 | CLIP |
| 桥接式 | 视觉编码器 + 投影层 + LLM | LLaVA, Qwen-VL |
| 原生多模态 | 从头训练的统一模型 | GPT-4V, Gemini |
| Flamingo 式 | Perceiver Resampler + 交叉注意力 | Flamingo |

### CLIP：对比学习对齐

图像编码器（ViT）和文本编码器（Transformer）分别编码两个模态，对比损失拉近匹配的图文对，推远不匹配的对：

$$L = -\log\frac{\exp(\text{sim}(I, T^+) / \tau)}{\sum_j \exp(\text{sim}(I, T_j) / \tau)}$$

零样本分类：用文本描述类别，选择与图像相似度最高的类别。

### LLaVA 架构

```
图像 -> ViT 视觉编码器 -> 视觉 Token -> MLP 投影 -> 接入 LLM 的 Token 序列
                                                        |
文本 -> Tokenization -> 文本 Token ---------------------> LLM -> 输出
```

简单有效：CLIP ViT 视觉编码器 + MLP 投影层 + 开源 LLM（Vicuna）。两阶段训练：图像描述对齐 -> 指令微调。

### 多模态对齐方法

| 方法 | 原理 | 代表 |
|------|------|------|
| 线性投影 | 简单 MLP 映射 | LLaVA |
| Q-Former | 可学习查询提取视觉特征 | BLIP-2 |
| 交叉注意力 | LLM 层间插入交叉注意力 | Flamingo |
| 原生融合 | 视觉 Token 直接作为序列一部分 | GPT-4V |

### 训练策略

1. **预训练对齐**：用大规模图文对训练视觉-语言对齐
2. **指令微调**：用多模态指令数据微调（GPT-4 生成指令）
3. **偏好对齐**：RLHF/DPO 适配多模态偏好

## 发展演进

CLIP（2021）-> Flamingo（2022）-> BLIP-2（2023）-> LLaVA（2023）-> GPT-4V/Gemini（2023）-> 原生多模态统一模型（2024+）

趋势：从桥接式架构走向原生多模态，最终实现 Any-to-Any 统一模型。

## 关键算法·模型

| 模型 | 架构特点 | 能力 |
|------|---------|------|
| CLIP | 双塔对比学习 | 零样本分类、图文检索 |
| BLIP-2 | Q-Former 桥接 | 图文理解 + 生成 |
| LLaVA | ViT + 投影 + Vicuna | 多模态对话 |
| Qwen-VL | ViT + Adapter + Qwen | 中文多模态、OCR |
| GPT-4V | 原生多模态 | 通用视觉理解 |
| Gemini | 原生多模态 | 图文音视频统一 |
| Flamingo | Perceiver Resampler + 交叉注意力 | 少样本图文理解 |

## 应用场景

- **多模态对话**：带图片的 ChatGPT
- **视觉问答（VQA）**：图像内容问答
- **OCR 与文档理解**：表格、公式、图表识别
- **多模态 Agent**：UI 操作、视觉导航
- **图像检索**：文本到图像搜索

- Transformer 是跨模态融合的统一架构
- VLM 扩展了 [[Agent]] 的感知能力

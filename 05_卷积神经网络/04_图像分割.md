# 图像分割

## 1. 概述

图像分割（Image Segmentation）是将图像划分为多个具有独立语义区域的任务。与目标检测用边界框粗略定位不同，分割要求在**像素级别**对每个像素进行分类，实现精确的轮廓描述。根据粒度和目标的不同，分割任务分为三个层次：

| 任务 | 目标 | 输出 |
|------|------|------|
| **语义分割** | 给每个像素分配类别标签 | $H \times W$ 的类别图 |
| **实例分割** | 区分同一类别的不同个体 | 每个实例的掩码 + 类别 |
| **全景分割** | 语义+实例的统一 | 所有stuff（背景）+things（物体） |

## 2. 发展历史

```
传统方法 (2000s)
  │  Graph Cut, GrabCut, 超像素分割 (SLIC)
  ▼
全卷积时代 (2015-2017)
  │  FCN → U-Net → SegNet → DeepLab v1/v2
  ▼
多尺度与注意力 (2017-2019)
  │  PSPNet → DeepLab v3/v3+ → DANet → CCNet
  ▼
实例分割突破 (2017-2020)
  │  Mask R-CNN → YOLACT → SOLO → BlendMask
  ▼
Transformer时代 (2021-至今)
     SETR → SegFormer → MaskFormer → SAM → Mask2Former
```

**关键转折**：
- **2015**: FCN首次实现端到端的像素级分类，舍弃全连接层
- **2015**: U-Net提出对称编码-解码器+跳跃连接，成为医学影像标准
- **2017**: Mask R-CNN在Faster R-CNN上增加掩码分支，实例分割进入实用阶段
- **2023**: SAM（Segment Anything Model）作为分割基础模型，支持开放世界的零样本分割

## 3. 核心概念

### 3.1 评估指标

| 指标 | 公式/含义 |
|------|-----------|
| **Pixel Accuracy** | 正确分类像素 / 总像素 |
| **mIoU (mean Intersection over Union)** | 各类别IoU的平均值，最主流的语义分割指标 |
| **Dice Coefficient** | $2 \times |A \cap B| / (|A| + |B|)$，医学影像常用 |
| **AP (Average Precision)** | 实例分割的COCO评估标准（类似检测的mAP） |

### 3.2 核心挑战

- **细节保留**：连续池化导致空间信息丢失，小物体边界模糊
- **多尺度物体**：同一场景中大物体（道路）和小物体（行人）共存
- **类别不平衡**：背景像素远多于前景物体像素
- **实时性**：高分辨率分割的计算负担

## 4. 技术原理

### 4.1 语义分割

#### FCN（Fully Convolutional Network, 2015）

**核心创新**：
1. 将分类网络（VGG等）的全连接层替换为 $1\times1$ 卷积，实现全卷积化
2. 通过**转置卷积**将低分辨率特征图上采样到原图尺寸
3. **跳跃连接**（Skip Connection）：融合浅层精细空间信息和深层语义信息

**三种融合策略**：
- **FCN-32s**：一次32×上采样（结果模糊）
- **FCN-16s**：融合pool4特征后16×上采样（中等质量）
- **FCN-8s**：融合pool3和pool4后8×上采样（最优）

**历史意义**：开创了全卷积的像素级预测范式，但上采样过程粗糙，边界精度有限。

#### U-Net（2015）

**核心设计——对称编码-解码器**：
- **编码器**（收缩路径）：连续卷积+池化，逐步提取语义信息
- **解码器**（扩展路径）：连续上采样+卷积，逐步恢复空间分辨率
- **跳跃连接**：编码器每层特征直接拼接到解码器对应层

```
编码器: C1 -> C2 -> C3 -> C4 -> C5 (Bottleneck)
  |       |     |     |          ^
解码器: C1'<- C2'<- C3'<- C4'<- C5'
   (<- 表示跳跃连接的拼接)
```

**为什么适合医学影像**：
- 医学数据量小，U-Net的密集跳跃连接可在有限样本下有效学习
- 对称结构自然地处理任意尺寸输入
- 边框镜像填充策略避免边界效应

**后续影响**：U-Net的编码-解码+跳跃连接架构成为分割任务的事实标准，衍生出3D U-Net、V-Net、Attention U-Net、U-Net++等变体。

#### DeepLab系列

| 版本 | 核心创新 |
|------|----------|
| **DeepLab v1** (2015) | 空洞卷积 + 全连接CRF后处理精修边界 |
| **DeepLab v2** (2016) | **ASPP**（Atrous Spatial Pyramid Pooling）：不同膨胀率的并行空洞卷积捕捉多尺度上下文 |
| **DeepLab v3** (2017) | 改进ASPP（加入全局平均池化+1×1卷积分支）+ 去除CRF |
| **DeepLab v3+** (2018) | 引入**编码-解码结构**：DeepLabv3作为编码器 + 简单解码器恢复细节 |

**ASPP模块原理**：在不同膨胀率（如6, 12, 18）下并行进行空洞卷积，让网络在每个像素位置都能同时看到不同尺度的上下文信息——大膨胀率看全局，小膨胀率看局部。

**贡献**：空洞卷积成为语义分割的标准组件，在不大幅增加计算量的前提下扩大感受野。

#### PSPNet（2017）

**核心模块——金字塔池化模块（PPM）**：将特征图在多个尺度上进行全局池化，上采样后与原特征拼接。与ASPP的并行空洞卷积异曲同工。

### 4.2 实例分割

#### Mask R-CNN（2017）

> 论文：He, Gkioxari, Dollar, Girshick. Mask R-CNN. ICCV 2017

在**Faster R-CNN**的基础上增加一个**掩码预测分支**（小型FCN），与分类和边界框回归并行：

```
Backbone -> FPN -> RPN -> RoIAlign -> [分类 (FC)]
                                       [边界框回归 (FC)]
                                       [掩码预测 (小型FCN)]
```

**RoIAlign 的关键改进**：
- Faster R-CNN的**RoI Pooling**有两次量化（浮点坐标->整数），导致特征和原图的**不对齐**（Misalignment），对分割任务致命
- **RoIAlign**使用双线性插值，保持浮点精度，显著提升掩码质量（mAP提升~3%）

**解耦设计**：分类和掩码预测独立——掩码分支只预测"前景/背景"（二值化），不涉及类别竞争，使每个实例掩码的预测更精准。

#### YOLACT（2019）

首个**实时实例分割**（30+ FPS）。核心思想：生成一组原型掩码和一组掩码系数，通过线性组合得到最终实例掩码。

#### SAM（Segment Anything Model, 2023）

> 论文：Kirillov et al., Meta AI. Segment Anything.

基于ViT的**分割基础模型**，支持点/框/文本提示下的零样本分割。在SA-1B数据集（1100万图像，10亿+掩码）上训练，将分割从监督学习推向**提示驱动的通用分割**范式。

SAM的关键启示：分割可以像LLM一样，作为基础能力预训练，然后通过提示适应任意下游任务。

## 5. 关键方法对比

| 方法 | 任务 | 年份 | 核心设计 | 特点 |
|------|------|------|----------|------|
| FCN | 语义分割 | 2015 | 全卷积+跳跃连接 | 开创性，边界粗糙 |
| U-Net | 语义分割 | 2015 | 对称编码-解码 | 医学影像标准 |
| DeepLab v3+ | 语义分割 | 2018 | 空洞ASPP+编解码 | 通用SOTA |
| PSPNet | 语义分割 | 2017 | 金字塔池化 | 多尺度上下文 |
| SegFormer | 语义分割 | 2021 | Transformer编码+MLP解码 | 简洁高效 |
| Mask R-CNN | 实例分割 | 2017 | RPN+RoIAlign+掩码分支 | 实例分割标杆 |
| SAM | 通用分割 | 2023 | 提示驱动+ViT | 基础模型范式 |

## 6. 优势与局限

### 语义分割
- 精确的像素级理解
- 适合场景级的全局分析（自动驾驶可行驶区域、遥感土地分类）
- 无法区分同类物体个体
- 边界精度仍然是挑战

### 实例分割
- 同时完成检测+分割，信息最丰富
- Mask R-CNN框架通用性强，易于扩展（+关键点=姿态估计）
- 计算量大于单纯检测或分割
- 密集场景中实例重叠难以分离

### 统一挑战
- 人工标注成本极高（像素级标注耗时是边界框标注的10-100倍）
- 大规模预训练数据构建困难

## 7. 应用场景

| 领域 | 任务 | 常用方法 |
|------|------|----------|
| **医学影像** | 器官分割、肿瘤勾画 | U-Net、nnU-Net、V-Net(3D) |
| **自动驾驶** | 道路/车道线/行人分割 | DeepLab、BiSeNet |
| **遥感** | 建筑物提取、土地分类 | DeepLab、U-Net |
| **视频编辑** | 人像抠图、背景替换 | MODNet、RVM |
| **工业质检** | 缺陷区域分割 | U-Net变体 |
| **卫星** | 云检测、变化检测 | 多时相U-Net |

## 9. 前沿发展

1. **视觉基础模型**：SAM的成功开启了分割的"基础模型+提示"范式，类似NLP的LLM。SAM 2将能力扩展到视频分割
2. **开放词汇分割**：CLIP+分割头，支持任意文本描述的目标分割（如OVSeg、OpenSeg）
3. **Transformer统一架构**：MaskFormer/Mask2Former将语义和实例分割统一为掩码分类问题
4. **交互式分割**：用户点击+模型即时响应（SAM的交互模式），降低标注成本
5. **3D分割**：NeRF/3DGS场景中的神经辐射场语义分割
6. **实时高分辨率分割**：BiSeNet、STDC、PIDNet等双分支设计平衡速度与精度

# 卷积神经网络

## 背景与发展

[[卷积神经网络]]（CNN）是专门处理网格结构数据（图像、视频）的[[深度学习]]架构。其思想源于 Hubel 和 Wiesel 对猫视觉皮层的研究——局部感受野和层次化处理。LeCun 于 1998 年提出 LeNet-5 用于手写数字识别，2012 年 AlexNet 在 ImageNet 竞赛上的突破引领了深度学习革命。此后 VGG、GoogLeNet、ResNet 等架构不断推进性能边界。

## 核心思想

利用**局部连接**和**权值共享**两大核心机制，大幅减少参数量同时保留空间结构信息。通过多层卷积堆叠，从低级边缘特征逐步构建到高级语义特征，形成层次化的特征表示。

## 技术原理

### 核心组件

| 组件 | 作用 | 关键参数 |
|------|------|---------|
| **卷积层** | 特征提取 | 卷积核大小、步长、填充、通道数 |
| **池化层** | 下采样、平移不变性 | 池化窗口大小、步长（最大池化/平均池化） |
| **全连接层** | 分类/回归 | 输出维度 |
| **BatchNorm** | 加速训练、稳定收敛 | — |
| **激活函数** | 引入非线性 | ReLU 为主 |

### 卷积运算

输出尺寸公式：$O = \frac{I - K + 2P}{S} + 1$

其中 $I$=输入尺寸，$K$=卷积核大小，$P$=填充，$S$=步长。

### 感受野

感受野（Receptive Field）指特征图上某个点对应输入图像的区域大小。深层网络通过层层堆叠扩大感受野，使网络能捕获更大的图像区域。计算公式：

$$RF_l = RF_{l-1} + (K_l - 1) \times \prod_{i=1}^{l-1} S_i$$

### 残差连接（ResNet）

$$\mathbf{y} = F(\mathbf{x}, \{W_i\}) + \mathbf{x}$$

残差连接使梯度能直接反向传播到浅层，解决了深层网络梯度消失问题，使训练上百层的网络成为可能。这一思想被后续的 [[Transformer]] 架构继承。

## 发展演进

LeNet（1998）$\to$ AlexNet（2012，深度学习革命）$\to$ VGG/GoogLeNet（2014）$\to$ ResNet（2015，残差连接）$\to$ DenseNet（2017，密集连接）$\to$ EfficientNet（2019，复合缩放）$\to$ Vision Transformer 挑战（2020+）$\to$ ConvNeXt（2022，CNN 回归）

## 关键算法·模型

| 架构 | 年份 | 核心创新 |
|------|------|---------|
| **LeNet-5** | 1998 | 首个 CNN，手写数字识别 |
| **AlexNet** | 2012 | ReLU、Dropout、GPU 训练 |
| **VGG** | 2014 | 小卷积核（3x3）堆叠 |
| **GoogLeNet** | 2014 | Inception 模块，并行卷积 |
| **ResNet** | 2015 | 残差连接，解决梯度消失 |
| **DenseNet** | 2017 | 密集连接，特征复用 |
| **EfficientNet** | 2019 | 复合缩放策略 |
| **ConvNeXt** | 2022 | 用 CNN 对标 Transformer 性能 |

## 应用场景

- **图像分类**：ImageNet、细粒度分类
- **目标检测**：YOLO、Faster R-CNN、DETR
- **图像分割**：U-Net（医学）、DeepLab（语义分割）
- **特征提取**：作为多模态模型的视觉编码器
- **视频分析**：3D CNN（动作识别）

## 与其他技术关系

- CNN 的卷积思想与线性代数的矩阵运算密切相关
- ResNet 的残差连接思想被[[Transformer]]继承
- CNN 作为视觉编码器，与多模态模型结合
- 训练技巧详见[[训练技巧与稳定性]]
- 在图像领域面临 Vision Transformer 的竞争，但 ConvNeXt 证明 CNN 仍有竞争力

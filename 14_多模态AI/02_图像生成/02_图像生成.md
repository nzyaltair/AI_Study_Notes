# 图像生成

## 背景与发展

图像生成是生成式 AI 最引人注目的应用之一。从 GAN（2014）到 [[扩散模型]]（2020），再到 DALL-E 3 和 Stable Diffusion，AI 图像生成质量实现了从"能看"到"好看"再到"精确控制"的飞跃。扩散模型已成为图像生成的主流范式，潜在扩散模型（LDM）使其在消费级 GPU 上可行。

## 核心思想

扩散模型通过前向加噪和反向去噪过程学习数据分布。训练时学习从噪声中逐步恢复图像，推理时从随机噪声开始逐步生成图像。潜在扩散模型将扩散过程从像素空间转移到低维潜在空间，大幅减少计算量。

## 技术原理

### 扩散模型原理

**前向过程**（加噪）：

$$q(\mathbf{x}_t | \mathbf{x}_0) = \mathcal{N}(\mathbf{x}_t; \sqrt{\alpha_t}\mathbf{x}_0, (1-\alpha_t)\mathbf{I})$$

**反向过程**（去噪）：

$$p_\theta(\mathbf{x}_{t-1} | \mathbf{x}_t) = \mathcal{N}(\mathbf{x}_{t-1}; \mu_\theta(\mathbf{x}_t, t), \Sigma_\theta(\mathbf{x}_t, t))$$

**训练目标**：预测噪声 $\epsilon$：

$$L = \mathbb{E}_{t, \mathbf{x}_0, \epsilon}\left[\|\epsilon - \epsilon_\theta(\mathbf{x}_t, t)\|^2\right]$$

### 潜在扩散模型（LDM）

Stable Diffusion 的核心创新：

1. 用 VAE 将图像压缩到低维潜在空间
2. 在潜在空间执行扩散去噪
3. VAE 解码器恢复到像素空间

```
图像 -> VAE编码器 -> 潜在表示 -> U-Net去噪 -> 去噪潜在表示 -> VAE解码器 -> 生成图像
```

### 条件控制

| 方法 | 原理 | 特点 |
|------|------|------|
| 文本条件 | CLIP 文本编码器 + 交叉注意力 | DALL-E, Stable Diffusion |
| ControlNet | 额外条件控制网络 | 精确结构控制 |
| LoRA 微调 | 低秩适配特定风格 | 个性化生成 |
| IP-Adapter | 图像作为条件 | 风格/内容迁移 |
| Inpainting | 只重绘指定区域 | 图像编辑 |

### Classifier-Free Guidance

无分类器引导，平衡质量与多样性：

$$\hat{\epsilon} = (1+w)\epsilon_\theta(x_t, c) - w\epsilon_\theta(x_t, \varnothing)$$

### 采样加速

DDIM、DPM++、Euler 等采样器减少去噪步数，从 1000 步降到 20-50 步。

## 发展演进

GAN（2014）-> VAE -> 扩散模型（DDPM, 2020）-> 潜在扩散（LDM, 2021）-> Stable Diffusion（2022）-> DALL-E 3（2023）-> FLUX（2024）

## 关键算法·模型

| 模型 | 架构 | 特点 |
|------|------|------|
| Stable Diffusion | LDM + U-Net | 开源、生态丰富 |
| DALL-E 3 | 扩散模型 + LLM | 文本理解强 |
| Midjourney | 扩散变体 | 艺术风格出色 |
| FLUX | Rectified Flow | 新一代开源 SOTA |
| GAN | 生成器 + 判别器 | 快速生成、训练不稳定 |

U-Net 是扩散模型的主力去噪网络，支持多尺度特征提取和跳跃连接。

## 应用场景

- **创意设计**：插画、海报、概念设计
- **内容创作**：营销素材、社交媒体内容
- **数据增强**：为 ML 训练生成合成数据
- **图像编辑**：修复、扩展、风格转换
- **多模态应用**：文生图作为多模态系统的输出能力

## 与其他技术关系

- 扩散模型的 U-Net 基于 CNN 架构
- 文本条件依赖 CLIP（[[视觉语言模型]] 技术）
- 训练技巧（如 EMA、梯度裁剪）对扩散模型重要
- 图像生成能力扩展了 [[Agent]] 的工具集
- [[视频生成]] 建立在图像生成技术之上

# 生成模型综述

## 1. 概述

生成模型（Generative Models）是一类**学习数据分布并从中生成新样本**的机器学习模型。与判别模型专注于建模 $p(y|x)$ 不同，生成模型直接建模数据的联合分布 $p(x)$ 或条件分布 $p(x|c)$，从而具备"创造"新数据的能力。

生成模型解决的核心问题：
- **密度估计**：从样本中学习数据的概率分布 $p_{data}(x)$
- **样本生成**：从学习到的分布中采样，生成逼真的新样本
- **表示学习**：学习数据的潜在结构和语义表示
- **条件生成**：在给定条件（文本、类别、图像等）下生成符合要求的样本

生成模型在图像合成、文本生成、音频合成、视频生成、药物设计、数据增强等领域有广泛应用。

## 3. 核心概念

### 3.1 生成模型的分类

按建模方式可分为两大范式：

**显式模型（Explicit Models）**：直接建模并优化数据分布 $p_\theta(x)$
- 似然最大化方法：VAE、归一化流、自回归模型
- 能量模型 / Score 模型：EBM、Diffusion Score Model

**隐式模型（Implicit Models）**：不显式建模分布，通过采样过程隐式匹配数据分布
- 代表：GAN

按生成过程可分为：

| 类型 | 原理 | 代表 |
|------|------|------|
| 对抗生成 | 生成器 vs 判别器博弈 | GAN 系列 |
| 变分推断 | 编码器-解码器 + ELBO | VAE 系列 |
| 可逆变换 | 可逆映射 + 变量替换 | Normalizing Flows |
| 迭代去噪 | 前向加噪 + 逆向去噪 | Diffusion Models |
| 自回归 | 逐 token/像素预测 | PixelCNN, GPT, ImageGPT |
| 能量模型 | 学习能量函数 | EBM, Score Matching |

### 3.2 关键术语

- **似然（Likelihood）**：$p_\theta(x)$，模型对数据的概率密度估计
- **KL 散度**：$D_{KL}(p \| q)$，衡量两个分布的差异
- **ELBO（证据下界）**：$\log p(x) \geq \text{ELBO}$，VAE 优化的目标
- **Score 函数**：$\nabla_x \log p(x)$，对数概率密度的梯度
- **模式崩溃（Mode Collapse）**：GAN 中生成器只产生有限种类样本
- **分类器引导 / 无分类器引导**：条件生成中控制质量-多样性平衡的技术
- **潜空间（Latent Space）**：数据被编码到的低维表示空间

### 3.3 理论框架统一

各类生成模型可以从**概率密度建模**的角度统一理解：

- **VAE**：最大化 $\log p(x)$ 的下界（ELBO）
- **归一化流**：精确计算 $\log p(x)$，通过可逆变换
- **GAN**：隐式匹配 $p_\theta$ 与 $p_{data}$，通过对抗损失
- **扩散模型**：通过去噪 score matching 学习 $\nabla_x \log p(x)$
- **自回归模型**：将 $p(x)$ 分解为 $\prod p(x_i | x_{<i})$

## 4. 核心数学基础

### 4.1 最大似然估计

$$\theta^* = \arg\max_\theta \mathbb{E}_{x \sim p_{data}}[\log p_\theta(x)]$$

等价于最小化 $D_{KL}(p_{data} \| p_\theta)$。

### 4.2 KL 散度与变分推断

$$\log p(x) = \text{ELBO} + D_{KL}(q_\phi(z|x) \| p(z|x))$$

由于 $D_{KL} \geq 0$，ELBO 是 $\log p(x)$ 的下界。

### 4.3 Score Matching

$$\mathcal{L}_{SM} = \mathbb{E}_{x \sim p_{data}}\left[\|\nabla_x \log p_\theta(x) - \nabla_x \log p_{data}(x)\|^2\right]$$

噪声条件版本（NCSN）通过多尺度加噪实现可扩展训练。

### 4.4 GAN Minimax 博弈

$$\min_G \max_D V(D, G) = \mathbb{E}_{x \sim p_{data}}[\log D(x)] + \mathbb{E}_{z \sim p_z}[\log(1 - D(G(z)))]$$

### 4.5 扩散模型

前向：$x_t = \sqrt{\bar\alpha_t} x_0 + \sqrt{1 - \bar\alpha_t} \epsilon$

训练损失：$\mathcal{L}_{simple} = \mathbb{E}_{t, x_0, \epsilon}[\|\epsilon - \epsilon_\theta(x_t, t)\|^2]$

## 5. 关键方法与模型

### 5.1 各类模型对比

| 特性 | GAN | VAE | Flow | Diffusion | Autoregressive |
|------|-----|-----|------|-----------|----------------|
| 似然估计 | 隐式 | 下界 | 精确 | 下界/精确 | 精确 |
| 采样速度 | 极快（1步） | 快（1步） | 快（1步） | 慢（多步） | 慢（逐token） |
| 样本质量 | 高 | 中 | 中-高 | 高 | 高 |
| 训练稳定性 | 不稳定 | 稳定 | 稳定 | 稳定 | 稳定 |
| 模式覆盖 | 差 | 好 | 好 | 好 | 好 |
| 可逆性 | 否 | 否 | 是 | 是 | 否 |

### 5.2 代表模型

- **GAN 系列**：DCGAN, WGAN, StyleGAN, BigGAN, CycleGAN
- **VAE 系列**：β-VAE, VQ-VAE, NVAE
- **归一化流**：RealNVP, Glow, Neural Spline Flows
- **扩散模型**：DDPM, DDIM, Score SDE, Stable Diffusion, DALL-E 2/3, DiT, FLUX
- **自回归**：PixelCNN, ImageGPT, DALL-E 1
- **新范式**：Consistency Models, Flow Matching, Rectified Flow

## 6. 优势与局限

### 6.1 整体优势

- **创造力**：从学习到的分布中生成全新的、逼真的样本
- **数据增强**：为下游任务生成合成训练数据
- **表示学习**：学习数据的潜在结构和语义
- **条件控制**：通过条件输入精确控制生成内容

### 6.2 共性挑战

- **评估困难**：样本质量难以用单一指标衡量（FID/IS 各有局限）
- **计算成本**：高质量生成模型（尤其扩散模型）训练和推理代价高
- **模式覆盖 vs 质量**：GAN 质量高但覆盖差，扩散模型覆盖好但慢
- **可控性**：精确控制生成内容仍是开放问题
- **安全性**：Deepfake、版权、偏见等伦理问题

## 8. 与其他技术关系

- **数学基础**：概率论、信息论（KL散度、Jensen不等式）是所有生成模型的理论基石
- **深度学习基础**：CNN（DCGAN、U-Net）、Transformer（DiT）是生成模型的核心网络架构
- **Transformer 与注意力机制**：DiT 将 Transformer 引入扩散模型，CLIP/T5 提供文本条件
- **多模态 AI**：CLIP 跨模态对齐是条件生成的核心；图像/视频生成是多模态应用的关键能力
- **大语言模型**：LLM 作为文本编码器引导图像生成（DALL-E 3）；自回归生成是 LLM 的核心范式
- **机器学习**：无监督学习中的密度估计是生成模型的理论基础

## 9. 前沿发展

- **统一生成框架**：Flow Matching 统一了扩散模型和连续归一化流
- **少步采样**：Consistency Models、Rectified Flow 实现 1-4 步高质量生成
- **Transformer 架构**：DiT 架构逐步取代 U-Net 成为主流
- **大规模训练**：Stable Diffusion 3、FLUX 展示了规模定律在生成模型中的适用性
- **多模态统一**：Any-to-Any 统一生成模型（文本+图像+音频+视频）
- **可控性增强**：ControlNet、IP-Adapter、InstantID 等精细控制方法
- **视频生成**：时空扩散模型实现高质量长视频生成（Sora）
- **3D 生成**：Score Distillation Sampling (SDS) 将 2D 生成先验提升到 3D
- **科学应用**：蛋白质设计、材料发现中的生成模型应用

## 当前发展状态

截至2026年，生成模型领域呈现以下状态：

- **扩散模型主导图像生成**：Stable Diffusion 3 / FLUX 基于 Rectified Flow 架构成为开源主流；DiT 架构逐步取代 U-Net。商业产品（DALL-E 3、Midjourney V6、Imagen 3）已大规模产品化。
- **视频生成突破**：Sora（2024）展示了分钟级高质量视频生成能力；Runway Gen-3、Kling、Veo 等产品跟进。视频生成仍是计算密集型任务，时长与一致性是主要瓶颈。
- **少步采样成熟**：Consistency Models 和 Rectified Flow 实现 1-4 步高质量生成，逐步缩小与多步扩散的质量差距。
- **Flow Matching 统一框架**：Flow Matching 框架统一了扩散模型和连续归一化流，成为新一代生成模型的理论基础。
- **3D 与科学应用**：Score Distillation Sampling (SDS) 将 2D 生成先验提升到 3D；RFdiffusion 在蛋白质设计上取得突破。

## 未来趋势

1. **统一多模态生成**：Any-to-Any 统一生成模型（文本+图像+音频+视频）逐步成熟，单一模型处理跨模态生成。
2. **架构收敛**：Transformer-based（DiT）+ Flow Matching 可能成为统一架构范式，U-Net 逐步退出。
3. **视频生成可商用化**：长视频一致性、物理合理性改善，视频生成从演示走向生产工具。
4. **可控性增强**：ControlNet、IP-Adapter 等精细控制方法与生成模型深度集成，实现像素级可控生成。
5. **效率优化**：蒸馏、量化、潜空间压缩降低推理成本，使高质量生成可在端侧运行。
6. **安全与版权**：生成内容水印、来源溯源、训练数据版权合规成为工程必需。

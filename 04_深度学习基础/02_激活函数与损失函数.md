# 激活函数与损失函数

## 1. 概述

激活函数（Activation Function）和损失函数（Loss Function）是神经网络训练的两个核心组件：激活函数为网络引入非线性表达能力，使其能够逼近任意复杂函数；损失函数定义了优化目标，为参数更新提供梯度信号。二者共同决定了一个神经网络"能学什么"（表达容量）和"在学什么"（优化方向）。

- **激活函数**：作用于每个神经元的输出端，将仿射变换的结果映射到特定的取值空间。其导数特性直接影响反向传播的梯度流
- **损失函数**：量化模型预测分布与真实分布之间的差异，是连接模型输出与任务目标的桥梁

## 2. 发展历史

### 激活函数的演进

```
Sigmoid (1980s) → Tanh (1990s) → ReLU (2010, Nair & Hinton)
    → Leaky ReLU / PReLU (2015) → ELU (2016)
    → GELU (2016, Hendrycks & Gimpel) → Swish (2017, Ramachandran et al.)
    → SwiGLU (2020, Shazeer, PaLM) → 成为现代LLM标配
```

### 损失函数的演进

```
MSE (经典回归) → Cross-Entropy (分类标配)
    → Focal Loss (2017, Lin et al., 解决类别不平衡)
    → InfoNCE (2018, Oord et al., 对比学习奠基)
    → 现代LLM: Next-Token Cross-Entropy + 强化学习对齐损失
```

## 3. 核心概念

### 3.1 激活函数的设计要求

一个理想的激活函数应满足：
- **非线性**：否则深层复合等价于单层线性变换
- **可微性**：需支持梯度下降（ReLU在0处不可导，但实践中不影响）
- **单调性**（非必需但有帮助）：保证单层网络的损失函数为凸函数
- **计算高效**：每轮前向+反向传播需要多次评估激活函数及其导数

### 3.2 损失函数的设计要求

- **一致性**：损失函数的优化目标应与任务的评价指标一致
- **可微性**：必须能对模型输出求导，为反向传播提供梯度
- **数值稳定性**：避免溢出和梯度消失（如交叉熵搭配Softmax时需Log-Sum-Exp技巧）
- **鲁棒性**：对标签噪声和样本不平衡有一定容忍度

## 4. 技术原理

### 4.1 饱和激活函数

#### Sigmoid

$$f(x) = \frac{1}{1 + e^{-x}}, \quad f'(x) = f(x)(1 - f(x)) \leq 0.25$$

- **输出范围**：$(0, 1)$，非零中心（zero-centered）
- **梯度特性**：$|x| > 5$ 时导数趋近于0（**饱和现象**），深层网络中链式法则的连乘导致梯度指数级衰减
- **现状**：在现代深层网络中基本被淘汰，仅在二分类输出层偶有使用

#### Tanh（双曲正切）

$$f(x) = \tanh(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}, \quad f'(x) = 1 - f(x)^2$$

- **输出范围**：$(-1, 1)$，零中心化，收敛速度快于Sigmoid
- **缺点**：仍未解决两侧饱和导致的梯度消失
- **现状**：RNN/LSTM中仍有使用，但深层前馈网络中已不再使用

### 4.2 非饱和激活函数

#### ReLU（Rectified Linear Unit）

$$f(x) = \max(0, x), \quad f'(x) = \begin{cases} 1 & x > 0 \\ 0 & x \leq 0 \end{cases}$$

- **核心优势**：
  1. 正半区导数恒为1，**彻底消除饱和导致的梯度消失**
  2. 计算极其简单（单次比较操作），无需指数/三角函数
  3. 诱导稀疏激活：约50%神经元输出为0，具有隐式正则化效果
- **Dead ReLU问题**：若某神经元对全体训练数据的加权输入恒为负，其梯度恒为0，权重永不再更新，该神经元"死亡"

#### Leaky ReLU

$$f(x) = \max(\alpha x, x), \quad 0 < \alpha \ll 1$$

- **设计动机**：负半区引入微小斜率 $\alpha$（通常取0.01），确保神经元恒有非零梯度，缓解Dead ReLU
- **PReLU（Parametric ReLU）**：将 $\alpha$ 作为可学习参数

#### ELU（Exponential Linear Unit）

$$f(x) = \begin{cases} x & x > 0 \\ \alpha(e^x - 1) & x \leq 0 \end{cases}$$

- **设计动机**：负半区使用指数函数，使输出均值更接近0，加速收敛；但计算开销高于ReLU

### 4.3 现代平滑激活函数（LLM标配）

#### GELU（Gaussian Error Linear Unit）

$$f(x) = x \cdot \Phi(x) = x \cdot P(X \leq x), \quad X \sim \mathcal{N}(0, 1)$$

$$f(x) \approx 0.5x\left(1 + \tanh\left[\sqrt{\frac{2}{\pi}}(x + 0.044715x^3)\right]\right)$$

- **设计动机**：将随机正则化思想与激活函数融合。根据输入值的大小，以概率决定是否激活。输入越小，被"门控"概率越高，起到自适应正则化效果
- **应用**：BERT, GPT-2/3, ViT等2018-2022年主流架构的默认选择

#### SwiGLU（Swish-Gated Linear Unit）

$$\text{SwiGLU}(\mathbf{x}) = \text{Swish}_{\beta}(\mathbf{x}\mathbf{W}) \odot (\mathbf{x}\mathbf{V})$$

其中 $\text{Swish}_{\beta}(x) = x \cdot \sigma(\beta x)$，$\sigma$ 为Sigmoid。

- **设计动机**：基于门控线性单元（GLU）范式。一个分支通过Swish进行非线性门控，与另一个线性分支做Hadamard积
- **核心优势**：门控机制提供了比GELU更精细的信息筛选能力，训练更稳定
- **应用**：LLaMA, PaLM, Gemini等 2023+ 现代LLM的标准FFN组件

## 5. 关键方法/模型

### 5.1 回归损失函数

#### 均方误差（MSE / L2 Loss）

$$\mathcal{L}_{\text{MSE}} = \frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2$$

- **梯度**：$\frac{\partial \mathcal{L}}{\partial \hat{y}_i} = \frac{2}{n}(\hat{y}_i - y_i)$，误差越大梯度越大
- **缺点**：对离群点（outlier）极敏感——误差平方项使离群点主导梯度

#### 平均绝对误差（MAE / L1 Loss）

$$\mathcal{L}_{\text{MAE}} = \frac{1}{n} \sum_{i=1}^{n} |y_i - \hat{y}_i|$$

- **梯度**：$\frac{\partial \mathcal{L}}{\partial \hat{y}_i} = \pm \frac{1}{n}$，梯度恒定为常数
- **优点**：对离群点鲁棒；**缺点**：梯度不连续，零点附近训练不稳定

#### Huber Loss（平滑L1）

$$\mathcal{L}_{\delta}(y, \hat{y}) = \begin{cases} \frac{1}{2}(y - \hat{y})^2 & |y - \hat{y}| \leq \delta \\ \delta(|y - \hat{y}| - \frac{1}{2}\delta) & |y - \hat{y}| > \delta \end{cases}$$

- 结合MSE和MAE的优点：小误差时用MSE（平滑可导），大误差时用MAE（鲁棒）

### 5.2 分类损失函数

#### 交叉熵损失（Cross-Entropy Loss）

**二分类**：

$$\mathcal{L} = -\frac{1}{n} \sum_{i=1}^{n} \left[y_i \log(\hat{y}_i) + (1 - y_i)\log(1 - \hat{y}_i)\right]$$

**多分类**（搭配Softmax）：

$$\mathcal{L} = -\sum_{c=1}^{C} y_c \log\left(\frac{e^{z_c}}{\sum_{j=1}^{C} e^{z_j}}\right)$$

- **梯度特性**：Softmax + Cross-Entropy组合的梯度为 $\hat{y}_c - y_c$，形式极其简洁
- **数值稳定性**：使用Log-Sum-Exp技巧：$\log\sum e^{z_j} = m + \log\sum e^{z_j - m}$，其中 $m = \max_j z_j$

#### 焦点损失（Focal Loss）

$$\mathcal{L}_{\text{focal}} = -\alpha_t (1 - p_t)^\gamma \log(p_t)$$

其中 $p_t = \hat{y}$（正类）或 $1 - \hat{y}$（负类）。

- **调制因子** $(1 - p_t)^\gamma$：预测置信度越高（$p_t$ 接近1），权重越小，使训练聚焦于难分类样本
- **应用**：目标检测（RetinaNet）、严重类别不平衡的分类场景

### 5.3 对比学习损失

#### InfoNCE（Information Noise-Contrastive Estimation）

$$\mathcal{L}_{\text{InfoNCE}} = -\log \frac{\exp(\text{sim}(\mathbf{q}, \mathbf{k}_+) / \tau)}{\exp(\text{sim}(\mathbf{q}, \mathbf{k}_+) / \tau) + \sum_{i=1}^{K} \exp(\text{sim}(\mathbf{q}, \mathbf{k}_i^-) / \tau)}$$

- **物理意义**：最大化查询 $\mathbf{q}$ 与正样本 $\mathbf{k}_+$ 的互信息下界（通过温度 $\tau$ 控制分布锐度）
- **核心论文**：Oord et al. (2018), *Representation Learning with Contrastive Predictive Coding*
- **应用**：CLIP, SimCLR, MoCo等自监督/多模态预训练的基础损失函数

## 6. 优势与局限

### 激活函数对比总结

| 激活函数 | 优点 | 缺点 | 适用场景 |
|:---|:---|:---|:---|
| Sigmoid | 输出有界，概率解释清晰 | 梯度消失，非零中心 | 二分类输出层 |
| Tanh | 零中心，梯度比Sigmoid强 | 仍有饱和梯度消失 | RNN/LSTM |
| ReLU | 无饱和梯度消失，计算快 | Dead ReLU | CNN, MLP（2012-2018） |
| Leaky ReLU | 缓解Dead ReLU | 负半区斜率需手动设定 | CNN, 生成模型 |
| GELU | 自适应正则化，平滑 | 计算开销高于ReLU | Transformer（BERT, GPT-2/3） |
| SwiGLU | 门控精细筛选，训练稳定 | 参数量翻倍（两个权重矩阵） | 现代LLM（LLaMA, PaLM） |

### 损失函数选择指南

| 任务类型 | 推荐损失函数 | 注意 |
|:---|:---|:---|
| 回归（无离群点） | MSE | 梯度随误差线性增长 |
| 回归（有离群点） | Huber Loss | 均衡MSE和MAE |
| 二分类 | BCE (Binary Cross-Entropy) | 搭配Sigmoid输出 |
| 多分类 | CCE (Categorical Cross-Entropy) | 搭配Softmax输出 |
| 类别严重不平衡 | Focal Loss | $\gamma=2, \alpha=0.25$ 为标准配置 |
| 自监督/对比学习 | InfoNCE | 温度 $\tau$ 是关键超参 |
| LLM预训练 | Next-Token Cross-Entropy | 对所有token位置取平均 |

## 7. 应用场景

- **视觉分类**：CNN中的ReLU + 交叉熵（ImageNet时代标配）
- **大语言模型**：Transformer FFN中的SwiGLU + Next-Token交叉熵（LLaMA/Gemini标配）
- **目标检测**：Focal Loss解决正负样本极度不平衡
- **多模态对比学习**：InfoNCE损失驱动图文对齐（CLIP）
- **回归预测**：MSE/Huber用于股价、温度、传感器数值等连续值预测

## 8. 与其他技术关系

- **与反向传播的关系**：激活函数的导数 $\frac{\partial a}{\partial z}$ 直接出现在BP递推公式 $\boldsymbol{\delta}^{(l)} = f'(\mathbf{z}^{(l)}) \odot ((\mathbf{W}^{(l+1)})^T \boldsymbol{\delta}^{(l+1)})$ 中，其数值特性决定了梯度流是否健康（详见 [[03_反向传播与自动微分]]）
- **与参数初始化的关系**：激活函数的输出方差特性影响初始化策略——ReLU的负半区清零使He初始化方差需扩大2倍（详见 [[04_参数初始化与归一化]]）
- **与归一化的关系**：归一化技术（BN/LN）改变了激活函数的输入分布，影响其有效工作区间（详见 [[04_参数初始化与归一化]]）

## 9. 前沿发展

1. **自适应激活函数**
   - **KAN (Kolmogorov-Arnold Networks)**（2024，MIT）：用可学习的B样条函数替代固定激活函数，每个连接边上有独立的激活函数。在PDE求解等科学计算任务上，以更少的参数量达到与MLP持平或更优的性能，且具有天然的可解释性

2. **损失函数的理论深化**
   - 对比学习损失的信息论基础（互信息最大化）持续深化，从InfoNCE到SimCLR/CLIP的损失函数设计推动自监督学习成为大模型预训练的核心范式
   - RLHF中的偏好损失和DPO中的偏好对比损失定义了新一代LLM对齐的优化目标

3. **高效门控机制**
   - 从SwiGLU到更精简的门控变体（如ReGLU、GeGLU），学术界和工业界持续在大模型FFN的计算效率和表达能力之间寻找更优的帕累托前沿

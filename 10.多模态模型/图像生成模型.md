# 图像生成模型

## 背景

图像生成是生成式AI最引人注目的应用之一。从GAN（2014）到扩散模型（2020），再到DALL-E 3和Stable Diffusion，AI图像生成质量实现了从"能看"到"好看"再到"精确控制"的飞跃。扩散模型已成为图像生成的主流范式。

## 核心思想

**扩散模型**：通过前向加噪和反向去噪过程学习数据分布。训练时学习从噪声中逐步恢复图像，推理时从随机噪声开始逐步生成图像。

## 技术原理

### 扩散模型原理

**前向过程**（加噪）：
$$q(\mathbf{x}_t | \mathbf{x}_0) = \mathcal{N}(\mathbf{x}_t; \sqrt{\alpha_t}\mathbf{x}_0, (1-\alpha_t)\mathbf{I})$$

**反向过程**（去噪）：
$$p_\theta(\mathbf{x}_{t-1} | \mathbf{x}_t) = \mathcal{N}(\mathbf{x}_{t-1}; \mu_\theta(\mathbf{x}_t, t), \Sigma_\theta(\mathbf{x}_t, t))$$

**训练目标**：预测噪声 $\epsilon$
$$L = \mathbb{E}_{t, \mathbf{x}_0, \epsilon}\left[\|\epsilon - \epsilon_\theta(\mathbf{x}_t, t)\|^2\right]$$

### 潜在扩散模型（LDM）

Stable Diffusion的核心创新：
1. **编码到潜在空间**：用VAE将图像压缩到低维潜在空间
2. **在潜在空间执行扩散**：计算量大幅减少
3. **解码回像素空间**：VAE解码器恢复图像

```
图像 → VAE编码器 → 潜在表示 → U-Net去噪 → 去噪潜在表示 → VAE解码器 → 生成图像
```

### 条件控制

| 方法 | 原理 | 特点 |
|------|------|------|
| **文本条件** | CLIP文本编码器 + 交叉注意力 | DALL-E, SD |
| **ControlNet** | 额外条件控制网络 | 精确结构控制 |
| **LoRA微调** | 低秩适配特定风格 | 个性化生成 |
| **IP-Adapter** | 图像作为条件 | 风格/内容迁移 |
| **Inpainting** | 只重绘指定区域 | 图像编辑 |

### 主流模型对比

| 模型 | 架构 | 特点 |
|------|------|------|
| **Stable Diffusion** | LDM + U-Net | 开源、生态丰富 |
| **DALL-E 3** | 扩散模型 + LLM | 文本理解强 |
| **Midjourney** | 未知（扩散变体） | 艺术风格出色 |
| **FLUX** | Rectified Flow | 新一代开源SOTA |
| **GAN** | 生成器+判别器 | 快速生成、训练不稳定 |

### 关键技术

- **U-Net**：扩散模型的主力去噪网络
- ** Classifier-Free Guidance**：无分类器引导，平衡质量与多样性
  - $\hat{\epsilon} = (1+w)\epsilon_\theta(x_t, c) - w\epsilon_\theta(x_t, \varnothing)$
- **采样加速**：DDIM、DPM++、Euler等采样器减少步数
- **潜在空间操作**：Img2Img、图生图、风格混合

## 发展演进

GAN（2014）→ VAE → 扩散模型（DDPM, 2020）→ 潜在扩散（LDM, 2021）→ Stable Diffusion（2022）→ DALL-E 3（2023）→ FLUX（2024）→ 统一多模态生成

## 应用领域

- **创意设计**：插画、海报、概念设计
- **内容创作**：营销素材、社交媒体内容
- **数据增强**：为[[机器学习|ML]]训练生成合成数据
- **图像编辑**：修复、扩展、风格转换
- **[[多模态模型|多模态]]应用**：文生图作为多模态系统的输出能力

## 与其他技术关系

- 扩散模型的U-Net基于[[卷积神经网络|CNN]]架构
- 文本条件依赖CLIP（[[视觉语言模型|VLM]]技术）
- [[训练技巧与稳定性|训练技巧]]（如EMA、梯度裁剪）对扩散模型重要
- 图像生成能力扩展了[[Agent智能体|Agent]]的工具集
- [[视频生成模型|视频生成]]建立在图像生成技术之上

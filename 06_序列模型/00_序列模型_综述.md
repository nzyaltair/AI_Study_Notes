# 序列模型综述

## 1. 概述

序列模型（Sequence Models）是处理具有时序依赖关系数据的神经网络架构。其核心目标是建模序列中元素之间的时间或顺序依赖性（Temporal/Sequential Dependency），使模型能够利用历史（或双向）上下文信息影响当前时刻的预测。

- **解决的问题**：传统前馈神经网络假设输入之间相互独立，无法处理变长序列和时序依赖。序列模型通过引入循环连接（Recurrent Connections），在每个时间步维护一个**隐藏状态（Hidden State）**，作为对过去信息的压缩记忆，从而实现对任意长度序列的建模。
- **核心价值**：序列模型是自然语言处理（NLP）、语音识别、时间序列预测等领域的基石技术。尽管 Transformer 在 2017 年后逐渐成为序列建模的主流架构，RNN 及其变体的设计思想（门控、隐状态传递、Seq2Seq 框架、注意力机制）仍是理解现代大模型的关键前置知识。
- **知识体系范围**：循环神经网络（RNN）、长短期记忆网络（LSTM）、门控循环单元（GRU）、Seq2Seq 架构、注意力机制（Attention Mechanism）、双向/深层 RNN、CTC（连接时序分类）、状态空间模型（SSM：S4/Mamba）。

## 2. 发展历史

序列模型的发展经历了从简单循环连接到高效并行架构的深刻演进，见证了"循环连接 → 门控机制 → 注意力机制 → 状态空间模型"四次范式跃迁。

|:---|:---|:---|
| **1982** | Hopfield 网络 (Hopfield) | 提出具有联想记忆能力的递归神经网络，开创了循环连接的研究范式 |
| **1986** | Jordan 网络 (Jordan) | 首次将隐藏状态用作"记忆单元"，存储过去输出信息 |

| **1991** | 梯度消失问题 (Hochreiter) | 在毕业论文中首次发现并系统分析 RNN 的梯度消失/爆炸问题（"长时滞"问题） |

| **2015** | Luong 注意力 (Luong et al.) | 提出多种注意力分数计算方式（dot、general、concat），丰富注意力机制的理论框架 |
| **2016** | PixelRNN / WaveNet | 将序列建模扩展到图像和音频生成领域 |
| **2017** | Transformer (Vaswani et al.) | 以自注意力全面取代循环连接，几乎终结了 RNN 在 NLP 领域的主流地位 |

| **2023** | Mamba (Gu & Dao) | 引入选择性 SSM（Selective State Space Model），通过输入依赖参数实现线性时间的高效序列建模 |
| **2024** | Mamba-2 / Jamba | SSM 与注意力机制的混合架构，探索"循环+注意力"的融合路径 |
| **2024** | RWKV-v6 (Peng et al.) | 线性注意力 RNN 架构持续演进，在保持线性复杂度的同时接近 Transformer 性能 |

## 3. 核心概念

### 3.1 隐藏状态（Hidden State）

隐藏状态是 RNN 的核心抽象：$h_t = f(h_{t-1}, x_t)$。它是模型对过去所有输入的压缩表示，在每个时间步更新并传递到下一时刻。RNN 的能力上限取决于隐藏状态能否有效编码长期依赖。

### 3.2 时间展开（Unrolling）与 BPTT

前向传播时，RNN 沿时间轴展开为一个深度等于序列长度的前馈网络（权重共享）。反向传播时，梯度沿时间轴反向流动（BPTT），应用链式法则计算各时刻参数的梯度。这是 RNN 训练的核心算法，也是梯度消失/爆炸的根源。

### 3.3 门控机制（Gating Mechanism）

通过可微的门控单元（Sigmoid 激活 + 逐元素乘法），动态控制信息的流动——哪些信息需要保留、遗忘或输出。LSTM 的遗忘门/输入门/输出门和 GRU 的更新门/重置门是门控机制的经典实现。

### 3.4 长期依赖（Long-Term Dependency）

当序列中的关键信息与预测目标之间存在较长时间跨度的依赖关系时，模型需要能够将信息跨越多步传递。RNN 因梯度连乘导致的指数衰减或增长，难以捕捉长期依赖；LSTM/GRU 通过加法型细胞状态更新缓解此问题。

### 3.5 Seq2Seq 框架

编码器（Encoder）将可变长输入序列编码为固定长度的上下文向量（Context Vector），解码器（Decoder）从该向量自回归地生成输出序列。该框架是机器翻译、文本摘要等序列转换任务的标准范式。

### 3.6 注意力机制（Attention Mechanism）

解码器在每一步生成时，不依赖于单一的固定长度上下文向量，而是动态地计算输入序列所有位置隐藏状态的加权和，权重由当前解码状态与各输入位置的"相关性"决定。这是 Transformer 自注意力机制的直接前身。

### 3.7 状态空间模型（State Space Model, SSM）

源于控制论的连续时间线性系统：$\dot{h}(t) = A h(t) + B x(t), y(t) = C h(t)$，经离散化后可用于序列建模。SSM 以线性复杂度处理序列，并在长程依赖建模上展现了与注意力机制互补的优势。

## 4. 技术原理

### 4.1 简单 RNN 的前向与反向传播

**前向传播**：

$$h_t = \tanh(W_{hh} h_{t-1} + W_{xh} x_t + b_h)$$
$$y_t = \text{softmax}(W_{hy} h_t + b_y)$$

**BPTT 梯度**：设总损失 $\mathcal{L} = \sum_{t=1}^T \mathcal{L}_t$，对隐藏权重 $W_{hh}$ 的梯度为：

$$\frac{\partial \mathcal{L}}{\partial W_{hh}} = \sum_{t=1}^T \sum_{k=1}^{t} \frac{\partial \mathcal{L}_t}{\partial h_t} \left(\prod_{i=k+1}^{t} \frac{\partial h_i}{\partial h_{i-1}}\right) \frac{\partial h_k}{\partial W_{hh}}$$

关键项 $\prod_{i=k+1}^{t} \frac{\partial h_i}{\partial h_{i-1}}$ 中的雅可比矩阵的谱半径若显著小于 1 则梯度消失，若显著大于 1 则梯度爆炸。

### 4.2 梯度消失与梯度爆炸的数学分析

- **梯度消失**：当激活函数采用 Sigmoid/Tanh 时，$\sigma'(x) \leq 0.25$（Sigmoid）或 $\leq 1$（Tanh）。经过 $T$ 步连乘，梯度呈指数级衰减：$\|\frac{\partial \mathcal{L}_T}{\partial h_1}\| \propto \gamma^T$，其中 $\gamma < 1$。
- **梯度爆炸**：当权重矩阵 $W_{hh}$ 的谱半径 $\rho(W_{hh}) > 1$ 时，梯度呈指数增长。
- **应对策略**：梯度裁剪（Gradient Clipping）缓解爆炸；LSTM/GRU 的门控结构从根源上缓解消失。

### 4.3 LSTM 核心机制的数学推导

LSTM 的关键创新在于**细胞状态（Cell State）** $C_t$ 的加法更新机制：

**遗忘门**：$f_t = \sigma(W_f [h_{t-1}, x_t] + b_f)$
**输入门**：$i_t = \sigma(W_i [h_{t-1}, x_t] + b_i)$
**候选细胞状态**：$\tilde{C}_t = \tanh(W_C [h_{t-1}, x_t] + b_C)$
**细胞状态更新**：$C_t = f_t \odot C_{t-1} + i_t \odot \tilde{C}_t$
**输出门**：$o_t = \sigma(W_o [h_{t-1}, x_t] + b_o)$
**隐藏状态**：$h_t = o_t \odot \tanh(C_t)$

核心洞察：$\frac{\partial C_t}{\partial C_{t-1}} = f_t$。当 $f_t \approx 1$ 时，梯度可以无损地在细胞状态中传递，从根本上解决了梯度消失问题。这是 LSTM 最重要的设计思想。

### 4.4 GRU 的简化设计

GRU 合并了 LSTM 的遗忘门和输入门为**更新门** $z_t$，并引入**重置门** $r_t$：

**更新门**：$z_t = \sigma(W_z [h_{t-1}, x_t])$
**重置门**：$r_t = \sigma(W_r [h_{t-1}, x_t])$
**候选隐藏状态**：$\tilde{h}_t = \tanh(W [r_t \odot h_{t-1}, x_t])$
**隐藏状态更新**：$h_t = (1 - z_t) \odot h_{t-1} + z_t \odot \tilde{h}_t$

GRU 去除了独立的细胞状态，直接对隐藏状态进行线性插值式的更新。当 $z_t \approx 0$ 时，梯度可以跨时间步自由流动。

### 4.5 注意力机制的数学形式

**Bahdanau 注意力（Additive Attention）**：

$$e_{t,i} = v_a^T \tanh(W_a s_{t-1} + U_a h_i)$$
$$\alpha_{t,i} = \frac{\exp(e_{t,i})}{\sum_{j=1}^T \exp(e_{t,j})}$$
$$c_t = \sum_{i=1}^T \alpha_{t,i} h_i$$

其中 $s_{t-1}$ 是解码器状态，$h_i$ 是编码器第 $i$ 步的隐藏状态，$c_t$ 是上下文向量。

**Luong 注意力（Multiplicative Attention）**：

- Dot: $e_{t,i} = s_t^T h_i$
- General: $e_{t,i} = s_t^T W_a h_i$
- Concat: 与 Bahdanau 类似

### 4.6 CTC 损失函数

CTC 通过在标签集中引入空白符（blank），建立输入帧序列与目标标签序列之间的多对一映射关系。设输入序列长度为 $T$，目标标签序列长度为 $U$（$U \leq T$），则：

$$p(\mathbf{y} | \mathbf{x}) = \sum_{\pi \in \mathcal{B}^{-1}(\mathbf{y})} \prod_{t=1}^T p(\pi_t | \mathbf{x})$$

其中 $\mathcal{B}^{-1}(\mathbf{y})$ 是目标标签的所有合法对齐路径的集合。该求和通过**前向-后向算法（Forward-Backward Algorithm）** 进行动态规划计算，将 $\mathcal{O}(2^T)$ 的暴力枚举降至 $\mathcal{O}(TU)$。

### 4.7 状态空间模型的离散化

连续时间 SSM：$h'(t) = A h(t) + B x(t), \quad y(t) = C h(t)$

通过零阶保持（Zero-Order Hold, ZOH）离散化：

$$\bar{A} = \exp(\Delta \cdot A), \quad \bar{B} = (\Delta A)^{-1}(\exp(\Delta \cdot A) - I) \cdot \Delta B$$

得到离散递推：$h_t = \bar{A} h_{t-1} + \bar{B} x_t, \quad y_t = C h_t$

**Mamba 的选择性机制**：使参数 $B, C, \Delta$ 依赖于输入 $x_t$（即 $B(x_t), C(x_t), \Delta(x_t)$），打破了 LTI（线性时不变）假设，使 SSM 能够根据输入内容动态调整状态更新行为。同时，通过**并行扫描算法（Parallel Scan）** 在 GPU 上高效实现选择性 SSM 的计算。

## 6. 优势与局限

### 优势

1. **处理变长序列**：RNN 天然支持任意长度的输入和输出序列，不受固定维度限制
2. **参数共享**：所有时间步共享同一组权重参数，模型参数量与序列长度无关，具有极高的参数效率
3. **理论上的图灵完备性**：RNN 在理论上可以模拟任意算法（Siegelmann & Sontag, 1995）
4. **长期记忆能力**：LSTM/GRU 的门控机制使其在实践中能够建模数百步跨度的依赖关系
5. **线性推理复杂度**：每个时间步的推理计算量为 $\mathcal{O}(1)$，推理效率高于自注意力（逐 token 解码时）
6. **状态空间模型突破**：Mamba 等 SSM 实现了训练和推理均为线性复杂度的序列建模

### 局限

1. **串行计算瓶颈**：训练时无法并行化（逐时间步依赖），序列长度增加导致训练时间线性增长
2. **长期记忆仍不完美**：即使 LSTM，在超过数百步的依赖关系上也存在退化
3. **Transformer 的取代**：自注意力机制在建模全局依赖和训练并行化方面全面超越 RNN
4. **训练不稳定**：RNN 容易遭遇梯度消失/爆炸，需要梯度裁剪和精心设计的初始化
5. **信息压缩瓶颈**：单向量隐藏状态在编码长序列时存在信息损失

## 8. 与其他技术关系

- **与深度学习基础的关系**：RNN 的训练依赖于反向传播和梯度下降；序列模型中的梯度消失/爆炸问题是深层网络梯度问题的极端体现。训练稳定性依赖于梯度裁剪、层级归一化（LayerNorm）和正确的参数初始化
- **与 Transformer 的关系**：RNN 的固有缺陷（串行计算、长程依赖退化）直接催生了 Transformer；Seq2Seq 的注意力机制是 Transformer 自注意力的直接前身，理解注意力从 Seq2Seq 语境中的诞生过程是理解 Transformer 动机的关键
- **与卷积神经网络（CNN）的关系**：时间卷积网络（TCN）在一定程度上与 RNN 互为替代；CNN-LSTM 混合架构在视频理解和语音识别中广泛应用
- **与数学基础的关系**：BPTT 的数学核心是链式法则和雅可比矩阵分析；CTC 的理论基础是动态规划和隐马尔可夫模型的扩展；SSM 源自控制论中的状态空间理论
- **与预训练语言模型的关系**：虽然现代 LLM 以 Transformer 为主，但 SSM（Mamba）正成为新一代高效 LLM 架构的重要方向
- **与强化学习的关系**：RNN 在部分可观测马尔可夫决策过程（POMDP）中作为策略网络的记忆组件

## 9. 前沿发展

### 9.1 状态空间模型的崛起

以 Mamba 为代表的选择性 SSM，通过输入依赖的参数化机制和高效的并行扫描算法，实现了线性时间的序列建模。Mamba-2 进一步通过矩阵变换统一了 SSM 和线性注意力的理论框架。Jamba 等混合架构探索了 SSM 与注意力机制的融合路径，在长上下文任务中展现了优异性能。

### 9.2 线性 RNN 复兴

RWKV、RetNet、HGRN 等线性 RNN 架构，通过在循环连接中引入线性注意力或门控线性变换，结合 Chunk-wise 并行训练策略，实现了训练并行化与推理线性的双重优势。这类架构正在缩小与 Transformer 的性能差距。

### 9.3 大规模 SSM 语言模型

Mamba、Mamba-2、Jamba 等架构正在被扩展到数十亿参数规模，挑战 Transformer 在大语言模型领域的主导地位。研究焦点包括：混合专家（MoE）与 SSM 的结合、长上下文扩展策略、与 KV-Cache 推理优化的兼容性。

### 9.4 RNN 与 Transformer 的融合

Hyena、H3、Monarch Mixer 等工作探索了将卷积、注意力和循环结构融合的新型序列混合器。Mamba-2 的 SSD（State Space Duality）框架揭示了 SSM 和线性注意力在矩阵乘法语义下的深层等价关系。

### 9.5 硬件适配与高效推理

针对 SSM 和线性 RNN 的专用 CUDA 内核优化（如 Mamba 的选择性扫描内核），以及在边缘设备上的轻量化部署，正成为工程实践的重要方向。FlashAttention 的衍生工作也在探索类似优化应用于 SSM 场景。

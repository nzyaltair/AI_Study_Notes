# 视觉模型蒸馏

## 1. 概述

视觉模型蒸馏将知识蒸馏应用于计算机视觉领域，包括图像分类、目标检测、语义分割等任务。视觉模型（CNN/ViT）的蒸馏是知识蒸馏最早和最广泛的应用领域之一——FitNets、Attention Transfer、RKD 等经典方法均首先在视觉任务上验证。

视觉蒸馏的特点：中间层特征丰富（特征图、注意力图），适合特征蒸馏和关系蒸馏；视觉任务对实时性要求高，压缩需求强烈。

## 2. 发展历史

| 年代 | 里程碑 | 核心意义 |
|:---|:---|:---|
| 2014 | FitNets (Romero et al.) | CNN 中间层 hint 训练 |
| 2015 | Hinton KD 在 CNN 上应用 | 软标签蒸馏用于图像分类 |
| 2016 | Attention Transfer (Zagoruyko) | 注意力图转移 |
| 2017 | NST (Huang & Wang) | Gram 矩阵匹配激活分布 |
| 2017 | YOLO 蒸馏 | 检测模型蒸馏 |
| 2018 | 自蒸馏 (BAN) | 同架构自蒸馏在视觉上验证 |
| 2019 | RKD (Park et al.) | 距离和角度关系蒸馏 |
| 2020 | CRD (Tian et al.) | 对比表示蒸馏 |
| 2020 | Defeat (Chen et al.) | 检测专用蒸馏 |
| 2021 | ReviewKD (Chen et al.) | 多级特征回顾蒸馏 |
| 2022 | ViT 蒸馏 | Vision Transformer 蒸馏 |
| 2023 | DeiT III | ViT 数据高效训练蒸馏 |

## 3. 核心概念

### 3.1 CNN 蒸馏的层级特征

CNN 的不同层级捕获不同抽象程度的特征：

| 层级 | 特征类型 | 蒸馏方式 |
|:---|:---|:---|
| 浅层（Conv1-2） | 边缘、纹理 | 特征图对齐 |
| 中层（Conv3-4） | 局部模式、部件 | 注意力图转移 |
| 深层（Conv5+） | 语义概念 | 响应蒸馏 + 关系蒸馏 |

### 3.2 ViT 蒸馏的特殊性

Vision Transformer 的蒸馏与 CNN 有所不同：
- **CLS Token 蒸馏**：ViT 的 CLS token 作为分类特征
- **Patch Token 蒸馏**：所有 patch token 的特征对齐
- **注意力蒸馏**：ViT 的多头自注意力矩阵对齐
- **DeiT 方式**：在 ViT 中加入 distillation token，教师为 CNN 或 ViT

### 3.3 密集预测任务蒸馏

检测和分割任务中，特征图的空间信息至关重要：
- **特征图蒸馏**：逐空间位置对齐特征
- **ROI 蒸馏**：对检测框区域特征进行蒸馏
- ** logits 蒸馏**：逐像素分类 logits 蒸馏

## 4. 技术原理

### 4.1 FitNets 在 CNN 上的应用

$$\mathcal{L}_{FitNets} = \| \phi(F^S_{guided}) - F^T_{hint} \|_2^2$$

其中 $F$ 为 CNN 中间层特征图，$\phi$ 为 1×1 卷积适配通道数。

### 4.2 Attention Transfer 的注意力图

Zagoruyko & Komodakis 定义了三种注意力图：

$$A_{sum}(F) = \sum_{c=1}^{C} |F_c|, \quad A_{sum^2}(F) = \sum_{c=1}^{C} |F_c|^2, \quad A_{max}(F) = \max_{c} |F_c|$$

注意力图 $A \in \mathbb{R}^{H \times W}$ 编码了模型在空间上"看哪里"的信息。

### 4.3 ReviewKD——多级特征回顾

Chen et al. (2021) 提出学生向教师"回顾"——将学生多级特征融合后与教师对齐：

1. 学生浅层特征通过融合模块向深层传递
2. 融合后的特征与教师对应层对齐
3. 利用多尺度信息增强蒸馏效果

### 4.4 DeiT——ViT 蒸馏

Touvron et al. (2021) 在 ViT 中引入蒸馏 token：

```
输入 → Patch Embedding + CLS Token + Dist Token → Transformer → 输出
```

- CLS Token 的输出用于标准分类损失
- Dist Token 的输出用于蒸馏损失（软标签）
- 教师：RegNet（CNN）或教师 ViT

**关键发现**：CNN 教师比 ViT 教师在 ViT 蒸馏中效果更好——CNN 的归纳偏置帮助 ViT 学习更好的空间表示。

### 4.5 检测蒸馏（FGD/Defeat）

目标检测蒸馏的特殊设计：

**前景感知蒸馏**：只对有目标的区域蒸馏，忽略背景：

$$\mathcal{L}_{det} = \sum_{i \in \text{foreground}} \| F^S_i - \phi(F^T_i) \|_2^2$$

**注意力引导**：用教师的注意力图加权学生的特征蒸馏。

## 5. 方法对比

| 方法 | 任务 | 知识类型 | 教师架构 | 特点 |
|:---|:---|:---|:---|:---|
| FitNets | 分类 | 特征 | CNN | hint 训练 |
| AT | 分类 | 注意力 | CNN | 注意力图转移 |
| RKD | 分类/检索 | 关系 | CNN | 距离+角度 |
| CRD | 分类/检索 | 关系 | CNN | 对比学习 |
| ReviewKD | 分类 | 特征 | CNN | 多级回顾 |
| DeiT | 分类 | 响应+特征 | CNN/ViT | dist token |
| FGD | 检测 | 特征 | CNN | 前景感知 |
| Freenet | 分割 | 特征 | CNN | 逐像素蒸馏 |

## 6. 优势与局限

### 优势
- **视觉模型压缩需求强烈**：实时检测/分割对延迟敏感
- **中间层特征丰富**：适合多种蒸馏方案
- **可叠加量化/剪枝**：蒸馏+量化+剪枝组合压缩
- **跨架构蒸馏**：CNN→ViT 或 ViT→CNN 蒸馏

### 局限
- **密集预测任务蒸馏复杂**：检测/分割的蒸馏方案设计复杂
- **大分辨率特征图开销大**：高分辨率特征对齐消耗显存
- **ViT 蒸馏仍在发展**：ViT 的蒸馏不如 CNN 成熟

## 7. 应用场景

- **移动端视觉**：手机上的图像分类、人脸识别
- **自动驾驶**：实时目标检测压缩
- **视频监控**：实时分割和检测
- **AR/VR**：低延迟视觉推理
- **工业检测**：边缘设备上的缺陷检测

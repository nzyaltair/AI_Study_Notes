# 图像分割

## 1. 概述

图像分割（Image Segmentation）是计算机视觉中比目标检测更精细的任务：为图像中的每个像素分配一个类别标签或实例标识。如果说检测回答"物体在哪里"，那么分割回答"每个像素属于什么"。

图像分割按任务粒度分为三个层次：
- **语义分割**（Semantic Segmentation）：为每个像素分配类别标签，同类物体不区分个体
- **实例分割**（Instance Segmentation）：在语义分割基础上区分同类物体的不同实例
- **全景分割**（Panoptic Segmentation）：统一语义分割和实例分割，覆盖所有像素

图像分割在医学影像、自动驾驶场景理解、遥感分析等领域有广泛应用。

## 3. 核心概念

### 3.1 三种分割任务对比

| 任务 | 输出 | 同类区分 | 背景处理 |
|:---|:---|:---|:---|
| 语义分割 | 每像素类别 | 不区分 | 归为"背景"类 |
| 实例分割 | 每像素实例 ID | 区分 | 不处理 |
| 全景分割 | 每像素类别 + 实例 | 可数类区分 | 归为"stuff" |

### 3.2 评估指标

**mIoU（平均交并比）**——语义分割标准指标：

$$\text{IoU}_c = \frac{TP_c}{TP_c + FP_c + FN_c}, \quad \text{mIoU} = \frac{1}{C} \sum_{c=1}^{C} \text{IoU}_c$$

**AP（Average Precision）**——实例分割标准指标，使用 Mask IoU 替代 Box IoU 计算。

**PQ（Panoptic Quality）**——全景分割指标：

$$PQ = \underbrace{\frac{\sum_{(p,g) \in TP} \text{IoU}(p,g)}{|TP| + \frac{1}{2}|FP| + \frac{1}{2}|FN|}}_{\text{识别质量}} \times \underbrace{\frac{|TP|}{|TP| + \frac{1}{2}|FP| + \frac{1}{2}|FN|}}_{\text{分割质量}}$$

### 3.3 数据集

| 数据集 | 任务 | 类别 | 特点 |
|:---|:---|:---|:---|
| PASCAL VOC | 语义/实例 | 21 | 早期基准 |
| COCO | 实例/全景 | 80/133 | 检测分割标准 |

## 4. 技术原理

### 4.1 全卷积网络（FCN）

FCN 是深度学习语义分割的开创性工作，将分类 CNN 的全连接层替换为卷积层，实现像素级预测：

```
输入图像 → CNN backbone（逐步下采样）→ 特征图
                                          ↓
                                    反卷积/上采样
                                          ↓
                                    像素级分类图
```

**问题**：下采样丢失空间细节，边界模糊。

### 4.2 U-Net：编码器-解码器 + 跳跃连接

```
输入 → [Encoder: 下采样] → 瓶颈层
         ↓ (跳跃连接)        ↑
         [Decoder: 上采样] → 输出
```

- **编码器**：逐步下采样提取语义特征
- **解码器**：逐步上采样恢复空间分辨率
- **跳跃连接**：将编码器特征直接拼接到解码器对应层，补充空间细节

U-Net 因其简洁高效成为医学图像分割的标准架构。

### 4.3 DeepLab 系列：空洞卷积 + ASPP

**空洞卷积（Dilated/Atrous Convolution）**：

在不增加参数量和计算量的前提下扩大感受野：

$$O(i,j) = \sum_m \sum_n I(i + r \cdot m, j + r \cdot n) \cdot K(m,n)$$

其中 $r$ 为空洞率（dilation rate），$r=1$ 时为普通卷积。

**ASPP（Atrous Spatial Pyramid Pooling）**：

并行使用不同空洞率（如 $r=6, 12, 18$）的卷积捕获多尺度上下文：

```
输入特征 → [1×1 conv]  ─┐
         → [3×3, r=6]   ├─ 拼接 → 1×1 conv → 输出
         → [3×3, r=12]  │
         → [3×3, r=18]  │
         → Global AvgPool ┘
```

### 4.4 Mask R-CNN：实例分割

在 Faster R-CNN 基础上增加 mask 预测分支：

```
图像 → backbone → RPN → RoI Align → 分类头
                                  → 回归头
                                  → Mask 分支（FCN 生成二值掩码）
```

每个 RoI 预测 $K$ 个 $m \times m$ 的 mask（$K$ 为类别数），使用逐像素 Sigmoid 二分类。

### 4.5 全景分割

将图像分为**可数物体（things）**和**不可数背景（stuff）**：

- **things**：人、车、动物等可数目标 → 实例分割
- **stuff**：天空、道路、草地等不可数区域 → 语义分割

**Panoptic FPN**：使用共享 FPN backbone，同时输出实例和语义分割结果。

### 4.6 Transformer 分割

**Mask2Former**：统一的 Transformer 分割框架：

- 将分割建模为 mask 预测问题
- 使用 $N$ 个 object query 预测 $N$ 个 mask
- 二分图匹配损失训练
- 统一处理语义/实例/全景分割

### 4.7 SAM（Segment Anything Model）

Meta 发布的通用分割基础模型，支持多种提示方式：

```
提示（点/框/文本/掩码）→ Prompt Encoder
                              ↓
图像 → Image Encoder → Image Embedding
                              ↓
                    Mask Decoder → 掩码预测
```

**核心特点**：
- 零样本分割：无需微调即可分割任意物体
- 多种提示：点击、框选、文本提示
- 模糊感知：一个提示点可能对应多个层级的物体，SAM 输出多个 mask
- SA-1B 数据集：10 亿级掩码，最大分割数据集

## 6. 优势与局限

### 优势

1. **像素级精度**：比检测框更精细的目标边界
2. **场景理解**：全景分割提供完整场景理解
3. **医学应用**：器官/病灶的精确分割对诊断至关重要
4. **基础模型突破**：SAM 实现零样本分割，大幅降低应用门槛

### 局限

1. **计算密集**：像素级预测比检测框计算量大
2. **边界精度**：细小物体和复杂边界仍然困难
3. **类别覆盖**：传统方法限于预定义类别集
4. **标注成本高**：像素级标注远比检测框昂贵

## 7. 应用场景

## 9. 前沿发展

- **视频分割**：SAM 2 将 SAM 扩展到视频时序分割

- **高效分割**：轻量化分割网络用于移动端实时分割

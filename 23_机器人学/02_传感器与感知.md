# 传感器与感知

## 一句话理解

从多种传感器（里程计、IMU、激光雷达、相机）获取数据并融合估计机器人自身状态和环境状态——是机器人自主行动的前提。

## 背景与发展

机器人感知技术随传感器硬件进步而发展。从早期的编码器+接触传感器，到IMU、激光雷达、深度相机，再到多传感器融合和深度学习驱动的视觉感知，感知能力持续提升。

| 年代 | 里程碑 | 意义 |
|:---|:---|:---|
| 1960s | 编码器+电位计 | 早期关节角度测量 |
| 1980s | 超声波传感器 | 简单避障 |
| 1990s | 激光雷达（LIDAR） | 精确距离感知 |
| 2000s | MEMS IMU普及 | 低成本惯性测量 |
| 2010 | Kinect深度相机 | 低成本RGB-D感知 |
| 2015 | 深度学习视觉感知 | 语义分割、目标检测 |
| 2020s | 多模态融合 | 视觉+激光+IMU融合 |

## 核心概念

### 常见传感器

| 传感器 | 测量量 | 优缺点 |
|:---|:---|:---|
| 轮式编码器 | 轮子转角→里程 | 短期精度高，长期累积漂移 |
| IMU | 加速度+角速度 | 高频更新，但漂移严重 |
| 激光雷达 | 距离点云 | 精度高，但成本高、无色彩 |
| 单目相机 | RGB图像 | 信息丰富，但无直接深度 |
| 立体相机 | RGB+视差→深度 | 有深度，但受光照影响 |
| RGB-D相机 | RGB+深度 | 直观，但范围有限 |
| 超声波 | 距离 | 成本低，但精度差、速度慢 |
| GPS | 全局位置 | 室外有效，室内不可用 |

### 里程计（Odometry）

通过积分轮子运动估计机器人位姿。差速驱动机器人的运动学模型：

$$\dot{x} = v \cos\theta, \quad \dot{y} = v \sin\theta, \quad \dot{\theta} = \omega$$

其中 $v$ 为线速度，$\omega$ 为角速度。里程计短期精度高但长期累积误差。

### IMU（惯性测量单元）

包含加速度计（测量线加速度）和陀螺仪（测量角速度），有些还包含磁力计。

IMU积分：
- 角速度积分 → 姿态
- 加速度二次积分 → 位置（但漂移严重）

### 传感器融合

不同传感器互补——IMU高频但漂移，里程计/视觉低频但稳定。融合方法：

| 方法 | 原理 | 适用 |
|:---|:---|:---|
| 互补滤波 | 频域互补 | 简单场景 |
| 卡尔曼滤波（KF） | 最优线性估计 | 线性高斯系统 |
| 扩展卡尔曼滤波（EKF） | 线性化非线性系统 | 非线性系统（主流） |
| 无迹卡尔曼滤波（UKF） | 无迹变换 | 强非线性系统 |
| 粒子滤波 | 蒙特卡洛采样 | 非高斯、多模分布 |

### 卡尔曼滤波

最优线性估计器。状态空间模型：

$$\mathbf{x}_k = A\mathbf{x}_{k-1} + B\mathbf{u}_k + \mathbf{w}_k$$
$$\mathbf{z}_k = H\mathbf{x}_k + \mathbf{v}_k$$

**预测**：

$$\hat{\mathbf{x}}_{k|k-1} = A\hat{\mathbf{x}}_{k-1} + B\mathbf{u}_k$$
$$P_{k|k-1} = AP_{k-1}A^T + Q$$

**更新**：

$$K_k = P_{k|k-1}H^T(HP_{k|k-1}H^T + R)^{-1}$$
$$\hat{\mathbf{x}}_k = \hat{\mathbf{x}}_{k|k-1} + K_k(\mathbf{z}_k - H\hat{\mathbf{x}}_{k|k-1})$$
$$P_k = (I - K_k H)P_{k|k-1}$$

EKF通过雅可比线性化将KF推广到非线性系统。

### 深度学习感知

现代机器人广泛使用深度学习进行视觉感知：
- **目标检测**（YOLO/DETR）：识别环境中的物体
- **语义分割**：像素级场景理解
- **深度估计**：单目深度预测
- **光流**：运动估计

## 应用场景

| 场景 | 传感器配置 |
|:---|:---|
| 室内移动机器人 | 激光雷达+IMU+里程计 |
| 无人机 | IMU+GPS+相机 |
| 自动驾驶 | 激光雷达+多相机+雷达+GPS+IMU |
| 机械臂 | 关节编码器+力/力矩传感器+视觉 |
| 人形机器人 | IMU+深度相机+力传感器 |

## 优势与不足

### 优势
- 多传感器互补，提高感知鲁棒性
- 卡尔曼滤波提供数学最优估计
- 深度学习增强语义理解能力

### 不足
- 传感器标定和同步复杂
- 融合算法计算开销大
- 极端环境（弱光、雨雪）下传感器失效

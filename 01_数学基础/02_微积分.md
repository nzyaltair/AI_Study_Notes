# 微积分

## 1. 概述

微积分刻画函数的局部变化行为。在 AI 中，梯度指向函数值增长最快的方向，负梯度方向即为参数优化的方向；链式法则将复合函数的梯度计算分解为每层的局部梯度，使得深层神经网络的训练成为可能。自动微分技术将链式法则系统化、自动化，是 PyTorch/TensorFlow 等深度学习框架的底层核心。

## 2. 发展历史

| 年代 | 里程碑 | 意义 |
|------|--------|------|
| 1666-84 | 牛顿与莱布尼茨独立发明微积分 | 现代数学的起点 |
| 1821 | Cauchy 严格化 ($\epsilon$-$\delta$ 语言) | 微积分的严格基础 |
| 1847 | Cauchy 提出梯度下降 | 一阶优化方法 |
| 1860s | Weierstrass 多变量微积分 | 多元函数微分学 |
| 1947 | Wengert 自动微分前向模式 | 自动微分的开创 |
| 1970 | Linnainmaa 反向模式自动微分 | 反向传播的算法雏形 |
| 1986 | Rumelhart, Hinton & Williams 推广反向传播 | 神经网络训练的突破 |
| 2010s | PyTorch/TensorFlow 自动微分引擎 | 深度学习框架普及 |
| 2018 | Baydin et al. 自动微分综述 | 统一自动微分理论框架 |

## 3. 核心概念

### 导数与微分

- **导数定义**：$f'(x_0) = \lim_{h \to 0} \frac{f(x_0 + h) - f(x_0)}{h}$，表示瞬时变化率，几何意义为切线斜率
- **基本公式**：$(x^n)' = nx^{n-1}$，$(e^x)' = e^x$，$(\ln x)' = 1/x$，$(\sin x)' = \cos x$，$(\cos x)' = -\sin x$
- **乘积法则**：$(fg)' = f'g + fg'$
- **商法则**：$(f/g)' = (f'g - fg')/g^2$

### 偏导数与梯度

- **偏导数**：$\frac{\partial f}{\partial x_i}$，其他变量固定时函数对某一变量的变化率
- **梯度**：$\nabla f(\mathbf{x}) = \left[\frac{\partial f}{\partial x_1}, \dots, \frac{\partial f}{\partial x_n}\right]^T$，指向函数值增长最快的方向
- **方向导数**：$D_{\mathbf{u}} f = \nabla f \cdot \mathbf{u}$，沿任意方向的变化率

### 链式法则

- **单变量**：$\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx}$
- **多变量**：$\frac{\partial y}{\partial x_j} = \sum_{i=1}^m \frac{\partial y}{\partial u_i} \cdot \frac{\partial u_i}{\partial x_j}$

链式法则是反向传播算法的数学核心：将复合函数的梯度分解为每层的局部梯度乘积。

### 雅可比与黑塞矩阵

- **雅可比矩阵**：$J_{ij} = \frac{\partial f_i}{\partial x_j}$，向量值函数 $\mathbf{f}: \mathbb{R}^n \to \mathbb{R}^m$ 的一阶导数
- **黑塞矩阵**：$H_{ij} = \frac{\partial^2 f}{\partial x_i \partial x_j}$，标量函数的二阶导数，对称矩阵
- **正定性判定**：$H$ 正定 → 局部极小；负定 → 局部极大；不定 → 鞍点

## 4. 技术原理

### 向量与矩阵微分

| 函数形式 | 梯度结果 | 条件 |
|---------|---------|------|
| $f(\mathbf{x}) = \mathbf{a}^T \mathbf{x}$ | $\nabla f = \mathbf{a}$ | $\mathbf{a}$ 常数 |
| $f(\mathbf{x}) = \mathbf{x}^T \mathbf{x}$ | $\nabla f = 2\mathbf{x}$ | — |
| $f(\mathbf{x}) = \mathbf{x}^T \mathbf{A} \mathbf{x}$ | $\nabla f = (\mathbf{A} + \mathbf{A}^T)\mathbf{x}$ | 一般情形 |
| $f(\mathbf{x}) = \|\mathbf{A}\mathbf{x}-\mathbf{b}\|^2$ | $\nabla f = 2\mathbf{A}^T(\mathbf{A}\mathbf{x}-\mathbf{b})$ | — |

### Taylor 展开

$$f(\mathbf{x}) \approx f(\mathbf{a}) + \nabla f(\mathbf{a})^T(\mathbf{x}-\mathbf{a}) + \frac{1}{2}(\mathbf{x}-\mathbf{a})^T \mathbf{H} (\mathbf{x}-\mathbf{a})$$

- 一阶近似：梯度下降的理论基础
- 二阶近似：牛顿法的理论基础

### 反向传播的数学推导

对两层神经网络，损失 $L = \frac{1}{2}\|\mathbf{y} - \hat{\mathbf{y}}\|^2$，激活函数 $\sigma$：

- 前向传播：$\mathbf{z}^{(l)} = \mathbf{W}^{(l)}\mathbf{a}^{(l-1)} + \mathbf{b}^{(l)}$，$\mathbf{a}^{(l)} = \sigma(\mathbf{z}^{(l)})$
- 输出层梯度：$\boldsymbol{\delta}^{(L)} = (\hat{\mathbf{y}} - \mathbf{y}) \odot \sigma'(\mathbf{z}^{(L)})$
- 隐藏层梯度（链式法则）：$\boldsymbol{\delta}^{(l)} = (\mathbf{W}^{(l+1)T} \boldsymbol{\delta}^{(l+1)}) \odot \sigma'(\mathbf{z}^{(l)})$
- 权重梯度：$\frac{\partial L}{\partial \mathbf{W}^{(l)}} = \boldsymbol{\delta}^{(l)} \mathbf{a}^{(l-1)T}$

### 自动微分

自动微分（AD）不同于数值微分（差分近似）和符号微分（表达式展开），它精确地计算导数：

- **前向模式**：沿计算图从输入到输出传播导数，适合输入维度小于输出维度的场景
- **反向模式**（Reverse-mode AD）：沿计算图从输出到输入传播梯度，适合输出维度远小于输入维度的场景（如损失函数标量输出 → 数百万参数梯度）
- **计算图**：将计算过程表示为有向无环图（DAG），节点为运算，边为数据流
- PyTorch 的 `autograd` 和 TensorFlow 的 `GradientTape` 均基于反向模式 AD

### 数值稳定性

- **Log-Sum-Exp 技巧**：$\text{softmax}(x_i) = \frac{e^{x_i - \max(x)}}{\sum_j e^{x_j - \max(x)}}$，防止指数溢出
- **梯度裁剪**：$\mathbf{g} \leftarrow \frac{c}{\max(\|\mathbf{g}\|, c)} \cdot \mathbf{g}$，限制梯度最大范数，防止梯度爆炸
- **梯度消失/爆炸**：深层网络中梯度连乘导致，解决方案包括残差连接、合适的权重初始化、归一化层

## 5. 关键方法/模型

### 梯度下降

- **Cauchy (1847)**: *Méthode générale pour la résolution des systèmes d'équations simultanées* — 最早提出梯度下降
- 沿负梯度方向迭代更新参数，是一阶优化的基础

### 反向传播算法

- **Rumelhart, Hinton & Williams (1986)**: *Learning Representations by Back-propagating Errors* — 系统阐述反向传播
- **Linnainmaa (1970)**: 最早实现反向模式自动微分
- 反向传播 = 链式法则 + 动态规划（避免重复计算）

### 自动微分框架

- **Baydin et al. (2018)**: *Automatic Differentiation in Machine Learning: a Survey* — 统一 AD 理论框架
- PyTorch autograd：动态计算图，define-by-run
- JAX：函数式自动微分，支持 `grad`、`vmap`、`jit` 组合

## 6. 优势与局限

**优势**：
- 链式法则使任意深度网络的可训练成为可能
- 自动微分实现精确导数计算，无需手动推导
- 反向模式 AD 对高维参数的梯度计算效率高（$O(n)$ 一次遍历）

**局限**：
- 反向模式需要存储中间激活值，内存开销大（需梯度检查点技术缓解）
- 二阶导数（Hessian）计算代价过高，实际中极少直接使用
- 数值不稳定问题（梯度爆炸/消失）需要工程手段处理
- 非光滑函数（如 ReLU 在 0 处）需要次梯度或次微分处理

## 7. 应用场景

- **[[深度学习]]**：反向传播算法、自动微分框架（PyTorch autograd、JAX）
- **[[优化方法]]**：梯度下降及其变体（SGD、Adam 等）依赖梯度计算
- **[[Transformer]]**：注意力权重的梯度计算、位置编码的导数
- **[[概率与统计]]**：最大似然估计的梯度优化
- **物理信息神经网络（PINN）**：将微分方程约束嵌入损失函数

## 8. 与其他技术关系

- 依赖 [[线性代数]] 中的矩阵运算（矩阵微积分）
- 为 [[优化方法]] 提供梯度计算的理论基础
- 为 [[概率与统计]] 中最大似然估计提供优化工具
- 链式法则是 [[深度学习]] 反向传播算法的数学核心
- 自动微分是 PyTorch/TensorFlow 等深度学习框架的底层技术

## 9. 前沿发展

- **高阶自动微分**：JAX 支持任意阶微分，推动元学习（MAML）和二阶优化方法
- **隐式微分**：对不动点/均衡问题求导，用于隐式层和均衡模型
- **可微编程**：将整个程序（包括控制流、循环）变为可微的，推动可微物理模拟、可微渲染
- **梯度检查点（Gradient Checkpointing）**：以时间换空间，减少反向传播内存开销
- **流水线并行中的微批次梯度累积**：大模型分布式训练中的梯度计算策略

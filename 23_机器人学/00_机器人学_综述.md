# 机器人学综述

## 1. 概述

机器人学（Robotics）是研究机器人感知、决策和控制的综合学科，融合了人工智能、机械工程、控制理论和计算机科学。机器人系统需要在物理世界中自主感知环境、规划行为并执行动作，是AI从虚拟世界走向物理世界的关键桥梁。

- **解决的问题**：如何让机器人在不确定的物理环境中自主感知、决策和行动。
- **核心价值**：机器人是AI的物理具身（Embodiment），将智能从数字世界延伸到物理世界。机器人学推动了感知、规划、控制和学习的深度融合。
- **知识体系范围**：运动学与动力学 → 传感器与感知 → 运动规划 → SLAM → 机器人控制与ROS。

## 3. 核心概念

### 感知-决策-行动循环

机器人系统的核心架构是**感知-决策-行动循环**（Sense-Plan-Act Cycle）：

```
传感器数据 → 感知/估计 → 规划/决策 → 控制/执行 → 机器人运动
    ↑                                                        │
    └────────────────── 环境反馈 ──────────────────────────┘
```

- **感知**：从传感器数据估计机器人状态和环境状态
- **决策**：根据当前状态和目标规划行为
- **行动**：将行为转化为电机指令执行

参见各子主题笔记。

### 运动学 vs 动力学

| 维度 | 运动学 | 动力学 |
|:---|:---|:---|
| 研究对象 | 位置/速度/加速度（无受力） | 力/力矩与运动的关系 |
| 核心问题 | 正/逆运动学 | 牛顿-欧拉方程、拉格朗日方程 |
| 典型应用 | 轨迹规划 | 力控、阻抗控制 |

参见 [[01_运动学与动力学]]。

### SLAM

同步定位与地图构建（Simultaneous Localization and Mapping）——机器人在未知环境中同时估计自身位姿和构建环境地图。是自主移动机器人的核心技术。参见 [[04_SLAM同步定位与地图构建]]。

### 运动规划

在考虑运动学/动力学约束的前提下，规划从起点到目标点的无碰撞路径/轨迹。参见 [[03_运动规划]]。

## 4. 与其他方向关系

- **与强化学习的关系**：RL是机器人学习运动策略的核心方法；机器人是RL在物理世界的最大应用场景。参见 [[13_强化学习]]。
- **与搜索与规划的关系**：机器人任务规划是符号AI规划的应用；运动规划与A*/RRT等搜索算法直接相关。参见 [[22_符号AI与知识表示/05_搜索算法]]、[[22_符号AI与知识表示/06_规划算法]]。
- **与计算机视觉的关系**：视觉感知是机器人感知的核心组件——目标检测、深度估计、视觉SLAM。参见 [[15_计算机视觉]]。
- **与大模型的关系**：大模型驱动的具身智能（VLA模型、RT-2）正在改变机器人学的范式。参见 [[14_多模态AI]]。

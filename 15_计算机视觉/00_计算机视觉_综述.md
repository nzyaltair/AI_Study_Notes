# 计算机视觉综述

## 1. 概述

计算机视觉（Computer Vision, CV）是人工智能的重要分支，研究如何让计算机从图像和视频数据中**获取、处理、理解和分析视觉信息**，实现对视觉世界的感知与理解。其核心目标是赋予机器"看"的能力——从像素到语义，从感知到认知。

- **解决的问题**：如何让机器自动提取图像/视频中的结构化信息，包括物体识别、定位、分割、场景理解、三维重建和运动分析等。覆盖自动驾驶、医学影像、安防监控、工业检测、增强现实等广泛场景。
- **核心价值**：计算机视觉是连接物理世界与数字世界的桥梁。它是机器感知能力的基石，也是多模态 AI、具身智能、机器人等前沿领域的前提。从 2012 年 AlexNet 开启深度学习革命以来，CV 一直是 AI 研究最活跃、应用最成熟的领域之一。
- **知识体系范围**：图像处理基础 → 传统视觉特征 → 图像分类 → 目标检测 → 图像分割 → 视觉 Transformer → 视频分析 → 三维视觉 → 自监督视觉学习 → 工程实践与前沿。

## 3. 核心概念

### 3.1 图像表示

图像在计算机中的基本表示形式：

- **灰度图像**：二维矩阵 $I(x,y) \in [0, 255]$，每个像素值表示亮度
- **彩色图像**：三维张量 $I(x,y,c) \in \mathbb{R}^{H \times W \times 3}$，常用 RGB 色彩空间
- **深度图像**：每个像素存储场景深度信息，用于 3D 视觉
- **频域表示**：通过傅里叶变换将图像从空间域转换到频率域

### 3.2 特征（Feature）

特征是图像中具有区分性的信息模式，是视觉理解的基石：

- **底层特征**：边缘、角点、纹理、颜色分布
- **中层特征**：局部区域描述子（SIFT/HOG/ORB）
- **高层特征**：语义概念（物体类别、场景类型）

传统 CV 依赖手工设计特征；深度学习通过端到端训练自动学习层次化特征。

### 3.3 视觉任务层级

计算机视觉任务按理解深度递进：

| 层级 | 任务 | 输出 | 说明 |
|:---|:---|:---|:---|
| 低层视觉 | 滤波、增强、超分辨率 | 图像 | 像素级处理 |
| 中层视觉 | 特征提取、分割 | 区域/掩码 | 区域级分析 |
| 高层视觉 | 分类、检测、识别 | 类别/位置 | 语义级理解 |
| 视频视觉 | 跟踪、动作识别 | 轨迹/动作 | 时序级分析 |
| 三维视觉 | 重建、SLAM | 3D 模型/位姿 | 空间级理解 |

### 3.4 评估指标

| 任务 | 核心指标 | 说明 |
|:---|:---|:---|
| 图像分类 | Top-1/Top-5 Accuracy | 分类正确率 |
| 目标检测 | mAP (mean Average Precision) | 多类别平均精度，常用 mAP@0.5 和 mAP@0.5:0.95 |
| 图像分割 | mIoU (mean Intersection over Union) | 预测与真实掩码的交并比 |
| 实例分割 | AP_mask | 基于 Mask IoU 的平均精度 |
| 目标跟踪 | MOTA, IDF1 | 多目标跟踪准确率与身份一致性 |
| 动作识别 | Clip Accuracy, Top-1 | 视频片段分类准确率 |

### 3.5 数据集

| 数据集 | 任务 | 规模 | 特点 |
|:---|:---|:---|:---|
| ImageNet (2009) | 图像分类 | 1400 万图 / 1000 类 | 深度学习催化剂 |
| COCO (2014) | 检测/分割 | 33 万图 / 80 类 | 密集标注，行业标准 |
| PASCAL VOC (2007-2012) | 检测/分割 | 1.2 万图 / 20 类 | 早期检测基准 |
| CIFAR-10/100 | 分类 | 6 万图 / 10 或 100 类 | 小尺寸，快速实验 |
| Cityscapes (2016) | 场景分割 | 2.5 万图 / 19 类 | 城市驾驶场景 |
| Kinetics (2017) | 动作识别 | 30 万视频 / 400-700 类 | 大规模视频动作基准 |
| ADE20K (2016) | 场景解析 | 2.5 万图 / 150 类 | 场景理解基准 |

## 4. 方法分类总览

| 类别 | 代表方法 | 特点 | 适用场景 |
|:---|:---|:---|:---|
| 传统图像处理 | 高斯滤波、Canny、形态学 | 数学理论清晰，无需训练 | 预处理、特征提取 |
| 传统特征匹配 | SIFT、SURF、ORB、HOG | 手工设计，几何不变性好 | 图像拼接、配准、跟踪 |
| 经典检测 | Viola-Jones、DPM | Haar/HOG + 机器学习 | 人脸检测、行人检测 |
| CNN 分类 | AlexNet、VGG、ResNet、EfficientNet | 端到端学习，层次化特征 | 图像分类 |
| CNN 检测 | R-CNN 系列、YOLO、SSD、RetinaNet | 区域提议 vs. 单阶段 | 目标检测 |
| 分割网络 | FCN、U-Net、DeepLab、Mask R-CNN | 像素级预测 | 语义/实例分割 |
| Vision Transformer | ViT、Swin、DeiT | 全局建模、可扩展 | 分类、检测、分割 |
| 视频理解 | C3D、I3D、SlowFast、TimeSformer | 时空特征提取 | 动作识别、视频分类 |
| 三维视觉 | NeRF、3DGS、SLAM | 从 2D 重建 3D | 场景重建、定位 |
| 自监督学习 | SimCLR、MoCo、MAE、DINO | 无需标注预训练 | 视觉表征学习 |
| 视觉基础模型 | SAM、CLIP、Grounding DINO | 通用化、可提示 | 开放词汇视觉任务 |

## 5. 核心公式速览

### 卷积运算

$$O(i,j) = \sum_m \sum_n I(i+m, j+n) \cdot K(m,n)$$

### Canny 边缘梯度

$$|\nabla I| = \sqrt{G_x^2 + G_y^2}, \quad \theta = \arctan\left(\frac{G_y}{G_x}\right)$$

### SIFT DoG 极值

$$D(x,y,\sigma) = L(x,y,k\sigma) - L(x,y,\sigma) = (G(k\sigma) - G(\sigma)) * I$$

### IoU（交并比）

$$\text{IoU} = \frac{|A \cap B|}{|A \cup B|}$$

### FPN 横向连接

$$C_i \xrightarrow{1\times1 \text{ conv}} P_i, \quad P_i = P_i + \text{Upsample}(P_{i+1})$$

### NeRF 体渲染方程

$$C(\mathbf{r}) = \int_{t_n}^{t_f} T(t) \sigma(\mathbf{r}(t)) \mathbf{c}(\mathbf{r}(t), \mathbf{d}) \, dt$$

$$T(t) = \exp\left(-\int_{t_n}^t \sigma(\mathbf{r}(s)) \, ds\right)$$

### ViT 分类

$$\mathbf{z}_0 = [x_{cls}; E \cdot x_1^p; \cdots; E \cdot x_N^p] + E_{pos}$$

$$\mathbf{z}_l' = \text{MSA}(\text{LN}(\mathbf{z}_{l-1})) + \mathbf{z}_{l-1}, \quad \mathbf{z}_l = \text{MLP}(\text{LN}(\mathbf{z}_l')) + \mathbf{z}_l'$$

### MAE 重建损失

$$\mathcal{L} = \frac{1}{|\Omega_M|} \sum_{i \in \Omega_M} \| x_i - \hat{x}_i \|^2$$

## 7. 与其他方向关系

- **与卷积神经网络的关系**：CNN 是现代 CV 的核心工具，从 AlexNet 到 ResNet 的架构演进深刻塑造了 CV 的发展。参见 [[05_卷积神经网络]]。
- **与深度学习基础的关系**：CV 深度学习模型依赖反向传播、BN、优化器等深度学习基础设施。参见 [[04_深度学习基础]]。
- **与 Transformer 的关系**：ViT 将自注意力机制引入视觉，Swin/DETR 等扩展到密集预测任务。参见 [[08_Transformer与注意力机制]]。
- **与多模态 AI 的关系**：CLIP 统一视觉与语言表示，视觉语言模型（LLaVA/GPT-4V）将 CV 作为感知组件。参见 [[14_多模态AI]]。
- **与生成模型的关系**：GAN/VAE/扩散模型在图像生成领域广泛应用，是 CV 的重要分支。参见 [[07_生成模型]]。
- **与模型部署与工程化的关系**：CV 模型的推理优化、量化、边缘部署催生了 TensorRT/ONNX 等工程实践。参见 [[18_模型部署与工程化]]。
- **与 LLM 应用工程的关系**：视觉感知是 Agent 系统的重要能力，多模态 Agent 需要视觉理解。参见 [[19_LLM应用工程]]。

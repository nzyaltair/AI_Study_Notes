# 深度学习框架

## 背景

深度学习框架提供了自动微分、GPU加速、模型抽象等基础设施，使开发者无需从零实现反向传播和矩阵运算。框架的选择直接影响开发效率和部署方案，是[[深度学习|深度学习]]工程实践的基础工具。

## 核心思想

通过计算图抽象和自动微分，让开发者只需定义前向传播逻辑，框架自动处理梯度计算和参数更新。

## 技术原理

### 计算图与自动微分

- **静态图**（TensorFlow 1.x）：先编译图再执行，优化空间大但调试困难
- **动态图**（PyTorch/TF 2.x Eager）：边执行边构建图，灵活易调试
- **自动微分**：前向传播记录操作，反向传播沿计算图自动求导

### PyTorch核心概念

```python
import torch
import torch.nn as nn

# 定义模型
class MLP(nn.Module):
    def __init__(self):
        super().__init__()
        self.fc1 = nn.Linear(784, 256)
        self.fc2 = nn.Linear(256, 10)
    
    def forward(self, x):
        x = torch.relu(self.fc1(x))
        return self.fc2(x)

# 训练循环
model = MLP().cuda()
optimizer = torch.optim.AdamW(model.parameters(), lr=1e-3)
loss_fn = nn.CrossEntropyLoss()

for epoch in range(epochs):
    for x, y in dataloader:
        x, y = x.cuda(), y.cuda()
        logits = model(x)
        loss = loss_fn(logits, y)
        optimizer.zero_grad()
        loss.backward()      # 自动微分
        optimizer.step()     # 参数更新
```

## 发展演进

Theano（2007）→ Caffe（2013）→ TensorFlow（2015）→ PyTorch（2016）→ TF 2.0（2019，Keras整合）→ JAX（2020）→ PyTorch 2.0（2023，torch.compile）

PyTorch在学术研究中占据主导地位，TensorFlow在企业生产部署中仍有优势，JAX在函数式和TPU场景中增长。

## 应用领域

- **学术研究**：PyTorch为主，论文复现首选
- **[[大语言模型|LLM]]开发**：PyTorch + HuggingFace Transformers + DeepSpeed
- **计算机视觉**：PyTorch + torchvision / TensorFlow
- **边缘部署**：TensorFlow Lite / PyTorch Mobile
- **科学计算**：JAX在物理模拟、分子动力学中增长

# 图像生成

## 1. 概述

图像生成是生成式 AI 最引人注目的应用之一，目标是从学习到的数据分布中生成逼真、高质量且符合条件要求的图像。从 GAN（2014）到 [[04_扩散模型|扩散模型]]（2020），再到 Stable Diffusion 和 FLUX，AI 图像生成质量实现了从"能看"到"好看"再到"精确控制"的飞跃。

图像生成解决的核心问题：
- **条件生成**：根据文本、图像、草图等条件生成符合要求的图像
- **可控生成**：精确控制生成图像的结构、风格、内容、身份等属性
- **图像编辑**：对已有图像进行修改、扩展、风格转换
- **高质量样本合成**：生成训练数据以增强下游模型

扩散模型已成为图像生成的主流范式，潜在扩散模型（LDM）使其在消费级 GPU 上可行。最新的 Rectified Flow 和 DiT 架构进一步提升了生成质量和效率。

## 3. 核心概念

### 3.1 扩散模型

扩散模型通过两个过程工作：
- **前向过程（加噪）**：逐步向数据添加高斯噪声，最终变为纯噪声
- **反向过程（去噪）**：学习从噪声中逐步恢复数据

扩散模型的理论基础参见 [[07_生成模型/04_扩散模型]]，此处聚焦于图像生成的工程实现。

### 3.2 潜在扩散模型（LDM）

Stable Diffusion 的核心创新——将扩散过程从像素空间转移到低维潜在空间：

```
图像 → VAE编码器 → 潜在表示(64×64×4) → U-Net/DiT去噪 → 去噪潜在表示 → VAE解码器 → 生成图像
```

优势：
- 计算量降低约 48 倍（512×512×3 → 64×64×4）
- 在消费级 GPU 上可训练和推理
- 潜在空间保留了图像的语义结构

### 3.3 条件控制

图像生成的核心是条件控制——如何让生成结果符合用户意图：

| 方法 | 原理 | 控制粒度 | 代表 |
|:---|:---|:---|:---|
| 文本条件 | CLIP/T5 编码文本 + 交叉注意力 | 语义级 | SD, DALL·E |
| ControlNet | 额外条件控制网络 | 像素级 | ControlNet |
| LoRA 微调 | 低秩适配特定风格 | 风格级 | LoRA |
| IP-Adapter | 图像嵌入作为条件 | 内容/风格 | IP-Adapter |
| T2I-Adapter | 轻量条件适配器 | 结构级 | T2I-Adapter |
| Inpainting | 只重绘指定区域 | 区域级 | SD Inpaint |

### 3.4 采样加速

扩散模型需要多步迭代去噪，采样加速是关键工程问题：

| 采样器 | 原理 | 步数 |
|:---|:---|:---|
| DDPM | 随机采样 | ~1000 步 |
| DDIM | 确定性采样 | 20-50 步 |
| DPM++ | 高阶 ODE 求解 | 20-30 步 |
| Euler a | 一阶 Euler 方法 | 15-30 步 |
| LCM | 潜在一致性模型 | 4-8 步 |
| Turbo | 蒸馏 + 对抗训练 | 1-4 步 |

## 4. 技术原理

### 4.1 扩散模型数学基础

**前向过程**（加噪）：

$$q(\mathbf{x}_t | \mathbf{x}_0) = \mathcal{N}(\mathbf{x}_t; \sqrt{\bar{\alpha}_t}\mathbf{x}_0, (1 - \bar{\alpha}_t)\mathbf{I})$$

等价表示：$\mathbf{x}_t = \sqrt{\bar{\alpha}_t}\mathbf{x}_0 + \sqrt{1 - \bar{\alpha}_t}\epsilon$，其中 $\epsilon \sim \mathcal{N}(0, \mathbf{I})$

**训练目标**：预测噪声 $\epsilon$：

$$\mathcal{L}_{simple} = \mathbb{E}_{t, \mathbf{x}_0, \epsilon}\left[\|\epsilon - \epsilon_\theta(\mathbf{x}_t, t, c)\|^2\right]$$

其中 $c$ 为条件（如文本嵌入）。

### 4.2 Classifier-Free Guidance (CFG)

无分类器引导，平衡生成质量与条件一致性：

$$\hat{\epsilon} = \epsilon_\theta(z_t, t, \varnothing) + w \cdot \left(\epsilon_\theta(z_t, t, c) - \epsilon_\theta(z_t, t, \varnothing)\right)$$

- $w = 0$：无条件生成
- $w = 1$：标准条件生成
- $w > 1$：增强条件一致性，提高质量但降低多样性
- 典型值：$w = 7.5$（Stable Diffusion 默认）

### 4.3 U-Net 架构

U-Net 是扩散模型的主力去噪网络：

```
输入 z_t + t + c
    ↓
  编码器(下采样) ←—— 跳跃连接 ——→ 解码器(上采样)
    ↓                                    ↓
  瓶颈层(最低分辨率) ─────────────────→ 
    ↓
  输出预测噪声 ε
```

关键组件：
- **ResBlock + Time Embedding**：注入时间步信息
- **Cross-Attention**：注入文本条件信息
- **Self-Attention**：捕获空间长程依赖
- **跳跃连接**：保留多尺度细节

### 4.4 DiT 架构

DiT（Diffusion Transformer）用 Transformer 替代 U-Net 进行去噪，是最新趋势：

- 将潜在表示分割为 token 序列（类似 ViT）
- 用 Transformer blocks 处理（Self-Attention + Cross-Attention + FFN）
- 通过 adaLN-Zero 注入时间和条件信息
- 优势：更好的规模定律（scaling law），更大模型更好性能

**MMDiT（Multi-Modal DiT）**：SD3 和 FLUX 使用，文本和图像 token 在同一 Transformer 中联合处理。

### 4.5 Rectified Flow

FLUX 和 SD3 采用的新生成框架，统一了扩散模型和连续归一化流：

$$\frac{d\mathbf{x}_t}{dt} = v_\theta(\mathbf{x}_t, t, c)$$

从噪声 $\mathbf{x}_1$ 到数据 $\mathbf{x}_0$ 的直线传输路径，比传统扩散的曲线路径更高效。

### 4.6 VAE 编解码器

潜在扩散模型的 VAE 负责：
- **编码器**：将 $512 \times 512 \times 3$ 图像压缩为 $64 \times 64 \times 4$ 潜在表示
- **解码器**：将潜在表示恢复为像素空间图像
- 压缩比约 48 倍，同时保留视觉质量

## 5. 关键方法与模型

### 5.1 模型对比

| 模型 | 架构 | 文本编码器 | 分辨率 | 特点 |
|:---|:---|:---|:---|:---|
| Stable Diffusion 1.5 | LDM + U-Net | CLIP ViT-L | 512×512 | 经典开源，生态最丰富 |
| Stable Diffusion XL | LDM + U-Net | CLIP + OpenCLIP | 1024×1024 | 高分辨率开源 |
| SD3 Medium | MMDiT + Rectified Flow | CLIP + T5 | 1024×1024 | 多模态 Transformer |
| FLUX.1 [dev] | DiT + Rectified Flow | T5 + CLIP | 1024-2048 | 新一代开源 SOTA |
| DALL·E 3 | 扩散 + LLM | GPT-4 | 1024×1024 | 文本一致性最高 |
| Midjourney V6 | 扩散变体 | 专有 | 2048×2048 | 艺术风格最佳 |
| Imagen | 级联扩散 | T5-XXL | 1024×1024 | 文本理解强 |

### 5.2 ControlNet：精确条件控制

ControlNet 通过在 U-Net 的编码器上添加可训练副本，实现精确结构控制：

```
条件图(边缘/深度/姿态) → ControlNet(可训练) → 条件特征
                                              ↓
噪声潜在表示 → U-Net(冻结) ←── 条置特征注入 → 去噪
```

支持的输入条件：
- Canny 边缘、HED 边缘、Scribble 草图
- Depth 深度图、Normal 法线图
- OpenPose 姿态、Human Segmentation
- MLSD 直线检测（建筑）
- Tile/Pixel（图像重绘）

### 5.3 LoRA 微调

LoRA（Low-Rank Adaptation）通过低秩矩阵分解实现高效微调：

$$W' = W + \alpha \cdot B A, \quad B \in \mathbb{R}^{d \times r}, A \in \mathbb{R}^{r \times d}, r \ll d$$

在图像生成中的应用：
- 风格微调：训练特定画风/角色/概念
- 只需少量图片（5-20 张）和几分钟训练
- 文件极小（2-100 MB），可热插拔

### 5.4 IP-Adapter：图像提示

IP-Adapter 将参考图像作为条件，实现风格/内容迁移：
- 通过图像编码器提取参考图像嵌入
- 将图像嵌入与文本嵌入一起注入交叉注意力
- 支持风格迁移、角色一致性、参考图生成

## 6. 优势与局限

### 6.1 优势

- **高质量生成**：扩散模型生成质量已达到照片级
- **灵活条件控制**：文本、图像、草图等多种条件输入
- **开源生态**：Stable Diffusion 系列拥有最丰富的开源工具链
- **可控性强**：ControlNet、LoRA、IP-Adapter 等精细控制方法
- **多分辨率**：支持从 256 到 2048+ 多种分辨率

### 6.2 局限

- **推理速度**：扩散模型需要多步迭代，比 GAN 慢
- **文本渲染**：生成图像中的文字常常错误（虽在改善）
- **精确构图**：空间布局、物体数量控制仍有困难
- **训练成本**：高质量模型训练需要大量 GPU 资源
- **安全风险**：Deepfake、版权侵权、偏见等
- **一致性**：同一概念在多次生成中难以保持完全一致

## 9. 前沿发展

- **DiT 架构统一**：Transformer 架构逐步取代 U-Net，更好的规模定律（FLUX, SD3）
- **Rectified Flow**：更高效的生成路径，减少采样步数
- **少步生成**：Consistency Models、LCM、Turbo 实现 1-4 步高质量生成
- **文本渲染**：结合字符级渲染能力，解决文字生成错误问题
- **3D 生成**：Score Distillation Sampling (SDS) 将 2D 生成先验提升到 3D
- **身份一致性**：InstantID、PhotoMaker 等人脸身份保持技术
- **多图合成**：支持多参考图的组合生成
- **可控编辑**：自然语言驱动的图像精确编辑
- **统一模型**：生成与理解统一在一个模型中（如 FLUX 同时支持理解和生成）

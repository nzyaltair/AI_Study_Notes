# 微积分

## 背景

微积分是机器学习与深度学习的核心数学基础，为AI算法提供了优化参数、计算梯度和分析模型性能的理论工具。神经网络训练中参数调整的方向和幅度完全依赖于对损失函数的梯度信息。

## 核心思想

微积分研究函数的变化率（导数）和累积（积分）。在AI中，导数和梯度用于优化模型参数，链式法则是反向传播算法的数学基础。

## 技术原理

### 导数与微分

- **导数定义**：$f'(x_0) = \lim_{h \to 0} \frac{f(x_0 + h) - f(x_0)}{h}$，表示函数在某点的瞬时变化率
- **几何意义**：函数曲线在该点的切线斜率
- **基本公式**：
  - $(x^n)' = nx^{n-1}$
  - $(e^x)' = e^x$
  - $(\ln x)' = 1/x$
  - $(\sin x)' = \cos x$，$(\cos x)' = -\sin x$

### 偏导数与梯度

- **偏导数**：$\frac{\partial f}{\partial x_i}$，其他变量固定时函数对某一变量的变化率
- **梯度**：$\nabla f(x) = \left( \frac{\partial f}{\partial x_1}, \frac{\partial f}{\partial x_2}, ..., \frac{\partial f}{\partial x_n} \right)^T$
  - 方向：指向函数值增长最快的方向
  - 负梯度方向：函数值下降最快的方向（梯度下降的理论基础）
- **性质**：
  - 线性性：$\nabla(f + g) = \nabla f + \nabla g$
  - 链式法则：$\nabla(f(g(x))) = f'(g(x)) \cdot \nabla g(x)$

### 链式法则

- **单变量**：$\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx}$
- **多变量**：$\frac{\partial y}{\partial x_j} = \sum_{i=1}^m \frac{\partial y}{\partial u_i} \cdot \frac{\partial u_i}{\partial x_j}$
- **神经网络中的应用**：神经网络是复合函数，链式法则将梯度计算分解为每层的局部梯度计算，这是[[神经网络基础|反向传播]]的核心

### 向量与矩阵微分

- **雅可比矩阵**：向量函数 $\mathbf{f}(\mathbf{x})$ 对 $\mathbf{x}$ 的导数，$J_{ij} = \frac{\partial f_i}{\partial x_j}$
- **常用公式**：

| 函数形式 | 梯度结果 | 条件 |
|---------|---------|------|
| $f(\mathbf{x}) = \mathbf{a}^T \mathbf{x}$ | $\nabla f = \mathbf{a}$ | $\mathbf{a}$ 为常数向量 |
| $f(\mathbf{x}) = \mathbf{x}^T \mathbf{x}$ | $\nabla f = 2\mathbf{x}$ | - |
| $f(\mathbf{x}) = \mathbf{x}^T A \mathbf{x}$ | $\nabla f = (A + A^T)\mathbf{x}$ | - |
| $f(\mathbf{x}) = \mathbf{x}^T A \mathbf{x}$ | $\nabla f = 2A\mathbf{x}$ | $A$ 对称 |
| $f(X) = \text{Tr}(A^T X)$ | $\nabla_X f = A$ | - |

### 反向传播的数学推导

对于两层神经网络，损失 $L = \frac{1}{2} \| \mathbf{y} - \hat{\mathbf{y}} \|^2$：

- 输出层梯度：$\frac{\partial L}{\partial \mathbf{z}^{(2)}} = (\hat{\mathbf{y}} - \mathbf{y}) \odot \sigma'(\mathbf{z}^{(2)})$
- 隐藏层梯度：$\frac{\partial L}{\partial \mathbf{z}^{(1)}} = (\mathbf{W}^{(2)T} \frac{\partial L}{\partial \mathbf{z}^{(2)}}) \odot \sigma'(\mathbf{z}^{(1)})$
- 权重梯度：$\frac{\partial L}{\partial \mathbf{W}^{(2)}} = \frac{\partial L}{\partial \mathbf{z}^{(2)}} \mathbf{a}^{(1)T}$

### 数值稳定性

- **Log-Sum-Exp技巧**：$\text{softmax}(x_i) = \frac{e^{x_i - \max(x)}}{\sum_j e^{x_j - \max(x)}}$，防止指数溢出
- **梯度裁剪**：限制梯度最大范数，防止梯度爆炸
- **梯度消失/爆炸**：深层网络中梯度连乘导致的问题，解决方案包括残差连接、合适的权重初始化

## 发展演进

牛顿和莱布尼茨独立发明微积分 → 多变量微积分 → 矩阵微积分 → 自动微分（深度学习时代的核心工具）。自动微分结合数值微分和符号微分的优点，是PyTorch/TensorFlow等框架的底层技术。

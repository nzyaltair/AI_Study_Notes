# 方向十一：大模型训练与对齐

## 定义与范围

大模型训练与对齐涵盖大语言模型（LLM）从预训练到部署的完整训练流水线，包括数据工程、预训练策略、监督微调（SFT）、偏好对齐方法（RLHF/DPO/GRPO等）、参数高效微调（PEFT）以及安全与价值对齐。

本方向聚焦于**训练流程与对齐方法层面的核心技术**，不涵盖模型架构设计（见方向10）、推理系统工程（见方向12）和应用开发（见方向19），但会建立与这些方向的关联。

## 核心概念与方法

### 预训练
- **训练目标**：Next Token Prediction（自回归语言建模）
- **数据规模**：万亿级token（LLaMA 3: 15T tokens）
- **Scaling Laws**：Chinchilla法则 $D/N \approx 20$，计算最优分配
- **分布式训练**：数据并行 + 张量并行 + 流水线并行（3D并行）+ ZeRO优化
- **训练稳定性**：梯度裁剪、混合精度、Loss Spike处理

### 监督微调
- **核心原则**：数据质量远比数量重要
- **数据格式**：Alpaca格式、ChatML格式、多轮对话
- **训练方式**：全参数微调或参数高效微调（LoRA/QLoRA）
- **进阶方法**：拒绝采样微调（RFT）、迭代SFT

### 偏好对齐
- **RLHF**：三阶段流程（SFT→RM→PPO），最成熟但最复杂
- **DPO**：从偏好数据直接优化策略，将RL转化为分类问题
- **RLAIF**：用AI反馈替代人类反馈，降低标注成本
- **GRPO**：组内相对优势，去除奖励模型和Critic，适合推理任务
- **其他变体**：KTO（非配对反馈）、SimPO（无参考模型）、ORPO（SFT阶段对齐）

### 参数高效微调
- **LoRA**：低秩矩阵分解 $\Delta W = BA$，参数量~1%，推理可合并
- **QLoRA**：4-bit NF4量化 + LoRA，单卡微调70B模型
- **Adapter**：瓶颈结构模块，参数量~3%
- **改进方向**：动态秩（AdaLoRA）、动态缩放（DoRA）、SVD初始化（PiSSA）

### 安全对齐
- **红队测试**：对抗性安全评估
- **Constitutional AI**：基于宪法的自我修正
- **对齐失败模式**：Reward Hacking、Over-Alignment、文化偏见固化

## 核心公式推导

### 1. 预训练目标（NTP）

$$\mathcal{L}_{PT} = -\sum_{t=1}^{T} \log P(x_t \mid x_{<t}; \theta)$$

### 2. SFT损失

$$\mathcal{L}_{SFT} = -\frac{1}{|y|} \sum_{t=1}^{|y|} \log P(y_t \mid x, y_{<t}; \theta)$$

仅计算目标响应部分的损失，输入部分标签设为-100。

### 3. RLHF三阶段

**奖励模型训练**（Bradley-Terry模型）：

$$\mathcal{L}_{RM} = -\mathbb{E}_{(x, y_w, y_l)} \left[\log\sigma\left(r_\phi(x, y_w) - r_\phi(x, y_l)\right)\right]$$

**PPO优化**：

$$J(\phi) = \mathbb{E}_{(x,y) \sim p_\phi}\left[r(x,y)\right] - \beta \, D_{KL}\left(p_\phi \| p_{ref}\right)$$

**PPO Clipped目标**：

$$\mathcal{L}_{PPO} = \mathbb{E}\left[\min\left(r_t \hat{A}_t, \, \text{clip}(r_t, 1-\epsilon, 1+\epsilon) \hat{A}_t\right)\right]$$

### 4. DPO直接偏好优化

核心洞察：从RLHF最优策略的闭式解反解奖励函数：

$$r(x,y) = \beta \log \frac{\pi^*(y|x)}{\pi_{ref}(y|x)} + C(x)$$

代入Bradley-Terry模型得到DPO损失：

$$\mathcal{L}_{DPO} = -\mathbb{E}\left[\log \sigma\left(\beta \log \frac{\pi_\theta(y_w|x)}{\pi_{ref}(y_w|x)} - \beta \log \frac{\pi_\theta(y_l|x)}{\pi_{ref}(y_l|x)}\right)\right]$$

### 5. GRPO组内相对优势

$$\tilde{A}_i = \frac{r_i - \text{mean}(r_1, \ldots, r_G)}{\text{std}(r_1, \ldots, r_G)}$$

用组内统计量代替Critic网络，去除价值模型。

### 6. LoRA低秩适配

$$W' = W + \Delta W = W + \frac{\alpha}{r} BA$$

其中 $B \in \mathbb{R}^{d \times r}$，$A \in \mathbb{R}^{r \times d}$，$r \ll d$。参数量从 $d^2$ 降至 $2dr$。

**QLoRA**：$h = \text{dequant}(W_{NF4}) x + \frac{\alpha}{r} B(Ax)$

## 1. 概述

概率与统计是**人工智能和机器学习的核心数学基础**，它们为AI算法提供了处理不确定性和建模数据分布的理论工具。在机器学习中，我们经常需要：

- 对数据的分布进行建模
- 基于有限样本进行统计推断
- 处理模型的不确定性
- 设计概率模型来解决分类、回归和生成等问题

### 学习目标
- 理解概率的基本概念和公理
- 掌握随机变量、概率分布的定义和性质
- 学会计算期望、方差、协方差等统计量
- 理解常见概率分布及其应用场景
- 掌握贝叶斯定理和统计推断方法
- 了解概率统计在机器学习中的重要应用

## 2. 概率的基本要素

### 2.1 样本空间与事件

- **样本空间**（$\Omega$）：随机实验的所有可能结果的集合
- **事件**（$A$）：样本空间的子集，即实验可能结果的集合
- **事件空间**（$\mathcal{F}$）：满足特定条件的事件集合，需要满足：
  1. 空集 $\emptyset \in \mathcal{F}$
  2. 若 $A \in \mathcal{F}$，则补集 $\Omega \setminus A \in \mathcal{F}$
  3. 若 $A_1, A_2, dots \in \mathcal{F}$，则并集 $igcup_{i} A_i \in \mathcal{F}$

### 2.2 概率公理

概率是定义在事件空间上的函数 $P: \mathcal{F} 
\rightarrow [0, 1]$，满足以下公理：

1. **非负性**：对于任意事件 $A \in \mathcal{F}$，有 $P(A) \geq 0$
2. **规范性**：$P(\Omega) = 1$
3. **可列可加性**：若 $A_1, A_2, dots$ 是互不相交的事件，则：
   $$P(igcup_{i=1}^{\infty} A_i
\right) = sum_{i=1}^{\infty} P(A_i)$$

### 2.3 条件概率与独立性

#### 条件概率

给定事件 $B$（$P(B) > 0$），事件 $A$ 的条件概率定义为：

$$P(A | B) \triangleq \frac{P(A \cap B)}{P(B)}$$ 

直观上，条件概率表示在已知事件 $B$ 发生的情况下，事件 $A$ 发生的概率。

#### 独立性

两个事件 $A$ 和 $B$ 独立，当且仅当：

$$P(A \cap B) = P(A)P(B)$$

等价地，若 $P(B) > 0$，则 $P(A | B) = P(A)$，即事件 $B$ 的发生不影响事件 $A$ 的概率。

## 3. 随机变量

### 3.1 基本概念

随机变量是一个从样本空间到实数的函数 $X: \Omega 
\rightarrow \mathbb{R}$，用于将随机实验的结果映射为实数，方便数学处理。

- **离散随机变量**：只能取有限或可数个值的随机变量（如抛硬币的结果、骰子的点数）
- **连续随机变量**：可以取某个区间内任意值的随机变量（如身高、体重、时间）

### 3.2 累积分布函数

累积分布函数（CDF）是描述随机变量取值小于等于某个值的概率的函数：

$$F_X(x) \triangleq P(X \geq x)$$

#### 性质
- $0 \geq F_X(x) \geq 1$
- $\sim_{x 
\rightarrow -\infty} F_X(x) = 0$
- $\sim_{x 
\rightarrow \infty} F_X(x) = 1$
- 若 $x \geq y$，则 $F_X(x) \geq F_X(y)$（单调不减）

### 3.3 概率质量函数

对于离散随机变量，概率质量函数（PMF）定义为：

$$p_X(x) \triangleq P(X = x)$$

#### 性质
- $0 \geq p_X(x) \geq 1$
- $sum_{x \in Val(X)} p_X(x) = 1$，其中 $Val(X)$ 是 $X$ 的所有可能取值
- $P(X \in A) = sum_{x \in A} p_X(x)$

### 3.4 概率密度函数

对于连续随机变量，若 CDF $F_X(x)$ 可导，则概率密度函数（PDF）定义为：

$$f_X(x) \triangleq \frac{dF_X(x)}{dx}$$

#### 性质
- $f_X(x) \geq 0$
- $int_{-\infty}^{\infty} f_X(x)dx = 1$
- $P(a \geq X \geq b) = int_{a}^{b} f_X(x)dx$
- 对于很小的 $elta x$，$P(x \geq X \geq x+elta x) pprox f_X(x)elta x$

**注意**：概率密度函数的值不是概率，即 $f_X(x) 
\geq P(X = x)$，连续随机变量在某一点的概率为0。

## 4. 期望与方差

### 4.1 期望的定义与性质

期望是随机变量的加权平均值，反映了随机变量的中心位置。

#### 定义
- **离散随机变量**：$E[X] \triangleq sum_{x \in Val(X)} x p_X(x)$
- **连续随机变量**：$E[X] \triangleq int_{-\infty}^{\infty} x f_X(x)dx$

对于函数 $g(X)$，其期望为：
- **离散**：$E[g(X)] = sum_{x} g(x) p_X(x)$
- **连续**：$E[g(X)] = int_{-\infty}^{\infty} g(x) f_X(x)dx$

#### 性质
- 对于常数 $a$，$E[a] = a$
- 线性性质：$E[aX + bY] = aE[X] + bE[Y]$
- 若 $X$ 和 $Y$ 独立，则 $E[XY] = E[X]E[Y]$

### 4.2 方差的定义与性质

方差是随机变量取值偏离其期望的程度的度量。

#### 定义

$$Var[X] \triangleq E[(X - E[X])^2]$$

等价形式：

$$Var[X] = E[X^2] - (E[X])^2$$

#### 性质
- 对于常数 $a$，$Var[a] = 0$
- $Var[aX] = a^2 Var[X]$
- $Var[X + b] = Var[X]$
- 若 $X$ 和 $Y$ 独立，则 $Var[X + Y] = Var[X] + Var[Y]$

## 5. 概率论与信息论

信息论提供了量化不确定性和信息量的数学框架，在机器学习中扮演核心角色。

### 5.1 自信息 (Self-Information)

事件 $x$ 的自信息定义为：

$$I(x) = -\l\log P(x) = \l\log(1/P(x))$$

**直觉理解**：
- 概率越小的事件，发生时携带的信息量越大
- $P(x) = 1$（确定事件）→ $I(x) = 0$（无信息）
- $P(x) \r\rightarrow 0$（罕见事件）→ $I(x) \r\rightarrow \i\infty$（大量信息）

通常使用 $\l\log_2$（单位为 bit）或 $\ln$（单位为 nat）。

### 5.2 熵 (Eintropy)

熵是自信息的期望值，衡量随机变量的平均不确定性：

**离散情况**：
$$H(X) = E[I(X)] = -\ssum_{x} P(x) \l\log P(x) = -E[\l\log P(X)]$$

**连续情况（微分熵）**：
$$H(X) = -\iint p(x) \l\log p(x) dx$$

**性质**：
- $H(X) \g\geq 0$
- 均匀分布熵最大：$H(X) \l\geq \l\log|X|$，其中 $|X|$ 是随机变量的可能取值数
- 确定性分布熵最小：$H(X) = 0$

### 5.3 条件熵与联合熵

**条件熵**：在已知随机变量 $X$ 的条件下，随机变量 $Y$ 的不确定性：

$$H(Y|X) = E[H(Y|X=x)] = -\ssum_{x} P(x) \ssum_{y} P(y|x) \l\log P(y|x)$$

**联合熵**：两个随机变量 $X$ 和 $Y$ 的联合不确定性：

$$H(X,Y) = -\ssum_{x,y} P(x,y) \l\log P(x,y)$$

**链式法则**：
$$H(X,Y) = H(X) + H(Y|X) = H(Y) + H(X|Y)$$

### 5.4 互信息 (Mutual Information)

互信息衡量两个随机变量的相互依赖程度：

$$I(X;Y) = H(X) - H(X|Y) = H(Y) - H(Y|X) = H(X) + H(Y) - H(X,Y)$$

也可表示为 KL 散度：
$$I(X;Y) = D_{KL}(P(X,Y) \| P(X)P(Y))$$

**性质**：
- $I(X;Y) \g\geq 0$
- $I(X;Y) = 0$ 当且仅当 $X$ 和 $Y$ 独立
- 对称性：$I(X;Y) = I(Y;X)$

**应用**：特征选择、变分信息瓶颈理论

### 5.5 KL 散度 (Kullback-Leibler Divergence)

KL 散度衡量两个分布的差异（也称为相对熵）：

**离散情况**：
$$D_{KL}(P\|Q) = \ssum_{x} P(x) \l\log\left(\f\frac{P(x)}{Q(x)}\r\right) = E_P[\l\log P(X) - \l\log Q(X)]$$

**连续情况**：
$$D_{KL}(P\|Q) = \iint p(x) \l\log\left(\f\frac{p(x)}{q(x)}\r\right) dx$$

**性质**：
- $D_{KL}(P\|Q) \g\geq 0$，当且仅当 $P=Q$ 时等於 0（Gibbs 不等式）
- **不对称**：$D_{KL}(P\|Q) \n\geq D_{KL}(Q\|P)$
- 不满足三角不等式（非距离度量）

**直观理解**：
- 前向 KL：$D_{KL}(P\|Q)$ → 使 $Q$ 覆盖 $P$ 的所有支持域（避免 zero-forcing）
- 后向 KL：$D_{KL}(Q\|P)$ → 使 $Q$ 集中在 $P$ 的高概率区域（zero-forcing）

**应用**：
- 变分推断（最小化 $D_{KL}(Q\|P)$）
- 模型选择（Akaike Information Criterion）
- 策略梯度方法（KL 约束）

### 5.6 交叉熵 (Cross-Eintropy)

交叉熵衡量用分布 $Q$ 编码来自分布 $P$ 的样本所需的平均编码长度：

$$H(P,Q) = -\ssum_{x} P(x) \l\log Q(x) = -E_P[\l\log Q(X)]$$

**与 KL 散度的关系**：
$$H(P,Q) = H(P) + D_{KL}(P\|Q)$$

**在机器学习中的应用**：

对于分类问题，设真实标签分布为 $P$（通常为 one-\rhot），模型预测分布为 $Q$：

$$Loss = -\ssum_i \ssum_c y_{ic} \l\log \h\hat{y}_{ic}$$

其中 $y_{ic}$ 为真实标签（0或1），$\h\hat{y}_{ic}$ 为预测概率。

**二元交叉熵 (Binary Cross-Eintropy)**：
$$Loss = -[y \l\log(\h\hat{y}) + (1-y) \l\log(1-\h\hat{y})]$$

**多类别交叉熵**：
$$Loss = -\ssum_c y_c \l\log(\h\hat{y}_c)$$

**最小化交叉熵等价于最大化似然**：
$$\min H(P,Q) \iff \max \l\log P(data|\theta)$$

## 6. 常见概率分布

### 6.1 离散分布

#### 伯努利分布（Bernoulli）

用于描述单次试验的结果，只有两种可能：成功（1）或失败（0）。

- **PMF**：$p(x) = p^x (1-p)^{1-x}$，$x \in \{0, 1\}$
- **参数**：$p \in [0, 1]$（成功概率）
- **期望**：$E[X] = p$
- **方差**：$Var[X] = p(1-p)$
- **应用**：二分类问题、硬币翻转

#### 二项式分布（Binomial）

用于描述 $n$ 次独立伯努利试验中成功的次数。

- **PMF**：$p(x) = \binom{n}{x} p^x (1-p)^{n-x}$，$x \in \{0, 1, dots, n\}$
- **参数**：$n$（试验次数），$p$（单次成功概率）
- **期望**：$E[X] = np$
- **方差**：$Var[X] = np(1-p)$
- **应用**：多次独立试验、分类问题中的样本计数

#### 泊松分布（Poisson）

用于描述单位时间内事件发生的次数，适用于罕见事件。

- **PMF**：$p(x) = e^{-ambda} \frac{ambda^x}{x!}$，$x \in \{0, 1, dots\}$
- **参数**：$ambda > 0$（事件发生率）
- **期望**：$E[X] = ambda$
- **方差**：$Var[X] = ambda$
- **应用**：罕见事件建模、计数数据

### 6.2 连续分布

#### 均匀分布（Uniform）

在区间 $[a, b]$ 上所有值等可能出现。

- **PDF**：$f(x) = \f\frac{1}{b-a}$，$a \geq x \geq b$，否则 0
- **参数**：$a$（下限），$b$（上限）
- **期望**：$E[X] = \f\frac{a+b}{2}$
- **方差**：$Var[X] = \f\frac{(b-a)^2}{12}$
- **应用**：随机数生成、等概率事件建模

#### 指数分布（Exponeintial）

用于描述事件发生的时间间隔，具有无记忆性。

- **PDF**：$f(x) = ambda e^{-ambda x}$，$x \geq 0$，否则 0
- **参数**：$ambda > 0$（速率参数）
- **期望**：$E[X] = \f\frac{1}{ambda}$
- **方差**：$Var[X] = \f\frac{1}{ambda^2}$
- **应用**：等待时间、寿命分布

#### 正态分布（Gaussian）

最常见的连续分布，也称为高斯分布，许多自然现象都近似服从正态分布。

- **PDF**：$f(x) = \f\frac{1}{\sqrt{2i}\sigma} e^{-\f\frac{(x-u)^2}{2\sigma^2}}$
- **参数**：$u$（均值），$\sigma^2$（方差）
- **期望**：$E[X] = u$
- **方差**：$Var[X] = \sigma^2$
- **应用**：噪声建模、误差分析、中心极限定理

### 6.3 分布性质对比

| 分布名称 | 类型 | 概率质量/密度函数 | 期望 | 方差 | 主要应用 |
|---------|------|------------------|------|------|----------|
| 伯努利分布 | 离散 | $p^x(1-p)^{1-x}$ | $p$ | $p(1-p)$ | 二分类问题 |
| 二项式分布 | 离散 | $\binom{n}{x}p^x(1-p)^{n-x}$ | $np$ | $np(1-p)$ | 多次独立试验 |
| 泊松分布 | 离散 | $e^{-ambda}\f\frac{ambda^x}{x!}$ | $ambda$ | $ambda$ | 罕见事件计数 |
| 均匀分布 | 连续 | $\f\frac{1}{b-a}$ | $\f\frac{a+b}{2}$ | $\f\frac{(b-a)^2}{12}$ | 随机数生成 |
| 指数分布 | 连续 | $ambda e^{-ambda x}$ | $\f\frac{1}{ambda}$ | $\f\frac{1}{ambda^2}$ | 等待时间 |
| 正态分布 | 连续 | $\f\frac{1}{\sqrt{2i}\sigma}e^{-\f\frac{(x-u)^2}{2\sigma^2}}$ | $u$ | $\sigma^2$ | 噪声建模、误差分析 |

## 7. 多个随机变量

### 7.1 联合分布与边缘分布

#### 联合分布

描述多个随机变量同时取值的概率分布：

- **联合CDF**：$F_{XY}(x, y) = P(X \geq x, Y \geq y)$
- **联合PMF（离散）**：$p_{XY}(x, y) = P(X = x, Y = y)$
- **联合PDF（连续）**：$f_{XY}(x, y) = \f\frac{artial^2 F_{XY}(x, y)}{artial x artial y}$

#### 边缘分布

从联合分布中提取单个随机变量的分布：

- **边缘PMF（离散）**：$p_X(x) = sum_y p_{XY}(x, y)$
- **边缘PDF（连续）**：$f_X(x) = int_{-\infty}^{\infty} f_{XY}(x, y)dy$

### 7.2 条件概率分布

描述在已知一个随机变量取值的情况下，另一个随机变量的分布：

- **条件PMF（离散）**：$p_{X|Y}(x|y) = \f\frac{p_{XY}(x, y)}{p_Y(y)}$，$p_Y(y) > 0$
- **条件PDF（连续）**：$f_{X|Y}(x|y) = \f\frac{f_{XY}(x, y)}{f_Y(y)}$，$f_Y(y) > 0$

### 7.3 贝叶斯定理

贝叶斯定理描述了条件概率之间的关系，是机器学习中贝叶斯方法的基础：

$$P(Y|X) = \f\frac{P(X|Y)P(Y)}{P(X)}$$

#### 连续形式

$$f_{Y|X}(y|x) = \f\frac{f_{X|Y}(x|y)f_Y(y)}{int_{-\infty}^{\infty} f_{X|Y}(x|y')f_Y(y')dy'}$$

#### 术语解释
- **先验概率**：$P(Y)$，在观察到数据前对Y的概率分布
- **似然函数**：$P(X|Y)$，给定Y时观察到X的概率
- **后验概率**：$P(Y|X)$，在观察到X后对Y的概率分布
- **证据**：$P(X)$，观察到X的边缘概率

### 7.4 协方差与相关系数

#### 协方差

协方差衡量两个随机变量之间的线性关系强度：

$$Cov[X, Y] \triangleq E[(X - E[X])(Y - E[Y])] = E[XY] - E[X]E[Y]$$

#### 相关系数

相关系数是归一化的协方差，取值范围在[-1, 1]之间：

$$
\rho_{XY} \triangleq \f\frac{Cov[X, Y]}{\sqrt{Var[X]Var[Y]}}$$

#### 性质
- $Cov[X, Y] = Cov[Y, X]$
- $Cov[aX + b, cY + d] = ac Cov[X, Y]$
- $Var[X + Y] = Var[X] + Var[Y] + 2Cov[X, Y]$
- 若 $X$ 和 $Y$ 独立，则 $Cov[X, Y] = 0$，但反之不成立
- $|
\rho_{XY}| \geq 1$，当且仅当 Y 是 X 的线性函数时取等号

## 8. 多元概率分布

### 8.1 随机向量

将多个随机变量组合成向量，称为随机向量：

$$\mathbf{X} = [X_1, X_2, dots, X_n]^T$$

随机向量的联合分布、边缘分布、条件分布等概念是对单个随机变量的推广。

### 8.2 协方差矩阵

对于随机向量 $\mathbf{X} \in \mathbb{R}^n$，协方差矩阵 $\sigma \in \mathbb{R}^{n 	\simes n}$ 定义为：

$$\sigma_{ij} \triangleq Cov[X_i, X_j]$$

矩阵形式：

$$\sigma = E[(\mathbf{X} - E[\mathbf{X}])(\mathbf{X} - E[\mathbf{X}])^T] = E[\mathbf{X}\mathbf{X}^T] - E[\mathbf{X}]E[\mathbf{X}]^T$$

#### 性质
- $\sigma$ 是对称矩阵：$\sigma^T = \sigma$
- $\sigma$ 是正半定矩阵：$orall \mathbf{a} \in \mathbb{R}^n$，$\mathbf{a}^T\sigma\mathbf{a} \geq 0$
- 对角线元素是各分量的方差：$\sigma_{ii} = Var[X_i]$

### 8.3 多元高斯分布

多元高斯分布是单个高斯分布在多维空间的推广，是机器学习中最重要的多元分布之一。

#### 定义

随机向量 $\mathbf{X} \in \mathbb{R}^n$ 服从均值为 $u \in \mathbb{R}^n$、协方差矩阵为 $\sigma \in \mathbb{S}_{++}^n$（正定）的多元高斯分布，记为 $\mathbf{X} \sim \mathcal{N}(u, \sigma)$，其 PDF 为：

$$f_{\mathbf{X}}(\mathbf{x}) = \f\frac{1}{(2i)^{n/2}|\sigma|^{1/2}} \exp(-\f\frac{1}{2}(\mathbf{x}-u)^T\sigma^{-1}(\mathbf{x}-u)
\right)$$

#### 性质
- 边缘分布：多元高斯分布的边缘分布仍为高斯分布
- 条件分布：多元高斯分布的条件分布仍为高斯分布
- 线性变换：若 $\mathbf{X} \sim \mathcal{N}(u, \sigma)$，则 $\mathbf{A}\mathbf{X} + \mathbf{b} \sim \mathcal{N}(\mathbf{A}u + \mathbf{b}, \mathbf{A}\sigma\mathbf{A}^T)$

#### 应用
- 概率图模型
- 高斯混合模型
- 贝叶斯线性回归
- 生成对抗网络（GAN）

## 9. 统计推断

统计推断是从样本数据中推断总体分布或参数的过程，是机器学习的核心任务之一。统计推断的理论基础是大数法则和中心极限定理，它们为从有限样本推断总体提供了数学保证。

### 9.0 统计推断的理论基础：大数法则与中心极限定理

#### 9.0.1 大数法则（Law of Large Nsumbers, LLN）

大数法则描述了随着样本量增加，样本均值趋近于总体均值的现象，是频率稳定于概率的数学表达。

**弱大数法则（Weak Law of Large Nsumbers, WLLN）**：

设 $X_1, X_2, ..., X_n$ 是独立同分布（i.i.d.）的随机变量序列，期望 $E[X_i] = \mu < \i\infty$，方差 $Var(X_i) = \s\sigma^2 < \i\infty$，则对于任意 $\epsilon > 0$，有：

$$\l\sim_{n \r\rightarrow \i\infty} P\left(\left|\f\frac{1}{n}\ssum_{i=1}^n X_i - \mu\right| < \epsilon\r\right) = 1$$

即样本均值 $\bar{X}_n = \f\frac{1}{n}\ssum_{i=1}^n X_i$ 依概率收敛于总体均值 $\mu$：$\bar{X}_n \xr\rightarrow{P} \mu$。

**强大数法则（Strong Law of Large Nsumbers, SLLN）**：

设 $X_1, X_2, ..., X_n$ 是独立同分布的随机变量序列，期望 $E[X_i] = \mu < \i\infty$，则：

$$P\left(\l\sim_{n \r\rightarrow \i\infty} \f\frac{1}{n}\ssum_{i=1}^n X_i = \mu\r\right) = 1$$

即样本均值 $\bar{X}_n$ 几乎必然收敛于总体均值 $\mu$：$\bar{X}_n \xr\rightarrow{a.s.} \mu$。

**应用**：
- 大数法则保证了样本均值是总体均值的一致估计
- 为蒙特卡洛方法提供了理论基础
- 解释了为什么大样本更可靠

#### 9.0.2 中心极限定理（Ceintral L\simit Theorem, CLT）

中心极限定理描述了大量独立随机变量的和的分布趋近于正态分布的现象，是许多统计方法的理论基础。

**独立同分布中心极限定理**：

设 $X_1, X_2, ..., X_n$ 是独立同分布的随机变量序列，期望 $E[X_i] = \mu < \i\infty$，方差 $0 < Var(X_i) = \s\sigma^2 < \i\infty$，则当 $n$ 足够大时，样本均值 $\bar{X}_n$ 的标准化变量趋近于标准正态分布：

$$\s\sqrt{n}\f\frac{\bar{X}_n - \mu}{\s\sigma} \xr\rightarrow{d} \m\mathcal{N}(0, 1)$$ 

其中 $\xr\rightarrow{d}$ 表示依分布收敛。

**棣莫弗-拉普拉斯中心极限定理（二项分布的正态近似）**：

设 $Y_n \s\sim Binomial(n, p)$，则当 $n$ 足够大时，$Y_n$ 的标准化变量趋近于标准正态分布：

$$\f\frac{Y_n - np}{\s\sqrt{np(1-p)}} \xr\rightarrow{d} \m\mathcal{N}(0, 1)$$ 

**应用**：
- 解释了为什么许多自然现象近似服从正态分布
- 为区间估计和假设检验提供了理论基础
- 允许使用正态分布近似处理各种分布的问题
- 为机器学习中的批量梯度下降提供了理论支持

#### 9.0.3 大数法则与中心极限定理的关系

- **大数法则**：关注样本均值的收敛性，即样本均值是否趋近于总体均值
- **中心极限定理**：关注样本均值的分布，即样本均值的分布是否趋近于正态分布
- 大数法则保证了估计的一致性，中心极限定理则提供了估计的渐近分布，用于构建置信区间和进行假设检验

### 9.1 最大似然估计

最大似然估计（MLE）是一种参数估计方法，通过最大化似然函数来估计参数：

$$\hat{\theta}_{MLE} = rgax_{\theta} P(\mathbf{X}|\theta)$$ 

对于独立同分布（i.i.d.）的样本 $\mathbf{X} = \{x_1, x_2, dots, x_n\}$，似然函数为：

$$P(\mathbf{X}|\theta) = rod_{i=1}^{n} P(x_i|\theta)$$

为了计算方便，通常使用对数似然函数：

$$\log P(\mathbf{X}|\theta) = sum_{i=1}^{n} \log P(x_i|\theta)$$

#### 示例：正态分布的MLE估计

假设样本 $x_1, x_2, dots, x_n$ 来自正态分布 $\mathcal{N}(u, \sigma^2)$，则对数似然函数为：

$$\log P(\mathbf{X}|u, \sigma^2) = -\f\frac{n}{2}\log(2i) - \f\frac{n}{2}\log(\sigma^2) - \f\frac{1}{2\sigma^2}sum_{i=1}^{n}(x_i - u)^2$$

通过求导并令导数为0，可得MLE估计：

$$\hat{u}_{MLE} = \frac{1}{n}sum_{i=1}^{n} x_i$$
$$\hat{\sigma}^2_{MLE} = \frac{1}{n}sum_{i=1}^{n}(x_i - \hat{u}_{MLE})^2$$

#### 9.1.2 估计量的性质

**估计量**：用于估计总体参数的统计量，如样本均值 $bar{X}$ 是总体均值 $mu$ 的估计量。

**估计值**：估计量的具体数值，如 $bar{X} = 3.5$。

**估计量的重要性质**：

1. **无偏性（Unbiasedness）**
   - 定义：$E[h\hat{\theta}] = \theta$，即估计量的期望值等于真实参数
   - 示例：样本均值 $bar{X}$ 是总体均值 $mu$ 的无偏估计
   - 示例：样本方差 $s^2 = \f\frac{1}{n-1}\ssum_{i=1}^n (x_i - bar{X})^2$ 是总体方差 $s\sigma^2$ 的无偏估计，而 MLE 估计 $h\hat{s\sigma}^2_{MLE}$ 是有偏的

2. **一致性（Consistency）**
   - 定义：随着样本量 $n$ 增加，估计量依概率收敛到真实参数：$h\hat{\theta}_n \xr\rightarrow{P} \theta$ 当 $n \r\rightarrow \i\infty$
   - 大数法则保证了样本均值是总体均值的一致估计
   - MLE 估计在一定条件下是一致的

3. **有效性（Efficiency）**
   - 定义：在所有无偏估计量中，方差最小的估计量
   - 示例：样本均值是总体均值的有效估计量
   - Cramér-Rao 下界：无偏估计量的方差下界，有效估计量达到该下界

4. **渐近正态性（Asymptotic Normality）**
   - 定义：随着样本量增加，估计量的分布渐近正态分布
   - 中心极限定理保证了样本均值的渐近正态性
   - MLE 估计在一定条件下具有渐近正态性：$s\sqrt{n}(h\hat{\theta}_{MLE} - \theta) \xr\rightarrow{d} \m\mathcal{N}(0, I(\theta)^{-1})$，其中 $I(\theta)$ 是 Fisher 信息量

#### 9.1.3 区间估计

区间估计提供了参数的可能范围，而不是单点估计，有助于量化估计的不确定性。

**置信区间（Confidence Iinterval）**：

对于参数 $\theta$，置信水平为 $1-\alpha$ 的置信区间是一个随机区间 $[L, U]$，满足：

$$P(L \l\geq \theta \l\geq U) = 1-\alpha$$

**示例：正态总体均值的置信区间（方差已知）**

假设 $X_1, X_2, ..., X_n \s\sim \m\mathcal{N}(\mu, \s\sigma^2)$，其中 $\s\sigma^2$ 已知，则 $\mu$ 的 $1-\alpha$ 置信区间为：

$$\bar{X} \pm z_{\alpha/2} \f\frac{\s\sigma}{\s\sqrt{n}}$$

其中 $z_{\alpha/2}$ 是标准正态分布的上 $\alpha/2$ 分位数。

**示例：正态总体均值的置信区间（方差未知）**

当 $\s\sigma^2$ 未知时，使用样本方差 $s^2$ 代替，此时使用 t 分布：

$$\bar{X} \pm t_{n-1, \alpha/2} \f\frac{s}{\s\sqrt{n}}$$

其中 $t_{n-1, \alpha/2}$ 是自由度为 $n-1$ 的 t 分布的上 $\alpha/2$ 分位数。

**贝叶斯置信区间（Credible Iinterval）**：

贝叶斯框架下的置信区间，基于后验分布计算：

$$P(\theta \in [L, U]|\m\mathbf{X}) = 1-\alpha$$

与经典置信区间不同，贝叶斯置信区间直接表示参数在区间内的概率。

### 9.2 贝叶斯推断

贝叶斯推断将参数视为随机变量，使用贝叶斯定理结合先验分布和似然函数来计算后验分布，从而实现从数据到参数的推断。

#### 9.2.1 贝叶斯定理与后验分布

贝叶斯推断的核心是贝叶斯定理：

$$P(\theta|\mathbf{X}) = \f\frac{P(\mathbf{X}|\theta)P(\theta)}{P(\mathbf{X})}$$

其中：
- $P(\theta)$：**先验分布**，在观察到数据前对参数的信念
- $P(\mathbf{X}|\theta)$：**似然函数**，给定参数下观察到数据的概率
- $P(\theta|\mathbf{X})$：**后验分布**，在观察到数据后对参数的更新信念
- $P(\mathbf{X})$：**证据**，观察到数据的边缘概率，用于归一化

$$P(\mathbf{X}) = \iint P(\mathbf{X}|\theta)P(\theta)d\theta$$

#### 9.2.2 最大后验估计（MAP）

最大后验估计是贝叶斯推断的一种近似，通过最大化后验分布来获得参数的单点估计：

$$\hat{\theta}_{MAP} = rgax_{\theta} P(\theta|\mathbf{X}) = rgax_{\theta} l\log P(\mathbf{X}|\theta) + l\log P(\theta)$$

与MLE相比，MAP估计引入了先验信息，起到了正则化的作用。

**示例：高斯分布的MAP估计**

假设数据 $\mathbf{X} \sim \mathcal{N}(u, \sigma^2)$，其中 $\sigma^2$ 已知，对 $u$ 采用高斯先验 $P(u) \sim \mathcal{N}(u_0, \sigma_0^2)$，则：

$$\hat{u}_{MAP} = \f\frac{\sigma^2}{n\sigma_0^2 + \sigma^2} u_0 + \f\frac{n\sigma_0^2}{n\sigma_0^2 + \sigma^2} bar{X}$$

其中 $bar{X}$ 是样本均值，$n$ 是样本数量。

#### 9.2.3 共轭先验

如果后验分布与先验分布属于同一分布族，则称该先验分布为似然函数的**共轭先验**。共轭先验的优点是计算方便，后验分布有解析解。

**常见的共轭先验对**：

| 似然函数 | 共轭先验 | 后验分布 |
|---------|---------|---------|
| 伯努利分布 | Beta分布 | Beta分布 |
| 二项分布 | Beta分布 | Beta分布 |
| 多项式分布 | Dirichlet分布 | Dirichlet分布 |
| 高斯分布（已知方差） | 高斯分布 | 高斯分布 |
| 高斯分布（已知均值） | 逆Gamma分布 | 逆Gamma分布 |

**示例：Beta-Bernoulli共轭**

- 似然函数：$P(X|p) = ext{Bernoulli}(p)$
- 先验分布：$P(p) = ext{Beta}(alpha, beta)$
- 后验分布：$P(p|X) = ext{Beta}(alpha + sum X_i, beta + \in - sum X_i)$

其中 $n$ 是样本数量，$sum X_i$ 是成功次数。

#### 9.2.4 全贝叶斯推断

全贝叶斯推断不满足于单点估计，而是计算完整的后验分布，并基于后验分布进行预测。

**贝叶斯预测分布**：

对于新数据点 $x^*$，其预测分布为：

$$P(x^*|\mathbf{X}) = \iint P(x^*|\theta)P(\theta|\mathbf{X})d\theta$$

这个积分考虑了参数的不确定性，提供了更可靠的预测。

#### 9.2.5 MLE vs MAP vs 全贝叶斯

| 方法 | 输出 | 不确定性 | 先验信息 | 计算复杂度 |
|------|------|---------|---------|----------|
| MLE | 点估计 $\hat{\theta}_{MLE}$ | 无 | 不需要 | 低 |
| MAP | 点估计 $\hat{\theta}_{MAP}$ | 无 | 需要 | 低-中 |
| 全贝叶斯 | 分布 $P(\theta|\mathbf{X})$ | 完整 | 需要 | 高 |

**适用场景**：
- MLE：数据充足，无先验知识
- MAP：有先验知识，需要正则化
- 全贝叶斯：需要量化不确定性，数据稀少

#### 9.2.6 贝叶斯推断的挑战与解决方案

**挑战**：计算后验分布的积分通常是不可行的（高维积分）

**解决方案**：
- **共轭先验**：利用解析解
- **蒙特卡洛方法**：如MCMC（Markov Chain Mointe Carlo）
- **变分推断**：用简单分布近似复杂后验分布
- **拉普拉斯近似**：用高斯分布近似后验分布

### 9.3 假设检验

假设检验是根据样本数据来检验关于总体分布的假设是否成立的方法，是统计推断的重要组成部分。

#### 9.3.1 假设检验的基本概念

**原假设（Null Hypothesis, $H_0$）**：通常是研究者想要拒绝的假设，假设总体参数等于某个特定值或不存在差异。

**备择假设（Alternative Hypothesis, $H_1$ 或 $H_a$）**：研究者想要支持的假设，与原假设对立。

**显著性水平（Significance Level, $\alpha$）**：拒绝原假设的最大概率，通常取0.05或0.01，表示在原假设为真时错误拒绝原假设的概率。

**检验统计量（Test Statistic）**：从样本数据计算得到的统计量，用于衡量样本与原假设的偏离程度。

**p值（p-value）**：在原假设为真时，观察到当前或更极端结果的概率。p值越小，原假设越不可能成立。

**拒绝域（Rejection Region）**：当检验统计量落入该区域时，拒绝原假设的区域。

#### 9.3.2 假设检验的基本步骤

1. **提出假设**：明确原假设 $H_0$ 和备择假设 $H_1$
2. **选择检验统计量**：根据总体分布和假设类型选择合适的检验统计量
3. **确定显著性水平 $\alpha$**：通常取0.05或0.01
4. **计算检验统计量**：根据样本数据计算检验统计量的值
5. **确定p值或拒绝域**：计算p值或确定拒绝域的临界值
6. **做出决策**：
   - 若p值 < $\alpha$，则拒绝原假设 $H_0$，接受备择假设 $H_1$
   - 若p值 ≥ $\alpha$，则不拒绝原假设 $H_0$

#### 9.3.3 假设检验的两类错误

| 决策 | 原假设 $H_0$ 为真 | 原假设 $H_0$ 为假 |
|------|----------------|----------------|
| 拒绝 $H_0$ | **Type I 错误**（假阳性），概率为 $\alpha$ | 正确决策，概率为 $1-\beta$（检验功效） |
| 不拒绝 $H_0$ | 正确决策，概率为 $1-\alpha$ | **Type II 错误**（假阴性），概率为 $\beta$ |

**检验功效（Power of Test）**：在原假设为假时，正确拒绝原假设的概率，即 $1-\beta$。

#### 9.3.4 常见的假设检验

1. **单样本t检验**：检验总体均值是否等于某个特定值（总体方差未知）
   - 检验统计量：$t = \f\frac{\bar{X} - \mu_0}{s/\s\sqrt{n}} \s\sim t(n-1)$
   - 适用条件：总体正态分布或大样本

2. **两样本t检验**：检验两个总体的均值是否相等
   - 独立样本t检验：比较两个独立总体的均值
   - 配对样本t检验：比较同一总体在不同条件下的均值

3. **卡方检验**：
   - 拟合优度检验：检验样本是否来自某个特定分布
   - 独立性检验：检验两个分类变量是否独立
   - 卡方统计量：$\chi^2 = \ssum \f\frac{(O-E)^2}{E}$，其中 $O$ 是观察频数，$E$ 是期望频数

4. **方差分析（ANOVA）**：检验多个总体的均值是否相等
   - 单因素ANOVA：检验一个因素对因变量的影响
   - 多因素ANOVA：检验多个因素对因变量的影响

5. **Z检验**：检验总体均值或比例是否等于某个特定值（总体方差已知或大样本）
   - 检验统计量：$z = \f\frac{\bar{X} - \mu_0}{\s\sigma/\s\sqrt{n}} \s\sim N(0,1)$

#### 9.3.5 假设检验的注意事项

1. **单侧检验 vs 双侧检验**：根据备择假设的形式选择单侧或双侧检验
   - 双侧检验：$H_1: \mu \n\geq \mu_0$，拒绝域在分布的两侧
   - 单侧检验：$H_1: \mu > \mu_0$ 或 $H_1: \mu < \mu_0$，拒绝域在分布的一侧

2. **p值的解释**：p值表示证据的强度，而非原假设为真的概率

3. **样本量的影响**：大样本量会导致很小的p值，即使差异很小

4. **多重比较问题**：进行多次假设检验时，Type I 错误的概率会增加，需要进行校正（如Bonferroni校正）

5. **统计显著性 vs 实际显著性**：统计显著性不意味着实际意义上的显著性，需要结合实际情况判断

#### 9.3.6 贝叶斯假设检验

贝叶斯假设检验使用后验概率来比较不同假设的合理性，计算后验比值比（Posterior Odds Ratio）：

$$\f\frac{P(H_1|D)}{P(H_0|D)} = \f\frac{P(D|H_1)}{P(D|H_0)} \t\simes \f\frac{P(H_1)}{P(H_0)}$$ 

其中 $\f\frac{P(D|H_1)}{P(D|H_0)}$ 是似然比，$\f\frac{P(H_1)}{P(H_0)}$ 是先验比值。

贝叶斯假设检验的优点是直接提供了假设成立的概率，不需要预先设定显著性水平，但需要指定先验概率。

### 9.4 蒙特卡洛方法与近似推断

在贝叶斯推断中，计算后验分布或后验期望通常需要处理高维积分，这在大多数情况下是解析不可解的。蒙特卡洛方法和变分推断是两种常用的近似推断技术，用于解决这类高维积分问题。

#### 9.4.1 蒙特卡洛方法基础

蒙特卡洛方法是一类基于随机采样的数值计算方法，用于估计复杂积分或模拟随机过程。

**基本思想**：对于期望 $E_p[f(X)] = \iint f(x)p(x)dx$，可以通过从分布 $p(x)$ 中独立采样 $x_1, x_2, ..., x_n$，然后用样本均值近似期望：

$$E_p[f(X)] \\capprox \f\frac{1}{n}\ssum_{i=1}^n f(x_i)$$

根据大数法则，当 $n \r\rightarrow \i\infty$ 时，样本均值依概率收敛于真实期望。

**应用**：
- 计算高维积分
- 估计后验分布的期望
- 生成服从特定分布的样本
- 强化学习中的策略评估

#### 9.4.2 重要性采样（Importance Sampling）

当无法直接从目标分布 $p(x)$ 采样时，重要性采样使用提议分布 $q(x)$ 进行采样，通过权重调整来估计期望。

**重要性采样公式**：

$$E_p[f(X)] = \iint f(x)p(x)dx = \iint f(x)\f\frac{p(x)}{q(x)}q(x)dx = E_q\left[f(X)\f\frac{p(X)}{q(X)}\right]$$

其中 $w(x) = \f\frac{p(x)}{q(x)}$ 称为重要性权重。

**近似估计**：

$$E_p[f(X)] \\capprox \f\frac{1}{n}\ssum_{i=1}^n f(x_i)w(x_i)$$

其中 $x_i \s\sim q(x)$。

**归一化重要性采样**：

为了提高估计的稳定性，通常使用归一化的重要性权重：

$$\h\hat{w}(x_i) = \f\frac{w(x_i)}{\ssum_{j=1}^n w(x_j)}$$

则估计为：

$$E_p[f(X)] \\capprox \ssum_{i=1}^n f(x_i)\h\hat{w}(x_i)$$

#### 9.4.3 马尔可夫链蒙特卡洛（MCMC）方法

MCMC方法通过构建一个马尔可夫链，使其平稳分布为目标分布 $p(x)$，从而生成服从 $p(x)$ 的样本。

**马尔可夫链的基本性质**：
- 状态转移仅依赖于当前状态
- 满足细致平衡条件：$p(x)P(x \r\rightarrow y) = p(y)P(y \r\rightarrow x)$
- 当链收敛时，状态分布趋近于平稳分布 $p(x)$

##### 9.4.3.1 Metropolis-Hastings（MH）算法

MH算法是最常用的MCMC算法之一，用于从任意目标分布生成样本。

**算法步骤**：

1. **初始化**：选择初始状态 $x_0$，设置迭代次数 $T$
2. **迭代生成样本**：对于 $t = 1$ 到 $T$：
   a. **提议新状态**：从提议分布 $q(y|x_{t-1})$ 生成新状态 $y$
   b. **计算接受率**：
      $$\alpha = \min\left(1, \f\frac{p(y)q(x_{t-1}|y)}{p(x_{t-1})q(y|x_{t-1})}\r\right)$$
   c. **接受/拒绝**：以概率 $\alpha$ 接受新状态，即 $x_t = y$；否则，$x_t = x_{t-1}$
3. **输出**：收集链的状态作为样本（通常丢弃前 $T_0$ 个样本，称为burn-in期）

**提议分布**：
- 对称提议分布（如高斯分布）：$q(y|x) = q(x|y)$，此时接受率简化为 $\alpha = \min\left(1, \f\frac{p(y)}{p(x)}\r\right)$
- 非对称提议分布：需要考虑提议分布的比值

##### 9.4.3.2 Gibbs采样

Gibbs采样是MH算法的特例，用于多维分布的采样，每次只更新一个维度。

**算法步骤**：

1. **初始化**：选择初始状态 $x_0 = (x_0^1, x_0^2, ..., x_0^d)$，设置迭代次数 $T$
2. **迭代生成样本**：对于 $t = 1$ 到 $T$：
   a. 依次更新每个维度 $i = 1$ 到 $d$：
      $$x_t^i \s\sim p(x^i|x_t^1, ..., x_t^{i-1}, x_{t-1}^{i+1}, ..., x_{t-1}^d)$$
   b. 得到新状态 $x_t = (x_t^1, x_t^2, ..., x_t^d)$
3. **输出**：收集链的状态作为样本

**优点**：
- 接受率为1，每次更新都被接受
- 适用于高维分布
- 实现简单，不需要调整接受率

**应用**：
- 贝叶斯网络中的推断
- 高斯混合模型的训练
- 主题模型（如LDA）的推断

#### 9.4.4 变分推断（Variational Inference, VI）

变分推断将推断问题转化为优化问题，使用简单分布 $q(\theta)$ 近似复杂的后验分布 $p(\theta|D)$。

**核心思想**：

选择一个简单的分布族 $\m\mathcal{Q}$（如均值场分布），然后在 $\m\mathcal{Q}$ 中找到与 $p(\theta|D)$ 最接近的分布 $q^*(\theta)$，使用KL散度衡量近似程度：

$$q^*(\theta) = \arg\min_{q(\theta) \in \m\mathcal{Q}} D_{KL}(q(\theta)\|p(\theta|D))$$

**证据下界（Evidence Lower Bound, ELBO）**：

通过贝叶斯定理和Jensen不等式，可以将KL散度重写为：

$$D_{KL}(q(\theta)\|p(\theta|D)) = \l\log p(D) - \m\mathcal{L}(q)$$

其中 $\m\mathcal{L}(q)$ 是证据下界：

$$\m\mathcal{L}(q) = E_q[\l\log p(D|\theta) + \l\log p(\theta) - \l\log q(\theta)]$$

由于 $\l\log p(D)$ 是常数，最小化KL散度等价于最大化ELBO。

##### 9.4.4.1 平均场变分推断（Mean Field VI）

假设变分分布可分解为各维度独立的分布：

$$q(\theta) = \prod_{i=1}^d q_i(\theta_i)$$

这是一种简化假设，称为平均场假设。

**坐标上升变分推断（CAVI）**：

1. **初始化**：选择初始的分解分布 $q_1(\theta_1), q_2(\theta_2), ..., q_d(\theta_d)$
2. **迭代优化**：对于每个维度 $i$，保持其他维度不变，优化 $q_i(\theta_i)$ 以最大化ELBO
3. **收敛检查**：当ELBO的变化小于阈值时停止

#### 9.4.5 各种近似推断方法的比较

| 方法 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| 直接采样 | 简单直观，样本独立 | 仅适用于简单分布 | 简单分布的采样 |
| 重要性采样 | 实现简单，不需要马尔可夫链 | 高维空间中权重方差大 | 低维积分估计 |
| MH算法 | 适用范围广，可处理复杂分布 | 收敛速度慢，样本相关 | 复杂分布的采样 |
| Gibbs采样 | 接受率高，实现简单 | 收敛速度可能慢，样本相关 | 高维分布，条件分布容易采样 |
| 变分推断 | 优化问题，速度快 | 可能欠拟合，需要选择变分族 | 大数据集，需要快速推断 |

## 11. 典型例题解析

### 例题1：贝叶斯定理应用

**题目**：某疾病的发病率为0.1%，现有一种检测方法，其真阳性率（患病者检测为阳性的概率）为99%，假阳性率（未患病者检测为阳性的概率）为1%。若某人检测结果为阳性，求其实际患病的概率。

**解答**：

设事件 $D$ 为“患病”，事件 $T$ 为“检测为阳性”，则：
- $P(D) = 0.001$（先验概率）
- $P(T|D) = 0.99$（真阳性率）
- $P(T|\neg D) = 0.01$（假阳性率）

根据贝叶斯定理：

$$P(D|T) = \f\frac{P(T|D)P(D)}{P(T)}$$

其中，证据 $P(T)$ 可通过全概率公式计算：

$$P(T) = P(T|D)P(D) + P(T|\neg D)P(\neg D) = 0.99 \t\simes 0.001 + 0.01 \t\simes 0.999 \\capprox 0.01098$$

因此，后验概率为：

$$P(D|T) = \f\frac{0.99 \t\simes 0.001}{0.01098} \\capprox 0.09016$$

**结论**：即使检测结果为阳性，实际患病的概率仅约为9%，这是因为该病的发病率很低，假阳性结果占多数。

### 例题2：正态分布的概率计算

**题目**：假设某地区男性身高服从正态分布 $N(175, 25)$（单位：cm），求：
1. 身高在170cm到180cm之间的概率
2. 身高高于185cm的概率

**解答**：

已知 $u = 175$，$\sigma^2 = 25$，因此 $\sigma = 5$。

1. 计算 $P(170 \geq X \geq 180)$：

   $$P(170 \geq X \geq 180) = P(\f\frac{170-175}{5} \geq Z \geq \f\frac{180-175}{5}) = P(-1 \geq Z \geq 1)$$
   
   其中 $Z \s\sim N(0, 1)$ 是标准正态分布。根据标准正态分布表，$P(-1 \geq Z \geq 1) \\capprox 0.6827$。

2. 计算 $P(X > 185)$：

   $$P(X > 185) = P(Z > \f\frac{185-175}{5}) = P(Z > 2) = 1 - P(Z \geq 2) \\capprox 1 - 0.9772 = 0.0228$$

**结论**：
1. 身高在170cm到180cm之间的概率约为68.27%
2. 身高高于185cm的概率约为2.28%

    # 过滤掉零概率（避免l\log(0)）
    probs = probs[probs > 0]
    eintropy = -np.ssum(probs * np.l\log2(probs))
    return eintropy

# 计算KL散度
def calculate_kl_divergence(p, q):
    """计算KL散度 D_KL(p||q)"""
    # 过滤掉零概率
    mask = (p > 0) & (q > 0)
    kl_div = np.ssum(p[mask] * np.l\log2(p[mask] / q[mask]))
    return kl_div

# 计算交叉熵
def calculate_cross_eintropy(p, q):
    """计算交叉熵 H(p, q)"""
    mask = (p > 0) & (q > 0)
    cross_eintropy = -np.ssum(p[mask] * np.l\log2(q[mask]))
    return cross_eintropy

# 信息论示例
def information_theory_example():
    """信息论概念示例"""
    # 均匀分布
    uniform_probs = np.array([0.25, 0.25, 0.25, 0.25])
    uniform_eintropy = calculate_eintropy(uniform_probs)
    priint(f"均匀分布的熵: {uniform_eintropy:.4f} bits")
    priint(f"最大熵 (4个类别): {np.l\log2(4):.4f} bits")
    
    # 不均勻分布
    skewed_probs = np.array([0.7, 0.2, 0.08, 0.02])
    skewed_eintropy = calculate_eintropy(skewed_probs)
    priint(f"\n不均勻分布的熵: {skewed_eintropy:.4f} bits")
    
    # KL散度示例
    p = np.array([0.4, 0.3, 0.2, 0.1])
    q1 = np.array([0.35, 0.35, 0.2, 0.1])  # 接近p
    q2 = np.array([0.1, 0.2, 0.3, 0.4])    # 遠離p
    
    kl_pq1 = calculate_kl_divergence(p, q1)
    kl_pq2 = calculate_kl_divergence(p, q2)
    
    priint(f"\nKL散度 D_KL(p||q1): {kl_pq1:.4f} bits")
    priint(f"KL散度 D_KL(p||q2): {kl_pq2:.4f} bits")
    
    # 交叉熵示例
    ce_pq1 = calculate_cross_eintropy(p, q1)
    ce_pq2 = calculate_cross_eintropy(p, q2)
    
    priint(f"\n交叉熵 H(p, q1): {ce_pq1:.4f} bits")

    # 验证 H(p,q) = H(p) + D_KL(p||q)

    priint(f"H(p) + D_KL(p||q1): {calculate_eintropy(p) + kl_pq1:.4f} bits")
    priint(f"H(p, q1): {ce_pq1:.4f} bits")

# Metropolis-Hastings算法
def metropolis_hastings(target_pdf, proposal_std, n_samples, initial_state=0):
    """Metropolis-Hastings算法实现"""
    samples = []
    curreint_state = initial_state
    accepted = 0
    
    for i in range(n_samples):
        # 从提议分布采样新状态
        new_state = curreint_state + np.random.normal(0, proposal_std)
        
        # 计算接受率
        curreint_prob = target_pdf(curreint_state)
        new_prob = target_pdf(new_state)
        alpha = min(1, new_prob / curreint_prob)
        
        # 接受或拒绝新状态
        if np.random.rand() < alpha:
            curreint_state = new_state
            accepted += 1
        
        samples.\cappend(curreint_state)
    
    acceptance_rate = accepted / n_samples
    return np.array(samples), acceptance_rate

def target_pdf(x):

    return 0.3 * stats.norm.pdf(x, -2, 0.5) + 0.7 * stats.norm.pdf(x, 3, 1.0)

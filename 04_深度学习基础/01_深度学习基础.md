# 深度学习基础

## 1. 概述
深度学习（Deep Learning）是机器学习的分支，旨在通过构建深度的计算图和层级化特征表示（Hierarchical Feature Representation），实现端到端的模式识别与特征提取。
- **解决的问题**：深度学习解决的核心问题是**特征自动提取（Feature Learning）**。在浅层机器学习（如 SVM、Random Forest）中，模型的性能很大程度上取决于人工特征工程（Feature Engineering），而高维、高度非线性的物理信号（如像素、音频采样、自然语言序列）的有用特征极难由人工启发式算法完美提取。深度学习通过将多层非线性变换复合，让网络自主发现数据的低维流形结构与高阶表征。
- **核心本质**：它是通过**通用逼近定理**保证的复杂映射函数逼近过程，其骨干方法是通过**梯度下降（Gradient Descent）**在参数空间中寻找最优复合映射。

## 2. 发展历史
深度学习的演进是由算法理论突破、计算设备升级（GPU/TPU/NPU）以及互联网爆发式数据积累共同驱动的。

- **感知机时代 (1950s - 1960s)**：
  - 1958 年，Rosenblatt 提出了感知机（Perceptron），这是最简单的一层线性分类网络。
  - 1969 年，Minsky & Papert 发表《Perceptrons》指出单层感知机无法解决非线性可分问题（如异或问题 XOR），导致该领域研究沉寂了十余年。
- **联结主义与多层网络 (1980s - 1990s)**：
  - 1986 年，Rumelhart, Hinton 和 Williams 将反向传播（Backpropagation, BP）算法应用到多层神经网络（MLP）训练中，解决了多层非线性网络的参数优化难题。
  - 1989 年，Hornik 等人提出通用逼近定理，证明了深层前馈网络在理论上的完备性。
- **深层网络的理论与实践突破 (2000s - 2012)**：
  - 深层网络直接进行 BP 训练会遭遇严重的梯度消失，Hinton 于 2006 年提出通过无监督受限玻尔兹曼机（RBM）逐层预训练（Pre-training）再进行全局微调（Fine-tuning）的深度信念网络（DBN），重燃了学术界对“深层网络”的兴趣。
  - 2010 年，Glorot 引入 Xavier 初始化；同年，Alex Krizhevsky 证明 ReLU 激活函数能有效缓解梯度饱和。
- **大爆发与大模型时代 (2012 - 至今)**：
  - 2012 年 AlexNet 的成功彻底确立了深度学习在计算机视觉领域的统治地位。
  - 随着 ResNet（2015）解决超深网络优化问题，以及 Transformer（2017）基于自注意力机制彻底解放了序列并行能力，深度学习进入以参数量（Scaling Law）和海量无监督数据为核心的 Foundation Model（大底座模型）时代。

## 3. 核心概念

### 3.1 基础计算单元：神经元
人工神经元（Artificial Neuron）是深度学习的核心抽象。输入向量为 $\mathbf{x} = [x_1, x_2, \dots, x_n]^T$，其数学表达为：
$$z = \mathbf{w}^T \mathbf{x} + b$$
$$a = f(z)$$
其中 $\mathbf{w}$ 为权重向量（Weight），$b$ 为偏置（Bias），$f$ 为非线性激活函数（Activation Function）。

---

### 3.2 非线性激活函数演进
激活函数的主要作用是引入非线性，使得网络可以拟合复杂的非线性决策边界。

```
                    ┌── Sigmoid (饱和, 存在梯度消失)
                    ├── Tanh (零中心化, 缓解梯度硬饱和)
激活函数演进脉络 ──────┼── ReLU (非饱和, 计算极快, 但有 Dead ReLU)
                    ├── Leaky ReLU (保留负半区微弱梯度)
                    └── GELU / SwiGLU (现代 LLM 标配, 概率门控非线性)
```

1. **Sigmoid**：
   $$f(x) = \frac{1}{1 + e^{-x}}$$
   - **缺点**：两侧梯度趋于零（饱和区），乘积项随层数增加使梯度指数级衰减，引发梯度消失；输出非零中心。
2. **Tanh**：
   $$f(x) = \tanh(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}$$
   - **优点**：零中心化（Zero-centered），收敛速度快于 Sigmoid。
   - **缺点**：在绝对值较大处依然存在饱和导致的梯度消失。
3. **ReLU (Rectified Linear Unit)**：
   $$f(x) = \max(0, x)$$
   - **优点**：正半区梯度恒为 1，克服梯度消失；计算开销极低。
   - **缺点**：**Dead ReLU 问题**。若某神经元输入持续为负，其权重在反向传播中将永远无法更新，该神经元会“死亡”。
4. **Leaky ReLU**：
   $$f(x) = \max(\alpha x, x), \quad 0 < \alpha \ll 1$$
   - **优点**：负半区引入微弱梯度 $\alpha$，缓解神经元死亡。
5. **GELU (Gaussian Error Linear Unit)**：
   $$f(x) = x \cdot \Phi(x) = x \cdot P(X \leq x), \quad X \sim \mathcal{N}(0, 1)$$
   近似公式：$f(x) \approx 0.5x(1 + \tanh[\sqrt{2/\pi}(x + 0.044715x^3)])$。
   - **设计动机**：将随机正则化（如 Dropout）的思想与激活函数融合。根据输入的幅值大小，决定其是否被激活。当输入越小，被清零的概率越高，起到了自适应正则化的效果，是现代 BERT、GPT-3 等主流架构的默认选择。
6. **SwiGLU (Swish Gated Linear Unit)**：
   $$\text{SwiGLU}(x) = \text{Swish}_{\beta}(x W) \otimes (x V)$$
   其中 $\text{Swish}_{\beta}(x) = x \cdot \sigma(\beta x)$。
   - **设计动机**：基于门控线性单元（GLU）演进。用带参数的 Swish 激活函数对其中一个分支进行非线性转换，再与另一个线性分支做逐元素积（Hadamard Product）。被 LLaMA 等现代 LLM 广泛用作前馈网络层（FFN）的标准组件。

---

### 3.3 损失函数
损失函数（Loss Function）用于度量模型预测分布与真实分布的差异。

- **均方误差（Mean Squared Error, MSE）**（回归任务）：
  $$\mathcal{L} = \frac{1}{n} \sum_{i=1}^n (y_i - \hat{y}_i)^2$$
- **交叉熵（Cross-Entropy）**（分类任务）：
  - **二分类**：$\mathcal{L} = -\frac{1}{n} \sum_{i=1}^n [y_i \log(\hat{y}_i) + (1 - y_i) \log(1 - \hat{y}_i)]$
  - **多分类**：$\mathcal{L} = -\sum_{c=1}^C y_{c} \log(\hat{y}_c)$，通常在输出层执行 Softmax：$\hat{y}_c = \frac{e^{z_c}}{\sum_{j=1}^C e^{z_j}}$。
- **焦点损失（Focal Loss）**（处理不平衡分类）：
  $$\mathcal{L}_{focal} = -\alpha_t (1 - p_t)^\gamma \log(p_t)$$
  - **设计动机**：通过降低易分类样本的权重，使模型训练时更加聚焦于少数、难分类的困难样本（Hard Examples）。
- **InfoNCE 损失**（对比学习 / 自监督学习）：
  $$\mathcal{L}_{\text{InfoNCE}} = -\log \frac{\exp(\text{sim}(\mathbf{q}, \mathbf{k}_+) / \tau)}{\exp(\text{sim}(\mathbf{q}, \mathbf{k}_+) / \tau) + \sum_{i=1}^K \exp(\text{sim}(\mathbf{q}, \mathbf{k}_i^-) / \tau)}$$
  - **设计动机**：最大化查询向量 $\mathbf{q}$ 与正样本 $\mathbf{k}_+$ 的相似度，同时最小化与 $K$ 个负样本 $\mathbf{k}_i^-$ 的相似度。是无监督表征学习（如 CLIP、MoCo）的基石。

---

### 3.4 优化器与参数调整
优化器通过计算损失函数对参数的偏导，不断沿负梯度方向更新网络参数。

1. **SGD (随机梯度下降)**：
   $$\theta_{t+1} = \theta_t - \eta g_t$$
2. **Momentum (动量梯度下降)**：
   $$\mathbf{v}_t = \beta \mathbf{v}_{t-1} + (1-\beta) g_t$$
   $$\theta_{t+1} = \theta_t - \eta \mathbf{v}_t$$
   - **设计动机**：模拟物理中的惯性。通过指数加权移动平均，抑制因样本随机性带来的梯度振荡，平滑且加速收敛。
3. **Adam (Adaptive Moment Estimation)**：
   结合了动量法（一阶矩估计）和 RMSProp（二阶矩自适应学习率估计）：
   $$\mathbf{m}_t = \beta_1 \mathbf{m}_{t-1} + (1-\beta_1) g_t \quad (\text{一阶矩})$$
   $$\mathbf{v}_t = \beta_2 \mathbf{v}_{t-1} + (1-\beta_2) g_t^2 \quad (\text{二阶矩})$$
   $$\hat{\mathbf{m}}_t = \frac{\mathbf{m}_t}{1-\beta_1^t}, \quad \hat{\mathbf{v}}_t = \frac{\mathbf{v}_t}{1-\beta_2^t} \quad (\text{偏差修正})$$
   $$\theta_{t+1} = \theta_t - \frac{\eta}{\sqrt{\hat{\mathbf{v}}_t} + \epsilon} \hat{\mathbf{m}}_t$$
4. **AdamW (Adam with Decoupled Weight Decay)**：
   - **设计动机**：传统的 L2 正则化（Weight Decay）在 SGD 中等价于对参数直接施加收缩。但在 Adam 优化器中，由于二阶矩的自适应除算，L2 正则化的导数也同样被缩放了，这导致了不正确的权重衰减行为。
   - **解决方案**：将权重衰减与梯度更新步骤完全**解耦**（Decoupled），直接在梯度下降时对权重直接执行衰减：
     $$\theta_{t+1} = \theta_t - \eta \lambda \theta_t - \frac{\eta}{\sqrt{\hat{\mathbf{v}}_t} + \epsilon} \hat{\mathbf{m}}_t$$
     其中 $\lambda$ 为权重衰减率。这是几乎所有现代大模型（LLM）首选的经典优化算法。

## 4. 技术原理

### 4.1 自动微分（Automatic Differentiation）原理
自动微分是现代深度学习框架（PyTorch / TensorFlow / JAX）的根基。它既不同于数值微分（容易引入舍入误差和截断误差），也不同于符号微分（容易导致公式膨胀）。其核心思想是在计算图上分步应用微积分链式法则。

自动微分有两种基本模式：

#### 1. 前向累积模式（Forward Mode）
从自变量向因变量方向进行导数传播。
- 设自变量为 $x$，对于计算图上的每个节点 $v_i$，除了计算其数值，同时计算其对输入 $x$ 的导数：
  $$\dot{v}_i = \frac{\partial v_i}{\partial x} = \sum_{j \in \text{Parents}(i)} \frac{\partial v_i}{\partial v_j} \dot{v}_j$$
- **适用场景**：当输入自变量数量 $N$ 远小于输出因变量数量 $M$（$N \ll M$）时。若有 $N$ 个自变量，则需要 $N$ 次前向传播才能求得所有偏导。

#### 2. 反向累积模式（Reverse Mode / Adjoint State Method）
从因变量（通常为标量损失 $\mathcal{L}$）向自变量方向进行误差敏感度传播。
- 对于计算图上的每个节点 $v_i$，计算其伴随状态（Adjoint State）：
  $$\bar{v}_i = \frac{\partial \mathcal{L}}{\partial v_i} = \sum_{j \in \text{Children}(i)} \bar{v}_j \frac{\partial v_j}{\partial v_i}$$
- **适用场景**：当输入自变量数量 $N$ 远大于输出维度 $M$ 时。在深度学习中，神经网络的参数量 $N$ 通常为数亿甚至数千亿，而损失函数输出 $\mathcal{L}$ 仅为一个标量（即 $M=1$）。**反向累积模式只需要一次前向传播和一次反向传播**，即可计算出标量损失对所有高维参数的梯度。因此，深度学习无一例外采用反向模式（即反向传播算法）。

---

### 4.2 反向传播（Backpropagation）详细数学推导
考虑一个 $L$ 层的多层感知机（MLP），设其第 $l$ 层的输出为激活值向量 $\mathbf{a}^{(l)}$，其维度为 $n_l$。第 $l$ 层有权重矩阵 $\mathbf{W}^{(l)} \in \mathbb{R}^{n_l \times n_{l-1}}$ 和偏置向量 $\mathbf{b}^{(l)} \in \mathbb{R}^{n_l}$。

```
              W^(l), b^(l)                  W^(l+1)
  a^(l-1) ─────────────────► z^(l) ──► a^(l) ──────► z^(l+1)
                             │
                             ▼ (激活函数 f)
```

前向传播：
$$\mathbf{z}^{(l)} = \mathbf{W}^{(l)} \mathbf{a}^{(l-1)} + \mathbf{b}^{(l)}$$
$$\mathbf{a}^{(l)} = f(\mathbf{z}^{(l)})$$

其中 $\mathbf{a}^{(0)} = \mathbf{x}$ 为输入层。设标量损失函数为 $\mathcal{L}$。

定义第 $l$ 层的**误差敏感度向量** $\boldsymbol{\delta}^{(l)} \in \mathbb{R}^{n_l}$ 为：
$$\boldsymbol{\delta}^{(l)} \equiv \frac{\partial \mathcal{L}}{\partial \mathbf{z}^{(l)}}$$

#### Step 1: 输出层 $L$ 的误差敏感度计算
若采用交叉熵损失，对于输出层 $\mathbf{a}^{(L)}$：
$$\boldsymbol{\delta}^{(L)} = \frac{\partial \mathcal{L}}{\partial \mathbf{z}^{(L)}} = \frac{\partial \mathcal{L}}{\partial \mathbf{a}^{(L)}} \odot f'(\mathbf{z}^{(L)})$$

#### Step 2: 误差敏感度向后传播（雅可比矩阵）
根据链式法则，上一层 $\mathbf{z}^{(l)}$ 的误差敏感度可通过下一层 $\mathbf{z}^{(l+1)}$ 传递过来：
$$\boldsymbol{\delta}^{(l)} = \frac{\partial \mathcal{L}}{\partial \mathbf{z}^{(l)}} = \left( \frac{\partial \mathbf{z}^{(l+1)}}{\partial \mathbf{z}^{(l)}} \right)^T \frac{\partial \mathcal{L}}{\partial \mathbf{z}^{(l+1)}} = \left( \frac{\partial \mathbf{z}^{(l+1)}}{\partial \mathbf{z}^{(l)}} \right)^T \boldsymbol{\delta}^{(l+1)}$$

因为 $\mathbf{z}^{(l+1)} = \mathbf{W}^{(l+1)} \mathbf{a}^{(l)} + \mathbf{b}^{(l+1)} = \mathbf{W}^{(l+1)} f(\mathbf{z}^{(l)}) + \mathbf{b}^{(l+1)}$，偏导部分（雅可比矩阵）计算为：
$$\frac{\partial \mathbf{z}^{(l+1)}}{\partial \mathbf{z}^{(l)}} = \mathbf{W}^{(l+1)} \text{diag}\left(f'(\mathbf{z}^{(l)})\right)$$

代入后得：
$$\boldsymbol{\delta}^{(l)} = \left( \mathbf{W}^{(l+1)} \text{diag}\left(f'(\mathbf{z}^{(l)})\right) \right)^T \boldsymbol{\delta}^{(l+1)} = \text{diag}\left(f'(\mathbf{z}^{(l)})\right) \left(\mathbf{W}^{(l+1)}\right)^T \boldsymbol{\delta}^{(l+1)}$$
写成按元素乘积（Hadamard Product）的形式：
$$\boldsymbol{\delta}^{(l)} = f'(\mathbf{z}^{(l)}) \odot \left( \left(\mathbf{W}^{(l+1)}\right)^T \boldsymbol{\delta}^{(l+1)} \right)$$

#### Step 3: 对可学习参数的梯度求解
由于 $\mathbf{z}^{(l)} = \mathbf{W}^{(l)} \mathbf{a}^{(l-1)} + \mathbf{b}^{(l)}$，可以分别求对矩阵元 $W_{ij}^{(l)}$ 和 $b_{i}^{(l)}$ 的偏导：
$$\frac{\partial \mathcal{L}}{\partial W_{ij}^{(l)}} = \frac{\partial \mathcal{L}}{\partial z_i^{(l)}} \frac{\partial z_i^{(l)}}{\partial W_{ij}^{(l)}} = \delta_i^{(l)} a_j^{(l-1)}$$
$$\frac{\partial \mathcal{L}}{\partial b_{i}^{(l)}} = \delta_i^{(l)}$$

写成矩阵乘积的形式：
$$\frac{\partial \mathcal{L}}{\partial \mathbf{W}^{(l)}} = \boldsymbol{\delta}^{(l)} \left(\mathbf{a}^{(l-1)}\right)^T$$
$$\frac{\partial \mathcal{L}}{\partial \mathbf{b}^{(l)}} = \boldsymbol{\delta}^{(l)}$$

## 5. 关键方法/模型

### 5.1 多层感知机（Multi-Layer Perceptron, MLP）
多层感知机是最基础的深层前馈网络，由输入层、一个或多个隐藏层以及输出层组成。层与层之间全连接。

#### 模型表示
$$h_1 = f(W_1 x + b_1)$$
$$h_2 = f(W_2 h_1 + b_2)$$
$$\hat{y} = g(W_3 h_2 + b_3)$$

#### 通用逼近定理 (Universal Approximation Theorem, UAT)
通用逼近定理表明，对于任意闭区间上的连续函数 $F(x)$，一个仅包含**单隐藏层**的神经网络，只要激活函数是非线性的，并且隐藏层的神经元数量足够多，就可以以任意精度逼近 $F(x)$。
- **理论意义**：证明了前馈网络极其强大的表达能力潜力。
- **局限性**：UAT 仅是一个存在性证明（Existence Theorem），它并没有说明：
  1. 如何用有限的数据和多项式时间内训练找到这些权重；
  2. 神经元的数量是否会呈现指数级增长。
- **深度的必要性**：经验与后续理论证明，增加网络深度（层数）相较于增加单层宽度，能够以**指数级更少的参数**表达高度复杂的函数（通过特征复用机制）。

---

### 5.2 经典模型里程碑
在深度前馈网络的基础上，通过引入局部诱导偏置（Inductive Bias），演化出以下经典模型结构：

| 经典模型 | 代表论文 / 核心贡献 | 核心架构特性与 Inductive Bias |
| :--- | :--- | :--- |
| **AlexNet** (2012) | *ImageNet Classification with Deep Convolutional Neural Networks* | 引入多层大卷积、ReLU 激活函数、Dropout，首次引入 GPU 并行计算。 |
| **VGG** (2014) | *Very Deep Convolutional Networks for Large-Scale Image Recognition* | 提出使用微小的 $3\times3$ 卷积核代替大卷积核，增加深度（16-19层）并保持相同的感受野。 |
| **ResNet** (2015) | *Deep Residual Learning for Image Recognition* | 提出 Residual Connection（跳跃连接），使信息直接绕过某些层传递。从数学上解决了超深层网络（可达上千层）的退化和难以收敛的危机。 |
| **Transformer** (2017) | *Attention Is All You Need* | 摒弃了循环神经网络（RNN）的时序迭代依赖，全靠自注意力（Self-Attention）机制捕捉全局依赖，具有极高的并行计算效能。 |

## 6. 优势与局限

### 6.1 核心优势
1. **层次化表征能力（Feature Hierarchy）**：浅层网络学习低阶物理特征（如边缘、纹理），深层网络将低阶特征拼接组合，自动学得高阶语义特征（如部件、物体轮廓）。
2. **端到端（End-to-End）联合训练**：将特征提取与下游分类/回归任务合并进同一个损失函数中。参数可在单一目标的梯度反向传播中同步更新，避免了多阶段流水线带来的“局部最优累积误差”。
3. **极佳的扩展律（Scaling Law）**：大模型的涌现能力表明，当数据量、计算量和模型规模突破一定阈值时，模型的常识、推理和泛化能力展现出质的飞跃。

---

### 6.2 理论与工程局限
1. **缺乏可解释性（Interpretability / Black-box Problem）**：神经网络由数以亿计的参数复合而成，难以进行可追溯的逻辑推导和数学形式的可解释性证明。
2. **过拟合与数据饥渴**：参数规模大而样本先验低，若没有海量语料或强大的正则化约束，极易发生“过拟合”。
3. **脆弱性与对抗攻击（Adversarial Vulnerability）**：在输入数据中加入人类肉眼不可察觉的微小噪声（对抗样本），即可使最顶尖的分类网络输出完全荒谬的结果。
4. **灾难性遗忘（Catastrophic Forgetting）**：在连续学习（Continual Learning）新任务时，新梯度的更新会彻底破坏和覆盖之前任务在网络中留下的突触权重表征。

## 7. 应用场景
深度学习基础模型已被扩展到各行各业：
- **表格与结构化数据**：传统多层感知机（MLP）在精准营销、信用评分等表格场景中起到了基础分类作用（尽管当前逐步被 LightGBM / XGBoost 或新型表格 TabTransformer 替代）。
- **信号级建模**：如通过一维残差网络对传感器振动、心电图（ECG）及音频声谱等直接进行特征感知与异态分类。
- **大模型核心组件**：大语言模型（LLM）的底层 FFN（Feed-Forward Network）部分本质上就是一个隐藏层维度扩增 4 倍的高维 MLP 层，在大模型计算中占据了约 2/3 的参数量。

## 8. 与其他技术关系
- **与机器学习的关系**：深度学习是机器学习的高阶子集，消除了对手工特征的严重依赖。
- **与数学及优化理论的关系**：它极大地依赖矩阵代数的高效实现，而求解高维参数空间的最优解则直接建立在非凸约束优化理论（一阶随机梯度方法）之上。
- **与后续专题的关系**：
  - 本章介绍的前馈机制和自动微分是构建「卷积神经网络（CNN）」和「序列模型（RNN）」的必要前置知识。
  - 本章归一化与正则化机制的实现直接影响「训练技巧与稳定性」中所论述的工程调优策略。

## 9. 前沿发展
1. **新型前馈网络与非全连接架构**：
   - 诸如 **MLP-Mixer** 证明了即便完全不用卷积（CNN）或自注意力（Transformer），仅依靠在通道维度和空间维度交替做全连接转置投影的 MLP，也能在 CV 任务上取得一流的泛化性能，重塑了人们对基础 MLP 表达上限的认知。
2. **物理信息神经网络（Physics-Informed Neural Networks, PINN）**：
   - 将已知的物理先验（如 Navier-Stokes 方程、薛定谔方程）作为正则化项直接注入深度学习的损失函数中。这使得网络在数据极其稀缺时，依然能够严格在物理守恒律约束下拟合和预报流体力学、量子力学等系统状态。
3. **基于非凸优化几何学的大模型优化探索**：
   - 探索**神经切线核（Neural Tangent Kernel, NTK）**在超参数化（Over-parameterization）极限下的行为，试图从解析数学的视角揭开深层网络能在非凸面上规避局部极小值、寻找到泛化性良好全局最优解的底层物理谜题。

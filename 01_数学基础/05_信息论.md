# 信息论

## 1. 概述

信息论量化信息和不确定性。核心概念是熵——衡量随机变量的平均不确定性，以及 KL 散度——衡量两个概率分布的差异。最小化交叉熵等价于最大化似然，这一联系将信息论与统计推断统一起来。在 AI 中，信息论为损失函数设计、模型评估、知识蒸馏和分布对齐提供核心理论工具。

## 3. 核心概念

### 自信息与熵

- **自信息**：$I(x) = -\log P(x)$，概率越小的事件携带信息量越大
- **熵**（离散）：$H(X) = -\sum_{x} P(x) \log P(x) = -E[\log P(X)]$
- **微分熵**（连续）：$H(X) = -\int p(x) \log p(x)\,dx$
- 性质：$H(X) \geq 0$（离散）；均匀分布熵最大；确定性分布熵为 0
- **单位**：以 2 为底 → bit；以 $e$ 为底 → nat；以 10 为底 → dit

### 条件熵与联合熵

- **条件熵**：$H(Y|X) = -\sum_{x,y} P(x,y) \log P(y|x)$，已知 X 时 Y 的剩余不确定性
- **联合熵**：$H(X,Y) = -\sum_{x,y} P(x,y) \log P(x,y)$
- **链式法则**：$H(X,Y) = H(X) + H(Y|X)$

### 互信息

$$I(X;Y) = H(X) - H(X|Y) = D_{KL}(P(X,Y) \| P(X)P(Y))$$

- 性质：$I(X;Y) \geq 0$；$I(X;Y) = 0$ 当且仅当 $X$ 与 $Y$ 独立；对称性 $I(X;Y) = I(Y;X)$
- 度量两个随机变量共享的信息量

### KL 散度

$$D_{KL}(P\|Q) = \sum_{x} P(x) \log\frac{P(x)}{Q(x)} = E_P\left[\log \frac{P(x)}{Q(x)}\right]$$

- **非负**：$D_{KL}(P\|Q) \geq 0$（Gibbs 不等式），当且仅当 $P=Q$ 时为 0
- **不对称**：$D_{KL}(P\|Q) \neq D_{KL}(Q\|P)$
- **前向 KL**（$D_{KL}(P\|Q)$，mean-seeking）：使 Q 覆盖 P 的所有支持域
- **后向 KL**（$D_{KL}(Q\|P)$，mode-seeking）：使 Q 集中在 P 的高概率区域

### 交叉熵

$$H(P,Q) = -\sum_{x} P(x) \log Q(x) = H(P) + D_{KL}(P\|Q)$$

最小化交叉熵等价于最大化似然：$\min H(P,Q) \iff \max \log P(\text{data}|\theta)$

## 4. 技术原理

### 重要不等式

- **吉布斯不等式**：$D_{KL}(P\|Q) \geq 0$
- **Jensen 不等式**：对凸函数 $f$，$f(E[X]) \leq E[f(X)]$，KL 散度非负性的证明基础
- **数据处理不等式**：对马尔可夫链 $X \to Y \to Z$，$I(X;Z) \leq I(X;Y)$（信息处理后不会增加）
- **最大熵原理**：在给定约束下，熵最大的分布是最无偏的分布（如已知均值方差时，最大熵分布为高斯分布）

### 信道容量

$$C = \max_{P(X)} I(X;Y)$$

信道容量是通信信道无差错传输信息的最大速率。虽源自通信理论，但其思想影响了 AI 中的信息瓶颈方法。

### 信息瓶颈（Information Bottleneck）

- **Tishby et al. (1999)**: *The Information Bottleneck Method*
- 目标：找到表示 $T$，最大化 $I(T;Y)$（保留标签信息），同时最小化 $I(T;X)$（压缩输入信息）
- 用于解释深度学习：网络逐层压缩输入中的冗余信息

### 扩展散度度量

- **JS 散度**（Jensen-Shannon）：对称化的 KL 散度

$$D_{JS}(P\|Q) = \frac{1}{2}D_{KL}(P\|M) + \frac{1}{2}D_{KL}(Q\|M), \quad M = \frac{P+Q}{2}$$

  - 对称、有界、有定义（即使 $Q$ 支持域不覆盖 $P$）
  - GAN 的原始目标函数

- **Wasserstein 距离**：衡量将一个分布"搬运"到另一个分布的最小代价
  - 解决 GAN 中 JS 散度的梯度消失问题

- **证据下界（ELBO）**：

$$\log p(x) \geq \mathbb{E}_{q(z)}[\log p(x,z) - \log q(z)] = \text{ELBO}$$

  - 变分推断的核心，VAE 的优化目标
  - $\log p(x) = \text{ELBO} + D_{KL}(q(z) \| p(z|x))$

## 5. 关键方法/模型

### 交叉熵损失

- 分类任务的标准损失函数：$L = -\sum_i y_i \log \hat{y}_i$
- 与最大似然估计等价

### 困惑度（Perplexity）

$$\text{PPL} = 2^{H(p,q)} = \exp\left(-\frac{1}{N}\sum_{i=1}^N \log P(x_i|context)\right)$$

- 语言模型评估的核心指标
- 困惑度越低，模型对数据拟合越好

### 知识蒸馏

- **Hinton et al. (2015)**: *Distilling the Knowledge in a Neural Network*
- 使用 KL 散度让学生模型逼近教师模型的软标签分布
- 温度参数 $T$ 控制软标签的平滑程度

### DPO 中的 KL 约束

- **Rafailov et al. (2023)**: *Direct Preference Optimization*
- 在偏好对齐中，用 KL 散度约束策略模型不偏离参考模型太远

### ELBO 与 VAE

- **Kingma & Welling (2013)**: *Auto-Encoding Variational Bayes*
- 最大化 ELBO 等价于最小化 $D_{KL}(q_\phi(z|x) \| p(z|x))$

## 6. 优势与局限

**信息论的优势**：
- 提供分布间差异的统一度量框架
- 交叉熵损失有明确的概率解释（最大似然）
- 困惑度为语言模型提供可解释的评估指标
- 信息瓶颈等为深度学习提供理论分析工具

**局限**：
- KL 散度不对称，不同方向的散度有不同的行为特性
- KL 散度要求两个分布支持域一致（否则无定义），Wasserstein 距离无此限制
- 高维空间中互信息估计困难
- 熵和互信息的实际计算依赖分布已知或可精确估计

## 7. 应用场景

- **损失函数设计**：交叉熵用于分类，KL 散度用于生成模型和蒸馏
- **模型压缩**：知识蒸馏用 KL 散度传递大模型知识到小模型
- **[[大语言模型]] 评估**：困惑度衡量语言模型性能
- **变分推断**：ELBO 是 VAE 的理论基础
- **[[强化学习]]**：熵正则化平衡探索与利用
- **特征选择**：互信息最大化选择与目标最相关的特征
- **大模型对齐**：DPO 中的 KL 约束防止策略过度偏离

## 8. 与其他技术关系

- 建立在 [[概率与统计]] 基础上（熵和 KL 散度均定义在概率分布上）
- 交叉熵损失是 [[深度学习]] 分类任务的标准损失
- KL 散度是 [[大语言模型]] 对齐方法（如 DPO）的核心度量
- ELBO 是变分推断和 VAE 的理论基础
- 与 [[优化方法]] 密切相关：最小化交叉熵等价于最大似然估计
- 信息瓶颈理论为 [[深度学习]] 的泛化性提供信息论解释

## 9. 前沿发展

- **信息瓶颈与深度学习**：Tishby 的"深度学习中的信息瓶颈"理论引发对泛化机制的热烈讨论
- **互信息估计**：MINE（Mutual Information Neural Estimation）、InfoNCE 对比学习中的互信息下界
- **最小描述长度（MDL）**：将模型复杂度与信息编码代价联系，用于模型选择
- **大模型校准**：温度缩放、弹性校准等信息论方法改善模型置信度估计
- **压缩与智能的关系**：Hutter 奖推动数据压缩与通用智能的理论关联
- **速率-失真理论在生成模型中的应用**：扩散模型的速率-失真解释

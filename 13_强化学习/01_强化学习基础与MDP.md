# 强化学习基础与 MDP

## 1. 概述

强化学习（Reinforcement Learning, RL）研究智能体（Agent）在与环境（Environment）的交互过程中，通过试错学习最优决策策略，以最大化长期累积奖励。与监督学习不同，强化学习没有"正确答案"的标注，而是通过环境的奖励信号（Reward Signal）来评估行为的好坏。

本篇介绍强化学习的基本框架、核心术语、以及其数学形式化——马尔可夫决策过程（Markov Decision Process, MDP）。MDP 是所有强化学习算法的理论基础，理解 MDP 及其 Bellman 方程是学习后续所有 RL 方法的前提。

## 3. 核心概念

### 3.1 智能体-环境接口

强化学习的基本框架是**智能体-环境交互循环**：

```
        动作 a_t
智能体 ───────────→ 环境
  ↑                   │
  │    状态 s_t        │
  │    奖励 r_t        │
  └───────────────────┘
```

在每个时间步 $t$：
1. 智能体观察环境状态 $S_t \in \mathcal{S}$
2. 智能体根据策略选择动作 $A_t \in \mathcal{A}$
3. 环境根据状态转移给出新状态 $S_{t+1}$ 和奖励 $R_{t+1}$
4. 智能体利用 $(S_t, A_t, R_{t+1}, S_{t+1})$ 更新策略

### 3.2 核心术语

| 术语 | 符号 | 含义 |
|:---|:---|:---|
| 状态（State） | $s \in \mathcal{S}$ | 环境的完整描述 |
| 动作（Action） | $a \in \mathcal{A}$ | 智能体的行为选择 |
| 奖励（Reward） | $r \in \mathcal{R}$ | 环境对动作的即时反馈标量 |
| 策略（Policy） | $\pi(a\|s)$ | 状态到动作的映射（概率分布或确定性） |
| 回报（Return） | $G_t$ | 从时刻 $t$ 起的累积折扣奖励 |
| 值函数（Value Function） | $V^\pi(s)$ / $Q^\pi(s,a)$ | 长期价值的期望 |
| 轨迹（Trajectory） | $\tau$ | 状态-动作-奖励序列 $(s_0, a_0, r_1, s_1, a_1, \ldots)$ |
| 回合（Episode） | — | 从初始状态到终止状态的完整轨迹 |
| 折扣因子（Discount Factor） | $\gamma$ | 未来奖励的折扣权重 |

### 3.3 回报与折扣

累积折扣回报（Discounted Return）定义为：

$$G_t = R_{t+1} + \gamma R_{t+2} + \gamma^2 R_{t+3} + \cdots = \sum_{k=0}^{\infty} \gamma^k R_{t+k+1}$$

折扣因子 $\gamma \in [0,1)$ 的作用：

- **数学必要性**：保证无限时域回报的级数收敛（当奖励有界时，$|G_t| \leq \frac{R_{\max}}{1-\gamma}$）
- **现实合理性**：未来奖励的不确定性随时间递增，近期奖励应更重要
- **避免循环**：在有环 MDP 中防止无限循环累积奖励

$\gamma$ 越接近 1，智能体越"远视"；$\gamma = 0$ 时退化为贪心策略（只看即时奖励）。

### 3.4 与监督学习的对比

| 维度 | 监督学习 | 强化学习 |
|:---|:---|:---|
| 数据来源 | 静态标注数据集 | 与环境动态交互产生 |
| 反馈类型 | 即时、精确的标签 | 延迟、稀疏的标量奖励 |
| 目标 | 最小化预测误差 | 最大化累积奖励 |
| 数据分布 | 独立同分布（i.i.d.） | 序列相关，智能体行为影响数据分布 |
| 探索 | 不需要 | 核心问题——需要主动探索未知 |
| 评估 | 即时可计算损失 | 需要完整轨迹才能评估决策好坏 |

## 4. 技术原理

### 4.1 马尔可夫决策过程（MDP）

MDP 由五元组 $\mathcal{M} = (\mathcal{S}, \mathcal{A}, P, R, \gamma)$ 定义：

- $\mathcal{S}$：状态空间（State Space）
- $\mathcal{A}$：动作空间（Action Space）
- $P(s'|s,a)$：状态转移概率（Transition Probability），$\sum_{s'} P(s'|s,a) = 1$
- $R(s,a,s')$（或 $R(s,a)$ 或 $R(s)$）：奖励函数（Reward Function）
- $\gamma \in [0,1)$：折扣因子

**马尔可夫性**：未来只依赖当前状态和动作，与历史无关：

$$P(S_{t+1} = s' | S_t = s, A_t = a, S_{t-1}, A_{t-1}, \ldots, S_0, A_0) = P(s'|s,a)$$

马尔可夫性是 RL 可处理的基石——它意味着状态 $s$ 包含了做决策所需的全部信息，无需维护完整历史。

### 4.2 部分可观测 MDP（POMDP）

当智能体无法观察完整状态，只能获得**观测（Observation）** $o_t$ 时，问题扩展为 POMDP $\mathcal{M} = (\mathcal{S}, \mathcal{A}, P, R, \gamma, \Omega, O)$：

- $\Omega$：观测空间
- $O(o|s',a)$：观测函数，给定真实状态和动作产生观测的概率

POMDP 中智能体需要从观测历史中推断**信念状态（Belief State）** $b_t(s) = P(s_t = s | o_0, a_0, \ldots, o_t, a_{t-1})$。大多数实际 RL 问题（如视觉 Atari、机器人传感器）都是 POMDP，但实践中常直接用观测近似状态。

### 4.3 Bellman 方程

Bellman 方程是 MDP 的核心，描述了值函数的递归结构。

#### Bellman 期望方程

状态值函数的 Bellman 方程：

$$V^\pi(s) = \sum_a \pi(a|s) \sum_{s'} P(s'|s,a) \left[ R(s,a,s') + \gamma V^\pi(s') \right]$$

动作值函数的 Bellman 方程：

$$Q^\pi(s,a) = \sum_{s'} P(s'|s,a) \left[ R(s,a,s') + \gamma \sum_{a'} \pi(a'|s') Q^\pi(s',a') \right]$$

**直觉**：当前状态的价值 = 即时奖励的期望 + 下一状态价值的折扣期望。

#### Bellman 最优方程

$$V^*(s) = \max_a \sum_{s'} P(s'|s,a) \left[ R(s,a,s') + \gamma V^*(s') \right]$$

$$Q^*(s,a) = \sum_{s'} P(s'|s,a) \left[ R(s,a,s') + \gamma \max_{a'} Q^*(s',a') \right]$$

最优策略可以从 $Q^*$ 直接导出：$\pi^*(s) = \arg\max_a Q^*(s,a)$。

### 4.4 状态访问分布

在策略 $\pi$ 下，从初始状态分布 $d_0$ 出发的**折扣状态访问分布**为：

$$d^\pi(s) = (1-\gamma) \sum_{t=0}^{\infty} \gamma^t P(s_t = s | \pi, d_0)$$

$d^\pi(s)$ 表示策略 $\pi$ 下状态 $s$ 被访问的频率权重。策略梯度的目标函数可写为：

$$J(\pi) = \sum_s d^\pi(s) \sum_a \pi(a|s) \sum_{s'} P(s'|s,a) R(s,a,s')$$

### 4.5 优化目标的形式

强化学习的优化目标根据设定不同有三种形式：

| 设定 | 目标 | 公式 |
|:---|:---|:---|
| 回合制（Episodic） | 最大化初始状态期望回报 | $J(\pi) = \mathbb{E}_{s_0 \sim d_0}[V^\pi(s_0)]$ |
| 平均奖励（Average Reward） | 最大化长期平均奖励率 | $\bar{r}^\pi = \lim_{T \to \infty} \frac{1}{T} \mathbb{E}\left[\sum_{t=0}^{T-1} R_t\right]$ |
| 折扣（Discounted） | 最大化期望折扣回报 | $J(\pi) = \mathbb{E}_{\tau \sim \pi}\left[\sum_{t=0}^{\infty} \gamma^t R_t\right]$ |

折扣设定是最常用的形式，平均奖励设定在继续型（continuing）任务中更为自然。

## 5. 关键方法/模型

### 5.1 GridWorld：RL 的"Hello World"

GridWorld 是一个经典的表格 MDP 示例：智能体在网格中移动，目标是到达终止状态（如终点），避开陷阱。状态是网格坐标，动作是上下左右，奖励通常为每步 -1（鼓励快速到达）或终点 +1。GridWorld 适合验证值迭代、Q-Learning 等算法的正确性。

### 5.2 Bandit 问题：单步 RL

多臂老虎机（Multi-Armed Bandit）是 MDP 的退化形式——只有一步决策，没有状态转移。它聚焦于探索与利用的核心矛盾：

- ε-greedy：以 $\epsilon$ 概率随机探索，$1-\epsilon$ 概率选择当前最优臂
- UCB（Upper Confidence Bound）：$a_t = \arg\max_a \left[ \hat{Q}(a) + c\sqrt{\frac{\ln t}{N(a)}} \right]$
- Thompson Sampling：从后验分布中采样选择臂

Bandit 是理解 RL 探索策略的最佳入门。参见 [[07_探索与利用]]。

### 5.3 策略评估与控制

给定 MDP 和策略 $\pi$：

- **策略评估（Prediction）**：计算 $V^\pi$ 或 $Q^\pi$
- **策略改进（Improvement）**：给定 $Q^\pi$，构造更优策略 $\pi' \geq \pi$
- **策略迭代（Control）**：交替进行评估和改进，收敛到最优策略

这是动态规划的核心框架。参见 [[02_值函数与动态规划]]。

## 6. 优势与局限

### 优势

1. **无需标注数据**：从环境奖励中自主学习，适合缺乏标注的场景
2. **序贯决策建模**：自然处理延迟奖励和长期规划问题
3. **通用性强**：从游戏到机器人控制到大模型对齐，统一框架
4. **自适应**：策略随环境动态调整

### 局限

1. **样本效率低**：通常需要大量交互数据，尤其在深度 RL 中
2. **奖励设计困难**：设计能正确反映目标的奖励函数是实际应用的主要挑战
3. **探索困难**：在稀疏奖励环境中，智能体可能长时间无法发现有价值的策略
4. **稳定性与复现性**：深度 RL 训练对超参数敏感，结果方差大
5. **安全性**：在线探索可能产生危险行为，限制在医疗、自动驾驶等领域的应用

## 8. 与其他技术关系

- **与监督学习的关系**：模仿学习（Imitation Learning）将专家示范作为"标签"，是监督学习到 RL 的桥梁。DAgger 等方法在两者之间迭代。
- **与动态规划的关系**：RL 是动态规划的推广——DP 需要完整 MDP 模型，RL 从数据中学习。参见 [[02_值函数与动态规划]]。
- **与最优控制的关系**：最优控制（LQR/MPC）和 RL 解决类似的序贯决策问题。最优控制通常假设已知线性模型，RL 处理非线性、未知模型。参见 [[06_基于模型的强化学习]]。
- **与大模型训练的关系**：RLHF 将 RL 的策略优化应用于语言模型偏好对齐，是当前大模型训练的核心环节。参见 [[10_强化学习与大模型]]。

## 9. 前沿发展

- **基础模型 + RL**：将预训练视觉/语言模型作为 RL 的感知模块或先验知识，加速学习（如 VPT、RT-2）
- **元强化学习**：学习快速适应新任务的策略，实现 few-shot RL
- **离线 RL 的成熟**：从大规模离线数据中学习策略，无需在线交互，推动 RL 落地实际应用
- **可证明安全的 RL**：在保证安全约束的前提下学习最优策略（Safe RL、Constrained MDP）
- **多模态 RL**：从视觉、语言、触觉等多模态输入中学习决策策略

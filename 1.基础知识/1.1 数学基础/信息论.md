# 信息论

## 背景

信息论由Claude Shannon于1948年创立，研究信息的度量、传输和处理。在AI中，信息论为损失函数设计、特征选择、模型压缩和概率建模提供了核心理论工具。

## 核心思想

信息论量化信息和不确定性。核心概念是熵——衡量随机变量的不确定性，以及KL散度——衡量两个概率分布的差异。

## 技术原理

### 自信息

$$I(x) = -\log P(x) = \log(1/P(x))$$

概率越小的事件，发生时携带的信息量越大。通常使用 $\log_2$（单位bit）或 $\ln$（单位nat）。

### 熵

随机变量的平均不确定性：

- **离散**：$H(X) = -\sum_{x} P(x) \log P(x) = -E[\log P(X)]$
- **连续（微分熵）**：$H(X) = -\int p(x) \log p(x) dx$

性质：$H(X) \geq 0$；均匀分布熵最大；确定性分布熵最小（$H(X)=0$）。

### 条件熵与联合熵

- **条件熵**：$H(Y|X) = -\sum_{x,y} P(x,y) \log P(y|x)$
- **联合熵**：$H(X,Y) = -\sum_{x,y} P(x,y) \log P(x,y)$
- **链式法则**：$H(X,Y) = H(X) + H(Y|X) = H(Y) + H(X|Y)$

### 互信息

衡量两个随机变量的相互依赖程度：

$$I(X;Y) = H(X) - H(X|Y) = H(Y) - H(Y|X) = D_{KL}(P(X,Y) \| P(X)P(Y))$$

性质：$I(X;Y) \geq 0$；$I(X;Y) = 0$ 当且仅当 $X$ 和 $Y$ 独立；对称性 $I(X;Y) = I(Y;X)$。

### KL散度

衡量两个概率分布的差异（相对熵）：

$$D_{KL}(P\|Q) = \sum_{x} P(x) \log\frac{P(x)}{Q(x)} = E_P[\log P(X) - \log Q(X)]$$

性质：
- 非负：$D_{KL}(P\|Q) \geq 0$（Gibbs不等式），当且仅当 $P=Q$ 时为0
- 不对称：$D_{KL}(P\|Q) \neq D_{KL}(Q\|P)$
- 不满足三角不等式

直观理解：
- 前向KL $D_{KL}(P\|Q)$：使Q覆盖P的所有支持域（避免zero-forcing）
- 后向KL $D_{KL}(Q\|P)$：使Q集中在P的高概率区域（zero-forcing）

### 交叉熵

$$H(P,Q) = -\sum_{x} P(x) \log Q(x) = -E_P[\log Q(X)]$$

与KL散度的关系：$H(P,Q) = H(P) + D_{KL}(P\|Q)$

最小化交叉熵等价于最大化似然：$\min H(P,Q) \iff \max \log P(data|\theta)$

### 信息不等式

1. **吉布斯不等式**：$D_{KL}(P\|Q) \geq 0$
2. **数据处理不等式**：对马尔可夫链 $X \to Y \to Z$，$I(X;Z) \leq I(X;Y)$
3. **最大熵原理**：在给定约束下，熵最大的分布是最无偏的分布

### 扩展概念

- **JS散度**：对称化的KL散度，$D_{JS}(P\|Q) = \frac{1}{2}D_{KL}(P\|m) + \frac{1}{2}D_{KL}(Q\|m)$，其中 $m = \frac{P+Q}{2}$
- **证据下界（ELBO）**：$\log p(x) \geq \mathbb{E}_{q(z)}[\log p(x,z) - \log q(z)]$，变分推断的核心

## 应用领域

- **损失函数设计**：交叉熵损失用于[[机器学习|分类任务]]，KL散度损失用于生成模型
- **特征选择**：互信息最大化选择与目标变量最相关的特征
- **模型压缩**：知识蒸馏使用KL散度将大模型知识传递到小模型
- **[[大语言模型]]评估**：困惑度（Perplexity = $2^{H(p,q)}$）衡量语言模型性能
- **变分推断**：ELBO是[[概率与统计|变分自编码器]]的理论基础
- **强化学习**：熵正则化平衡探索与利用

## 与其他技术关系

- 信息论建立在[[概率与统计]]基础上
- 交叉熵损失是[[深度学习]]分类任务的标准损失函数
- KL散度是[[大语言模型]]训练中对齐方法（如[[DPO]]）的核心度量
- 互信息在[[神经网络基础|自监督学习]]中用于表示学习
- ELBO是变分推断和VAE的理论基础

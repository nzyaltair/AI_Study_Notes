# 知识表示与推理

## 背景

知识表示与推理是[[符号主义AI]]的核心问题。如何将人类知识以机器可处理的形式表示，并基于这些知识进行逻辑推理，是早期AI研究的核心挑战。

## 核心思想

将领域知识形式化为符号结构，通过推理引擎（Inference Engine）应用逻辑规则从已知知识推导新知识。

## 技术原理

### 一阶谓词逻辑

- **原子公式**：$P(x)$，如 $Person(Socrates)$
- **逻辑连接词**：$\land$（与）、$\lor$（或）、$\neg$（非）、$\rightarrow$（蕴含）
- **量词**：$\forall$（全称）、$\exists$（存在）
- **推理规则**：
  - Modus Ponens：从 $P \rightarrow Q$ 和 $P$ 推出 $Q$
  - Modus Tollens：从 $P \rightarrow Q$ 和 $\neg Q$ 推出 $\neg P$
  - 归结（Resolution）：自动定理证明的基础

### 语义网络

- 节点表示概念，边表示关系（is-a, has-a, part-of等）
- 支持继承推理：子概念继承父概念属性

### 框架表示

```
框架名: Car
  is-a: Vehicle
  wheels: 4
  fuel: gasoline
  default-speed: 60km/h
```

### 本体论（Ontology）

- 定义领域内概念及其关系的正式描述
- OWL（Web Ontology Language）、RDF（Resource Description Framework）
- 现代应用：知识图谱

## 发展演进

一阶逻辑（1950s）→ 语义网络（1960s）→ 框架系统（1970s）→ 描述逻辑（1980s）→ 知识图谱（2000s+）→ 大语言模型中的隐式知识表示（2020s）

现代[[大语言模型]]通过分布式表示隐式存储知识，与符号主义的显式知识表示形成对比。知识图谱与LLM的结合是当前研究热点。

## 与其他技术关系

- 知识表示是[[符号主义AI]]的核心技术
- 知识图谱是现代版的知识表示，被用于[[RAG系统]]
- 本体论影响了[[Agent智能体|Agent]]的知识管理
- 逻辑推理与[[搜索与规划]]密切相关

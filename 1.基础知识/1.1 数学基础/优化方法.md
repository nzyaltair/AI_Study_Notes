# 优化方法

## 背景

优化方法是AI模型训练的核心算法。在机器学习和深度学习中，通过优化算法最小化损失函数来训练模型。从经典梯度下降到现代AdamW优化器，优化方法的发展直接推动了深度学习的成功。

## 核心思想

优化方法寻找使目标函数最小（或最大）的参数值。在AI中，目标函数通常是损失函数，优化过程通过迭代地沿梯度反方向更新参数来最小化损失。

## 技术原理

### 优化问题基本概念

- **目标函数**：需要最小化的函数 $f(x)$
- **全局极小值**：$f(x^*) \leq f(x)$ 对所有 $x$ 成立
- **局部极小值**：$f(x^*) \leq f(x)$ 在 $x^*$ 邻近区域成立
- **鞍点**：梯度为零但非极值点，深度学习中最常见的临界点类型
- **凸问题**：局部极小值即为全局极小值；深度学习问题多为非凸

### 一阶优化方法

**梯度下降**：$x \leftarrow x - \eta \nabla f(x)$

变体：
- **批量梯度下降（BGD）**：使用所有样本，稳定但计算量大
- **随机梯度下降（SGD）**：每次使用一个样本，快但震荡大
- **小批量梯度下降**：平衡效率与稳定性，最常用

**动量法**：
$$v \leftarrow \beta v + (1-\beta) \nabla f(x)$$
$$x \leftarrow x - \eta v$$

**Nesterov加速梯度（NAG）**：先按动量更新再计算梯度，更好地预测下一步方向

### 自适应学习率方法

| 优化器 | 核心思想 | 优点 | 缺点 |
|-------|---------|------|------|
| **Adagrad** | 累积梯度平方，对频繁更新参数用小学习率 | 适合稀疏数据 | 学习率单调下降，过早停滞 |
| **RMSprop** | 梯度平方的指数移动平均 | 解决Adagrad衰减过快 | 可能收敛到次优解 |
| **Adam** | 结合动量和RMSprop，跟踪一阶矩和二阶矩 | 收敛快，对超参数不敏感 | 可能泛化性能不佳 |
| **AdamW** | Adam + 权重衰减分离 | 更好的正则化效果 | 计算复杂度略高 |

**Adam更新规则**：
$$m_t = \beta_1 m_{t-1} + (1-\beta_1) \nabla L(\theta)$$
$$v_t = \beta_2 v_{t-1} + (1-\beta_2) (\nabla L(\theta))^2$$
$$\hat{m}_t = m_t / (1-\beta_1^t), \quad \hat{v}_t = v_t / (1-\beta_2^t)$$
$$\theta \leftarrow \theta - \eta \cdot \hat{m}_t / (\sqrt{\hat{v}_t} + \epsilon)$$

### 二阶方法

- **牛顿法**：$x \leftarrow x - H^{-1} \nabla f(x)$，利用Hessian矩阵，二阶收敛但计算成本高
- **拟牛顿法（BFGS/L-BFGS）**：用梯度信息近似Hessian逆矩阵

### 学习率调度策略

1. **阶梯衰减**：每N个epoch乘以衰减因子
2. **指数衰减**：$\eta_t = \eta_0 \times \text{decayrate}^t$
3. **余弦退火**：学习率随epoch余弦变化
4. **One-Cycle**：先增大再减小，配合动量反向变化
5. **学习率预热**：初始阶段线性增大，避免不稳定性

### 约束优化

- **Lagrange乘子法**：将约束融入目标函数
- **KKT条件**：约束优化问题的最优性条件，包括互补松弛、梯度条件和约束条件

### 正则化技术

- **L1正则化**：$\lambda \sum |\theta_i|$，产生稀疏解
- **L2正则化**：$\lambda \sum \theta_i^2$，防止过拟合
- **Dropout**：训练时随机失活神经元
- **权重衰减**：在损失函数中加入权重平方项

## 发展演进

梯度下降（Cauchy, 1847）→ 随机梯度下降（SGD）→ 动量法 → 自适应方法（Adagrad → RMSprop → Adam → AdamW）。优化器的发展趋势是从固定学习率到自适应学习率，从一阶到近似二阶。

## 应用领域

- **[[神经网络基础|神经网络训练]]**：反向传播后的参数更新
- **[[深度学习]]**：CNN训练常用SGD+Momentum，Transformer训练常用AdamW
- **[[大语言模型]]训练**：大规模训练使用LARS/LAMB优化器
- **强化学习**：处理非平稳目标函数
- **超参数优化**：Optuna、Ray Tune等工具

# 深度强化学习

## 1. 概述

深度强化学习（Deep Reinforcement Learning, Deep RL）将深度神经网络作为函数近似器引入强化学习，用神经网络近似值函数 $Q_\theta(s,a)$ 或策略 $\pi_\theta(a|s)$，从而处理高维状态空间（如图像、点云）和连续动作空间。深度 RL 的突破始于 2013 年 DQN 在 Atari 游戏上的成功，此后迅速扩展到连续控制、机器人学习和大模型对齐等领域。

本篇聚焦于**值函数类深度 RL**——以 DQN 及其变体为核心，以及结合策略梯度和值函数的混合方法（DDPG/A3C/Rainbow）。纯策略梯度方法参见 [[04_策略梯度方法]]。

## 3. 核心概念

### 3.1 函数近似

表格 RL 只能处理有限小状态空间。深度 RL 用神经网络 $\hat{v}(s;\theta) \approx V^\pi(s)$ 或 $\hat{q}(s,a;\theta) \approx Q^\pi(s,a)$ 处理高维/连续状态空间。函数近似的挑战在于：

- **非平稳目标**：TD 学习的自举使学习目标随训练变化
- **相关性数据**：RL 的序列数据具有时间相关性，违反 i.i.d. 假设
- **致命三要素**：函数近似 + 自举 + off-policy 同时存在时可能发散

### 3.2 经验回放（Experience Replay）

DQN 的关键创新之一。将交互经验 $(s, a, r, s')$ 存入回放缓冲区 $\mathcal{D}$，训练时从中随机采样 mini-batch：

- **打破时间相关性**：随机采样近似 i.i.d.
- **提高数据利用率**：每条经验可被多次使用
- **稳定训练**：避免最新数据对网络的过度影响

### 3.3 目标网络（Target Network）

DQN 的另一关键创新。使用一个延迟更新的目标网络 $Q_{\theta^-}$ 计算 TD 目标：

$$\mathcal{L}(\theta) = \mathbb{E}_{(s,a,r,s') \sim \mathcal{D}} \left[ \left( r + \gamma \max_{a'} Q_{\theta^-}(s', a') - Q_\theta(s, a) \right)^2 \right]$$

目标网络 $\theta^-$ 每 $N$ 步或以 $\tau$ 的软更新频率同步：$\theta^- \leftarrow \tau \theta + (1-\tau) \theta^-$。

**作用**：如果用在线网络 $Q_\theta$ 同时计算当前值和目标值，TD 目标 $r + \gamma \max_{a'} Q_\theta(s', a')$ 会随 $\theta$ 变化剧烈波动，导致训练不稳定。目标网络固定目标，使优化更稳定。

## 4. 技术原理

### 4.1 DQN

**论文**：Mnih et al., *Human-level control through deep reinforcement learning* (Nature, 2015)

DQN = Q-Learning + 深度 CNN + 经验回放 + 目标网络。

**架构**：输入为游戏画面（84×84×4 堆叠帧），经过 CNN 提取特征，输出每个离散动作的 Q 值。

**训练流程**：
1. ε-greedy 采集经验 $(s, a, r, s')$，存入回放缓冲区
2. 从缓冲区随机采样 mini-batch
3. 计算 TD 目标 $y = r + \gamma \max_{a'} Q_{\theta^-}(s', a')$（对终止状态 $y = r$）
4. 最小化 $\mathcal{L}(\theta) = \mathbb{E}[(y - Q_\theta(s,a))^2]$
5. 每 C 步同步 $\theta^- \leftarrow \theta$

### 4.2 Double DQN

**论文**：van Hasselt et al., *Deep RL with Double Q-learning* (2016)

DQN 的 $\max_{a'} Q_{\theta^-}(s', a')$ 同时用同一网络选动作和评估值，导致过估计。

Double DQN 解耦选择和评估：

$$y = r + \gamma Q_{\theta^-}(s', \arg\max_{a'} Q_\theta(s', a'))$$

用在线网络 $\theta$ 选最优动作，用目标网络 $\theta^-$ 评估该动作的值。

### 4.3 Dueling DQN

**论文**：Wang et al., *Dueling Network Architectures for Deep RL* (2016)

将 Q 值分解为状态值和优势函数：

$$Q(s,a) = V(s) + A(s,a) - \frac{1}{|\mathcal{A}|} \sum_{a'} A(s,a')$$

网络输出两个分支：$V(s)$ 和 $A(s,a)$。减去均值保证可辨识性（否则 $V$ 和 $A$ 无法唯一确定）。

**优势**：当很多动作的值相近时，直接学习 $V(s)$ 更高效——不需要为每个动作都更新。

### 4.4 Prioritized Experience Replay (PER)

**论文**：Schaul et al., *Prioritized Experience Replay* (2016)

不是均匀采样，而是按 TD 误差的大小优先回放"重要"经验：

$$P(i) = \frac{|\delta_i|^\alpha}{\sum_j |\delta_j|^\alpha}$$

为修正优先采样引入的分布偏差，使用重要性采样权重：

$$w_i = \left( \frac{1}{N \cdot P(i)} \right)^\beta$$

$\beta$ 从 0 线性退火到 1，保证无偏性。

### 4.5 Distributional RL (C51 / QR-DQN)

**论文**：Bellemare et al., *A Distributional Perspective on RL* (2017)

传统 RL 学习 $Q(s,a) = \mathbb{E}[G_t | s, a]$（期望值）。分布式 RL 学习回报的**分布** $Z(s,a)$（随机变量），使得 $Q(s,a) = \mathbb{E}[Z(s,a)]$。

**C51**：将回报值域离散化为 51 个支撑点（atoms），输出每个支撑点的概率。

**QR-DQN**：用分位数回归近似回报分布，输出一组分位数 $\tau_1, \ldots, \tau_N$。

**优势**：分布信息提供更丰富的学习信号，在 Atari 上显著提升性能。学习分布还能更好地处理随机环境中的不确定性。

### 4.6 NoisyNet

**论文**：Fortunato et al., *Noisy Networks for Exploration* (2018)

在网络权重中加入可学习的噪声参数：

$$y = (\mu + \sigma \odot \epsilon) x$$

其中 $\mu, \sigma$ 可学习，$\epsilon$ 是随机噪声。网络自身的随机性驱动探索，替代 ε-greedy。

**优势**：参数空间探索比动作空间探索（ε-greedy）更高效，尤其在状态依赖的探索需求下。

### 4.7 Rainbow

**论文**：Hessel et al., *Rainbow: Combining Improvements in Deep RL* (2018)

Rainbow 整合了 DQN 的六大改进：

1. Double DQN（去过估计）
2. Dueling Networks（V/A 分解）
3. Prioritized Experience Replay（优先回放）
4. C51 Distributional RL（分布学习）
5. N-step Returns（多步回报）
6. NoisyNet（参数探索）

在 Atari 上大幅超越原始 DQN 和各单独改进，展示了组件化研究范式的有效性。消融实验表明各组件的贡献不同：优先回放和多步回报贡献最大。

### 4.8 DDPG（Deep Deterministic Policy Gradient）

**论文**：Lillicrap et al., *Continuous control with deep reinforcement learning* (2016)

DDPG 将 DQN 扩展到连续动作空间。核心思想是用确定性策略 $\mu_\theta(s)$ 直接输出连续动作，并用 Q 函数评估：

**Critic 更新**：$\mathcal{L}(\phi) = \mathbb{E}[(r + \gamma Q_{\phi^-}(s', \mu_{\theta^-}(s')) - Q_\phi(s,a))^2]$

**Actor 更新**：$\nabla_\theta J \approx \mathbb{E}[\nabla_a Q_\phi(s,a)|_{a=\mu_\theta(s)} \nabla_\theta \mu_\theta(s)]$

（确定性策略梯度定理）

DDPG 使用软更新 $\theta^- \leftarrow \tau \theta + (1-\tau)\theta^-$ 和 Ornstein-Uhlenbeck 噪声进行探索。

**局限**：超参数敏感，训练不稳定，被 SAC 取代为连续控制 SOTA。

### 4.9 A3C（Asynchronous Advantage Actor-Critic）

**论文**：Mnih et al., *Asynchronous Methods for Deep RL* (2016)

多个 worker 在不同环境实例上异步采集经验并更新全局参数：

- 每个 worker 独立探索，增加数据多样性
- 异步更新天然打破时间相关性，不需要经验回放
- 支持 CPU 多线程，无需 GPU 也能训练

A3C 的同步版本 A2C 在实践中也广泛使用。

### 4.10 IMPALA

**论文**：Espeholt et al., *IMPALA: Scalable Distributed Deep-RL* (2018)

IMPALA 使用 actor-learner 架构：大量 actor 采集经验发送给 learner 进行训练。引入 V-trace 修正 off-policy 偏差，支持大规模分布式训练。

## 5. 值函数方法 vs 策略梯度方法

| 维度 | 值函数方法（DQN 系列） | 策略梯度方法（PPO/SAC） |
|:---|:---|:---|
| 动作空间 | 离散 | 离散/连续 |
| 策略类型 | 隐式（$\arg\max Q$） | 显式参数化 |
| 随机策略 | 困难（需 Boltzmann） | 自然支持 |
| off-policy | 天然支持 | 需要额外机制（SAC） |
| 样本效率 | 高（经验回放） | 低（on-policy）/ 高（SAC） |
| 连续动作 | 不适用 | 适用 |

## 6. 优势与局限

### 优势

1. **高维状态处理**：神经网络处理图像、文本等高维输入
2. **端到端学习**：从原始感知到决策的端到端优化
3. **通用性强**：同一架构可应用于游戏、机器人、推荐等多种场景

### 屏限

1. **样本效率低**：需要海量交互（Atari 需数千万帧）
2. **训练不稳定**：超参数敏感，复现困难，不同种子方差大
3. **灾难性遗忘**：新策略可能遗忘旧技能
4. **探索不足**：ε-greedy 在稀疏奖励环境中探索效率低

| 场景 | 方法 | 说明 |

| Atari 游戏 | Rainbow / R2D2 | 视觉输入的离散控制 |

## 8. 与其他技术关系

- **与无模型方法的关系**：DQN 本质是 Q-Learning + 深度网络 + 工程技巧（经验回放/目标网络）。参见 [[03_无模型预测与控制]]。
- **与策略梯度方法的关系**：DDPG 是确定性策略梯度 + DQN 式训练；A3C 是 Actor-Critic 的深度化。参见 [[04_策略梯度方法]]。
- **与卷积神经网络的关系**：DQN 用 CNN 处理视觉输入。参见 [[05_卷积神经网络]]。
- **与大模型训练的关系**：深度 RL 的 PPO 算法被直接用于 RLHF 训练大语言模型。参见 [[10_强化学习与大模型]]。

## 9. 前沿发展

- **R2D2 / Agent57**：循环网络 + 分布式训练，在所有 Atari 游戏上超人类
- **Decision Transformer**：将 RL 重新表述为序列建模，用 Transformer 直接生成动作
- **EfficientZero**：基于 MuZero 的样本高效版本，1 小时内掌握 Atari
- **自监督预训练 + RL**：用自监督表示学习加速 RL 收敛
- **Sim-to-Real 迁移**：在模拟器中训练深度 RL，迁移到真实机器人

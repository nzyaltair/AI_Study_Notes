# 三维计算机视觉

## 1. 概述

三维计算机视觉（3D Computer Vision）研究如何从二维图像中恢复三维场景的结构信息，以及如何表示、理解和重建三维世界。它是计算机视觉中数学基础最深厚、工程挑战最大的分支之一。

三维视觉的核心问题包括：相机成像模型（从 3D 到 2D 的投影）、三维重建（从 2D 恢复 3D）、位姿估计（相机在哪里）、场景表示（如何编码 3D 场景）等。

近年来，神经辐射场（NeRF）和 3D 高斯泼溅（3DGS）等神经场景表示方法的出现，将深度学习与经典几何方法深度融合，开辟了三维视觉的新纪元。

## 3. 核心概念

### 3.1 相机模型

**针孔相机模型**：将 3D 世界点投影到 2D 图像平面：

$$\mathbf{p} = \mathbf{K} [\mathbf{R} | \mathbf{t}] \mathbf{P}$$

其中：
- $\mathbf{P} = [X, Y, Z, 1]^T$：世界坐标系下的 3D 点（齐次坐标）
- $\mathbf{K} = \begin{pmatrix} f_x & 0 & c_x \\ 0 & f_y & c_y \\ 0 & 0 & 1 \end{pmatrix}$：相机内参矩阵（焦距 $f$ + 主点 $c$）
- $[\mathbf{R} | \mathbf{t}]$：外参（旋转 $\mathbf{R}$ + 平移 $\mathbf{t}$），世界到相机的变换
- $\mathbf{p} = [u, v, 1]^T$：图像平面上的 2D 点

**畸变**：实际镜头有径向畸变和切向畸变，需要标定校正。

### 3.2 对极几何

两个相机视图之间的几何关系：

- **基本矩阵 $\mathbf{F}$**：$\mathbf{p}'^T \mathbf{F} \mathbf{p} = 0$，像素坐标系下的对极约束
- **本质矩阵 $\mathbf{E}$**：$\mathbf{E} = \mathbf{K}'^T \mathbf{F} \mathbf{K}$，归一化坐标系
- **对极线**：一个图像中的点在另一幅图像中的可能位置构成一条线

### 3.3 三角测量

已知两视图的相机参数和匹配点对，恢复 3D 点坐标：

给定两个投影方程 $\mathbf{p}_1 = \mathbf{P}_1 \mathbf{X}$ 和 $\mathbf{p}_2 = \mathbf{P}_2 \mathbf{X}$，通过最小二乘求解 $\mathbf{X}$。

### 3.4 3D 场景表示

| 表示方式 | 数据结构 | 优势 | 劣势 |
|:---|:---|:---|:---|
| 点云 | $(x,y,z)$ 点集 | 简单灵活 | 无拓扑/法线信息 |
| 体素网格 | 3D 网格 | 规则结构 | 内存消耗大 |
| 网格 | 顶点+面片 | 精确表面 | 难以优化 |
| 深度图 | 2D + 深度 | 与图像对齐 | 单视图有限 |
| 隐式表示 | 神经网络 | 连续、紧凑 | 训练慢 |
| 3D 高斯 | 高斯椭球集合 | 实时渲染 | 显存占用 |

## 4. 技术原理

### 4.1 运动恢复结构（SfM）

从多张 2D 图像恢复 3D 场景结构和相机位姿：

```
多张图像 → 特征提取 → 特征匹配 → 稀疏重建 → 稠密重建
                                    ↓
                              相机位姿 + 稀疏点云
                                    ↓
                              MVS → 稠密点云/网格
```

**增量式 SfM**：
1. 选择初始图像对，三角测量初始 3D 点
2. 逐张添加新图像，PnP 估计位姿
3. 三角测量新匹配点
4. Bundle Adjustment 全局优化

**Bundle Adjustment**：

$$\min_{\mathbf{P}_i, \mathbf{X}_j} \sum_{i,j} \|\mathbf{p}_{ij} - \pi(\mathbf{P}_i \mathbf{X}_j)\|^2$$

最小化所有观测的重投影误差。

### 4.2 视觉 SLAM

SLAM（Simultaneous Localization and Mapping）在未知环境中同时估计相机位姿和构建地图：

**ORB-SLAM 架构**：

```
视频流 → 跟踪（Track） → 局部建图（Map） → 闭环检测（Loop Closure）
         ↓                    ↓                    ↓
    位姿估计            局部地图优化           全局一致性
```

- **跟踪**：提取 ORB 特征，与上一帧匹配，估计位姿
- **局部建图**：新增关键帧，三角测量新地图点，局部 BA
- **闭环检测**：词袋模型检测回环，位姿图优化

**VIO（视觉惯性里程计）**：融合相机和 IMU，提供更鲁棒的位姿估计。

### 4.3 神经辐射场（NeRF）

NeRF 用 MLP 表示 3D 场景的辐射场（颜色 + 密度），通过体渲染从任意视角合成图像：

**场景表示**：

$$F_\theta: (\mathbf{x}, \mathbf{d}) \to (\mathbf{c}, \sigma)$$

- $\mathbf{x} = (x, y, z)$：3D 位置
- $\mathbf{d}$：观察方向
- $\mathbf{c}$：颜色（RGB）
- $\sigma$：体密度（不透明度）

**位置编码**（解决 MLP 学习高频细节的瓶颈）：

$$\gamma(\mathbf{x}) = [\mathbf{x}, \sin(2^0 \mathbf{x}), \cos(2^0 \mathbf{x}), \ldots, \sin(2^{L-1} \mathbf{x}), \cos(2^{L-1} \mathbf{x})]$$

**体渲染方程**：

$$C(\mathbf{r}) = \int_{t_n}^{t_f} T(t) \sigma(\mathbf{r}(t)) \mathbf{c}(\mathbf{r}(t), \mathbf{d}) \, dt$$

$$T(t) = \exp\left(-\int_{t_n}^t \sigma(\mathbf{r}(s)) \, ds\right)$$

**离散化**：

$$\hat{C}(\mathbf{r}) = \sum_{i=1}^{N} T_i (1 - \exp(-\sigma_i \delta_i)) \mathbf{c}_i$$

其中 $T_i = \exp\left(-\sum_{j=1}^{i-1} \sigma_j \delta_j\right)$。

**训练**：最小化渲染图像与真实图像的 MSE 损失：

$$\mathcal{L} = \sum_{\mathbf{r} \in \mathcal{R}} \| \hat{C}(\mathbf{r}) - C_{gt}(\mathbf{r}) \|^2$$

### 4.4 3D 高斯泼溅（3DGS）

3DGS 用一组可学习的 3D 高斯椭球表示场景，实现实时高质量渲染：

**3D 高斯表示**：

$$G(\mathbf{x}) = e^{-\frac{1}{2}(\mathbf{x} - \boldsymbol{\mu})^T \boldsymbol{\Sigma}^{-1} (\mathbf{x} - \boldsymbol{\mu})}$$

每个高斯的属性：
- 位置 $\boldsymbol{\mu}$
- 协方差矩阵 $\boldsymbol{\Sigma}$（控制形状和朝向）
- 不透明度 $\alpha$
- 球谐函数系数（控制视角相关颜色）

**泼溅渲染**：

$$C(\mathbf{u}) = \sum_{i \in N} \mathbf{c}_i \alpha_i \prod_{j=1}^{i-1} (1 - \alpha_j)$$

- 将 3D 高斯投影到 2D 屏幕空间
- 按深度排序
- Alpha 混合

**训练**：梯度下降优化高斯参数 + 自适应密度控制（分裂/克隆/删除）。

**NeRF vs. 3DGS**：

| 维度 | NeRF | 3DGS |
|:---|:---|:---|
| 表示 | 隐式 MLP | 显式高斯集合 |
| 渲染 | 体渲染（慢） | 光栅化（快） |
| 训练 | 数小时 | 分钟级 |
| 渲染速度 | 秒/帧 | 100+ FPS |
| 编辑 | 困难 | 直观 |

### 4.5 深度学习点云处理

**PointNet**：直接处理无序点云：

```
点云 (N×3) → 共享 MLP → Max Pooling → 全局特征 → 分类
                ↓
        逐点特征 → 分割
```

关键操作：Max Pooling 实现排列不变性（输入点的顺序不影响输出）。

**PointNet++**：层级化采样和分组，捕获局部几何结构。

## 6. 优势与局限

### 优势

1. **三维理解**：从 2D 图像恢复 3D 结构，是空间智能的基础
2. **几何精度**：基于严格数学理论，可达到高精度重建
3. **实时定位**：SLAM 实现实时位姿估计，支持 AR/机器人导航
4. **新视角合成**：NeRF/3DGS 从已有视图合成任意新视角

### 局限

1. **计算成本高**：3D 数据处理和渲染计算量大
2. **数据获取困难**：3D 标注（深度/网格）成本远高于 2D
3. **动态场景困难**：NeRF/3DGS 原生处理静态场景，动态扩展复杂
4. **大场景挑战**：城市级大规模重建的存储和计算挑战
5. **光照敏感**：室外场景的光照变化影响重建质量

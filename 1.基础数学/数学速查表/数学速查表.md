# 数学速查表

这是一份快速参考指南，包含AI学习中最常用的数学公式和概念，基于基础数学.md的核心内容整理而成。

## 📐 线性代数 (Linear Algebra)

### 基本运算

| 运算 | 符号 | 定义 | 维度 |
|------|------|------|------|
| 矩阵乘法 | C = AB | C_ij = Σ_k A_ik B_kj | (m×n)(n×p) → (m×p) |
| 转置 | A^T | (A^T)_ij = A_ji | (m×n) → (n×m) |
| 点积 | a·b | Σ_i a_i b_i | 标量 |
| 外积 | a⊗b | (a⊗b)_ij = a_i b_j | (n×1)(1×m) → (n×m) |
| Hadamard 积 | A⊙B | (A⊙B)_ij = A_ij B_ij | 逐元素相乘 |
| Trace | tr(A) | Σ_i A_ii | 对角线元素和 |

### 重要分解

**特征分解 (Eigendecomposition)**
```
A = QΛQ^(-1)
```
- Q: 特征向量矩阵
- Λ: 特征值对角矩阵
- 仅适用于方阵

**奇异值分解 (SVD)**
```
A = UΣV^T
```
- U: 左奇异向量 (m×m)
- Σ: 奇异值对角矩阵 (m×n)
- V: 右奇异向量 (n×n)
- 适用于任意矩阵

**Cholesky 分解**
```
A = LL^T
```
- 仅适用于对称正定矩阵
- L: 下三角矩阵

### 矩阵性质

| 性质 | 定义 | 条件 |
|------|------|------|
| 对称 | A = A^T | A_ij = A_ji |
| 正交 | Q^T Q = I | 列向量互相正交且单位化 |
| 正定 | x^T A x > 0 | 对所有非零 x |
| 半正定 | x^T A x ≥ 0 | 对所有 x |
| 满秩 | rank(A) = min(m,n) | 列/行向量线性独立 |

### 范数 (Norms)

**向量范数**
```
L0 范数: ||x||₀ = #{i : xᵢ ≠ 0}  (非零元素个数)
L1 范数: ||x||₁ = Σᵢ |xᵢ|
L2 范数: ||x||₂ = √(Σᵢ xᵢ²)
L∞ 范数: ||x||∞ = maxᵢ |xᵢ|
Lp 范数: ||x||p = (Σᵢ |xᵢ|^p)^(1/p)
```

**矩阵范数**
```
Frobenius 范数: ||A||_F = √(Σᵢⱼ Aᵢⱼ²) = √tr(A^T A)
核范数: ||A||_* = Σᵢ σᵢ  (奇异值和)
谱范数: ||A||₂ = σ_max  (最大奇异值)
```

### 常用恒等式

```
(AB)^T = B^T A^T
(AB)^(-1) = B^(-1) A^(-1)
tr(AB) = tr(BA)
tr(A) = tr(A^T)
det(AB) = det(A)det(B)
∂(x^T a)/∂x = a
∂(x^T A x)/∂x = (A + A^T)x = 2Ax  (若 A 对称)
∂(a^T X b)/∂X = ab^T
```

## 📈 微积分 (Calculus)

### 导数规则

**基本规则**
```
(c)' = 0
(x^n)' = nx^(n-1)
(e^x)' = e^x
(ln x)' = 1/x
(sin x)' = cos x
(cos x)' = -sin x
```

**运算规则**
```
(f + g)' = f' + g'
(cf)' = cf'
(fg)' = f'g + fg'  (乘积法则)
(f/g)' = (f'g - fg')/g²  (商法则)
(f∘g)' = f'(g(x))·g'(x)  (链式法则)
```

### 多变量微积分

**梯度 (Gradient)**
```
∇f = [∂f/∂x₁, ∂f/∂x₂, ..., ∂f/∂xₙ]^T
```

**Jacobian 矩阵**
```
J = [∂fᵢ/∂xⱼ]  (m×n 矩阵)
```

**Hessian 矩阵**
```
H = [∂²f/∂xᵢ∂xⱼ]  (n×n 对称矩阵)
```

**方向导数**
```
D_v f(x) = ∇f(x)·v  (v 为单位向量)
```

### 常用梯度

```
∇(a^T x) = a
∇(x^T A x) = (A + A^T)x
∇(||x||²) = 2x
∇(||Ax - b||²) = 2A^T(Ax - b)
```

### Taylor 展开

**一维**
```
f(x) ≈ f(a) + f'(a)(x-a) + (1/2)f''(a)(x-a)² + ...
```

**多维**
```
f(x) ≈ f(a) + ∇f(a)^T(x-a) + (1/2)(x-a)^T H(a)(x-a)
```

## 🎲 概率与统计 (Probability & Statistics)

### 基本概念

**概率公理**
```
P(A) ∈ [0, 1]
P(S) = 1  (S 为样本空间)
P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
```

**条件概率**
```
P(A|B) = P(A ∩ B) / P(B)
```

**贝叶斯定理**
```
P(A|B) = P(B|A)P(A) / P(B)
P(θ|D) = P(D|θ)P(θ) / P(D)
```

**全概率公式**
```
P(B) = Σᵢ P(B|Aᵢ)P(Aᵢ)
```

### 数值特征

**期望值 (Expectation)**
```
E[X] = Σₓ x P(x)  (离散)
E[X] = ∫ x p(x) dx  (连续)
E[g(X)] = Σₓ g(x) P(x)
E[aX + b] = aE[X] + b
```

**方差 (Variance)**
```
Var(X) = E[(X - μ)²] = E[X²] - (E[X])²
Var(aX + b) = a² Var(X)
```

**协方差 (Covariance)**
```
Cov(X,Y) = E[(X-E[X])(Y-E[Y])] = E[XY] - E[X]E[Y]
Cov(X,X) = Var(X)
```

**相关系数 (Correlation)**
```
ρ(X,Y) = Cov(X,Y) / (σₓ σᵧ)  ∈ [-1, 1]
```

### 常用分布

**Bernoulli 分布**
```
P(X=1) = p
E[X] = p
Var(X) = p(1-p)
```

**高斯/正态分布**
```
N(μ, σ²): p(x) = (1/√(2πσ²)) exp(-(x-μ)²/(2σ²))
E[X] = μ
Var(X) = σ²
```

**多变量高斯分布**
```
N(μ, Σ): p(x) = (1/√((2π)^d|Σ|)) exp(-1/2(x-μ)^T Σ^(-1)(x-μ))
```

**指数分布**
```
p(x|λ) = λe^(-λx),  x ≥ 0
E[X] = 1/λ
Var(X) = 1/λ²
```

### 信息论

**熵 (Entropy)**
```
H(X) = -Σₓ P(x) log P(x) = -E[log P(X)]
```

**交叉熵 (Cross-Entropy)**
```
H(P,Q) = -Σₓ P(x) log Q(x) = -E_P[log Q(X)]
```

**KL 散度 (KL Divergence)**
```
D_KL(P||Q) = Σₓ P(x) log(P(x)/Q(x))
            = H(P,Q) - H(P)
```

**互信息 (Mutual Information)**
```
I(X;Y) = H(X) - H(X|Y) = H(Y) - H(Y|X)
       = D_KL(P(X,Y) || P(X)P(Y))
```

### 估计

**最大似然估计 (MLE)**
```
θ_MLE = argmax_θ P(D|θ) = argmax_θ Πᵢ P(xᵢ|θ)
      = argmax_θ Σᵢ log P(xᵢ|θ)
```

**最大后验估计 (MAP)**
```
θ_MAP = argmax_θ P(θ|D) = argmax_θ P(D|θ)P(θ)
```

## ⚡ 优化方法 (Optimization)

### 梯度下降变体

**标准梯度下降 (GD)**
```
θ ← θ - η∇L(θ)
```

**随机梯度下降 (SGD)**
```
θ ← θ - η∇L(θ; xᵢ, yᵢ)
```

**动量法 (Momentum)**
```
v ← βv + (1-β)∇L(θ)
θ ← θ - ηv
```

**Nesterov 加速梯度 (NAG)**
```
v ← βv + (1-β)∇L(θ - ηβv)
θ ← θ - ηv
```

**AdaGrad**
```
Gₜ ← Gₜ₋₁ + (∇L(θ))²
θ ← θ - (η/√(Gₜ + ε))∇L(θ)
```

**RMSprop**
```
Eₜ ← βEₜ₋₁ + (1-β)(∇L(θ))²
θ ← θ - (η/√(Eₜ + ε))∇L(θ)
```

**Adam**
```
mₜ ← β₁mₜ₋₁ + (1-β₁)∇L(θ)
vₜ ← β₂vₜ₋₁ + (1-β₂)(∇L(θ))²
m̂ₜ ← mₜ/(1-β₁ᵗ)
v̂ₜ ← vₜ/(1-β₂ᵗ)
θ ← θ - η·m̂ₜ/(√v̂ₜ + ε)
```

**AdamW**
```
# 带权重衰减的 Adam，大模型训练常用
θ ← θ - η·m̂ₜ/(√v̂ₜ + ε) - η·λ·θ
```

### 凸优化条件

**一阶条件 (First-order)**
```
∇f(x*) = 0
```

**二阶条件 (Second-order)**
```
∇f(x*) = 0  且  H(x*) ⪰ 0  (半正定)
```

**KKT 条件 (约束优化)**
```
∇f(x*) + Σᵢ λᵢ∇gᵢ(x*) + Σⱼ μⱼ∇hⱼ(x*) = 0
gᵢ(x*) ≤ 0,  λᵢ ≥ 0,  λᵢgᵢ(x*) = 0
hⱼ(x*) = 0
```

## 🔥 深度学习特定公式

### 激活函数

**Sigmoid**
```
σ(x) = 1/(1 + e^(-x))
σ'(x) = σ(x)(1 - σ(x))
```

**Tanh**
```
tanh(x) = (e^x - e^(-x))/(e^x + e^(-x))
tanh'(x) = 1 - tanh²(x)
```

**ReLU**
```
ReLU(x) = max(0, x)
ReLU'(x) = 1_{x>0}
```

**Leaky ReLU**
```
LeakyReLU(x) = max(αx, x),  α ∈ (0,1)
```

**GELU**
```
GELU(x) = x·Φ(x)  (Φ 为标准正态 CDF)
GELU(x) ≈ x·σ(1.702x)  (近似)
```

**Softmax**
```
softmax(x)ᵢ = exp(xᵢ) / Σⱼ exp(xⱼ)
softmax(x)ᵢ = exp(xᵢ - max(x)) / Σⱼ exp(xⱼ - max(x))  (数值稳定版)
```

### 损失函数

**均方误差 (MSE)**
```
L = (1/n)Σᵢ(yᵢ - ŷᵢ)²
∂L/∂ŷᵢ = (2/n)(ŷᵢ - yᵢ)
```

**交叉熵 (Cross-Entropy)**
```
L = -(1/n)Σᵢ yᵢ log(ŷᵢ)  (分类)
L = -(1/n)Σᵢ[yᵢlog(ŷᵢ) + (1-yᵢ)log(1-ŷᵢ)]  (二元)
```

**KL 散度损失**
```
L = Σᵢ P(xᵢ) log(P(xᵢ)/Q(xᵢ))
```

### 正则化

**L1 正则化**
```
L_total = L_data + λ||θ||₁
```

**L2 正则化**
```
L_total = L_data + λ||θ||²
```

**Dropout**
```
训练: h = m ⊙ f(Wx)  (m ~ Bernoulli(p))
测试: h = p·f(Wx)
```

### 归一化

**批量归一化 (Batch Normalization)**
```
μ_B = (1/m)Σᵢ xᵢ
σ²_B = (1/m)Σᵢ(xᵢ - μ_B)²
x̂ᵢ = (xᵢ - μ_B)/√(σ²_B + ε)
yᵢ = γx̂ᵢ + β
```

**层归一化 (Layer Normalization)**
```
μ = (1/H)Σᵢ xᵢ
σ² = (1/H)Σᵢ(xᵢ - μ)²
x̂ᵢ = (xᵢ - μ)/√(σ² + ε)
yᵢ = γx̂ᵢ + β
```

### 注意力机制

**缩放点积注意力**
```
Attention(Q,K,V) = softmax(QK^T/√d_k)V
```

**多头注意力 (Multi-Head Attention)**
```
MultiHead(Q,K,V) = Concat(head₁,...,headₕ)W^O
headᵢ = Attention(QW^Q_i, KW^K_i, VW^V_i)
```

## 📊 数值稳定性技巧

**Log-Sum-Exp 技巧**
```
log Σᵢ exp(xᵢ) = a + log Σᵢ exp(xᵢ - a)
其中 a = max(x)
```

**Softmax 稳定计算**
```
softmax(x) = exp(x - max(x)) / Σ exp(x - max(x))
```

**梯度裁剪 (Gradient Clipping)**
```
# 范数裁剪
g ← (c/||g||)·g  若 ||g|| > c

# 值裁剪
g ← clip(g, -c, c)
```

## 🎯 常用不等式

**Jensen 不等式**
```
f(E[X]) ≤ E[f(X)]  (f 为凸函数)
```

**Cauchy-Schwarz 不等式**
```
|⟨x,y⟩| ≤ ||x||·||y||
```

**Hölder 不等式**
```
||xy||₁ ≤ ||x||_p · ||y||_q  其中 1/p + 1/q = 1
```

**三角不等式**
```
||x + y|| ≤ ||x|| + ||y||
```

## 💡 实用技巧

### 矩阵求逆的替代方法
- 使用 QR 分解或 SVD 代替直接求逆
- 更数值稳定，适用于条件数较大的矩阵

### 大矩阵乘法优化
- 使用 Einstein Summation (`np.einsum`)
- 注意运算顺序：(AB)C vs A(BC)
- 利用矩阵的稀疏性或低秩性质

### 内存优化
- 梯度检查点 (Gradient Checkpointing)
- 混合精度训练 (Mixed Precision)
- 动态计算图 vs 静态计算图

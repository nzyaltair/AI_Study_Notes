
# 数据集和模型


_**MNIST**_ **[11]** 包含 70,000 个 28 × 28 个数字 0 到 9 的灰度图像，即 10 个类别。我们随机选择 60,000 个样本作为训练集，剩余的 10,000 个样本作为测试集。我们在训练集上训练一个 LetNet-5 模型。学习率为 0.01，训练好的良性模型在测试集上的预测准确率为 99.12%。


_**CIFAR-10**_ _。_CIFAR-10 [29] 包含属于 10 个类别的 60,000 张图像。我们随机选择 50000 个样本作为训练集，其余 10000 个样本作为测试集。我们在训练集上训练 ResNet-18 300 个 epoch。学习率为 0.001，动量随机梯度下降为 0.9。经过训练的良性模型在测试集上的预测准确率为 90.90%。


_**CIFAR-100**_ _。_CIFAR-100 [29] 包括属于 100 个类别的 600,000 张图像。每个样本的维度为 32 × 32。我们训练一个 ResNet-50 模型 200 个 epoch。我们将学习率设置为 0.1，动量随机梯度下降设置为 0.9。为了进一步提高性能，在第 60 个、第 120 个和第 160 个周期中使用了_γ_为 0.2 的 MultiStepLR 调度器。经过训练的良性模型在测试集上的预测准确率为 78.95%。


_**ImageNette**_ _的_ImageNette [13] 是 ImageNet 的一个子集，广泛用于研究界 [41]、[63]。ImageNette 包括 9,469 个训练样本和 3,925 个测试样本。每个图像都具有高分辨率，尺寸为 224 × 224。我们训练一个 ResNet-18 网络 150 个 epoch。我们将学习率设置为 0.001，批量大小为 16（由于 GPU 源有限），随机梯度下降的动量为 0.95，权重衰减为 0.0005。经过训练的良性模型在测试集上的预测准确率为 86.15%。

此外，我们将 SAGE 与六种最先进的后门模型纯化方法进行了比较。我们使用基线的原始源代码实现基线，并确保与足够的 epoch 收敛。

- 对模型进行微调可以移除模型的后门 [32]、[37]。我们运行 50 个 epoch 的微调。
- 修剪也可用于禁用后门 [18]。我们运行 50 个 epoch 的修剪。
- Fine-pruning [37] 结合了 finetuning 和 model pruning 来净化后门模型。我们运行 50 个 epoch 的精细修剪。
- NAD [32] 首先使用一组干净的数据对后门模型进行微调，得到一个教师模型，然后使用教师模型通过注意力蒸馏过程来纠正后门模型（学生模型）。我们运行 50 个 epoch 来建立教师模型，运行 50 个 epoch 来净化学生模型。
- MCR [69] 提议在损失环境中采用_模式连接_ [15]，以使用一组干净的数据从模型中删除后门神经路径。我们运行 50 个 epoch 的微调、100 个 epoch 的曲线网络训练和 100 个 epoch 的模型更新。
- ANP（对抗神经元修剪）[62] 通过在对抗神经元扰动下修剪各种敏感神经元来纠正后门模型。我们运行 50 个 epoch 的掩码优化和 1000 个 epoch 的修剪。
